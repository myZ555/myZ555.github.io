# 臀部
浅层
- 皮神经
深层
- 肌肉
	- 浅层：臀大肌
	- 中层：臀中肌、臀小肌
	- 深层：梨状肌、上孖肌、闭孔内肌、下孖肌、股方肌
- “口”
	1. 梨状肌上孔
	2. 梨状肌下孔
	3. 坐骨小孔
*临床联系*
1. 梨状肌综合征
2. 臀部肌肉注射安全区
# 股后区
深层结构
- 骨筋膜鞘：股后股筋膜鞘
- 肌肉（腘绳肌）
	- 组成：股二头肌，半腱肌，半膜肌
	- 共同起点：坐骨结节
	- 功能：伸髋、屈膝
	- 神经支配：坐骨神经胫神经
- 血管神经束
	- 坐骨神经
	- 动脉供应：股深动脉穿动脉
# 膝后区
内容物
- 神经：胫神经、腓总神经
- 静脉：腘静脉
- 动脉：腘动脉
*临床联系*
1. 腓总神经损伤--足下垂、马蹄内翻足
2. 膝关节动脉网
# 小腿后区
浅层：
- 小隐静脉：伴行腓肠神经，汇入腘静脉
深层：
- 肌肉
	- 浅层：腓肠肌、比目鱼肌，共同形成跟腱，止于跟骨结节。【跖屈踝关节】（踮脚尖）
	- 深层：胫骨后肌、趾长屈肌、长屈肌
- 血管神经束：
	- 胫神经：伴行胫后动脉；穿过踝管进入足底
	- 胫后动脉
	- 胫后静脉
*临床联系*
1. 胫神经损伤--小腿后群肌瘫痪，足不能**跖屈**，出现“**钩状足**”畸形 (足呈背屈外翻状)
# 踝后区
**踝管**
- 构成
- 支持带……4个通道：
	1. 胫骨后肌腱
	2. 趾长屈肌腱
	3. 胫后动静脉及胫神经
	4. 踇长屈肌腱
# 足底
浅筋膜
深层：
- 足底肌
- 足底内外侧动脉
- 神经
