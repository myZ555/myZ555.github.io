# 深层
### 1. 肌层

| 腹外斜肌                                | 腹内斜肌        | 腹横肌      | 腹直肌   |
| ----------------------------------- | ----------- | -------- | ----- |
| 参与腹直肌鞘前层                            | 参与腹直肌鞘前层+后层 | 参与腹直肌鞘后层 | 腹直肌鞘中 |
| 腹股沟韧带<br>腔隙韧带(陷窝韧带)<br>耻骨梳韧带        | 腹股沟镰/联合腱    | 参与形成腹股沟镰 | 腱划    |
| **腹股沟管浅环**<br>- 男性：精索<br>- 女性：子宫圆韧带 | 提睾肌         | 参与构成提睾肌  | 锥状肌   |
> **腹股沟管浅环**：腹外斜肌腱膜在耻骨结节外上方形成三角形裂隙
> **腹股沟韧带**：腹外斜肌腱膜下缘伸张于骼前上棘至耻骨结节之间，向后卷曲反折增厚形成
> 	腔隙韧带：腹股沟韧带内侧端的一小部分纤维向下后方，并向外侧转折形成
> 	耻骨梳韧带(Cooper韧带)：腔隙韧带向外侧延续附着于耻骨梳的部分
> **腹股沟镰(联合腱)**：腹内斜肌腱膜与腹横肌腱膜在精索的内侧互相融合形成
> **提睾肌**：腹内斜肌的最下部发出一些细散的肌束，向下包绕精索和睾丸。收缩时上提睾丸
> **腱划**：结缔组织，为肌节愈合的痕迹。肌被3～5条腱划分成几个肌腹。
> **锥状肌**：腹直肌鞘内，腹直肌下部前面的三角形肌
> **白线**：位于腹前壁正中线上，为左右腹直肌鞘间的隔，由两侧的前外侧群肌的腱膜构成
> **半月线**：腹直肌外侧缘

###### ★腹直肌鞘
- 境界：
	- 前层--*腹外*斜肌腱膜与*腹内*斜肌腱膜的前层愈合而成
	- 后层--*腹内*斜肌腱膜的后层与*腹横肌*腱膜愈合而成
- ==弓状线arcuateline（半环线）==：在脐下4-5cm处
	- 腹内斜肌腱膜后层和腹横肌腱膜均转至腹直肌前面参与构成腹直肌鞘浅层，*腹直肌鞘后层在此形成一凹向下的弓状下缘![275](Pasted%20image%2020251031134755.png)

- 内容物：腹直肌、锥状肌、腹壁上下血管、下位肋间神经的末端

### ★手术切口

==正中切口==：皮肤-浅筋膜-腹白线-腹横筋膜-腹膜外筋膜-壁腹膜
- 损伤血管神经少，层次简单；血液供应差，易发生切口疝或切口裂开
==旁正中切口==：皮肤-浅筋膜-腹直肌鞘前层-腹直肌-腹直肌鞘后层(弓状线下无此层)-腹横筋膜-腹膜外筋膜-壁腹膜
==麦氏切口(阑尾切口)==：皮肤-浅筋膜-腹外斜肌剂及其腱膜-腹内斜肌-腹横肌-腹横筋膜-腹膜外筋膜-壁腹膜
- 损伤血管，神经和肌少，但显露手术野的范围小，不利于扩大延长
==腹直肌切口==：皮肤-浅筋膜-腹直肌鞘前层-腹直肌(游离其内侧缘拉向外侧)-腹直肌鞘后层(弓状线下无此层)-腹横筋膜-腹膜外筋膜-壁腹膜
- 术中损伤血管、神经和肌少，切口血液供应好,且有肌保护。

# 腹股沟区
> 此区是腹壁的薄弱区，其原因是:
> 	①腹外斜肌在此处移行为较薄的腹外斜肌腱膜,其下方还形成一裂口(腹股沟管浅环)；
> 	②腹内斜肌与腹横肌的下缘均未达到腹股沟韧带的内侧部，因而该区域没有肌遮盖;
> 	③男性有精索、女性有子宫圆韧带通过腹股沟管，在此形成解剖上的潜在性裂隙。
> 此外，当人体站立时，腹股沟区所承受的腹内压力比平卧时约高3倍。故腹壁疝多发生于此区。
### 1. 腹股沟管
- 是男性精索或女性子宫圆韧带所通过的一条肌和腱之间的裂隙。
- **位置**：腹股沟韧带内侧半上方，长约4～5cm。
- ==内口==：腹股沟管深(腹)环；为*腹横筋膜*随精索向外突出而成的一个卵圆形裂隙，位于腹股沟韧带中点上方1.5cm处（约一横指处）
- ==外口==：腹股沟管浅(皮下)环；为*腹外斜肌腱膜*在*耻骨结节外上方*的一个三角形裂隙
- ==前壁==：腹外斜肌腱膜形成，外1/3有腹内斜肌起始
- ==后壁==：腹横筋膜形成，内1/3有联合腱
- ==上壁==：*腹内斜肌和腹横肌构成的弓状下缘*
- ==下壁==：腹股沟韧带和腔隙（陷窝韧带）
- **内容**：精索、子宫圆韧带
> [!important] 关键韧带结构
> 1. 腹股沟韧带：腹外斜肌腱膜下缘增厚形成，构成腹股沟管下壁
> 2. 腔隙/陷窝韧带：腹股沟韧带内侧端向后上转折形成
> 3. 联合肌腱/腹股沟镰：腹内斜肌和腹横肌下缘融合形成

> [!warning] 结构薄弱点
> 1. 联合肌腱下缘与腹股沟韧带之间的间隙
> 2. 腹横筋膜在腹股沟中点上方2cm处的裂隙（内环）
> 3. 腹外斜肌腱膜在耻骨结节外上方的三角形裂隙（外环）
---
==**腹股沟三角/海氏三角**==
- 由腹直肌外侧缘、腹股沟韧带和腹壁下动脉围成；是腹前外侧壁的一个薄弱部位
- 层次：
	- 皮肤-浅筋膜-腹外斜肌腱膜、腹股沟管浅环-(腹股沟镰)-腹横筋膜-腹膜外组织-壁腹膜
### 2. 腹股沟疝
- 腹腔脏器从腹股沟韧带上方的腹股沟区脱出形成的疝

|          | 斜疝        | 直疝        |
| -------- | --------- | --------- |
| 脱出部位     | 腹股沟管深环    | 腹股沟三角     |
| 与腹壁下A的关系 | 疝囊位于A外侧   | 疝囊位于A内侧   |
| 与腹膜陷凹的关系 | 从腹股沟外侧窝脱出 | 从腹股沟内侧窝脱出 |
| 脱出方向     | 外上-->内下   | 向前脱出      |
**脱出路径**：
斜疝：腹壁下A外侧，自外上向内下；腹股沟管深环--腹股沟管--腹股沟管浅环--阴囊/大阴唇皮下
直疝：腹壁下A内侧，向前；腹股沟三角--腹股沟管浅环而出
