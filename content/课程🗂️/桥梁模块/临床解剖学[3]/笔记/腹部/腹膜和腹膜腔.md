> 腹膜：
> 	- 壁腹膜：衬于腹、盆腔壁内面和膈下面
> 	- 脏腹膜：覆盖于腹、盆腔脏器表面
> 腹膜腔：脏、壁腹膜所围成的潜在性腔隙，腹膜腔内有少量浆液
> 	- 男性腹膜腔--封闭腔隙
> 	- 女性腹膜腔--借输卵管腹腔口经输卵管、子宫、阴道与外界相通
> - **大腹膜腔**：网膜囊以外的腹膜腔
> - **小腹膜腔**：即**网膜囊**，位于胃和小网膜后方
> - *两者借网膜孔相交通*


| 腹膜内位                                     | 腹膜间位                       | 腹膜外位                                   |
| ---------------------------------------- | -------------------------- | -------------------------------------- |
| 表面几乎都被腹膜所覆盖                              | 表面大部分被腹膜所覆盖                | 仅有一面被腹膜覆盖                              |
| 胃、*十二指肠上部*、空肠、回肠、盲肠、阑尾、横结肠、乙状结肠、脾、卵巢、输卵管 | 肝、胆囊、升结肠、降结肠、直肠上部、子宫、充盈的膀胱 | 胰、*十二指肠降部和水平部*、直肠中部和下部、肾、肾上腺、输尿管、空虚的膀胱 |
> 腹膜间位器官无腹膜覆盖的部分
> - 肝-肝裸区
> - 胆囊-与肝相接部分
> - 升结肠、降结肠-与腹膜紧贴部分
> - 直肠上段-肛门
> - 子宫-与阴道项连
> - 膀胱-与尿道相连

==腹、盆腔脏器均位于腹腔之内、腹膜腔之外==

## 一、腹膜形成的结构
#### 1. 网膜
##### 1. 大网膜
- 连于胃大弯和横结肠之间的*双层腹膜*
1. ==胃结肠韧带==：前两层和后两层游离形成
##### ★2. 小网膜
- 连于膈、肝静脉韧带裂、肝门与胃小弯、十二指肠上部之间的双层腹膜
1. ==肝胃韧带==：从膈、肝静脉韧带裂连于胃小弯。内含*胃左右血管*、淋巴结等
2. ==肝十二指肠韧带==：从肝门连至十二指肠上部。内含*胆总管(右前方)、肝固有动脉(左前方)、肝门静脉(后方)*，还有淋巴管、神经等
##### 网膜孔(Winslow孔)
**★网膜孔的位置和毗邻？**
- 小网膜右侧游离缘后方
	- 上界--肝尾状叶
	- 下界--十二指肠上部
	- 后界--覆盖下腔静脉的腹膜
	- 前界--肝十二指肠韧带
- 是网膜囊与腹膜腔其余部分相通的唯一通道。

##### 网膜囊
**★网膜囊的毗邻？**
- 位于小网膜和胃后方的扁窄间隙，又称*小腹膜腔*，是腹膜腔的一部分
	- 上壁--肝尾状叶、膈下方的腹膜
	- 下壁--大网膜前后层愈着处
	- 前壁--上-->下：小网膜、胃后壁腹膜、大网膜前两层
	- 后壁--下-->上：大网膜后两层、横结肠及其系膜、覆盖胰/左肾/左肾上腺等处的腹膜
	- 左侧--脾、胃脾韧带、脾肾韧带
	- 右侧--*网膜孔通向大腹膜腔*
#### 2. 系膜
**★系膜内有什么结构？**
1. ==肠系膜==
	- 将空回肠系于腹后壁的双侧腹膜结构
	- *肠系膜根*：肠系膜附着于腹后壁的部分
	- 系膜的两层腹膜间含有*肠系膜上血管*的分支和属支、淋巴管、淋巴结、神经丛及脂肪
2. ==阑尾系膜==
	- 三角形，内有*阑尾血管*
3. ==横结肠系膜==
	- 自结肠右区至结肠左曲
	- 内有*中结肠血管*、淋巴管、淋巴结、神经
4. ==乙状结肠系膜==
	- 将乙状结肠固定于左下腹
	- 内含*乙状结肠和直肠上血管*
#### 3. 韧带
##### 1. 肝的韧带(8)
1. 镰状韧带
2. 肝圆韧带
3. 冠状韧带
4. 肝肾韧带
5. 左右三角韧带
6. 肝胃韧带
7. 肝十二指肠韧带
**★肝裸区**：冠状韧带上下两层之间相距较远，使肝后面位于冠状韧带两层之间的肝表面无腹膜覆盖，形成肝裸区。
##### 2. 胃的韧带(5)
1. 胃脾韧带
2. 胃结肠韧带
3. 胃胰韧带
4. 胃膈韧带
5. 肝胃韧带
**★胃裸区**：胃膈韧带两层相距较远，部分胃后壁缺少腹膜覆盖形成为裸区
##### 3. 脾的韧带(3)
1. 脾肾韧带
2. 膈脾韧带
3. 脾结肠韧带

#### 4. 皱襞
##### 1. 胃胰襞和肝胰襞
**胃胰襞**：胃左动脉从腹后壁走向胃小弯时形成
**肝胰襞**：肝总动脉或肝固有动脉从腹后壁向前进入小网膜时形成
##### 2. 十二指肠上下襞
**十二指肠上襞**：十二指肠升部左侧，相当于L2平面，半月形，下端游离。深面为*十二指肠上隐窝*，隐窝下有肠系膜下静脉通行
**十二指肠下襞**：自十二指肠升部向左延伸至腹主动脉，平对L3，三角形，上端游离。深面为*十二指肠下隐窝*

## 二、隐窝和陷凹
>- 在腹膜皱襞之间或皱襞与腹、盆壁之间的凹陷称隐窝(recess)
>- 比隐窝大的凹陷称陷凹(pouch) 

- **十二指肠上隐窝**--十二指肠上襞深面
- 十二指肠下隐窝--十二指肠下襞深面
- 盲肠后隐窝-------盲肠后方
- 乙状结肠间隐窝--乙状结肠左后方
- **肝肾隐窝**---------肝右叶后下方和左肾之间，*仰卧时腹膜腔的最低处*
---
- 直肠膀胱陷凹(男)--腹膜腔最低位
- 直肠子宫陷凹(女)--腹膜腔最低位
- 膀胱子宫陷凹(女)


## 三、腹膜腔的分区
>以横结肠及其系膜为界，分为结肠上区和结肠下区

#### ★结肠上区(膈下间隙)
![Pasted image 20251102161336](Pasted%20image%2020251102161336.png)
肝裸区与胃裸区：膈下腹膜外间隙

#### 结肠下区
1. ==左结肠旁沟==
	1. 上：膈结肠韧带，不与结肠上区相通
	2. 下通：盆腔
2. ==右结肠旁沟==
	1. 上通：肝肾隐窝
	2. ★下通：右髂窝、盆腔
3. ==左肠系膜窦==（**★毗邻**）
	- 内侧：肠系膜根
	- 外侧：降结肠
	- 上界：横结肠及其系膜的左1/3部
	- 下界：乙状结肠及其系膜
	- 后界：贴附于腹后壁的腹膜
4. ==右肠系膜窦==
	1. 内侧：肠系膜根
	2. 外侧：升结肠
	3. 上界：横结肠及其系膜的右2/3部
	4. 下界：回肠末端阻隔，近乎封闭
	5. 后界：贴附于腹后壁的腹膜

```dataviewjs
// =====================================================
// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);


```
