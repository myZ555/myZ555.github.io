##### 3. 血管神经束

^bb4ca6

###### 1. 桡血管神经束
- 行于肱桡肌内侧或深面
1. 桡动静脉
2. 桡神经浅支

- 桡动脉--触摸脉搏部位
- 损伤：
	- 肱骨外科颈：腋神经
	- 肱骨中部：桡神经
	- 肱骨内上髁：尺神经

###### 2. 尺血管神经束
- 行于尺侧腕屈肌桡侧
1. 尺动静脉
2. 尺神经
###### 3. 正中血管神经束
1. 正中神经
2. 正中血管
###### 4. 骨间前血管神经束
1. 骨间前神经
2. 骨间前动静脉
##### 4. 前臂屈肌后间隙

|     |            |
| --- | ---------- |

| 外侧  | 桡侧腕屈肌、前臂筋膜 |
| 前   | 指深屈肌、拇长屈肌腱 |
