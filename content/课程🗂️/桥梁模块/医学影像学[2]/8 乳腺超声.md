### 1. 正常解剖与超声声像图基础

浅表超声极其看重解剖层次和腺体评估。

| 评估维度                | 核心影像学表现 / 临床意义                                                                                            |
| :------------------ | :-------------------------------------------------------------------------------------------------------- |
| ==终末导管小叶单位 (TDLU)== | 正常成年女性乳腺的小叶与终末导管组成TDLU。典型超声呈**“网球拍”样**，且球拍柄总是**指向乳头方向**。大多数乳腺导管病变均起源于此。                                   |
| ==腺体组织成分 (GTC)==    | 评估低回声区（腺体组织）占纤维腺体组织（FGT）的比例。GTC分为4类：极少（<25%）、轻度（25%-49%）、中度（50%-74%）、**显著（≥75%）**。**GTC越高，未来发生乳腺癌的风险越大**。 |
| ==Cooper韧带==        | 连接正常腺体与皮下/胸肌的结缔组织，超声上呈线状高回声。**Cooper韧带缩短、变直或增厚是恶性肿瘤的特征**（结构扭曲）。                                           |

### 2. BI-RADS分类与良恶性鉴别

这是考试的绝对重点！必须熟练掌握这10个维度的对比描述。

####  BI-RADS 风险分级速记

- **1-2类**：良性（恶性率基本为 0%）。
- **3类**：可能良性，需短期随访（恶性率 ≤ 2%）。
- **4类 (a/b/c)**：可疑恶性，需组织病理学诊断（恶性率 2% - 95%）。
- **5类**：高度提示恶性（恶性率 ≥ 95%）。

#### 二维及CDFI观察指标对比

| 观察维度             | 良性倾向特征 (如纤维腺瘤)            | 恶性倾向特征 (如浸润性导管癌)                                       |
| :--------------- | :------------------------ | :----------------------------------------------------- |
| ==1. 形态==        | 规则、椭圆形或圆形、呈浅分叶状。          | **不规则**。                                               |
| ==2. 方位 (极其重要)== | **平行于皮肤**（长轴与皮肤平行，纵横比<1）。 | **不平行于皮肤**（前后径>水平径，纵横比>1），提示肿瘤垂直生长具有高度侵袭力。             |
| ==3. 边缘==        | 光整，全部边缘界限清楚，有假回声包膜。       | **不光整**，表现为模糊（高回声晕）、**成角**（锯齿状）、微小分叶、**毛刺**（放射状锐利线状物）。 |
| ==4. 内部回声==      | 均匀低回声。                    | **不均匀低回声**或极低回声。                                       |
| ==5. 后方回声==      | 增强或无明显改变。                 | **后方声影（衰减）**或混合性改变。                                    |
| ==6. 钙化==        | 少见，多为粗大、**爆米花状**钙化（退化所致）。 | 多见，**微小钙化 / 砂砾样钙化**（由于癌细胞生长过快坏死引起）。                    |
| ==7. 结构扭曲==      | 无。                        | 肿块周围导管扭曲，**Cooper韧带缩短、变直或增厚**，侵犯脂肪。                    |
| ==8. 皮肤改变==      | 无（或仅在急性乳腺炎/脓肿时出现）。        | **皮肤增厚**、**皮肤回缩**（表面下凹/牵拉）。                            |
| ==9. 水肿==        | 无。                        | 周围组织回声增强，呈**网格状**（代表扩张的淋巴管或间质积液），常见于**炎性乳癌**。          |
| ==10. 血管供应==     | 无或少血流，低速低阻型。边缘血供多见于脓肿。    | **内部血供丰富**，穿透边缘，呈高速高阻型。                                |

### 3. 常见疾病综合诊断


| 疾病名称            | 临床特征                                | 典型超声 / CDFI 表现                                                              |
| :-------------- | :---------------------------------- | :-------------------------------------------------------------------------- |
| ==急性乳腺炎 / 脓肿==  | 多发于产后哺乳期初产妇（乳汁淤积、金葡菌感染），表现为发热、红肿热痛。 | 腺体内见**边界欠清的不均匀低回声区**，血流丰富。形成脓肿时内部为无回声/弱回声，**探头加压有流动感**，同侧腋下淋巴结肿大。           |
| ==纤维腺瘤==        | 女性最好发的良性肿瘤，20-25岁最多见（与雌激素刺激有关）。     | **椭圆形、平行于皮肤、边缘光整有包膜**，内部回声均匀，后方回声增强。血流较少。                                   |
| ==浸润性乳腺癌==      | 最常见恶性肿瘤，可伴腋窝淋巴结受累。                  | 完美符合恶性特征：**形态不规则、纵径大于长径（不平行）、边缘模糊/毛刺、微小钙化、后方声影**。冠状面上可见破坏周围结构的特异性**“汇聚征”**。 |
| ==导管内癌 (DCIS)== | 原位癌。                                | 常表现为**非肿块病变**，不易察觉，大小难测，**节段性分布**，常伴有微小钙化。                                  |
