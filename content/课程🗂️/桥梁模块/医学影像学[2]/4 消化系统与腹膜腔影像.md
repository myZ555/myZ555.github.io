# 检查方法与正常表现

| 检查方法           | 在本系统的主要作用 / 地位                                 | 正常影像学表现简述                                                                                                                       |
| -------------- | ---------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **X线 (平片/造影)** | 钡餐/钡灌肠是观察消化道黏膜的首选；立位腹部平片是排查急腹症（穿孔/梗阻）的最简便手段。   | 钡剂造影须无穿孔、梗阻禁忌。正常食管有主动脉弓、左主支气管、左心房三个生理压迹。                                                                                        |
| **CT / CTA**   | 消化系统实质性与空腔脏器肿瘤分期及急腹症病因评估的主要手段。                 | **多期增强扫描**是肝脏病变诊断的核心：动脉期(20-25s)、门脉期(50-60s)和平衡期(110-120s)。正常胃壁充分扩张时厚度<5mm，小肠壁<3mm。                                             |
| **MRI / MRCP** | ==MRCP (磁共振胰胆管成像)==： 利用重T2WI水成像技术，完美无创显示胆道树全貌。 | **肝胆特异性对比剂 (普美显, 钆塞酸二钠)**：可进行肝胆期成像，对微小肝癌(HCC)敏感。<br>**化学位移成像 (同/反相位)**：对脂肪极其敏感，用于定量肝脏脂肪。                                        |
| **超声 (US)**    | 肝、胆、胰、脾的**首选筛查方法**。                            | **检查准备**：须空腹，以防胃肠气体干扰并确保胆囊充盈。<br>**正常测值**：肝右叶斜径<12-14cm，胆囊长7-9cm、壁厚<3mm。<br>**经典解剖**：肝门部可见门静脉、胆总管、肝动脉形成的 ==米老鼠征 (Mickey sign)== |
# 基本病变
| 基本病变          | X线 / CT 典型表现 | 发生意义                                               |
| ------------- | ------------ | -------------------------------------------------- |
| **轮廓改变 (X线)** | ==充盈缺损==     | 管壁向腔内的局限性突起（多为息肉或恶性肿瘤），致使局部不能充盈钡剂。                 |
| **轮廓改变 (X线)** | ==龛影==       | 管壁局限性溃烂缺损达一定深度，钡剂充填其内，形成突出于管腔外的钡斑影。                |
| **轮廓改变 (X线)** | ==憩室==       | 消化道管壁局部发育不良/薄弱，向壁外呈囊袋状膨出，**内有正常黏膜通入**（与龛影不同）。      |
| **超声伪像/特征**   | ==彗星尾征/ 声影== | 胆囊内，**胆固醇结晶/腺肌症**因多次反射常伴“彗星尾征”；而**真正的胆结石**后方伴“声影”。 |
# 常见疾病
### 胃肠道与食管

| 疾病名称             | 核心影像表现 (X线/CT)                                                                | 典型征象 / 鉴别要点                              |
| ---------------- | ----------------------------------------------------------------------------- | ---------------------------------------- |
| ==食管静脉曲张==       | 钡餐早期黏膜迂曲，典型呈**蚯蚓状或串珠状充盈缺损**。                                                  | 常继发于肝硬化门脉高压。                             |
| ==食管贲门失弛缓症==     | 钡餐示下段管壁狭窄呈**“漏斗状或鸟嘴状”**，上段极度扩张，管壁黏膜光整。                                        | 需与食管癌（管壁不规则、黏膜破坏僵硬）鉴别。                   |
| ==胃与十二指肠溃疡==     | 直接征象为突出腔外的**龛影**；间接征象包括黏膜水肿带（黏膜线、项圈征、狭颈征）及黏膜向龛影纠集。                            | 十二指肠溃疡好发于球部，常致球部恒久变形（如三叶草形）。良性龛影多突出于轮廓外。 |
| ==胃癌==           | X线可见不规则充盈缺损、管腔狭窄僵硬（全胃受累称*皮革胃*）。<br>溃疡型胃癌典型表现为 *半月综合征*（不规则半月形龛影位于轮廓内，伴环堤及指压痕）。 | CT/MRI可评估胃壁增厚及淋巴结/肝转移。                   |
| ==克罗恩病 (Crohn)== | 钡餐可见 *“鹅卵石征”* 及*跳跃性/节段性狭窄*。CT增强可见系膜血管增多呈 *“梳齿征 (Comb sign)”*                  | 多累及回肠末端，常呈跳跃性、非连续性分布，易生瘘管。               |
| ==溃疡性结肠炎 (UC)==  | 钡剂可见颗粒状黏膜、锯齿状边缘；晚期结肠袋消失呈 *铅管样改变*                                              | 呈连续性分布，主要累及直肠及左半结肠。                      |
| ==结直肠癌==         | X线钡灌肠/CT示肠壁环周浸润性狭窄，典型表现为 *苹果核征*                                               | 肿瘤沿肠壁环周浸润超2/3，两端隆起中央狭窄。                  |
### 肝脏疾病
| 疾病名称           | 影像学表现 (超声 + CT/MRI)                                                                  | 典型征象 / 鉴别要点                     |
| -------------- | ------------------------------------------------------------------------------------ | ------------------------------- |
| ==脂肪肝==        | CT示肝密度普遍减低，CT值低于脾脏，出现**“血管湮没/反转征”**。MRI在**反相位(Out-of-phase)**上肝信号明显衰减。               | 多无超声课件描述，但影像学以CT密度和MRI化学位移为金标准。 |
| ==肝海绵状血管瘤==    | 最常见良性肿瘤。CT/MRI动态增强呈 **“早出晚归” / “快进慢出”**（边缘结节状强化向中心填充）。MRI T2WI呈特征性 *灯泡征*（随TE延长信号极高）。 | 血窦内血液流动缓慢导致特殊的强化模式。             |
| ==肝细胞癌 (HCC)== | CT/MRI动态增强呈典型 **“快进快出”**（动脉期明显强化，门脉/延迟期迅速减退呈低信号）。**肝胆特异性MRI（钆塞酸二钠）**肝胆期呈低摄取。         | 常伴肝硬化背景，AFP升高。需与血管瘤的“快进慢出”鉴别。   |
| ==肝转移瘤==       | CT/MRI增强典型表现为肿块边缘环状强化，中央坏死无强化，呈 *牛眼征*。                                               | 常多发，多伴有胃肠、乳腺、肺等原发灶。             |
| ==肝硬化==        | 肝边缘凹凸不平、各肝叶比例失常，实质内见**再生结节(RN)**。常合并脾大、门脉高压、腹水。                                      | 晚期RN可能演变为不典型增生(DN)乃至HCC。        |
### 胆道与胆囊疾病
| 疾病名称        | 影像学表现 (超声核心 + CT/MRCP)                                         | 典型征象 / 鉴别要点                               |
| ----------- | -------------------------------------------------------------- | ----------------------------------------- |
| ==胆囊结石==    | **超声**（金标准）：腔内出现**强回声光团**，后方伴**干净声影**，且可随体位重力方向**移动**。         | 当结石嵌顿于胆囊颈部压迫肝总管引起扩张和黄疸时，称为 ==Mirizzi综合征== |
| ==胆囊息肉==    | **超声**：囊壁向腔内突出的等/强回声结节，**不随体位移动，后方无声影**。                       | 分为胆固醇性（多发，常伴彗星尾征）、炎症性及腺瘤性（宽基底、有血流，有恶变倾向）。 |
| ==胆囊腺肌症==   | **超声**：胆囊壁局限、节段或弥漫性增厚，增厚壁内见无回声或强回声点（胆固醇结晶在罗-阿氏窦内沉积），常伴**彗尾征**。 | 属于良性增生性病变，超声彩超可见血流闪烁伪像排除真血流。              |
| ==先天性胆管扩张== | 最常见为**I型**（胆总管囊状/梭形扩张）、**V型**（Caroli病，多发肝内胆管囊状扩张）。             | 青年/儿童多发，MRCP显示极佳。                         |
### 胰腺
| 疾病名称      | 影像学表现                                                                                   | 典型特征 / 临床结合                         |
| --------- | --------------------------------------------------------------------------------------- | ----------------------------------- |
| ==急性胰腺炎== | **超声**：水肿型体积增大边缘整齐；坏死型边缘模糊。可并发**假性囊肿**（囊实性无回声）。<br>**CT**：胰周渗出、左侧肾前筋膜增厚。坏死型胰腺组织不均匀/无强化。 | 暴饮暴食/胆系结石诱发，表现为剧烈上腹痛放射至背部，血/尿淀粉酶升高。 |
| ==胰头癌==   | 典型征象：压迫并引起胆总管与主胰管同时扩张，称为 *双管征 (Double duct sign)*。超声可见不规则低回声且血供匮乏。                      | 常伴无痛性进行性梗阻性黄疸。                      |
### 急腹症
| 疾病名称      | 影像学表现                                              | 典型特征 / 临床结合                    |
| --------- | -------------------------------------------------- | ------------------------------ |
| ==胃肠道穿孔== | **立位腹部平片**：可见典型 **膈下游离气体**（新月形透亮影）。                | 突发刀割样剧痛，全腹压痛及反跳痛（板状腹）。         |
| ==肠梗阻==   | **立位平片/CT**：肠腔明显扩张积气，腔内出现高低不等、长短不一的 **气液平面**（阶梯状）。 | 分为机械性（粪石/套叠）、动力性和血运性（肠系膜血管栓塞）。 |

# 鉴别诊断
| 鉴别诊断                               | 疾病 A 及典型征象                                                                           | 疾病 B 及典型征象                                                                        | 核心鉴别要点与首选手段                                |
| ---------------------------------- | ------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------- | ------------------------------------------ |
| **1. 胃溃疡的良恶性**<br>_(X线钡餐)_         | **良性胃溃疡**：<br>龛影**突出于胃轮廓外**；口部有黏膜水肿带（黏膜线、项圈征）；黏膜皱襞向龛影集中直达口部；附近胃壁柔软有蠕动。               | **恶性胃溃疡 (溃疡型胃癌)**：<br>龛影位于**胃轮廓之内**（半月综合征）；周围见环堤、指压痕、裂隙征；黏膜皱襞破坏中断；附近胃壁僵硬。         | **龛影与轮廓的位置关系** 是最重要的X线鉴别点。良性向外突，恶性向内凹。     |
| **2. 肝脏富血供肿瘤**<br>_(CT / MRI 增强)_  | **肝细胞癌 (HCC)**：<br>多期增强呈典型的**“快进快出”**（动脉期明显强化，门脉期迅速减退）；常伴有肝硬化背景、门静脉瘤栓；MRI 肝胆特异期呈低摄取。 | **肝海绵状血管瘤**：<br>多期增强呈典型的**“早出晚归 / 快进慢出”**（边缘结节状强化向中心逐渐填充）；MRI T2WI 呈特征性**“灯泡征”**。 | **静脉期/延迟期的造影剂减退情况** 是金标准。HCC 迅速变暗，血管瘤持续强化。 |
| **3. 胆囊内强/高回声**<br>_(超声)_          | **胆囊结石**：<br>腔内强回声光团，后方伴**干净声影**，且随体位改变而**移动**。                                      | **胆囊息肉 / 胆固醇结晶**：<br>壁上高回声光点，**不移动**。息肉**无声影**，常有血管蒂；胆固醇结晶/腺肌症后方伴**“彗星尾征”**。      | 探头压迫或让患者翻身，观察**是否移动**及**后方有无声影/彗尾**。       |
| **4. 炎症性肠病 (IBD)**<br>_(CT / 钡灌肠)_ | **克罗恩病 (Crohn)**：<br>病变呈**跳跃性/多节段性**分布；好发回肠末端；特征为“鹅卵石征”、梳齿征（CT系膜血管增多），易并发瘘管。         | **溃疡性结肠炎 (UC)**：<br>病变呈**连续性**分布；好发于直肠及左半结肠；特征为黏膜颗粒状/锯齿状，晚期呈“铅管样改变”。              | **病变分布（跳跃 vs 连续）****累及部位（小肠为主 vs 结直肠为主）**。 |
| **5. 梗阻性黄疸病因**<br>_(超声 / CT)_      | **胰头癌**：<br>胰头局限性增大，呈实性低回声/低密度乏血供肿块；同时引起胆总管与主胰管扩张（**双管征**）。                          | **胆管结石 / 胆管癌**：<br>结石表现为胆管内强回声伴声影；胆管癌表现为胆管内实性回声，但**通常不伴有胰头肿块和主胰管扩张**。             | 寻找梗阻端。若**胆管与胰管同时扩张**（双管征），高度提示梗阻部位在壶腹部或胰头。 |
