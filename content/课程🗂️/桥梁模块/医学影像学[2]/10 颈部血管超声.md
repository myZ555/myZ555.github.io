## 1. 正常解剖与超声声像图基础

| 评估维度            | 核心影像学表现 / 临床意义                                                                                              |
| --------------- | ----------------------------------------------------------------------------------------------------------- |
| ==血管壁三层结构==     | **内膜**：血管内壁，呈线样连续光滑的等回声光带；<br>**中膜**：平滑肌细胞，呈低回声；<br>**外膜**：致密结缔组织，呈强回声光带。                                   |
| ==颈动脉IMT测量==    | **IMT (内中膜厚度)** 是评估早期动脉粥样硬化的关键指标。测量部位通常在**颈总动脉远段**（分叉水平下方1-1.5cm处）。<br>*正常*：<1.0mm<br>*异常*：1.0mm≤IMT<1.5mm。 |
| ==正常血流 (CDFI)== | 彩色多普勒呈现单色、亮带、层流特征。                                                                                          |

## 2. 斑块检测与易损性评估 (稳定 vs 易损)

##### 斑块的诊断标准
满足以下任意一条即可诊断为斑块：
1. 颈动脉 IMT≥1.5mm
2. 凸入动脉管腔的局部结构变化，厚度至少为 0.5mm
3. 大于周边 IMT 的 50%

|              | 稳定斑块特征              | 易损斑块特征 (高危风险)                                           |
| ------------ | ------------------- | ------------------------------------------------------- |
| ==1. 形态学==   | 形态规则，表面光整。          | 形态不规则，表面不光整。                                            |
| ==2. 内部回声==  | **等、高或强回声**，表现为均质性。 | **低回声、低至无回声**（极低回声提示坏死的脂质核心和炎症），内部**不均质**（回声不一致超过20%）。  |
| ==3. 表面纤维帽== | 完整、连续光滑。            | 不完整或**溃疡形成**。当发生“火山口征”（长度与深度均 ≥2.0mm）时，CDFI 可见血流向斑块内灌注。 |
| ==4. 新生血管==  | 无明显新生血管。            | 斑块内部出现**新生血管形成**（可通过超声造影或超微血管成像显示），是引发斑块内出血和破裂的重要因素。    |

## 3. 常见病理状态：颈动脉狭窄程度分级 (流速法)

📌 颈内动脉狭窄的血流参数标准 (综合2003北美/2006宣武标准)

| 狭窄程度               | PSV (收缩期峰值流速) 改变阈值                     | EDV (舒张期末流速) 改变阈值                    |
| ------------------ | -------------------------------------- | ------------------------------------ |
| **正常或 < 50% 轻度狭窄** | PSV<125 cm/s （或 <155 cm/s）             | EDV<40 cm/s （或 <60 cm/s）             |
| **50%～69% 中度狭窄**   | PSV 介于 125∼<230 cm/s （或 155∼<220 cm/s） | EDV 介于 40∼<100 cm/s （或 60∼<100 cm/s） |
| **70%～99% 重度狭窄**   | PSV≥230 cm/s （或 ≥220 cm/s）             | EDV≥100 cm/s                         |
| **完全闭塞**           | 无血流信号                                  | 无血流信号                                |
