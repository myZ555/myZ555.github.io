# 超声
## 1. 正常解剖与超声声像图基础

| 评估维度              | 核心影像学表现 / 临床意义                                                                 |
| ----------------- | ------------------------------------------------------------------------------ |
| ==解剖位置与毗邻==       | 甲状腺分左右两叶，中间以峡部连接，借韧带悬吊于软骨上，因此**作吞咽动作时会随之上、下移动**。其淋巴液主要汇入颈深淋巴结、气管前/旁及喉返神经周围淋巴结。 |
| ==正常实质回声==        | 呈细而密集的点状回声，分布均匀。**正常回声呈“中等回声”：即略高于颈前肌群回声，但低于甲状腺轮廓线回声**。若低于颈前肌群则视为回声减低。         |
| ==正常超声测值==        | 侧叶前后径和左右径约为 **20mm**，上下径为 **40～60mm**；峡部厚度约为 **5mm**。                          |
| ==血流与多普勒 (CDFI)== | 正常呈稀疏点状/条状血流。甲状腺上下动脉收缩期峰值流速（PSV）约 **22～33cm/s**，阻力指数（RI）约 **0.55～0.66**。       |

## 2. TI-RADS 恶性征象与良恶性鉴别



| 观察维度               | 良性倾向特征 (如腺瘤、结节性甲状腺肿)     | 恶性倾向特征 (甲状腺癌 TI-RADS征象)                    |
| ------------------ | ------------------------ | ------------------------------------------ |
| ==1. 形态 (纵横比)==    | 椭圆形或扁圆形，**纵横比 < 1**。     | 直立状生长，**纵横比 > 1（A/T > 1）**。                |
| ==2. 边缘与边界==       | 边界清楚、光滑，常有完整纤细的包膜或假包膜。   | 边缘模糊，边界不光滑、不规则（表现为**成角或微小分叶**），常伴有甲状腺包膜侵犯。 |
| ==3. 内部回声==        | 中等回声、高回声或混合回声。           | **实性低回声或极低回声**。                            |
| ==4. 声晕 (Halo)==   | 常见**特征性完整声晕**。           | 无声晕，或声晕不完整、厚薄不均。                           |
| ==5. 内部结构==        | 可呈**海绵状**结节或发生**囊性变**。   | 实性结节为主。                                    |
| ==6. 钙化特征==        | 无钙化，或多为粗大、弧状的**蛋壳样粗钙化**。 | 常出现**微小钙化 / 砂砾体 (Psammoma bodies)**[^1]。       |
| ==7. 血管供应 (CDFI)== | 呈现边缘型（绕边血流）或无血流型。        | 呈**中央型血流**（血流集中在结节内部）。                     |


## 3. 常见疾病综合诊断


| 疾病名称                         | 临床特征与病史                               | 典型二维超声 / CDFI 表现                                                                       |
| ---------------------------- | ------------------------------------- | -------------------------------------------------------------------------------------- |
| ==弥漫性毒性甲状腺肿 (甲亢 / Graves病)== | 青年女性多见，高代谢状态及多器官受累。                   | 甲状腺对称均匀性肿大，实质回声可呈弥漫或散在性减低。CDFI 示血流极其丰富，呈典型的**“火海征”**；上动脉流速明显加快（**>50~120cm/s**），呈低阻频谱。 |
| ==亚急性甲状腺炎==                  | **病毒感染（感冒）前驱病史**，女性多见，伴颈部疼痛及**局部压痛**。 | 腺体内见**散在和融合的低回声病灶**（可延伸至表层与颈前肌群粘连）。CDFI 示**病灶周边血流丰富，内部无或仅有少许血流**。随访时病灶可缩小或完全消失。        |
| ==甲状腺腺瘤==                    | 20-40岁女性多发，病程缓慢，少数可引起继发性甲亢。           | 多为单发圆形/椭圆形实质肿块，等回声多见，**边界清楚有包膜，可见特征性“晕征”**。CDFI 典型表现为血管从边缘伸入中央，呈**“辐轮”状血流**。           |
| ==甲状腺乳头状癌 (PTC)==            | **最常见的甲状腺癌**（占成人60%，儿童100%），病程慢，预后较好。 | 完美符合恶性征象：**实性低回声/极低回声、纵横比>1 (直立状)、边缘不规则(微小分叶/成角)、内部含微钙化/砂砾体**，常伴颈部淋巴结转移。               |

# 核医学
| 检查方法                             | 核心机制与原理                                   | 临床意义与技术细节                                     |
| -------------------------------- | ----------------------------------------- | --------------------------------------------- |
| **甲状腺摄131I率试验**                  | 碘是合成激素原料，甲状腺具有特异性浓聚碘的功能。                  | 正常24h达高峰（25%-60%）。服用碘盐会使测定值降低。                |
| **甲状腺静态显像**<br>(99mTcO4- / 131I) | 探测显像剂分布以观察形态与功能。99mTcO4-可被摄取但**不参与激素合成**。 | 寻找异位甲状腺（如舌根部、胸骨后）；131I全身显像用于**寻找分化型甲状腺癌转移灶**。 |


| 疾病名称 / 分类         | 核医学典型表现                                                             | 典型征象 / 鉴别要点                            |
| ----------------- | ------------------------------------------------------------------- | -------------------------------------- |
| ==典型甲亢 (Graves)== | 摄131I率显著增高，且**高峰前移**。甲状腺显像呈弥漫性肿大且摄取增强。                              | /                                      |
| ==甲状腺结节功能分型==     | **热结节**（摄取>周边，恶变率仅1%）；**温结节**（摄取≈周边）；**冷/凉结节**（摄取<周边，恶变率最高可达18.3%）。 | 考点：**冷结节不一定是癌，但热结节几乎都是良性（腺瘤）**。        |
| ==亚急性甲状腺炎==       | 摄131I率显著降低，但血液中T3、T4升高。                                             | **分离现象**（破坏的甲状腺滤泡将激素释放入血，但剩余滤泡丧失摄碘能力）。 |
| ==甲状旁腺功能亢进==      | **99mTc-MIBI 双时相显像**：早期相甲状腺+甲状旁腺均显影，延迟相甲状腺影消退，**甲状旁腺腺瘤滞留显影**。       | 减影法：99mTc-MIBI影像 减去 99mTcO4-影像。        |

[^1]: 沙砾体：甲状腺乳头状癌特征性病理改变，是由于癌细胞生长过快导致“死亡钙化”形成的同心圆状钙化小体，超声上表现为微钙化。
