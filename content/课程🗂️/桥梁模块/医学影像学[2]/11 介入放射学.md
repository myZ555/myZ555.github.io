### 1. 基础概念

| 核心概念 / 维度      | 关键知识点与影像学表现                                                      |
| :------------- | :--------------------------------------------------------------- |
| **学科定位**       | 在影像设备引导下，采用经皮穿刺或经自然孔道，对疾病进行**微创诊断和治疗**的学科。与内科、外科并列为**三大治疗学科**。   |
| **介入导向设备**     | 包括 X线透视、**DSA（数字减影血管造影，最核心设备）**、超声、CT和MRI。                       |
| **常用耗材 (双链词)** | 导管 (Catheter)、导丝 (Guidewire)、球囊 (Balloon)、支架 (Stent)、滤器 (Filter) |


### 2. 核心技术分类 

| 介入分类        | 核心技术操作                                                                                                          | 常用治疗性耗材/药物                                               |
| :---------- | :-------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------- |
| **血管介入技术**  | 1. 经皮腔内血管成形术 (球囊扩张、支架植入)<br>2. 经导管**血管栓塞**及封堵术 (用于止血、肿瘤“饿死”)<br>3. 经导管动脉药物灌注术 (局部化疗、溶栓)                         | **栓塞剂**：微弹簧圈 (Coils)、PVA颗粒、明胶海绵、碘化油。<br>**对比剂**：碘剂、二氧化碳。 |
| **非血管介入技术** | 1. 经皮穿刺活检术<br>2. 经皮穿刺引流术 (如 PTCD 胆道引流[^1])<br>3. 非血管管腔扩张术 (消化道/气道狭窄撑开)<br>4. 经皮**肿瘤消融术** (冷冻、射频、微波等)<br>5. 放射性粒子植入术 | **引流管**、**消融针**、骨水泥、125I放射粒子。                            |

### 3. 各系统经典介入手术


#### 1. 血管-狭窄撑开与动脉瘤隔绝

| 典型疾病                    | 介入治疗术式                     | 影像与操作要点                              |
| :---------------------- | :------------------------- | :----------------------------------- |
| ==腹主动脉瘤 (AAA) / 主动脉夹层== | **覆膜支架腔内隔绝术 (EVAR/TEVAR)** | 著名考点！使用**覆膜支架**，将动脉瘤或夹层假腔与血流完全隔绝。    |
| ==颈动脉狭窄==               | 颈动脉支架植入术                   | 术中必须先放置**过滤器**，防止球囊扩张和放支架时斑块脱落导致脑梗死。 |
| ==下肢/肾动脉狭窄==            | 球囊扩张成形术 + 裸支架植入            | 使用球囊撑开狭窄处（如胫前动脉、肾动脉），必要时植入金属支架维持血流。  |

####  2. 血管-栓塞止血与肿瘤阻断

| 典型疾病           | 介入治疗术式        | 影像与操作要点                                                    |
| :------------- | :------------ | :--------------------------------------------------------- |
| ==消化道/呼吸道大出血== | 动脉栓塞止血术       | 如空肠间质瘤出血、鼻咽癌放疗后大出血。DSA找到造影剂外溢的破口，使用 **PVA 或微弹簧圈** 进行靶血管栓塞。 |
| ==妇产科大出血==     | 子宫动脉栓塞术 (UAE) | 如疤痕妊娠清宫后大出血，超选择插管至子宫动脉，注入栓塞剂迅速止血。                          |
| ==颅内动脉瘤==      | 前交通动脉瘤栓塞术     | 通过微导管向颅内动脉瘤腔内填入**微弹簧圈**，防止其破裂导致蛛网膜下腔出血。                    |

####  3. 门静脉高压与静脉回流障碍

| 典型疾病              | 介入治疗术式                   | 影像与操作要点                                                                                            |
| :---------------- | :----------------------- | :------------------------------------------------------------------------------------------------- |
| ==肝硬化门静脉高压==      | **经颈静脉肝内门体分流术 TIPS**[^2] | 🌟**介入放射学的“皇冠级手术”**。在肝静脉与门静脉之间打通一并植入**覆膜支架 (Viatorr支架)**。<br>**适应证**：食管胃底静脉曲张破裂出血、顽固性腹水/胸水、布加综合征等。 |
| ==下肢深静脉血栓 (DVT)== | 下腔静脉滤器 (IVCF) 植入术[^3]    | 在下腔静脉内放置伞状滤器，既能保持静脉通畅，又能拦截脱落的血栓，**预防致命性肺栓塞 (PE)**。                                                 |
| ==上腔静脉综合征==       | 上腔静脉支架植入术                | 恶性肿瘤压迫上腔静脉致面颈部水肿，球囊扩张后植入大支架恢复回流。                                                                   |

####  4. 非血管管道与脏器介入 -引流/成形

| 典型疾病          | 介入治疗术式                | 影像与操作要点                                           |
| :------------ | :-------------------- | :------------------------------------------------ |
| ==恶性梗阻性黄疸==   | 经皮肝穿刺胆道引流术 **(PTCD)** | 穿刺扩张的肝内胆管，置入引流管将胆汁引出体外，或植入 125I粒子胆道支架。            |
| ==食管癌狭窄与气管瘘== | 食管支架植入术               | 单纯狭窄可用**裸支架**；若合并食管气管瘘（吃东西漏进肺里），必须使用**覆膜支架**封堵瘘口。 |
| ==椎体压缩性骨折==   | **椎体成形术 (PVP)**       | 经皮穿刺病变椎体（如骨质疏松或转移瘤），注入**骨水泥**，迅速缓解疼痛并强化椎体。        |
| ==肝囊肿 / 肾囊肿== | 囊肿抽吸硬化术               | 超声引导下抽干囊液，并注入无水乙醇等硬化剂破坏囊壁内皮。                      |
