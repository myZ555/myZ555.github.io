## 检查方法与正常表现

| 检查方法 | 在本系统的主要作用/地位                                                         | 正常影像学表现简述                                                                  |
| ---- | -------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| X线   | **骨骼系统首选和最常用的检查方法**，经济方便。<br>前后重叠，软组织显示不佳。                           | 正常可见密质骨（如骨皮质）与松质骨（如海绵状骨小梁）；<br>儿童期可见骺板（线）。<br>X线上的“关节间隙”=关节软骨+真正关节间隙+少量滑液。 |
| CT   | 无重叠，密度分辨力高；<br>擅长显示小病灶、隐匿性骨折、复杂部位骨折重建（如骨盆、脊柱等）。                      | 断面成像，通过调整骨窗和软组织窗，可精细显示骨与周围结构的关系。                                           |
| MRI  | 无辐射，软组织分辨力极高。<br>**评估软组织（脂肪、肌肉、韧带、肌腱）、软骨、骨髓病变及骨挫伤的首选**；但钙化和骨膜反应显示欠佳。 | 软组织和软骨清晰，例如正常的膝关节半月板在MRI上呈均匀的低信号。                                          |
>[!Warning] 检查方法的优选原则
>- **X线平片是首选**：对于骨与关节创伤、骨骼基本病变，X线经济方便，永远是首选和最常用的检查方法。
>- **MRI的绝对优势**：X线和CT对软组织显示不佳，遇到**骨挫伤（骨髓水肿与出血）、椎间盘突出、半月板撕裂**以及评估肌肉、韧带、肌腱和软骨病变时，MRI是首选及最敏感的检查手段。
>- **CT的补充价值**：CT通常作为非常规检查，但在寻找隐匿性骨折、精细观察小骨质破坏区、小死骨以及复杂部位（如骨盆、脊柱）骨折的三维重建时具有不可替代的作用。
## 基本病变
1. 骨骼基本病变（6）
	1. **骨质疏松 (Osteoporosis)**：指一定单位体积内正常钙化的骨组织量减少，其特点是**有机成份和钙盐同比例减少**。
	2. **骨质软化 (Osteomalacia)**：指骨组织有机成份正常，**仅矿物质（钙盐）含量减少**。
	3. **骨质破坏 (Bone destruction)**：局部骨质为病理组织（如肿瘤、肉芽组织）所代替而消失。
	4. **骨质增生硬化 (Hyperostosis and osteosclerosis)**：一定单位体积内骨量的增多。
	5. **骨膜增生 / 骨膜反应 (Periosteal proliferation)**：骨膜受刺激水肿、增厚，导致成骨细胞活动增加形成的骨膜下新生骨。
	6. **骨质坏死 (Bone necrosis)**：骨组织局部代谢停止，坏死的骨质被称为**死骨**。
2. 关节基本病变（5）
	1. **关节肿胀 (Swelling of joint)**：由于关节积液或关节囊及周围软组织充血、水肿、出血引起，MRI是最佳观察手段。
	2. **关节破坏 (Destruction of joint)**：关节软骨及其下方的骨性关节面被病理组织侵犯、代替，导致关节间隙变窄和骨质缺损。
	3. **关节退行性变 (Degeneration of joint)**：以关节软骨退变、坏死和溶解为基础，中晚期出现关节面边缘**骨赘形成**（骨质增生），一般无明显骨质破坏。
	4. **关节强直 (Ankylosis of joint)**：关节破坏后骨端相互连接。化脓性关节炎愈合后常形成**骨性强直**（有骨小梁贯穿）；关节结核常形成**纤维性强直**（无骨小梁贯穿）。
	5. **关节脱位 (Dislocation of joint)**：关节骨端解剖位置的改变，可有半脱位或全脱位。
3. 软组织基本病变（4）
	1. **软组织肿胀**：因炎症、出血、水肿或脓肿等原因引起的软组织肿大膨胀
	2. **软组织肿块**：多因软组织肿瘤引起，骨肿瘤破坏骨皮质侵入软组织也可形成
	3. **软组织内钙化和骨化**：指肌肉、肌腱、关节囊、血管和淋巴结等软组织，由于出血、坏死、肿瘤、结核和寄生虫感染等病理变化而发生钙化或骨化
	4. **软组织内气体**：指软组织外伤、手术或产气杆菌感染等病理情况下导致的软组织内积气征象。

| 基本病变                | X线 / CT 典型表现                                                                        | 常见对应疾病 / 病理基础                                         |
| ------------------- | ----------------------------------------------------------------------------------- | ----------------------------------------------------- |
| ==骨质疏松==            | 骨密度减低，骨小梁变细减少，骨皮质变薄分层；<br>脊椎：锥体内结构纵行条纹；严重时椎体结构消失、上下缘内凹，椎间隙增宽-梭形                     | 有机物与钙盐**同比例减少**（常见于老年、骨折后）；                           |
| ==骨质软化==            | 骨密度普遍减低、腰椎和骨盆明显；骨小梁和骨皮质*边缘模糊*；承重骨骼变形-*三叶草样骨盆*；**假骨折线**[^4]                          | **仅钙盐减少**（见于佝偻病、肾性骨病）。                                |
| ==骨质破坏==            | *局限性*骨质密度减低。良性病变边缘锐利；<br>骨松质破坏：斑片状缺损<br>骨皮质破坏：筛孔样破坏和表面的虫蚀状改变<br>**恶性病变边缘模糊、骨皮质中断**。 | 局部骨质被病理组织（如肿瘤、化脓性或结核性肉芽组织）代替。<br><u>炎症、肉芽肿及肿瘤</u>     |
| ==骨质增生硬化==          | 骨密度增高；骨小梁增粗增多密集；骨皮质增厚致密。<br>发生于长骨可见骨干粗大、骨髓腔变窄/消失                                    | 成骨增多和/或破骨减少所致<br>局限性：慢性炎症、退行性变等<br>普遍性：代谢性骨病、骨/软骨发育异常 |
| ==骨膜增生 (骨膜反应)==[^5] | 早期为与骨皮质平行的线状致密影。<br>良性多呈线状、层状、花边状；<br>**恶性多呈针状、放射状或形成Codman三角（骨膜三角）**[^1]。          | 骨膜受刺激水肿后成骨细胞活动增加形成新生骨。<br>Codman三角提示肿瘤生长迅速。           |
| ==骨与软骨钙化==          | *骨内钙化*：斑片状无骨小梁结构的高密度影<br>*瘤软骨钙化*：颗粒状、小环或半环状致密影                                      | 骨内钙化：钙盐异常沉积；骨肿瘤、结核<br>软骨钙化：生理性（肋软骨）、病理性（瘤软骨）          |
| ==骨质坏死==            | 表现为局限性的**骨质密度增高（死骨）**。                                                              | 血液供应中断导致骨局部代谢停止，常见于化脓性骨髓炎、缺血性坏死。                      |


| 基本病变       | X线 / CT 典型表现                                                       | 常见对应疾病 / 病理基础                                              |
| ---------- | ------------------------------------------------------------------ | ---------------------------------------------------------- |
| ==关节肿胀==   | 关节周围软组织影增厚、密度增高<br>关节腔内*积液*表现为关节间隙增宽（X线）或不同类型的液体密度影（CT）            | 关节腔积液或关节囊及周围软组织充血、水肿、出血或关节来源的肿瘤性病变；<br>关节炎症、外伤、出血和滑膜来源肿瘤等  |
| ==关节破坏==   | *关节间隙变窄*（软骨破坏），骨性关节面及邻近松质骨破坏缺损。                                    | 病理组织侵犯代替软骨和关节面，见于*关节结核、类风湿、化脓性关节炎*等。                       |
| ==关节退行性变== | 早期：骨性关节面模糊中断消失<br>中晚期：关节间隙变窄、软骨下骨质囊变和骨赘形成等                         | 早期改变起源于软骨；软骨碎片->*关节内游离体/“关节鼠”*；关节间隙变窄等<br>**承重**如脊椎、髋膝关节常见 |
| ==关节强直==   | **骨性强直**：关节间隙明显变窄或消失，有**骨小梁贯穿**关节；<br>**纤维性强直**：关节间隙狭窄，**无**骨小梁贯穿。 | 骨性强直常见于**急性化脓性关节炎愈合后**；<br>纤维性强直常见于**关节结核**。               |
| ==关节脱位==   | 完全脱位：原相对的关节面不接触<br>半脱位：原相对的关节面尚有部分接触                               | 外伤性、先天性、病理性                                                |

| 基本病变         | X线 / CT 典型表现                                                                                                              |
| ------------ | ------------------------------------------------------------------------------------------------------------------------- |
| ==软组织肿胀==    | 密度略高于正常软组织、皮下脂肪层可出现网状结构影<br>脓肿-边界较清；结核性脓肿壁可发生钙化<br>CT更优；脓肿边界清、内可见液体密度区；血肿高密度                                              |
| ==软组织肿块==    | 良性：边界清；邻近骨质可出现*压迫性骨吸收*或*反应性骨硬化*<br>恶性：边缘模糊；邻近被侵蚀的骨皮质可出现骨质破坏<br>*脂肪瘤*-低密度；*软骨类肿瘤*-钙化影；*骨化性肌炎*-成熟骨组织影<br>CT更优；CT增强扫描可反映血供情况 |
| ==软组织钙化和骨化== | 形状各异的*钙质样高密度影*<br>CT效果最佳                                                                                                  |
| ==软组织内气体==   | 气体呈不同形状的*极低密度影*                                                                                                           |
# 常见疾病

| 疾病名称        | 核心影像表现 (X线/CT)                                                                                                | 典型征象 / 鉴别要点                                       |
| ----------- | ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| ==特殊类型骨折==  | **青枝骨折**：儿童多见，仅皮质皱折/凹陷，无透亮骨折线；<br>**Colles骨折**：桡骨远端骨折，远段向背侧移位；<br>**骨挫伤**：X线/CT常无异常，**MRI表现为T1WI低、T2WI高信号区**。 | 骨折的并发症包括延迟愈合、骨缺血性坏死、骨化性肌炎等。                       |
| ==化脓性骨髓炎==  | **急性期**以骨质破坏伴增生为主，层状/花边状骨膜反应，常见**大块状死骨**，晚期易导致**骨性关节强直**；**慢性期**以骨增生硬化为主，髓腔变窄。                                | 致病菌多为金葡菌，起病急，进展快，病灶常在持重部位。                        |
| ==骨/关节结核==  | 骨质疏松和破坏为主，多见**“泥沙样”死骨**。短骨干结核可表现为膨胀性的**“骨气鼓”**[^2]；晚期常表现为**纤维性关节强直**，常伴邻近肌肉萎缩。                                | 起病缓，病程长，常累及非持重部位，少有骨质增生。脊椎结核常伴椎旁冷脓肿。              |
| ==退行性骨关节病== | 关节软骨退变引起。主要表现为**关节面边缘骨赘形成（骨质增生）**、关节间隙变窄、软骨下骨质硬化及囊变。**无明显骨质破坏及骨质疏松**。                                         | 与类风湿关节炎（以侵蚀性滑膜炎为特点）不同，退行病变很少引起全身骨质疏松。             |
| ==骨软骨瘤==    | 骨皮质和骨松质均**与母骨相连续**的骨性突起，表面有软骨帽（内可见钙化）。                                                                        | 最常见的良性骨肿瘤，多见于干骺端，**背离关节生长**。                      |
| ==骨巨细胞瘤==   | 20-40岁多发；长骨骨端。呈偏心性、膨胀性生长，典型的**“皂泡样”**透亮区。                                                                     | 典型表现为**“三无”：无硬化边、无骨膜反应、瘤内无钙化（无瘤骨）**。              |
| ==骨肉瘤==     | 青少年长骨干骺端。关键特征为：**肿瘤骨（云絮状、针状、斑块状）**[^3]、骨质破坏、骨膜增生、软组织肿块。                                                       | 最常见的原发恶性骨肿瘤；特征性骨膜反应包括**葱皮样、日光放射征、Codman三角（袖口征）**。 |
| ==转移性骨肿瘤==  | **溶骨型（最常见）**：斑点状、虫蚀状破坏，无硬化边，少有骨膜反应；<br>**成骨型**：斑点或块状硬化，常见于前列腺癌、乳腺癌等转移。                                        | 最常见的恶性骨肿瘤。好发于中轴骨（脊柱、骨盆、肋骨等）。                      |

## 鉴别诊断
#### 慢性化脓性骨髓炎VS成骨型骨肉瘤

|         | 慢性化脓性骨髓炎    | 成骨型骨肉瘤         |
| ------- | ----------- | -------------- |
| 临床表现    | 反复发作、局部窦道流脓 | 快速进展、间歇性或持续性疼痛 |
| 骨质增生硬化  | 广泛          | 云絮状、斑片状、针状瘤骨   |
| 死骨      | 大块          | 无              |
| 骨膜新生骨   | 广泛且成熟       | 多见、不成熟且可悲破坏    |
| 周围软组织改变 | 常无明显肿胀      | 肿块，其内可有瘤骨      |

#### 良性VS恶性骨肿瘤

|        | 良性                | 恶性                                       |
| ------ | ----------------- | ---------------------------------------- |
| 生长速度   | 缓慢                | 迅速                                       |
| 生长方式   | 膨胀性               | 浸润性                                      |
| 骨质破坏   | 与正常组织分界清晰，常有*硬化边* | 与周边界限不清，无硬化边                             |
| 骨皮质    | 变薄、膨胀但多完整         | 虫蚀状破坏、缺损、中断                              |
| 骨膜反应   | 一般无（除非发生病理骨折）     | 各种形式；尤其是*Codman三角*（肿瘤破坏骨膜新生骨后残存的三角形高密度灶） |
| 肿瘤骨    | 无                 | 常见，针状、放射状等                               |
| 软组织肿块  | 少有，边界清楚           | 常有，且肿块内有瘤骨或瘤软骨，边界不清                      |
| 远隔器官转移 | 无                 | 常见                                       |
#### 化脓性关节炎VS关节结核

|        | 化脓性关节炎                | 关节结核                          |
| ------ | --------------------- | ----------------------------- |
| 病程与进展  | 起病急、病程短               | 起病缓、病程长                       |
| 骨与软骨破坏 | 发生较早，多位于持重部位，常伴*骨质增生* | 破坏较晚，多位于非持重部位，常伴有明显*骨质疏松*     |
| 死骨形态   | 大块状死骨                 | *“泥沙样*死骨                      |
| 晚期关节强直 | 愈后常形成*骨性强直*（有骨小梁贯穿）   | 常形成*纤维性强直*（无骨小梁贯穿，且常伴有邻近肌肉萎缩） |

#### 强直性脊柱炎VS类风湿关节炎

|         | 强直性脊柱炎                | 类风湿关节炎                     |
| ------- | --------------------- | -------------------------- |
| 家族史     | 有                     | 一般无                        |
| 发病年龄    | 青年                    | 30-50岁                     |
| 性别      | 男多见                   | 女多见                        |
| HLA-B27 | 阳性                    | 阴性                         |
| 类风湿因子   | 阴性                    | 阳性                         |
| 四肢关节    | 非对称、大关节多见，下肢≥上肢       | 对称性、小关节多见，下肢≤上肢            |
| 脊柱受累    | 自腰椎逐渐向上发展，“方椎”“竹节椎”多见 | 主要累及颈椎，脊椎小关节间隙狭窄，寰枢关节半脱位多见 |
| 关节强直    | 骨性强直                  | 纤维性强直                      |

# Glossary

[^1]: Codman三角/骨膜三角：一种典型的恶性骨膜反应。肿瘤迅速生长，累及并破坏骨膜形成的新生骨质，其边缘残存的骨质呈三角形高密度灶。常见于骨肉瘤。

[^2]: 骨气鼓：短骨干结核的特有表现；病变部位出现囊状骨质破坏，导致骨皮质变薄、整个骨干膨胀，形似充气的鼓。

[^3]: 肿瘤骨/瘤骨：骨肉瘤的关键诊断特征，指肿瘤细胞自身成骨形成的骨样组织，形态可呈云絮状、针状或斑块状。

[^4]: 假骨折线（Pseudofracture line）：表现为宽1-2cm的透明线，与骨皮质垂直，边缘稍致密；好发于耻骨支、肱骨、股骨上段和胫骨等。

[^5]: 骨膜增生/骨膜反应：因骨膜受刺激出现水肿、增厚，导致骨膜内层成骨细胞活动增加，最终形成骨膜下新生骨。
