# 检查方法与正常表现

| 检查方法                              | 在本系统的主要作用 / 地位                                      | 正常影像学表现简述                                                                                                        |
| --------------------------------- | --------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| **X线 (胸片)**                       | 胸部疾病的最常用初查方法（常规拍后前位与侧位）。                            | **肺野划分**：人为分为内、中、外带，及上、中、下野。<br>**纵隔划分**：分为上、下纵隔，及前、中、后纵隔。                                                       |
| **CT / HRCT**                     | 临床应用越来越广泛，逐渐成为**胸部最常用检查方法**。                        | **后处理技术**：*MPR (多平面重组)*、*VR (容积再现)*、最小密度投影（minIP）构建气管树。<br>**次级肺小叶**：肺的基本解剖单位，由HRCT显示。包括小叶核心（细支气管/动脉）、小叶实质与小叶间隔。 |
| **MRI**                           | 对肺部病变**不敏感**，但对纵隔、胸壁和心脏疾病（如囊肿定性）有价值。                | /                                                                                                                |
|                                   |                                                     |                                                                                                                  |
| **肺灌注显像**<br>(99mTc-MAA)          | **机制**：微粒（直径10-90μm）一过性**嵌顿在肺毛细血管床**，反映局部肺血流量。极度安全。 | 放射性分布均匀，反映肺血流灌注。                                                                                                 |
| **肺通气显像**<br>(99mTc-DTPA / 133Xe) | **机制**：吸入放射性气溶胶或惰性气体，随气流分布于肺泡内，反映局部通气量。             | 放射性分布与灌注基本一致。                                                                                                    |
# 基本病变
| 基本病变                    | X线 / CT 典型表现                                                                                         | 常见对应疾病 / 病理基础                       |
| ----------------------- | ---------------------------------------------------------------------------------------------------- | ----------------------------------- |
| ==肺实变 (Consolidation)== | 肺泡内空气被病理组织/液体代替，表现为致密影。若大病灶内见含气支气管影，称为 *空气支气管征 (Air bronchogram)*                                    | 常见于大叶性肺炎、肺出血、肺水肿。渗出局限于腺泡时呈“梅花瓣”状结节。 |
| ==支气管阻塞==               | 表现为透亮度增加（肺气肿）或密度增高（肺不张）。右上肺肿瘤伴不张时，下缘呈典型的 **反S征 (Golden S sign)**。                                    | 腔内异物、肿瘤或腔外压迫所致。                     |
| ==空洞 (Cavity)==         | **厚壁空洞 (≥3mm)**：外缘毛刺/内缘凹凸不平多为肺癌。<br>**薄壁空洞 (<3mm)**：内外缘清晰，多为结核/转移瘤。<br>**虫蚀样空洞**：无壁，多发大小不等，多见于干酪性肺炎。 | 肺内病变组织坏死并经支气管排出形成。                  |
| ==结节与肿块 (恶性征象)==        | 呈浸润性生长。特征征象包括：*分叶征*（波浪状凸起）、*毛刺征*（短直细小毛刺）、*胸膜凹陷征*（牵拉胸膜呈喇叭样凹陷）、*空泡征*（内含1-3mm低密度气影）。                    | 高度提示**周围型肺癌**。                      |
| ==钙化 (Calcification)==  | *爆玉米花样钙化*（错构瘤）；*蛋壳样钙化*（尘肺淋巴结）；*单发点状钙化*（周围型肺癌）；*层状/环状钙化*（结核球等良性病变）。                                   | CT窗上 CT值 >100HU。                    |
| ==网状与条索状影==             | *Kerley B线*：垂直于肋胸膜的细线状影（长约2cm），见于间质性肺水肿。晚期表现为胸膜下 **“蜂窝状影”**。                                         | 肺间质病变（纤维化、炎性浸润、渗出等）。                |
| ==胸腔积液==                | 少量积液表现为肋膈角变钝；中大量呈外高内低弧线影。                                                                            | 漏出液（心衰等）、渗出液/脓液（炎症）。                |
# 常见疾病
### 感染性疾病

| 疾病名称               | 核心影像表现 (X线/CT)                                                           | 典型征象 / 鉴别要点                  |
| ------------------ | ------------------------------------------------------------------------ | ---------------------------- |
| ==大叶性肺炎==          | 节段或大叶性均匀致密实变影，常可见 *空气支气管征*。                                              | 肺炎链球菌引起，突发高热、咳铁锈色痰。          |
| ==小叶性/间质性肺炎==      | **小叶性**：两肺中下野多发斑片状模糊影。<br>**间质性**：网状/小斑片影，HRCT示支气管血管束增粗。                 | COVID-19 早期表现为双肺外周磨玻璃影(GGO)。 |
| ==肺脓肿==            | 早期大片实变；中央坏死排出后形成**厚壁空洞，常伴有** **气-液平面**。                                  | 咳大量脓痰（静置分层）。血源性（金葡菌）呈多发外围空洞。 |
| ==原发型肺结核 (I型)==    | 典型表现为 *原发综合征*（原发灶、淋巴管炎、肿大淋巴结组成的**哑铃双极现象**）。                              | 初次感染，好发儿童。                   |
| ==血行播散型肺结核 (II型)== | **急性（粟粒型）**：具有 **“三个均匀”**（分布、大小、密度均均匀一致的粟粒结节）。<br>**慢性**：具有 **“三个不均匀”**。 | 急性为大量结核菌一次进入血流；慢性为少量多次进入。    |
| ==继发型肺结核 (III型)==  | 多在上叶尖后段及下叶背段。新旧病灶同存，可见 *结核球*（边缘光滑伴卫星灶），或大叶干酪性肺炎。                         | 成人最常见类型，好发锁骨下区。              |
### 肿瘤性疾病
| 疾病名称           | 影像学表现与技术选择                                                                 | 临床细节与定位                        |
| -------------- | -------------------------------------------------------------------------- | ------------------------------ |
| ==中心型肺癌==      | 肿瘤向腔内突出或沿管壁浸润。直接征象：**管腔息肉样缺损、壁增厚、鼠尾状/杯口状截断**。                              | 常并发阻塞性肺不张/肺炎。                  |
| ==周围型肺癌==      | 肺内结节/肿块，呈现经典的恶性肿瘤特征（**分叶、毛刺、胸膜凹陷征、空泡征**）。                                  | CT增强呈明显均匀或中心强化。                |
| ==肺转移瘤==       | 血行播散多表现为两肺中下野外带**多发大小不一的球形病灶**。                                            | 密度均匀，轮廓清楚。                     |
| ==纵隔肿瘤 (按部位)== | **前纵隔**：胸腺瘤、畸胎瘤（含脂肪/钙化/骨骼）。<br>**中纵隔**：支气管囊肿、心包囊肿。<br>**后纵隔**：神经鞘瘤等神经源性肿瘤。 | 淋巴瘤常见于年轻患者，表现为多组淋巴结肿大伴“血管漂浮征”。 |
### 弥漫性肺疾病与支气管病变
| 疾病名称              | 影像学表现 (主要依赖 HRCT)                                              | 临床特征与病理               |
| ----------------- | -------------------------------------------------------------- | --------------------- |
| ==支气管扩张==         | CT 是最佳检查方法。表现为**柱状**（双轨征/印戒征）、**曲张样**（串珠状）或**囊状**（一簇葡萄样）。      | 青年多发，反复咯血咳大量脓痰。       |
| ==特发性肺纤维化 (IPF)== | 早期见下肺磨玻璃影；晚期双肺弥漫网状结节影及**胸膜下多发囊状透光影**（*蜂窝肺*）。                   | 中老年男性多发，进行性呼吸困难。      |
| ==肺泡蛋白沉积症 (PAP)== | HRCT 特征性表现为地图状分布的磨玻璃影叠加小叶间隔增厚，形成 **铺路石征 (Crazy-paving sign)**。 | 肺泡内充满富磷脂蛋白物质，灌洗物呈牛奶状。 |

### 核医学诊断
| 疾病名称                | 核心影像表现 (V/Q 扫描)                        | 典型征象 / 鉴别要点                                |
| ------------------- | -------------------------------------- | ------------------------------------------ |
| **肺栓塞 (PE)**        | 灌注显像：出现局限性显像剂稀疏或缺损。<br>通气显像：气道畅通，分布正常。 | **不匹配征 (Mismatch)**。即“灌注缺损，但通气正常”，高度提示肺栓塞。 |
| **慢性阻塞性肺疾病 (COPD)** | 灌注与通气显像均出现异常缺损。                        | **匹配征 (Match)**：缺损区范围一致或通气缺损大于灌注缺损。        |

# 总结
#### 同病异影
| 疾病大类      | 亚型或阶段         | 核心影像学表现 (同病异影)                                       |
| --------- | ------------- | ---------------------------------------------------- |
| **肺结核**   | 原发综合征 (I型)    | 表现为典型的**哑铃双极现象**（原发灶 - 淋巴管炎 - 肿大淋巴结）。                |
|           | 急性血行播散型 (II型) | *急性粟粒型肺结核*，表现为经典的 **“三个均匀”**：病灶分布均匀、大小均匀、密度均匀。       |
|           | 慢性血行播散型 (II型) | 表现为**“三个不均匀”**：分布不均（上中肺野为主）、大小不均、密度不均（可同时有钙化和渗出）。    |
|           | 干酪性肺炎 (特殊类型)  | 表现为大片致密影（密度高于大叶性肺炎），其内可见多发无壁的*虫蚀样空洞*。                |
| **原发性肺癌** | 中心型肺癌         | 肿瘤向腔内突出/浸润，直接征象为管腔**息肉样缺损、鼠尾状/杯口状截断**；伴发肺不张时形成*反S征*。 |
|           | 周围型肺癌         | 表现为肺内孤立结节/肿块，呈现经典的恶性肿瘤特征（如分叶、毛刺、空泡征、胸膜凹陷征）。          |
| **肺炎**    | 大叶性肺炎         | 节段或大叶的均匀渗出和实变，常可见*空气支气管征*。                           |
|           | 小叶性肺炎         | 沿肺纹理分布的**多发斑片状模糊影**，可融合，支气管血管束增粗。                    |

#### 异病同影
|                             | 疾病 A                                                           | 疾病 B                                                     | 疾病 C                                                     | 鉴别定性                                                     |
| --------------------------- | -------------------------------------------------------------- | -------------------------------------------------------- | -------------------------------------------------------- | -------------------------------------------------------- |
| **1. 结节与肿块**<br>_(单发孤立性)_   | **周围型肺癌**：<br>边缘有 *分叶征* 和 *毛刺征*，伴 *胸膜凹陷征* 和*空泡征*。增强呈明显均匀或中心强化。 | **结核球**：<br>边缘光滑，内有点状/环状钙化，周围常见 *卫星灶*。增强无强化或仅见周边环形强化。    | **错构瘤**：<br>边缘光滑呈球形，内有脂肪密度（CT值为负），特征性表现为 *爆玉米花样钙化*。      | 观察边缘是否锐利（有无毛刺/分叶）、有无卫星灶、以及内部是否有特定的钙化或脂肪成分。               |
| **2. 厚壁空洞**<br>_(洞壁 ≥ 3mm)_ | **结核性空洞**：<br>外缘多整齐清楚，邻近常有散在斑片状卫星灶。多见于上叶尖后段及下叶背段。              | **癌性空洞**：<br>外缘和轮廓呈分叶、毛刺，**内缘凹凸不平或呈结节状（壁结节）**。壁厚常 >15mm。 | **肺脓肿空洞**：<br>急性期大片实变中央坏死形成，特征性表现为常伴有*气-液平面*。            | 观察**内壁是否光滑**（癌性常凹凸不平）、**有无液平**（脓肿常有）以及**周围有无卫星灶**（结核常有）。 |
| **3. 大片致密实变影**              | **大叶性肺炎**：<br>占一个肺段或肺叶，密度较均匀，内可见 *空气支气管征*，肺体积正常。               | **大叶性干酪性肺炎**：<br>与大叶性肺炎相似，但**密度更高**，且内部极易出现多发*虫蚀样空洞*。    | **阻塞性肺不张**：<br>致密影伴有**体积缩小的征象**（如肋间隙变窄、叶间裂向心移位、纵隔向患侧移位）。 | 寻找**容积改变的标志**（纵隔/叶间裂是否移位）以排查肺不张，寻找空洞以排查干酪性肺炎。            |
| **4. 纵隔肿瘤**<br>_(部位与密度)_    | **胸腺瘤 / 畸胎瘤**：<br>好发于**前纵隔**。畸胎瘤密度不均，内含牙齿、脂肪或骨骼。               | **支气管囊肿 / 淋巴瘤**：<br>好发于**中纵隔**。淋巴瘤表现为分叶状肿块，可见特征性“血管漂浮征”。 | **神经源性肿瘤**：<br>好发于**后纵隔**。                               | **病变在纵隔的解剖位置**是鉴别纵隔肿瘤的首要法则。                              |
