## 检查方法与正常表现

| 检查方法          | 在本系统的主要作用 / 地位                                                                                                                                                                    | 正常影像学表现简述                                                                                   |
| ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| X线            | 临床上很少用于脑部，主要用于评估颅骨骨折。脊椎平片常用于脊柱病变初查。                                                                                                                                               | CT/MRI普及后地位下降。                                                                              |
| CT            | **平扫是急性脑外伤、脑出血的首选常规检查**；CTA用于无创评估血管病变；[^2]CT灌注(CTP)反映脑微循环，用于急性脑缺血及肿瘤分级。                                                                                                           | 脑部CT常用横断面，垂体常用冠状面。可通过 *“窗技术”*[^1]（脑窗、骨窗）分别观察脑实质和颅骨。                                         |
| MRI (常规)      | 软组织分辨率极高，除钙化/骨折外，全面优于CT。平扫为常规，增强极常用。                                                                                                                                              | 需分清 T1WI 与 T2WI。T1WI 脑脊液呈低信号（黑），T2WI 脑脊液呈高信号（白）。                                            |
| MRI (功能/fMRI) | *核心考点*：<br>1. **DWI (扩散加权)**：**急性脑梗死早期诊断最敏感**。<br>2. **DTI (扩散张量)**：显示脑白质纤维束走向，明确肿瘤与神经传导束关系。<br>3. **PWI (灌注加权)**：脑缺血、肿瘤鉴别。<br>4. **BOLD**：定位脑皮质功能区（术前方案制定/癫痫灶定位），也用于老年痴呆、抑郁症等研究。 | DWI 上急性脑梗死呈明显高信号（亮白）。                                                                       |
| 1H-MRS (波谱)   | 用于活体检测组织化学成分，进行肿瘤的定性与鉴别诊断。                                                                                                                                                        | 关键代谢物峰值：**NAA** (2.04 ppm, 神经元标志物)、**Cho** (3.22 ppm, 细胞膜增殖标志物)、**Cr** (3.0/3.9 ppm, 能量代谢)。 |
| DSA (血管造影)    | **诊断脑血管疾病（动脉瘤、AVM[^3]）的“金标准”**，也是介入治疗的基础。                                                                                                                                         | 颈内动脉造影分五段：**C5岩骨段、C4海绵窦段、C3前膝段、C2视交叉池段、C1后膝段**。                                             |
| 超声 / 核医学      | 超声价值有限（主要用于前囟未闭的婴幼儿）；核医学（PET/CT、PET/MRI）正在迅速发展。                                                                                                                                   | /                                                                                           |

# 基本病变
| 基本病变          | X线 / CT 典型表现                       | 常见对应疾病 / 病理基础                    |
| ------------- | ---------------------------------- | -------------------------------- |
| **脑实质密度异常**   | CT图像出现**“岛带征”**（脑岛皮质灰白质界限模糊）。      | 提示**脑梗死**早期，病灶往往严格按**脑血管分布区**分布。 |
| **脑内占位与水肿**   | 局部可见异常密度/信号灶，伴随周围水肿，常引起**中线结构偏移**。 | 常见于恶性肿瘤（如胶质母细胞瘤）、巨大血肿或脓肿。        |
| **椎弓根改变**     | 脊柱X线平片示椎弓根间距增宽或骨质吸收。               | 提示**椎管内占位**病变。                   |
| **椎间孔扩大**     | 脊柱X线上局部椎间孔异常扩大。                    | 典型提示**神经源性肿瘤**（如神经鞘瘤通过椎间孔向外生长）。  |
| **椎骨破坏+椎旁肿块** | 椎体骨质破坏，常伴有椎旁软组织影（如冷脓肿）。            | 提示**脊柱结核**或**恶性肿瘤**转移。           |
## 常见疾病
| 疾病名称               | 核心影像表现 (X线/CT)                                                          | 典型征象 / 鉴别要点                              |
| ------------------ | ----------------------------------------------------------------------- | ---------------------------------------- |
| ==急性硬膜外血肿==        | CT示颅骨内板下**梭形（双凸透镜形）**高密度影，常伴发骨折。                                        | 脑膜中动脉破裂多见。_(题库：左额骨骨折+梭形不均匀高密度=急性硬膜外血肿)_。 |
| ==脑梗死==            | CT早期可呈**“岛带征”**，病灶严格按**脑血管分布区**分布；MRI-**DWI**是早期诊断最敏感序列。                | _(题库：风心病史+脑桥卵圆形低密度灶=脑梗死)_。               |
| ==脑脓肿==            | CT/MRI增强扫描典型表现为**壁光滑的环形强化**（或多房分隔）。                                     | 需与胶质母细胞瘤（花环样不规则强化）鉴别。                    |
| ==星形细胞瘤==          | 根据病理分I-IV级。**I级星形细胞瘤通常无增强**。                                            | 级别越高，强化和周围水肿越明显。                         |
| ==胶质母细胞瘤 (IV级)==   | CT/MRI示混杂密度/信号区，典型呈**花环样（不规则环形）强化**，周围水肿和占位效应极明显（中线移位）。                 | _(题库：65岁+额顶叶混杂密度+花环样强化+中线移位=胶质母细胞瘤)_。    |
| ==听神经瘤==           | 位于**桥小脑角区（中后颅窝）**，MRI示T1WI等信号、T2WI高信号，增强呈**均一强化**，常伴**岩骨尖骨质破坏**（内听道扩大）。 | _(题库：头晕耳鸣+中后颅窝卵圆形灶+岩骨尖破坏+均一强化=听神经瘤)_。    |
| ==垂体瘤==            | 鞍内实质性占位，可向上突入鞍上池。                                                       | **鞍区最常见的肿瘤**。                            |
| ==Chiari畸形 (先天性)== | 矢状位MRI可直观显示：**小脑扁桃体经枕骨大孔下疝入颈椎管内**。                                      | 常合并延髓/四脑室下移、**脊髓空洞症**和幕上脑积水。             |
>[!Warning] CNS定位诊断总结表

| 病变部位            | 疾病                | 影像特征与备注                                               |
| --------------- | ----------------- | ----------------------------------------------------- |
| **鞍区**          | ==垂体瘤==           | 这是**鞍区最常见的肿瘤**，常表现为鞍内实质性占位。                           |
| **颞叶、小脑**       | ==耳源性脑脓肿==        | 脓肿形成期在CT上可见等密度环，MRI增强呈光滑环形强化。                         |
| **额叶、顶叶**       | ==血源性脑脓肿、胶质母细胞瘤== | 额顶叶出现混杂密度区伴“花环样强化”和中线移位，高度提示胶质母细胞瘤。                   |
| **大脑前、中动脉供血区**  | ==动静脉畸形 (AVM)==   | 最常见的脑血管畸形，MRI可见扩张流空的畸形血管团及粗大迂曲的引流静脉。                  |
| **特定脑血管分布区**    | ==脑梗死==           | 病灶严格按脑血管分布区分布，CT早期表现可出现“岛带征”。                         |
| **颅骨内板下**       | ==急性硬膜外血肿==       | 典型表现为额顶部等区域的**梭形（双凸透镜形）**不均匀高密度影，常伴发相邻部位的颅骨骨折。        |
| **枕骨大孔及颈椎管内**   | ==Chiari畸形（先天性）== | 典型表现为**小脑扁桃体经枕骨大孔下疝**入颈椎管内，常合并脊髓空洞症和幕上脑积水。            |
| **椎管内 — 髓内**    | ==室管膜瘤、星形细胞瘤==    | 表现为脊髓本身增粗，周围的蛛网膜下腔对称性变窄或闭塞。                           |
| **椎管内 — 髓外硬膜内** | ==神经源性肿瘤、脊膜瘤==    | 表现为**患侧蛛网膜下腔增宽**，对侧变窄；脊髓受压向对侧移位。神经源性肿瘤还常引起椎间孔扩大（哑铃状）。 |
| **椎管内 — 硬膜外**   | ==转移瘤==           | 表现为蛛网膜下腔变窄，脊髓受压移位，常伴有椎骨的破坏。                           |

[^1]: 窗技术（Windowing）：CT成像中为显示不同组织结构而调整数据的技术，脑部常用脑窗、骨窗。

[^2]: CTP（CT Perfusion，CT灌注成像）：反映脑实质微循环和血流灌注情况，主要用于急性脑缺血的诊断，也可用于脑肿瘤病理级别判断。

[^3]: AVM（Arteriovenous Malformation，动静脉畸形）：最常见的脑/椎管内血管畸形，表现为扩张流空的畸形血管团及粗大的引流静脉。
