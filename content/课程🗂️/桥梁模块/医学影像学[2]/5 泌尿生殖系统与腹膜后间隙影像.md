## 检查方法与正常表现

| 检查方法               | 在本系统的主要作用 / 地位                                                                       | 正常影像学表现简述                                                                                 |
| ------------------ | ------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------- |
| **X线平片 (KUB/IVU)** | 常用于泌尿系结石术前定位的最简捷手段。                                                                  | 仰卧前后位，上界包肾上腺，下达耻骨联合。**无法检测阴性结石**。                                                         |
| **CT / CTU / CTA** | **目前泌尿系统的主要检查手段**。CTU（尿路成像）已基本取代传统的静脉尿路造影(IVU)。                                      | 肾脏增强采用多期：**皮髓质期**(25-60s)、**肾实质期**(2-3m)和**排泄期**(5-10m)。                                  |
| **MRI**            | 软组织分辨率极高。**mp-MRI（多参数MRI）是评估前列腺疾病最先进准确的手段**。在==女性生殖==肿瘤分期中价值高。                       | **子宫T2WI呈三层结构**：内膜(高信号)、结合带(低信号带)、肌外层(中等信号)。<br>**化学位移成像**：通过正/反相位对比，极度擅长鉴别脂肪与脂质（如肾上腺腺瘤）。 |
| **超声 (US)**        | 前列腺、妇科等病变的**筛查首选**。<br>妇科常分为**经腹部**（需憋尿，视野大）和**经阴道**（无需憋尿，探头贴近病灶，分辨率极高，但无性生活史者禁忌）检查。 | **前列腺正常测值**：横径约4cm，上下径约3cm，前后径约2cm。正常超声表现为类板栗状，轮廓清晰包膜完整。                                  |
## 基本病变
| 基本病变/分类系统    | X线 / CT 典型表现                                                           | 常见对应疾病 / 病理基础                        |
| ------------ | ---------------------------------------------------------------------- | ------------------------------------ |
| ==先天性肾异常==   | **孤立肾**（对侧缺如，患侧代偿增大）；**重复肾**（可见两组肾盂肾盏及双输尿管）；**马蹄肾**（最常见融合畸形，双肾下极峡部融合）。 | 常伴随泌尿系统排泄异常或感染。                      |
| *Bosniak 分类* | 针对==肾脏囊性病变==的分类，2019年Radiology更新版分为五类：**I类, II类, IIF类, III类, IV类**。    | 级别越高，恶性可能性越大。                        |
| *PI-RADS 评分* | ==前列腺==影像数据与报告系统（基于 mp-MRI）。                                           | 客观评估**前列腺癌**病灶性质及临床显著前列腺癌的可能性。       |
| *VI-RADS 评分* | ==膀胱==影像报告和数据系统（基于 mp-MRI）。                                            | 5分制评估，分数越高，**膀胱癌发生肌层（逼尿肌）浸润**的可能性越大。 |

# 常见疾病
### 泌尿系统

| 疾病名称               | 核心影像表现 (CT/MRI)                                                       | 典型征象                                                                                                         |
| ------------------ | --------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| ==泌尿系统结石==         | KUB平片可见阳性结石高密度影；**阴性结石在平片上不显影**，需靠CT明确确切位置及上游积水情况。                    | *肾结石*：结石位于肾影内；结石较大充满整个肾盂时，称为**铸形结石或鹿角状结石**。<br>*输尿管结石*：常滞留于三个生理狭窄处。X线/CT间接征象为结石上方肾盂、肾盏和输尿管扩张积液<br>*膀胱结石*（类似） |
| ==泌尿系统结核==         | 早期肾脏增大；干酪坏死排出后肾实质见空腔样病变（壁见钙化）；晚期肾萎缩、弥漫钙化。                             | 晚期典型征象称为 **“肾自截” (油灰肾, putty kidney)**[^1]。常继发于肺结核，表现为尿频尿急血尿脓尿。                                              |
| ==肾囊肿==            | 内部呈均一水样密度/信号，无强化。                                                     | 最常见的肾脏占位，评估时需结合**Bosniak 2019版分类**（分为I、II、IIF、III、IV类，级别越高恶性可能越大）。                                           |
| ==血管平滑肌脂肪瘤 (AML)== | **最常见的肾良性肿瘤**。CT示实质肿块内含有**极低密度的脂肪成分 (<-20HU)**；MRI脂肪抑制序列可见对应区域信号明显降低。 | 多发者常合并**结节性硬化**。CT值和MRI脂抑序列是确诊的“金标准”。                                                                        |
| ==肾细胞癌 (RCC)==     | 最常见原发恶性肿瘤。最常见亚型**透明细胞癌**常发生囊变坏死，增强扫描呈**明显强化**。                        | 易侵犯静脉形成**下腔静脉瘤栓**。典型“三联征”（血尿、腹痛、肿块）很少同时出现。                                                                   |
| ==尿路上皮癌==          | CT示基底位于尿路黏膜的软组织结节，有强化。具有**沿尿路播散**的特性，常多发。                             | 典型临床表现为**全程无痛性肉眼血尿**。膀胱癌的肌层浸润评估首选**多参数MRI (mpMRI)**，并采用**VI-RADS评分**客观评估。                                    |
### 男性生殖系统
| 疾病名称              | 核心影像表现 (以MRI为主)                                                        | 典型征象                                                               |
| ----------------- | ---------------------------------------------------------------------- | ------------------------------------------------------------------ |
| ==良性前列腺增生 (BPH)== | 主要位于**移行带**。T2WI上移行带增大，见多发高低混杂信号的增生结节，正常外周带受压变薄（但多仍呈高信号）。              | 60岁以上老年男性好发达75%，常导致尿急尿频排尿困难。**超声为筛查首选**。                           |
| ==前列腺癌==          | 95%为腺癌，主要位于**外周带**。MRI敏感度极高：T2WI上正常高信号的外周带内出现**低信号结节**；DWI呈高信号；增强早期强化。 | 易发生血行转移，尤其是**成骨性骨转移**。诊断高度依赖**mp-MRI**，并使用**PI-RADS评分系统**提供客观分级依据。 |


### 女性生殖系统
| 疾病名称       | 影像表现 (CT/MRI/超声)                                                                                          | 典型征象                                                                                                                                |
| ---------- | --------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| ==子宫平滑肌瘤== | MRI：T2WI典型表现为**明显低信号**，边界清楚。增强扫描有不同程度强化（略低于正常肌层）。<br><br>**超声**：类圆形实性低回声，切面呈特征性 **“漩涡状”结构**，周围有假包膜        | 当发生囊变时T2WI呈高信号；当发生**红色变性时T1WI信号略增高**。黏膜下肌瘤是引起月经过多的主要原因。<br><br>玻璃样变（最常见）、囊性变、**红色变性**（多发生于妊娠期，瘤体缺血坏死，内回声呈细花纹状）、脂肪或钙化变性（子宫石）、肉瘤变（恶变） |
| ==子宫颈癌==   | MRI：软组织肿块，T2WI上宫颈低信号基质层被破坏。<br><br>MRI软组织分辨率高，是评估局部T分期和侵犯范围的最佳手段。                                         | 最常见妇科恶性肿瘤，90%为鳞癌，早期表现为接触性出血。<br><br>常基于**FIGO分期标准**评估阴道下段或骨盆壁受累情况。                                                                  |
| ==卵巢囊腺瘤==  | 分为浆液性与黏液性。表现为单房或多房囊性肿块，壁和内隔较薄且均匀。**黏液性囊腺瘤**因含粘蛋白，T1WI和T2WI均可呈较高信号。<br><br>**超声/CT**：多房或单房囊性肿块，壁和内隔薄。      | **浆液性囊腺瘤**常含钙化，且恶变率较高（30%-50%）。<br>当囊壁不规则、分隔增厚伴实性强化时，需警惕**囊腺癌**。                                                                    |
| ==卵巢畸胎瘤==  | CT/MRI：混杂密度/信号肿块，典型特征是内含**脂肪和钙化/骨化**。<br><br>**超声**：含有多胚层组织。典型表现为**脂液分层征**、**面团征**、**瀑布征/垂柳征**（毛发）、杂乱结构征。 | 影像上常可见特征性的**脂肪-液体平面**或囊内漂浮物。<br><br>95%以上为良性，好发育龄年轻女性。                                                                              |
### 肾上腺和腹膜后间隙
| 疾病名称         | 核心影像表现 (CT/MRI)                                                          | 典型征象                                                  |
| ------------ | ------------------------------------------------------------------------ | ----------------------------------------------------- |
| ==肾上腺皮质增生==  | 肾上腺对称均匀性增粗（厚度>10mm或大于同层膈肌脚厚度）。                                           | 临床多伴有功能亢进（如原醛、Cushing综合征），但约50%增生在影像上形态可正常。           |
| ==肾上腺皮质腺瘤==  | 单侧类圆形结节。CT示密度较低（富含脂质）。**核心手段为MRI化学位移成像**：在反相位 (Out-of-phase) 上信号明显衰减。    | 分为无功能性（体检偶然发现）和功能性。反相位信号衰减是鉴别腺瘤与转移瘤的“金标准”。            |
| ==嗜铬细胞瘤==    | CT示肿块常巨大（>3-5cm），密度/信号极不均匀。特征性表现为：**实质部分极度明显强化 (>200HU)，且极易发生出血、坏死、囊变**。 | 著名的**“10%肿瘤”**（10%恶性、10%双侧、10%异位）。临床典型表现为阵发性高血压伴大汗淋漓。 |
| ==肾上腺转移瘤==   | 双侧多发不规则肿块，实性成分强化，坏死区不强化。                                                 | 肾上腺是全身肿瘤转移第4大好发部位（多来自肺癌、乳腺癌等），反相位上信号不衰减。              |
| ==原发性腹膜后肿瘤== | 位于腹膜后间隙脏器外，脂肪肉瘤、平滑肌肉瘤等。                                                  | 原发肿瘤恶性占多数，最常见的是脂肪肉瘤。                                  |
##### 核医学诊断

| 疾病名称 / 病变部位                  | 核心分子探针与机制                                     | 临床价值 / 典型征象                                  |
| ---------------------------- | --------------------------------------------- | -------------------------------------------- |
| ==肾上腺皮质病变==<br>(原醛症、库欣综合征)   | **胆固醇类似物**被皮质细胞特异性摄取。                         | 结合地塞米松抑制试验，有效鉴别功能性腺瘤与皮质增生。                   |
| ==肾上腺髓质病变==<br>(嗜铬细胞瘤、副神经节瘤) | **123I / 18F-MIBG** (去甲肾上腺素类似物) 被嗜铬细胞特异性摄取储存。 | 诊断 PPGLs 的**公认“金标准”**。对定位诊断，尤其是寻找异位病灶具有极高价值。 |
| ==肾上腺恶性肿瘤==鉴别                | **18F-FDG PET/CT**（葡萄糖代谢）。                    | SUVmax > 3.5 高度提示恶性。                         |
