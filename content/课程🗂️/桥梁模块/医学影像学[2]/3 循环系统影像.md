# 检查方法与正常表现

| 检查方法                       | 在本系统的主要作用 / 地位                      | 正常影像学表现简述                                                                                                                         |
| -------------------------- | ----------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| **X线 (胸片)**                | 评估心脏整体形态与肺血改变。                      | 正常心胸比率上限为 **0.5**。心脏基本形态分为三型：<br>1. 二尖瓣型心（梨形，肺动脉段凸出，见于二尖瓣病变/房缺）<br>2. 主动脉瓣型心（靴形，主动脉结增宽，见于高血压/主瓣病变）<br>3. 普大型心（向两侧均匀增大，见于心衰/心包积液）。 |
| **超声心动图 (Echo)**           | **心脏结构与功能的筛查首选**（极度依赖解剖切面）。         | **M型**：时间分辨率极高，用于测量室壁厚度和射血分数（EF）。<br>**二维 (2D)**：切面为王，掌握14个标准切面（如胸骨旁长/短轴、心尖四腔心）。<br>**多普勒 (Doppler)**：血流朝向探头运动频率增高（红），背向降低（蓝）。    |
| **CT / CTA**               | 显示冠状动脉狭窄程度最准确的无创方法。                 | 依赖心电门控技术。后处理技术包括：VR (容积再现)、MPR (多平面重组)和 CPR (曲面重组)                                                                                |
| **MRI**                    | 评估**心肌形态/功能**及**组织特征定性**的核心工具。      | 是心肌病表型分类、心肌瘢痕与组织学定性的“金标准”。                                                                                                        |
| **DSA / 冠脉造影**             | 冠脉血管狭窄评估与介入治疗的有创“金标准”。              | 可动态观察血管解剖结构与血流状态。                                                                                                                 |
| **心肌灌注显像 (MPI)**           | 利用心肌细胞选择性摄取。细胞正常时，**摄取量与冠脉血流量成正比**。 | 提供心肌缺血的**直接证据**。必须结合静息与负荷（运动/药物）显像对比。                                                                                             |
| **下肢深静脉显像**<br>(99mTc-MAA) | 穿刺双足背静脉并扎止血带注射，观察深静脉回流。是筛查DVT的首选。   | 血管影像光滑连贯，呈倒“丫”字形。                                                                                                                 |
##### MPI显像剂
| 显像剂类型          | 药代动力学特征                                                         | 显像方案特点                       |
| -------------- | --------------------------------------------------------------- | ---------------------------- |
| **201Tl (铊)**  | 具有特征性的**再分布 (Redistribution)**现象。正常心肌清除快，缺血心肌清除慢，3-4小时后放射性重新填充。 | 只需注射一次，进行初期和延迟两次显像。          |
| **99mTc-MIBI** | 进入线粒体后滞留，**没有再分布现象**。                                           | 必须注射两次药物，分别进行负荷和静息显像（常隔日进行）。 |

# 基本病变
### X线：肺血管改变

| 基本病变      | X线 / CT 典型表现                       | 常见对应疾病 / 病理基础             |
| --------- | ---------------------------------- | ------------------------- |
| ==肺血增多==  | 肺野清晰，肺动脉分支向外周伸展。                   | **左向右分流**的先天性心脏病（如房缺、室缺）。 |
| ==肺血减少==  | 肺野透明度增加，肺门动脉及血管纹理变细。               | 右心排血受阻或搏出减少的病变（如法洛四联症）。   |
| ==肺动脉高压== | 肺动脉段突出，右心室增大。                      | 肺心病、先心病肺血量增多者、肺栓塞。        |
| ==肺静脉高压== | 肺门及纹理增粗模糊，出现特征性 **Kerley A、B、C线**。 | **左心功能不全**、二尖瓣/主动脉瓣病变。    |
### 超声：多普勒与血流动力学
- **多普勒公式**：利用速度计算压差，评估瓣膜狭窄与肺动脉高压。
- **二尖瓣口面积估算 (MVA)**：利用压力减半时间 (PHT) 计算，MVA=220/PHT。
# 常见疾病
### 瓣膜病与先天性心脏病（超声）

| 疾病名称            | 核心影像表现 (超声)                                                                                      | 典型征象 / 鉴别要点                              |
| --------------- | ------------------------------------------------------------------------------------------------ | ---------------------------------------- |
| ==风湿性二尖瓣狭窄==    | **M型**：特征性二尖瓣前叶 **城墙样改变**，前后叶同向运动。<br>**二维**：瓣叶增厚钙化，舒张期呈**穹隆状改变**。<br>**多普勒**：瓣口五彩镶嵌血流，计算平均跨瓣压差。 | 继发征象：左房增大、左房血栓、肺动脉高压。瓣口面积 <1.0cm2 为重度狭窄。 |
| ==房间隔缺损 (ASD)== | 二维直接显示房间隔连续性中断；彩超显示**心房水平舒张期左向右穿隔血流**。                                                           | 间接征象：右心扩大（右房/右室/肺动脉扩大）。                  |
| ==室间隔缺损 (VSD)== | 二维直接显示室间隔中断；彩超显示**心室水平收缩期左向右五彩射流**。                                                              | 连续多普勒可测得高速分流频谱（>4m/s）。                   |
### 心肌病（超声-表型；MRI-病因）
| 心肌病表型         | 超声心动图核心特征                                          | 典型病因/辅助诊断指标                                |
| ------------- | -------------------------------------------------- | ------------------------------------------ |
| ==扩张型 (DCM)== | 心腔明显扩张（尤其是左室），**左室收缩功能（EF）显著减低**。                  | 需与重度瓣膜反流、晚期高血压心脏病、冠心病晚期缺血性心肌病等鉴别。          |
| ==肥厚型 (HCM)== | 左室任意节段厚度 >15mm。梗阻性者左室流出道压差 >30mmHg，频谱呈特征性 **匕首样**。 | 基因突变导致；极易引发流出道梗阻和猝死。                       |
| ==限制型 (RCM)== | 左室壁厚度 >12mm，双心房明显增大，左室**舒张功能显著减退**（2级或以上）。         | 淀粉样变 (ATTR-CA)：超声心肌应变牛眼图可见特征性 **心尖保留** 征象。 |
### 冠脉病变与大血管疾病（CT/CTA）
| 疾病名称               | 影像学表现与技术选择                                               | 临床细节与干预依据                                  |
| ------------------ | -------------------------------------------------------- | ------------------------------------------ |
| ==冠状动脉粥样硬化 (CAD)== | 首选 **CCTA** 评估管腔狭窄程度。狭窄 30%∼90% 者，需加测 **CT-FFR** 评估缺血程度。 | 当 CT-FFR ≤0.8 时，提示血流动力学显著异常，需行有创冠脉造影及支架干预。 |
| ==主动脉夹层==          | **CTA** 是绝对首选，可清晰显示内膜破裂片（内膜片）及真假腔。                       | 极其凶险的急症，需明确破口位置及受累血管范围以决定手术方式。             |
### 核医学诊断
| 疾病名称              | 核心影像表现                           | 典型征象 / 鉴别要点                                           |
| ----------------- | -------------------------------- | ----------------------------------------------------- |
| ==心肌缺血 (心绞痛)==    | 负荷状态下局部缺损，静息/延迟图像上**缺损恢复**。      | 表现为 **可逆性缺损**（再分布现象）。                                 |
| ==心肌梗死==          | 负荷与静息图像上均表现为放射性缺损。               | 表现为 **不可逆性缺损 (固定缺损)**。                                |
| ==心脏淀粉样变 (CA)==   | 使用 **99mTc-PYP** 显像。             | 出现**异常高摄取**（视觉评分>2级或心/对侧肺比值>1.5）高度提示 **ATTR型心肌淀粉样变**。 |
| ==下肢深静脉血栓 (DVT)== | 下肢深静脉显像见放射性充盈缺损、侧支循环或放射性滞留（血栓征）。 | 与超声符合率高（80.1%）。                                       |
