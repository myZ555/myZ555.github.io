# 概述

###### 体液的分布
![400](Pasted%20image%2020250930132927.png)
- 细胞内液（intracellular fluid, ICF)：**钾离子；磷酸氢根、蛋白质阴离子**
- 细胞外液（extracellular fluid, ECF) ：约占体重20%； 其中==血浆约占体重5%==，组织间液约占体重15%：**钠离子；氯离子**
	- 功能性细胞外液（90%）：能迅速与血管和细胞内液 交换，对维持体液平衡作用大；
	- 无功能性细胞外液（10%）：对体液平衡作用小，包 括结缔组织液、脑脊液、关节液、消化液

# 水钠代谢紊乱
## 脱水 Dehydration

低渗--130mmol/L--等渗--150mmol/L--高渗

##### 低渗性脱水（低容量性低钠血症）hypotonic dehydration

###### 特征：
- 失钠多于失水
- 脱水的特征：血钠<135mmol/L，血浆渗透压<280 mOsm/L 
- ==细胞外液↓，细胞内液↑==

###### 病因：
- 大量消化液（属于细胞外液）丢失而只补充水/葡萄糖溶液（渗透压=0）
- 液体在第三间隙集聚
- 长期连续应用排钠利尿剂
- 经皮肤丢失（大面积出汗/烧伤）

###### 临床表现：

| 轻度(<135mmol/L)     | 中度(<130mmol/L)                          | 重度(<120mmol/L)               |
| ------------------ | --------------------------------------- | ---------------------------- |
| 疲乏，头晕，手足麻木<br>尿钠减少 | 恶心呕吐，脉搏细速，血压不稳定/下降<br>视物模糊<br>尿小，几乎不含钠氯 | 神志不清，肌痉挛，腱反射减弱/消失<br>呼吸困难，昏迷 |
###### 治疗：
- 对病因治疗
- 轻症补充*等渗*盐水，重症补充*高渗*盐水
- 迅速纠正严重低钠血症可能导致*中央脑桥脱髓鞘*改变


##### 等渗性脱水 isotonic dehydration
###### 特征：
- 水、钠等比例丧失
- 脱水的特征是：血钠正常范围 135-150mmol/L；血浆渗透压 280-310 mOsm/L ；
- ==细胞外液↓，细胞内液-==；*血容量*减少
###### 病因：
- 任何等渗体液丢失，*短期内*均属等渗性脱水
- 等渗性脱水--(不处理)---[高渗性脱水]
			  --(只补水)---[低渗性脱水]--大量补水--水中毒
###### 治疗：
- 积极处理病因
- 补液：平衡盐或等渗盐水
##### 高渗性脱水（低容量性高钠血症）hypertonic dehydration

###### 特征：
- 失水多于失钠
- 脱水的特征是：血钠>150mmol/L，血浆渗透压>310mOsm/L 
- ==细胞内液和细胞外液量都减少==
###### 病因：
- 摄入水分不足
- 水丧失过多
- 呕吐、腹泻及消化道引流等导致等渗或含钠低的消化液丢失
- 经肾排出大量低渗性尿液
- 任何原因引起的过度通气
###### 临床表现：

| 轻度    | 中度                      | 重度                      |
| ----- | ----------------------- | ----------------------- |
| 2%-4% | 4%-6%                   | >6%                     |
| 口渴    | 乏力，尿少，皮肤失去弹性，烦躁不安，肌张力增高 | 中毒症状+：躁狂/幻觉/谵妄/昏迷，高热，死亡 |
- 尿比重正常，尿渗透压高
###### 治疗：
- 快速纠正细胞外液缺乏以改善组织灌注
- 纠正脱水的同时适当补钠

| ---- | ----------- | --------------- | --------------- |
| 失水钠  | 成比例         | 失钠              | 失水              |

## 水中毒和水肿

##### 水中毒（高容量性低钠血症）

###### 定义：
- 水潴留导致液体量明显增多，细胞外液低渗，细胞内液↑（细胞水肿）；钠总量正常或增多
###### 病因：
 - 急性肾衰竭，各种原因所致的==抗利尿激素==分泌过多
 - 继续性大量饮水或精神性饮水过量，静脉输入不含盐或含盐量少液体过多过快，超过肾脏排泄能力
###### 临床表现：
 - 急性水中毒：脑细胞肿胀可造成颅内压增高，引起一系列神经、精神症状。
 - 慢性水中毒：可被原发疾病的症状所掩盖，可有软弱无力、恶心、呕吐、嗜睡等。体重明显增 加，皮肤苍白而湿润

##### 水肿 edema
[../../诊断学[5.5](../../诊断学[5.5)
###### 定义：
组织间隙或体腔内积聚过多液体
###### 病因：
- 全身性水肿：多见于充血性心力衰竭、肾病综合征和肾炎、肝脏疾病。也见于营养不良和某些 内分泌疾病。
- 局限性水肿：常见于器官组织局部炎症，静脉或淋巴管阻塞等情况

[](3%20水电与酸碱平衡失调.pdf#page=38&selection=0,0,0,46|3%20水电与酸碱平衡失调,%20页面%2038)
5%葡萄糖：除了glucose没有其他溶质，相当于没有渗透压，等于纯水
B&C：C为更优解，氯较少，防止酸中毒

# 钾代谢紊乱
非常重要

正常：3.5-5.5mmol/L
![500](Pasted%20image%2020250930142406.png)

## 低钾血症

### 病因
1. 摄入不足
2. 丢失过多：消化道（呕吐/腹泻/胃肠引流）、肾（袢利尿剂/噻嗪类/醛固 酮等）
3. 钾向胞内转移：胰岛素+葡萄糖、碱中毒等
### 临床表现
- 最早的临床表现是肌无力：先是四肢肌无力，可延及 躯干和呼吸肌
### 治疗
- 见尿补钾
- 补钾浓度：外周输液<0.3%，补钾速度应<1～2g/h
- 若严重累及心脏，可更高浓度、更快，通过中心静脉补钾，严密监测
- 同时纠正*碱中毒*，**补镁**

## 高钾血症
### 病因
1. 摄入过多
2. 排出减少
3. 胞内钾外移
### 临床表现
### 治疗
- 促使钾转入细胞内
	- 钙剂：保护心脏（葡萄糖酸钙）
	- 高糖+胰岛素
	- 碳酸氢钠
- 利尿剂：袢利尿剂
若高于7.0：上述之外马上血透

# 镁及钙磷代谢紊乱

#### 镁代谢
- 肾脏调节
- 正常血清镁浓度：0.75-1.25mmol/L

|      | 低镁血症                                                                   | 高镁血症                                                            |
| ---- | ---------------------------------------------------------------------- | --------------------------------------------------------------- |
| 病因   | 摄入不足、胃肠道丢失、经肾脏丢失<br>高钙血症/其他疾病--肾小管重吸收减少<br>(甲状腺功能亢进、甲状旁腺功能减退、糖尿病酮症酸中毒) | 肾衰竭（最常见）<br>严重脱水伴少尿<br>肾上腺皮质功能减退、甲状腺功能减退、分解代谢亢进(…酮症酸中毒-胞内镁移至胞外) |
| 临床表现 | 肌震颤、手足抽搐；严重者癫痫大发作<br>肠麻痹：恶心/呕吐/腹胀<br>心脏受累：心律失常（PR间期和QT间期延长）            | 抑制内脏平滑肌功能：嗳气/呕吐/便秘/尿潴留<br>抑制神经兴奋性传递<br>心脏受累：心肌兴奋性↓，严重时心搏骤停      |
| 治疗   | 静脉补镁：25%硫酸镁5～10ml加入5%葡萄糖溶液缓慢滴注                                         | 有明显心血管症状--静脉注射钙剂                                                |

#### 钙磷代谢
- 正常血清钙浓度：2.25-2.75mmol/L
- 正常血清磷浓度：0.8-1.6mmol/L

|      | 低钙血症                         | 高钙血症                              |
| ---- | ---------------------------- | --------------------------------- |
| 病因   | VitD缺乏<br>甲状旁腺功能减退；急性胰腺炎     | 甲状旁腺功能亢进<br>白血病、多发性骨髓瘤等恶性肿瘤/~骨转移  |
| 临床症状 | 神经肌肉性分析升高<br>心脏：QT间期和ST段明显延长 | 神经肌肉症状(↓)；胃肠道症状；骨骼系统<br>心脏：QT间期缩短 |
| 治疗   | 出现手足抽搐、喉头痉挛立即处理<br>同时纠正低镁血症  | 尿钙排出↑；抑制骨吸收；减少肠道钙吸收               |

# 酸碱代谢紊乱

病理生理部分复习
	酸碱平衡是指体内氢离子浓度的平衡，维持pH在7.35-7.45之间。主要通过缓冲系统、呼吸系统和肾脏调节![Pasted image 20250930150139](Pasted%20image%2020250930150139.png)![Pasted image 20250930150247](Pasted%20image%2020250930150247.png)![Pasted image 20250930150149](Pasted%20image%2020250930150149.png)![Pasted image 20250930150157](Pasted%20image%2020250930150157.png)![Pasted image 20250930150205](Pasted%20image%2020250930150205.png)

**==酸碱平衡==**：缓冲系统、肺、肾脏、组织细胞交换![Pasted image 20250930150636](Pasted%20image%2020250930150636.png)
**==酸碱平衡失调==**：

| 指标              | 意义   | 正常值       | ↓       | ↑       |
| --------------- | ---- | --------- | ------- | ------- |
| pH              |      | 7.35~7.45 |         |         |
| 动脉血二氧化碳分压 PaCO2 | 呼吸因素 | 35~45mmHg | 呼碱/代酸代偿 | 呼酸/代碱代偿 |
| 碳酸氢根浓度 HCO3-    | 代谢因素 | 22~27mmHg | 代酸/呼碱代偿 | 代碱/呼酸代偿 |
| 剩余碱 BE          | 代谢因素 | -3~+3mmHg | 负值代酸    | 正值代碱    |


呼吸性--CO2-----↑[酸中毒]
代谢性--HCO3---↑[碱中毒]


| 失衡类型                         | 病因                                                                                                                                     | 临床表现                               | 治疗                   |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------- | -------------------- |
| ==代谢性酸中毒==<br>H+ ↑<br>HCO3-↓ | 1. 酸性物质产生过多(缺氧和组织低灌注)<br>2. 外源固定酸摄入过多(阿司匹林、氯化铵…)<br>3. H+排出减少（肾脏排酸保碱×)<br>4. HCO3-排出过多(碱性物质丢失过多-腹泻/肠瘘…)<br>5. 高钾血症(细胞外液K+与H+交换-胞外H+增加) | Kussmual呼吸-深大呼吸<br>脑-心脏-肺-胃肠道      | “宁酸勿碱”<br>碳酸氢钠-严重酸中毒 |
| ==代谢性碱中毒==<br>H+↓<br>HCO3-↑  | 1. 酸性物质丢失(呕吐-丢H+；腹泻-丢HCO3-)<br>2. 碱性物质产生过多<br>3. H+向胞内移动--*反常性酸性尿*(H+-K+交换↑)                                                           | 呼吸浅慢，换气减少<br>神经肌肉症状<br>心脏传导阻滞、心搏骤停 | 纠正低钾血症               |
| ==呼吸性酸中毒==<br>PaCO2↑         | 1. 呼吸系统疾病-(通气功能障碍)<br>2. 呼吸中枢抑制(高碳酸血症)<br>3. 医源性(CO2排出障碍)                                                                              | 胸闷紫绀<br>躁动不安<br>精神症状               | 改善通气                 |
| ==呼吸性碱中毒==<br>PaCO2↓         | 1. 过度通气                                                                                                                                | 神经肌肉兴奋性↑<br>神经系统功能障碍               | 纸袋罩住口鼻               |

// =====================================================
// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
