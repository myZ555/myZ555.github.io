#外科 
# （一）输血适应证
1. 大量失血（失血量<10%可自行代偿）
2. 贫血或低蛋白血症
	- 浓缩红细胞纠正贫血
	- 血浆或白蛋白纠正低蛋白血症
3. 重症感染
	- 少量多次输血可提高机体抵抗能力
4. 凝血异常
# （二）输血的不良反应及其防治
###### 1. 发热反应、变态反应、细菌污染反应

|     |  发热反应<br>（最常见早期并发症）  |                变态反应                 |       细菌污染反应       |
| --- | :------------------: | :---------------------------------: | :----------------: |
| 时间  |     输血开始后1.5-2h      |             数分钟后；中、后也可              |                    |
| 表现  |     寒战、高热，血压无变化      |        荨麻疹、支气管痉挛、休克等变态反应，无发热        | 污染轻发热；污染重立即致内毒素性休克 |
| 原因  | 免疫反应<br>致热原<br>污染和溶血 | 过敏体质<br>产生抗IgA抗体<br>免疫功能降低<br>IgA过敏 |     细菌污染血制品并繁殖     |
| 治疗  |    严重者停止输血<br>退热     |     组胺类抗过敏治疗<br>应用肾上腺皮质激素、糖皮质激素     |  中止输血；细菌涂片培养；抗生素   |

###### 2. 溶血反应（最严重）
1. 原因：血型不合；少数输入有缺陷的RBC引起非免疫性溶血；受血者患自身免疫性贫血
2. 临床表现：沿输血静脉的红肿热痛、呼吸困难、HR↑、BP↓、休克、血红蛋白尿和溶血性黄疸，甚至急性肾衰竭；术中病人最早征象是原因不明的血压下降和手术野渗血；另有延迟性溶血反应。
3. 治疗：停止输血；抗休克；5%碳酸氢钠碱化尿液促血红蛋白结晶溶解；血浆交换
###### 3. 循环超负荷
- 过多过快输血；表现为急性心衰和肺水肿
- 治疗：停止输血，强心，利尿
###### 4. 输血相关的急性肺损伤
- 病人体内存在白细胞凝集素和HLA特异性抗体

###### 8. 大量输血的影响
- 【低体温】短时间大量输入冷藏血
- 【低钙】大量输入含枸橼酸钠的血制品
- 【碱中毒】抗凝药枸橼酸钠在肝转化为碳酸氢钠（低钾-碱中毒）
- 【高钾血症】大量输入库存血，胞内钾释放到胞外
