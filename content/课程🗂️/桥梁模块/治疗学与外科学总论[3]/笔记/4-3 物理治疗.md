# （二）超声波疗法
- 声波是一种机械波；以压力和密度的变化传播；需要介质
- 根据声波的频率范围：
	- 低频声波
	- 中频声波（能够听到的）
	- 高频声波（高于20kHz，超声波）
1. 超声波的产生
	- **逆压电效应**（指压电晶体在外加电场的作用下，晶体会发生尺寸变化，从而振动产生声波。）
2. 超声波的生物物理特性
	1. 频率：1MHz~10MHz，频率越高，波长越短。医学常用：**800kHz~3000kHz**
	2. 声速：在不同组织中传播速度不同
	3. 衰减：随频率增加而增加（低频率-更好穿透组织-清晰图像）
	4. 散射：介质不均匀性
	5. 反射与折射
	6. 声压
	7. 干涉现象
	8. 驻波现象
3. 超声波治疗设备
4. 超声波的作用
	[](4-3%20物理治疗学.pdf#page=17&selection=2,0,2,8|4-3%20物理治疗学,%20页面%2017)
	- 可以打碎肾结石或膀胱结石
5. 超声波治疗的注意事项
6. 特殊超声波疗法
	1. 超声间动电疗法
	2. 超声药物透入疗法
	3. 超声雾化吸入疗法（*各种呼吸道疾病*）
	4. 超声空化效应清创（*清除创面中的坏死组织、细菌及其他污染物；适用于慢性创面、糖尿病性溃疡、压疮以及烧伤后的创面管理*）

# （三）光疗法
- 光—电磁波；波粒二象性
1. 红外线
	-  波长：780nm-1mm
	- 被生物组织吸收：能量--热能，组织温度升高；可用于深层组织
	1. 治疗作用：
		1. 加速血液循环
		2. 缓解肌肉疼痛
		3. 增强免疫功能
		4. 促进组织修复
		5. 减轻关节和软组织炎症
	2. 注意事项：
		- *增生期瘢痕应避免红外线照射*
2. 可见光
	- 400-760nm
	1. 作用：
		1. 光动力疗法 PDT
		2. 红光疗法
		3. 蓝光疗法（*痤疮*）
		4. 蓝紫光疗法（*银屑病、湿疹*）
		5. 多光谱疗法
		6. ==蓝紫光疗法治疗核黄疸==：蓝光波长460-490nm
	2. 注意事项
3. 紫外线
	- 400-180nm
	- 反射、散射、吸收、穿透
	- 紫外线引发的光化学反应主要发生在皮肤*浅层*
	1. 紫外线的**红斑反应**
		- 皮肤细胞损伤和血管扩张的直接结果
		- 存在潜伏期；短波--出现快且消退快
	2. 紫外线的色素沉着
		1. 类型：立即性、持久性、晚发性
		2. 色素沉着与紫外线波长的关系
		3. 机理：皮肤的保护性反应
			1. 黑色素生成
			2. 黑色素分布
			3. 炎症反应
			4. 长期变化
	3. 紫外线的杀菌作用
		1. 中波紫外线能直接被DNA吸收--DNA嘧啶碱基变化形成嘧啶二聚体--感染DNA复制和转录
		2. 紫外线--影响酶的活性
	4. 紫外线对钙磷代谢的影响
		- 促进维生素D合成
		- 治疗*小儿佝偻病和成人软骨病*
		- *紫外线脱敏治疗*：通过降低血管通透性和神经兴奋性，钙离子减轻过敏反应
	5. 紫外线对免疫功能的影响（双重作用）
		1. 适量紫外线照射--刺激免疫系统
		2. 过量紫外线暴露--损伤皮肤免疫细胞；改变抗原呈递方式影响T胸细胞活性；与皮肤癌发生有关
	6. 治疗技术
		1. 生物剂量：**最小红斑剂量**
			- 通过 观察人体特定部位在一定距离的光源照射后出现的最弱红斑反应（肉眼可见）所需的时间来定义
			- 时间长度以秒为单位，简称MED
	7. 生物学作用及治疗作用
		1. 维生素D合成
		2. 抗菌作用
		3. 治疗皮肤疾病
		4. 调节免疫系统
		5. 心理作用
		6. 皮肤癌风险
		7. 老化作用
		- 紫外线光敏疗法是目前治疗*银屑病和白癜风*的常规手段
	8. 注意事项
4. 激光
	- 特殊；一种电磁波、表现粒子性质；产生基于受激发射的原理
	- 特性：①高能量密度、②方向性、③单色性、④相干性
	1. 激光的生物学效应
		1. 热作用
		2. 光化作用
		3. 压强作用
		4. 电磁作用
		5. 生物刺激作用
	2. 高强度激光的应用
		- 高强度激光：对人体造成不可逆损伤；切除病变组织、导致受照射组织凝固/止血/融合/气化
		- **皮肤疾病**：疣及疣状痣、血管病变和色素性皮肤病、皮脂腺囊肿……
	3. 低强度激光应用
		- 非侵入性、副作用小
		- 疼痛管理、创面愈合、炎症减轻、物理康复、激活细胞功能、皮肤病治疗
# （四）电疗法
- 利用电流刺激身体组织
- 直流电疗法、交流电疗法；交流电疗法分为低/中/高频
1. 直流电疗法
	1. 直流电生理作用
		1. 促进组织修复
		2. 疼痛缓解
		3. 改善血液循环
		4. 肌肉刺激
		5. 促进骨骼愈合
		6. 电渗作用
	2. 直流电治疗作用
		1. 镇静和兴奋作用
		2. 对自主神经和内脏神经的调节作用
		3. 消炎作用
		4. 其他作用
	3. 直流电药物导入疗法
		- 利用电解作用和电渗作用促进药物离子穿透皮肤和黏膜；增强局部浓度，减少全身毒副作用。
		1. 电解作用：直流电使得药物溶液电离，产生药物离子
		2. 电渗作用：相同电荷的电极排斥药物离子--使其通过皮肤进入体内
	4. 注意事项
2. 低频电疗法
	1. 神经肌肉电刺激
		- 直接刺激肌肉使其产生收缩-增强肌肉力量
		1. 刺激失去神经支配的肌肉
		2. 减轻肌肉痉挛
		3. 刺激平滑肌
		4. 治疗废用性肌萎缩
	2. 经皮电刺激神经疗法 TENS
		- 止痛效果；治疗软组织损伤、神经痛以及手术后疼痛
3. 中频电疗法
	- 1000~10000Hz电流；此时运动神经和肌肉的兴奋不在遵循周期同步原则；脉冲周期短于运动神经和肌肉组织的绝对反应期
	1. 作用特点
		1. 组织电阻抵抗：较低频电更易穿过皮肤和其他组织，穿透组织更深
		2. 不发生电解
		3. 对神经肌肉的作用：神经兴奋后的*绝对不应期*大约1-2ms，大于1000Hz的电流不能引起神经肌肉收缩
		4. 对感觉神经的作用：与低频电刺激相比， 较强的中频电刺激在引起肌肉收缩时感觉更为舒适
		5. 提高局部血液循环
		6. 提高活性生物膜的通透性
	2. **音频电疗法**
		- 应用1000～20000Hz等幅正弦电流治疗疾病的方法
		- 治疗慢性疼痛、促进伤口愈合、缓解肌肉疲劳以及改善神经功能障碍
	3. **电脑调制中频疗法**
		- *镇痛*，锻炼骨骼肌，调节自主神经功能，促进血液循环，消散慢性炎症
		- 适应证和禁忌证[](4-3%20物理治疗学.pdf#page=74&selection=2,3,2,15|4-3%20物理治疗学,%20页面%2074)
4. 高频电疗法
	1. 生物物理学特性
		1. *热效应*
		2. 非热效应（生物刺激效应）
		3. 电场效应
		4. 无电解作用，对神经肌肉无兴奋作用，电极*不必接触皮肤*
	2. 分类
		1. 短波：10-100m（深部组织）【热效应为主】
			1. 治疗作用
				1. 改善血液及淋巴循环
				2. 镇静、止痛、缓解肌肉痉挛
				3. 改善器官功能
			2. 注意事项
				1. 周围设备使用绝缘隔离罩（形成法拉第笼），确保治疗仪正确接地
				2. 移除患者身上所有金属物品，避免在含有金属异物的部位治疗
		2. 超短波：1m-10m
		3. 微波疗法：1mm-1m
		4. 毫米波疗法：1-10mm（表浅）
