| 给药方式 | 药物吸收部位        | 吸收特点              | 适用情况               |
| ---- | ------------- | ----------------- | ------------------ |
| 静脉注射 | 直接进入血液循环      | *起效最快，剂量准确*       | 急救、重症、需要快速起效的情况    |
| 肌内注射 | 肌肉组织          | 慢于静注、快于皮下         |                    |
| 皮下注射 | 皮下组织          | 慢于肌内注射            | 胰岛素等               |
| 吸入   | 肺部            | 迅速进入血液循环，起效快      | 呼吸系统疾病治疗（哮喘）       |
| 舌下含服 | 快速、*避免肝脏首过效应* | 急救，如硝酸甘油          |                    |
| 直肠给药 | 直肠黏膜          | 快于口服              | 不宜口服，如儿童和呕吐患者      |
| 口服给药 | 胃肠道           | 起效相对较慢，受食物/药物性质影响 |                    |
| 皮肤给药 | 皮肤            | 吸收较慢              | 局部治理，部分药物可经皮发挥全身作用 |

###### 4. 剂量和剂量调整

| 概念   | 意义                          |
| ---- | --------------------------- |
| 治疗剂量 | 达到治疗效果而通常使用的药物量，包括最小和最大有效剂量 |
| 极量   | 是药物的剂量限制，超过这个量有中毒的危险        |
| 日剂量  | 指一天内给予的药物*总量*，日剂量可能分多次给予    |
| 负荷剂量 | 是指在治疗开始时给予的较大剂量，可*快速达到治疗浓度* |
| 初始剂量 | 是根据药物特性、患者情况和疾病严重程度确定的起始剂量  |
| 维持剂量 | 指达到治疗效果后，为维持药物浓度在有效范围内所需的剂量 |

###### 6. 药物不良反应
药物不良反应（Adverse Drug Reaction, ADR）：是指**合格药品**在**正常用法用量**下出现的与治疗目的无关的有害反应，任何药品都有可能引起不良反应。
###### 7. 药物相关的相互作用
- **药动学（Pharmacokinetics，PK）**：药物半衰期、清除率、分布溶剂、生物利用度；设计给药 方案、特殊人群剂量调整、药物相互作用排查、新药研发。。。
- **药效学(Pharmacodynamics, PD)**：药物作用的分子基础（受体、通道、转运蛋白。。。）、 药物对受体的作用类型：激动剂、部分激动剂、拮抗剂、反向激动剂。。）、信号转导与生理效 应、剂量-效应关系、剂型与药效特点。。。
- 药物间相互作用的*结果*：增强疗效/毒性；降低疗效/毒性。
- 药物间相互作用的*途径*：药动学相互作用；药效学相互作用。

# 三、临床常见症状和疾病的药物治疗
### 1. 发热和疼痛
==发热==：可以提高免疫细胞的活性，帮助身体更有效地对抗感染，还可以抑制病原体生长， 这有助于身体清除感染。
- **对乙酰氨基酚(扑热息痛)**：解热、镇痛，几乎无抗炎作用。解热作用缓和而持久，胃肠刺激小，较安全，适用于儿童和老年人
- **非甾体消炎药 NSAIDS**：常用有布洛芬、双氯芬酸、吲哚美辛等，解热、镇痛、抗炎。（阿司匹林-水杨酸类，目前基本不用于退热）
==疼痛==
- **解热镇痛药(对乙酰氨基酚、NSAIDS)**：“封顶效应”，*ADR:* 有消化性溃疡、消化道出血、血小板功能障碍、肝肾功能损伤
- **阿片类镇痛药(曲马多、吗啡、羟考酮、芬太尼 etc)**：关注成瘾风险。*ADR:* 呼吸抑制、便秘、恶心、呕吐、嗜睡、头晕、尿潴留等
- **辅助镇痛药**：抗抑郁药、抗惊厥药物、镇静催眠药物、肌肉松弛剂、糖皮质激素

##### 4. 肝肾功能不全
==肝功能不全==：药物*代谢*速率减慢，容易在体内积累，增加毒性风险
- 尽量选择不经肝脏代谢或肝毒性较低的药物
- 减少用药种类、减少肝脏代谢负担
- 主要经肝脏清除：大环内酯类（如红霉素）
==肾功能不全==：药物*排泄*受阻，引起药物蓄积和副作用增加
- 选择肾损较少的药物或调整用药方案
- 主要经肾脏排泄的抗生素：氨基糖苷类、某些头孢菌素类
# 五、药物治疗的个体化
###### 1. 外科手术对药物治疗的影响
- 慢性支气管炎患者：术后由于麻醉、卧床等因素，常出现急性发作或诱发呼吸衰竭， 需要加用*抗感染、化痰、平喘*的药物
- 糖尿病：外科手术作可使糖尿病患者血糖升高，可能需要使用*胰岛素等稳定血糖*
- 长期服用抗凝药物者：在术前准备时需要停药并根据患者出血风险来决定是否使用*低分子肝素*桥接
- 长期服用二甲双胍者：在外科术前行CT检查时，应在造影剂使用前后48小时停药，减少肾损伤的发生
- 脏器切除：可影响药物的吸收（胃、肠等），减慢药物清除（肝肾），改变药物的疗 效和安全性
###### 2. 肾脏替代治疗对药物作用的影响
- 经肾脏清除的药物，易被血液透析清除
- 分子量小的药物，易被透析清除
- 蛋白结合率高的药物通过透析清除较少
- 分布容积大的药物通过透析清除较少
*对清除率较高的药物，需要在透析后补充药物；对于血浆蛋白结合率较高不易被透析清除的药物，则无需补充*
