# 第三节 全身麻醉

>[!NOTE] 重点难点
掌握：全身麻醉的概念，气管内插管的目的和方法，全身麻醉的并发症及其防治
熟悉：全身麻醉的常用方法，全身麻醉深度的判断
了解：全身麻醉药的临床药理特点

==概念==：麻醉药经呼吸道吸入或静脉、肌内注射进入人体内，产生中枢神经系统的抑制， 临床表现为**神志消失、全身的痛觉丧失、遗忘、反射抑制和一定程度的肌肉松弛,** 这种方法称为全身麻醉

- 麻醉药对CNS 的抑制完全可逆
	1. 抑制程度与血药浓度有关
	2. 药物被代谢或排出，病人神志和各种反射逐渐恢复
---
## 一、全身麻醉药
- 真正意义上的全身麻醉药：吸入麻醉药、静脉麻醉药
- 全身麻醉术中不可或缺的：肌肉松弛药、麻醉性镇痛药
### （一）吸入麻醉药
- 指经呼吸道吸入进入人体内并产生全身麻醉作用的药物 可用于全身麻醉的诱导和维持
#### 1. 理化性质和药理性能
1. 吸入麻醉药的强度以**最低肺泡浓度**（minimum alveolar concentration ，**MAC**）衡量
	- *MAC是指某种吸入麻醉药在一个大气压下与纯氧同时吸入时，能使50% 病人在切皮时不发生摇头、四肢运动等反应时的最低肺泡浓度*
2. 吸入麻醉药的强度与其**油/气分配系数**成正比关系
	- 油/气分配系数（Oil/gas partition coefficient, λ油/气 ）：在平衡状态下， 药物在橄榄油和 大气中发布的比例，反映药物的脂溶性。
	- λ油/气越大其麻醉效能越强
3. 吸入麻醉药的可控性与其**血/气分配系数**相关
	- 血/气分配系数（Blood/gas partition coefficient, λ血/气 ）：吸入麻醉药在血液中的溶解度， 指吸入麻醉药的分压在血液和空气中相等时，血液中该吸入麻醉药的浓度与空气中该吸入麻醉药 的浓度比
#### 2. 影响肺泡药物浓度的因素
1. 通气效应
2. 浓度效应
3. 心排出量
4. 血/气分配系数
5. 麻醉药在肺泡和静脉血中的浓度差
#### 3. 代谢和毒性
1. 吸入麻醉药的绝大部分由呼吸道排出，仅小部分在体内代谢后随尿排出
2. 主要代谢场所是肝脏
3. 药物的代谢过程及其代谢产物对肝脏和肾脏的功能都有不同程度的影响
#### 4. 常用吸入麻醉药

| 药品           | 特点                       |
| ------------ | ------------------------ |
| 氧化亚氮（笑气 N2O） | 麻醉性能弱（MAC>100%），可导致弥散性缺氧 |
| 七氟烷/醚        | 麻醉性能强，副作用小，器官保护          |
| 地氟烷/醚        | 麻醉性能弱，特殊挥发罐              |
| 异氟烷/醚        |                          |
| 安氟烷/醚        |                          |

#### 5. 吸入麻醉优缺点

| 优点          | 缺点            |
| ----------- | ------------- |
| 1. 作用全面     | 1. 环境污染       |
| 2. 麻醉深度易于掌控 | 2. 肝毒性        |

|             | 4. 恶心呕吐       |

### （二）静脉麻醉药
- 指经静脉注射进入体内，通过血液循环作用于中枢神经系统而产生全身麻醉作用的药物 与吸入麻醉药相比，其优点为诱导快，对呼吸道无刺激，无环境污染，术后恶心、呕吐发生率低
#### 1. 静脉麻醉优缺点

| 优点                       | 缺点                                      |
| ------------------------ | --------------------------------------- |
| 1.麻醉诱导的速度快、诱导平稳，病人感觉舒适   | 1.作用的终止仅依赖于其药代动力学特性                     |
| 2.对呼吸道无刺激作用，对环境无污染       | 2.麻醉医生对其主动干预能力有限。                       |
| 3.不需要通过呼吸道给药，适用于气管及支气管手术 | 3.个体差异性大，可控性差                           |
| 4.对缺氧性肺血管收缩不产生抑制作用       | 4.无良好的镇痛作用，需同时给予麻醉性镇痛药 和肌松弛药，才能达到最佳麻醉状态 |
#### 2. 常用静脉麻醉药
| 药品            | 特点                                          |
| ------------- | ------------------------------------------- |
| 氯胺酮（ketamine） | 镇痛作用显著，临床可用于全麻诱导及维持；可产生分离麻醉；常用于小儿基础麻醉。毒品    |
| 依托咪酯          | 对循环影响小；用于全麻诱导，esp年老体弱和危重病人；长时间输注引起肾上腺皮质功能抑制 |
| 丙泊酚/异丙酚       | 全麻静脉诱导；可产生呼吸抑制；心血管抑制作用                      |
| 咪达唑仑          | 术前镇静、麻醉诱导和维持，局麻辅助和ICU镇静；可产生顺行性遗忘            |
| 右旋美托咪定        | 临床用于术中镇静、全麻辅助，机械通气病人镇静。副作用：心动过缓、心脏传导抑制、低血压  |
### （三）肌松药
#### 1. 作用原理和分类
- ==去极化肌松药==：其分子结构与**乙酰胆碱**相似， 能与终板的乙酰胆碱受体结合，引起运动终板去极化 使运动终板暂时丧失对乙酰 胆碱的正常反应，肌处于松 弛状态
- ==非去极化肌松药==：其与运动终板胆碱能受体结合 后，不改变运动终板的膜电位， 而是*妨碍乙酰胆碱与其受体的 结合*，使肌松弛。
#### 2. 常用肌松药

| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| 琥珀胆碱（司可林，Succinylcholine, scoline）                          | • 维库溴胺 Vecuronium • 阿曲库铵 Atracurium <br>• 顺阿曲库铵 Cisatracurium • 哌库溴胺 pipecuronum <br>• 罗库溴胺 Pancuronium |
| 应用：<br>- 有利于饱胃病人的气管内插管<br>- 短小操作中维持肌肉松弛                     | 应用：<br>- 全身麻醉时气管内插管<br>- 术中及ICU内维持肌肉松弛                                                                  |
| 副作用：<br>- 术后肌痛<br>- 心动过缓<br>- 高钾血症<br>- 升高颅内压和眼内压<br>- 恶性高热 | 拮抗药：<br>- 新斯的明<br>- Sugammadex——罗库溴胺 的特异性拮抗剂                                                            |

## 四、全身麻醉的并发症及其防治

##### （一）反流与误吸
- 贲门松弛--*胃食管反流*；一旦到达咽喉部可发生误吸，造成窒息或吸入性肺炎
- 最易发生在麻醉诱导、气管插管前或气管拔管后
- 处理：头偏一侧，头低脚高位，吸引器清除
- **==Mendelson综合征==**：吸入酸性胃液可导致*哮喘样发作*，并引发*吸入性肺炎*，临床表现为发绀、呼吸困难、呼吸浅速、心率增快、支气管痉挛

##### （二）呼吸道梗阻
- 以声门为界，上呼吸道/下呼吸道梗阻
###### 1.上呼吸道梗阻
- 常见原因：①机械性；不全-鼾声；完全-鼻翼扇动和三凹征。②喉痉挛-吸气性呼吸困难，缺氧发绀
###### 2. 下呼吸道梗阻
- 常见原因：支气管痉挛【哮喘史/COPD病人】，气管导管扭折，导管斜面堵塞等
##### （三）通气量不足
1. 主要表现为二氧化碳潴留，伴有低氧血症
	- 各种麻醉药产生的中枢性和外周的呼吸抑制
	- 处理：增加潮气量或呼吸频率
2. 全身麻醉后的通气量不足
	- 各种麻醉药物的残留作用，中枢性呼吸抑制和呼吸肌功能障碍
##### （四）低氧血症
##### （五）低血压
##### （六）高血压
##### （七）心律失常
##### （八）恶性高热
- 常见于小儿麻醉
- 遗传性
- 治疗恶性高热的特效药物是丹曲林

## 局麻药药理

### 作用机制
- 直接作用于==细胞膜Na＋通道特异性受体==，阻断Na+内流，从而阻断 神经冲动的传导，起到神经阻滞作用
### 化学结构及分类
![300](Pasted%20image%2020250916153754.png)
### 理化性质与临床麻醉特性
#### 1. 离解常数Pka：决定麻醉药起效的快慢及弥散性能
- 局麻药分子解离成具有药物活性的自由碱基才能透过各层组织屏障和生物膜,产生麻醉作用
- Pka越大,与组织PH（接近7.4）差值增大,有药物活性的自由碱基比例越少,穿透组织屏障和生 物膜的药物越少神经阻滞起效时间愈长，弥散性能愈差.
- 提高组织PH,缩短神经阻滞起效时间，增加弥散的比例
#### 2. 脂溶性：决定麻醉效能
- 脂溶性越大,越易穿透神经组织膜,麻醉效能越强
#### 3. 血浆蛋白结合率：与阻滞作用持续时间有关
- 结合率高则作用时间长

### 药代动力学
- 吸收：进入血液循环
	- 影响因素
		- A.给药途径:静脉（咽喉、气管、支气管粘膜）> 粘膜 > 肋间 > 皮下（皮内）
		- B.剂量和浓度：量大、浓度高，吸收快
		- C.扩血管作用：加肾上腺素
- 分布：再分布于其它组织，蛋白结合率高不易透过胎盘屏障
- 生物转化和排泄
	- 酯 类(肝外代谢)：假性胆碱酯酶
	-  酰胺类(肝内代谢)：肝细胞内质网微粒体酶
### 不良反应

1. **局麻药全身毒性反应 LAST**
	1. 常见原因：超过耐受、意外注入血管内、主要部位血供丰富
	2. 临床表现：中枢神经系统和心血管系统
	3. 预防和治疗：脂肪乳剂；麻醉前用药如地西泮或巴比妥类
2. **过敏反应**
	1. 临床表现：荨麻疹，咽喉水肿，支气管痉挛……
	2. 处理原则：停止用药，保持呼吸道通畅，维持循环稳定，同时应用糖皮质激素和抗组胺药

### 局麻方法
#### 1. 表面麻醉
- 黏膜表面；穿透力强
- 丁卡因、利多卡因
#### 2. 局部浸润麻醉
- 注射
- 普鲁卡因、利多卡因
#### 3. 区域阻滞
- 手术部位四周和底部注射局麻药
- 普鲁卡因、利多卡因
#### 4. 神经阻滞
- 神经干、丛、节周围注射

# 第五节 椎管内麻醉
- 将局麻药注入椎管内的不同腔隙，脊神经根受到阻滞使脊神经支配的区域产生 麻醉作用

- **蛛网膜腔下腔阻滞**(Subarachnoid block)：局麻药注入蛛网膜腔下腔产生的阻滞作用， 又称==腰麻或脊麻== 
- **硬膜外阻滞**(Epidural block )：局麻药注入硬膜外腔产生的阻滞作用 
- **脊麻-硬膜外联合阻滞**(Combined spinal-epidural block)：上述两种技术的联合应用

#### 椎管内麻醉的解剖基础
- 四个生理弯曲：颈，胸，腰，骶--影响麻药扩散
- 脊髓与被膜

#### 椎管内麻醉的生理
- 蛛网膜腔下腔：脊髓和脑脊液
	- 脑脊液：对腰麻时起稀释或扩散局麻药作用。
- 硬膜外腔：容积和压力随生理和病理状态而变
	- 容积：100ml
	- 压力：负压
- 药物作用的部位：
	- 腰麻：主要直接作用于N根，也作用于脊髓表面
	- 硬膜外麻醉，三个途径：
		- ①通过蛛网膜绒毛进入根蛛网膜下腔作用于N根 
		- ②渗出椎间孔，在椎旁阻滞脊N 
		- ③直接透过硬脊膜和蛛网膜，类似腰麻
- **神经阻滞顺序**
	- 交感N —冷觉—温觉—温度识别觉—钝痛觉—锐痛觉—触觉—运动 神经—压力—本体感觉消失，阻滞消退顺序与阻滞顺序相反
- **阻滞作用与麻醉平面**
	- 麻醉平面：用针刺法测痛觉消失范围，上下界称平面
	- 感觉神经阻滞——皮肤和肌肉不疼
	- 交感神经阻滞——降低内脏牵拉反应、血压、心率、汗腺分泌 
	- 运动神经阻滞——肌松作用
- 椎管内麻醉对机体的影响
	- 呼吸：完全取决于平面高低 
	- 循环：最为常见的是低血压，是因交感神经被阻滞所致 
	- 体温：中心体温下降（交感神经阻滞引起外周血管扩张）。麻醉后30-60min达高峰 
	- 其它：迷走神经亢进，胃肠蠕动增加，易恶心呕吐。腰麻、骶神经阻滞易生尿酯留
#### 椎管内麻醉方法

##### 蛛网膜下腔阻滞
- 禁忌症：
	- CNS疾患；休克；穿刺点感染；败血症；脊柱外伤、肿瘤或结核；急性心衰或冠心病发作
	- 老年、孕妇、心脏病、高血压慎用
##### 硬膜外腔阻滞
- 平面的调节：与腰麻截然不同
- 并发症：穿破硬脊膜、全脊髓麻醉、血压下降、呼吸抑制、脊神经根损伤、导管拔出困难或折断、硬膜外 血肿、毒性反应、恶心呕吐
