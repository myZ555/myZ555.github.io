# 概论
- **定义**：有效循环血量下降、组织灌流不足导致细胞代谢紊乱、功能异常的病理生理过程
	- 有效循环血量的维持条件：①充足的血容量；②足够的心排出量（CO）；③适当的外周血管张力。
	- 共同的病理生理基础：①有效循环血量锐减；②组织灌流不足；③炎症介质的产生。
- 本质：==氧供给不足和需求增加==
- 特征：产生炎症介质
- 分类：
	- 按病因：*低血容量性休克*（失血性休克、创伤性休克等）、*血管源性休克*（神经源性休克、过敏性休克、感染性休克等）、*心源性休克*。
- 治疗原则：本质：循环血量减少、氧供不足---首先补充血容量

1. **微循环的变化**

2. **代谢改变**
	1. 供氧不足→细胞无氧代谢→*乳酸↑*、丙酮酸↓（乳酸/丙酮酸比值 L/P↓）→*代谢性酸中毒*。
	2. 能量代谢障碍：交感-肾上腺髓质轴+下丘脑-垂体-肾上腺皮质轴→高血糖、蛋白分解、脂肪分解。
3. **炎症介质释放和缺血再灌注损伤**
4. **内脏器官损害**

|     |                              |                                                           |
| --- | ---------------------------- | --------------------------------------------------------- |
| 肺   | 急性窘迫综合症 ARDS                 | 肺泡表面活性物质减少、微循环栓塞、肺不张、肺水肿、 无效腔增加、肺分流等。常发生于休克期，或稳定后 48~72h。 |

# 低血容量性休克
- 原因：大量失血/体液*丢失*，或液体潴留于*第三间隙*
- ==**失血性休克**==：迅速失血超过全身血量20%
![625](Pasted%20image%2020251014142045.png)
	- 补液试验：等渗盐水250ml静脉注入，若血压升高而CVP不变，为血容量不足；若血压不变而CVP增高，为心功能不全
- ==**创伤性休克**==：
	- 特点：伤员；易发生多器官衰竭

# 感染性休克

- 病因：常继发于 **G-杆菌感染**（内毒素性休克），是机体对宿主-微生物应答失衡的表现。
- ==全身性炎症反应综合征（**SIRS**）==判断标准：
	- 体温高于38或低于36
	- 心率＞90
	- 呼吸＞20
	- PaCO2<4.3kPa
	- 血白细胞计数＞12×10^9 或<4×10^9
- 感染性休克诊断：*①SIRS；②感染的证据；③休克表现*
![625](Pasted%20image%2020251014142110.png)
- 治疗原则
	- 休克未纠正前：着重治疗休克，同时治疗感染
	- 休克纠正后：着重治疗感染
