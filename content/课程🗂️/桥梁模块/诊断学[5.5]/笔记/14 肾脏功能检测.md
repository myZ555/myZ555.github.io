# （一）肾小球功能

==肾小球滤过率 GFR==：双肾在单位时间内产生原尿的量
==肾清除率==：双肾在单位时间内，能将若干毫升血浆中所含的某物质全部加以清除![[Pasted image 20251208144048.png|400]]
*各种物质经肾排出的方式：*
1. 全部由肾小球滤过，肾小管既不吸收也不分泌【菊粉】（完全反映GFR）
2. 全部由肾小球滤过，不被肾小管重吸收，很少被肾小管排泌【**肌酐**】（基本代表GFR）
3. 全部由肾小球滤过后又被肾小管全部重吸收【葡萄糖】（肾小管最大吸收率）
4. 除肾小球滤过外，大部分通过肾小管周围毛细血管向肾小管分泌后排出（肾血流量）

### 1、血清肌酐测定 Cr
> 机体每20g肌肉每天代谢产生1mgCr，产生速率为1mg/min，每天Cr的生成量恒定
> 血中Cr主要由肾小球滤过排出体外，肾小管基本不重吸收且排泌量也较少
> 在外源性肌酐摄入稳定的情况下，血液中Cr的浓度取决于肾小球滤过能力

【临床意义】
- 增高：肾小球滤过功能减退
- 鉴别肾前性/肾实质性少尿：肾前性常<200μmol/L，肾性可>200μmol/L
- Bun/Cr(单位mg/dl)的意义
- 老年人、肌肉消瘦者肌酐可能偏低
### 2、内生肌酐清除率 Ccr

> 肾脏在单位时间内把若干毫升血液中的内在肌酐全部清除出去的能力-内生肌酐清除率

- 成人正常值为80~120ml/min，随年龄增长，有自然下降趋势
【临床意义】
1. 判断肾小球损害的程度：是反应GFR的灵敏指标；
2. 评估肾功能
3. 指导治疗
### 3、血尿素氮 BUN
- 蛋白质代谢的终末产物；
	- 生成：生成速率相对稳定；蛋白质（饮食摄入或体内组织蛋白分解）经肝脏代谢
	- 排泄：＞90%通过肾小球滤过进入原尿，原尿中的尿素约 50% 在肾小管被重吸收
【临床意义】
1. 器质性肾功能损害:肾功能下降
2. 肾前性少尿：扩容后，一般BUN自行下降
3. 蛋白质分解或摄入过多（*不敏感，特异性较低*）
### 4、肾小球滤过率测定
- GFR可以用$^{99m}\text{Tc-DTPA}$的最大清除率表示
- 方法：动态监测放射性强度变化
- 优点：*双侧分肾功能*
### 5、血β2微球蛋白 β2-MG
- 体内有核细胞包括淋巴细胞 、 血小板、多形核白细胞产生的一种小分子球蛋白
- 正常人血液队 -MG 浓度很低，可自由通过肾小球，然后在近端小管内几乎全部被重吸收
【临床意义】
1. 评价肾小球功能（血β2-MG 升高比血肌酐更灵敏）
### 6、血清胱抑素C
- 由人体所有有核细胞持续、稳定表达（无组织特异性）
- 不依赖肌肉量、年龄、性别
- 自由通过肾小球滤过膜，在近曲小管完全被重吸收并降解（不被肾小管分泌）
- *反映肾小球滤过功能（GFR）的理想内源性标志物*
# （二）肾小管功能

> [!肾小管功能]
> **远端小管**：
> - 肾脏浓缩-稀释功能试验
> - 尿渗量测定
> 
> **近端小管**：
> - 对小分子蛋白重吸收功能
> - 肾小管葡萄糖最大重吸收量试验(TmG)
> 
> **肾小管酸碱平衡调节**：
> - 尿PH
> - 尿中碳酸氢根、可滴定酸、尿铵测定

## 近端小管（重吸收、排泌）
### 1、尿β2-微球蛋白
- 体内有核细胞产生的小分子球蛋白，自由通过肾小球，几乎完全被肾小管重吸收
- 正常参考值：成人尿＜0.3mg/L，或以尿肌酐校正＜0.2mg/g

【临床意义】
- 近端肾小管重吸收功能受损，如肾小管-间质性疾病、药物或毒物所致早期肾小管损伤，以及肾移植后急性排斥反应早期
- 肾小球滤过功能：非重吸收功能原因
### 2、尿α1-微球蛋白
- 自由透过肾小球，约 99% 被近端肾小管上皮细胞以胞饮方式重吸收并分解，故仅微量从尿中排泄

【临床意义】
- 近端肾小管重吸收功能受损，如肾小管-间质性疾病、药物或毒物所致早期肾小管损伤，以及肾移植后急性排斥反应早期
- 肾小球滤过功能：非重吸收功能原因
### 3、尿视黄醇结合蛋白 RBP
- 游离的 RBP 由肾小球滤出，大部分由近端小管上皮细胞重吸收，并被分解成氨基酸供体内合成利用，仅有少量从尿中排泄
- 肾小管重吸收功能障碍时，尿中 RBP浓度升高，血清 RBP 浓度下降

【临床意义】
1. 尿液RBP升高：早期近端肾小管损伤
2. 血清RBP升高：肾小球滤过功能减退、肾功能衰竭
### 4、肾小管葡萄糖最大重吸收试验 TmG
### 5、肾小管对氨马尿酸最大排泌量测定 TmPAH
### 6、尿酶测定 NAG酶
- 当肾小管损伤时，尿NAG排出增多
- 对急性损伤和活动性病变特别敏感，可用于早期肾损伤的监测和病程的观察
## 远端小管（浓缩）
### 1、尿渗量测定 Uosm
【参考值】
- 正常人禁饮后尿渗量为600~1000mOsm/kg H2O
- 平均800mOsm/kg H2O
- 尿:血浆渗量为3~4.5:1

【临床意义】判断肾浓缩功能。
- 肾前性少尿：>450mOsm/kgH2O
- 肾性少尿：<350mOsm/kgH2O

### 2、浓缩稀释功能试验【莫氏试验】-昼夜尿比重
- 试验前日晚8时后禁食，试验当日正常进食，每餐含水分约500ml，不再饮任何液体。晨8时排尿弃去，于上午10时、12时，下午2、4、6、8时（日间尿）及次晨8时（夜间尿）各留尿1次，尿须排尽。准确测定各次尿量及比密。
【参考值】
- 昼尿量:夜尿量为3~4:1
- 尿液最高比重应在1.020以上
- 最高比重－最低比重>0.009
【临床意义】尿量升高，比重下降，夜尿多——*肾小管浓缩功能*差


# （三）血尿酸检测
==**尿酸 UA**==：核蛋白和核酸中嘌呤的代谢产物，包括内源性和外源性
成人酶法血清尿酸浓度男性：150~416umol/L，女性89~357umol/L

*==血尿酸浓度升高：==*
- 肾小球滤过功能受损
- 体内尿酸生成异常增多
	- 遗传性酶缺陷所致的原发性*痛风*，以多种血液病、恶性肿瘤等细胞大量破坏所致的继发性痛风。
	- 长期服用利尿剂及抗结核药物、慢性铅中毒等。

*==血尿酸浓度降低：==*
- 各种原因导致肾小管重吸收尿酸功能损害，尿中大量丢失，以及肝功能严重损害致尿酸合成减少。如范可尼综合征、急性重型肝炎，肝豆状核变性等。
# （四）肾小管性酸中毒 RTA

* **近端小管：**
	* 分泌 $\mathrm{H}^+$
	* 重吸收 $\mathrm{HCO}_3^-$
	* 分泌 $\mathrm{NH}_4^+$
	* 形成可滴定酸 ($\mathrm{TA}$)
* **髓襻：**
	* 重吸收 $\mathrm{HCO}_3^-$
	* 重吸收 $\mathrm{NH}_4^+$
* **远端小管：**
	* 重吸收或分泌 $\mathrm{NH}_4^+$
	* 重吸收或分泌 $\mathrm{HCO}_3^-$
	* 重吸收或分泌可滴定酸 ($\mathrm{TA}$)

### 1、碱负荷试验（碳酸氢根离子重吸收排泄试验）
- $\mathrm{HCO}_3^-$：80-90%由近端小管重吸收；肾阈值25-27mmol/L
- 正常值≈0，$\mathrm{HCO}_3^-$>15%可诊断为近曲肾小管酸中毒
### 2、酸负荷试验（氯化铵负荷试验）
- 远端小管排泌$\mathrm{H}^+$，与$\mathrm{NH}_3$结合为$\mathrm{NH}_4^+$，经尿排出，尿液明显酸化
- 如尿pH>5.5，可诊断为远曲小管酸中毒

// =====================================================
// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
