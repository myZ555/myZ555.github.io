> 尿液检测主要用于：
> 	1. 协助泌尿系统疾病的诊断、病情和疗效观察
> 	2. 协助其它系统疾病的诊断
> 	3. 职业病防治
> 	4. 用药的监护
> 	5. 健康人群的普查
> 尿液检测：尿液标本收集，尿液的一般性状，尿液化学检查，尿液显微镜检查；各项检测指标的参考值及增减变化的临床意义


> [!NOTE] 尿液的生成
> - 影响肾小球滤过的因素：
> 	- 滤过屏障的通透性和有效滤过面积
> 	- 有效滤过压和肾血流量
> - 肾小管和集合管重吸收及排泌
> -  ![[Pasted image 20251208140736.png|200]]
> 	- 近端肾小管：滤过液的重吸收
> 	- 髓襻：重吸收水和电解质（非比例性）
> 	- 远端小管、集合管：调节尿液最终成分
> - ==尿液==：血液经过肾小球滤过、肾小管和集合管重吸收和排泄所产生的终末代谢产物

# （一）尿液标本采集
![[Pasted image 20251208081228.png]]
==注意事项==
1. 晨尿标本价值最大
2. 陈旧性标本可因尿液 CO2 挥发或细菌生长而使 pH 增高
3. 细菌可使尿液葡萄糖降解为酸和乙醇，使其 pH 降低
4. 尿液标本在2h内检测完毕；
	1. 冷藏：2-8℃
	2. 化学防腐
# （二）尿液一般性状检查
![[Pasted image 20251208081409.png]]
### 尿量
###### 1. 多尿：
[[23 少尿、无尿与多尿]]
- **多尿**：成人24h尿量大于2500ml，儿童24h尿量大于3000ml
1. *生理性多尿*
2. *病理性多尿*![[Pasted image 20251208085211.png]]
	- 内分泌：
		- **糖尿病**：尿糖增多引起溶质性利尿
		- **尿崩症**：ADH不足或肾小管对ADH反应性降低【绝对/相对不足】
	- 肾脏疾病【小管】
###### 2. 少尿与无尿
- **少尿**：成人24h尿量<400ml或每小时少于17ml
- **无尿**：成人24h尿量<100ml，小儿少于30-50ml![[Pasted image 20251208085353.png]]
### 颜色
正常：淡黄色/橘黄色
1. **==红色==**
	- 以血尿最常见；1000ml尿液所含血量超过1ml，外观出现红色--肉眼血尿![[Pasted image 20251208085646.png]]
2. **==深黄色==**
	- 最常见：*胆红素尿*：含有大量结合胆红素的尿液。外观：深黄色豆油样，振荡后其泡沫仍呈黄色，胆红素定性检查阳性。
	- 常见于：胆汁淤积性黄疸及肝细胞性黄疸
	- 尿液放置过久，胆红素氧化为胆绿素，可使尿液呈棕绿色
		- *VitB2，利福平，呋喃唑酮* 也可使尿液外观呈黄色【VitB12：黄色荧光】
3. **==白色==**
	![[Pasted image 20251208085942.png]]
4. **==黑褐色==**
	- 见于重症血尿、变性血红蛋白尿；也可见于酪氨酸病、酚中毒、黑尿酸症或黑色素瘤等
5. **==蓝色==**
	- 主要见尿布蓝染综合征
6. **==淡绿色==**
	- 见于铜绿假单胞菌感染
### 气味
- 氨臭味：慢性膀胱炎、尿潴留或尿液长时间放置（非特异性）
- 蒜臭味：有机磷中毒
- 烂苹果味：糖尿病酮症酸中毒
- 鼠臭味：苯丙酮尿症
### 透明度
- 正常：清晰透明
- 浑浊：盐类结晶、红细胞、白细胞、细菌、乳糜
![[Pasted image 20251208090220.png]]
### 比重
- 指在4℃条件下尿液与同体积纯水的重量之比，是尿液中所含溶质浓度的指标
1. 比重增高：>1.025--高渗尿/高比重尿。见于血容量不足导致的*肾前性少尿*、糖尿病、急性肾小球肾炎、肾病综合征等
2. 比重降低：<1.015--低渗尿/低比重尿。见于大量饮水、慢性肾小球肾炎、尿崩症等。尿比重固定于1.010±0.003，提示*肾脏浓缩稀释功能丧失*
### 酸碱度
- PH降低：
	- ==肾小球滤过增加，肾小管保碱能力正常==
	- 食物：肉类及混合性食物等
	- 药物：氯化铵，维生素C
	- 疾病：酸中毒，高热，糖尿病，通风；*低钾性代谢性碱中毒*
- PH增高
	- ==肾小球滤过功能正常，肾小管保碱能力丧失==
	- 食物：蔬菜、水果
	- 药物：噻嗪类利尿剂、碳酸氢钠
	- 疾病：碱中毒、膀胱炎、肾小管性酸中毒；尿液放置过久致尿素分解释放氨
- 药物干预
	- 氯化铵酸化尿液--促使碱性药物排出
	- 碳酸氢钠碱化尿液

### 蛋白质

> - 定性：阴性
> - 定量：0-80mg/24h
> - 当蛋白质浓度＞100mg/L 或 150mg/24h 尿液，蛋白质定性检查呈阳性的尿液，称为蛋白尿

 [[13-1 尿液检查.pdf#page=29&selection=0,0,0,8|肾小球滤过膜]]
 
==蛋白尿产生的机制==
1. 肾小球毛细血管壁断裂或电荷屏障改变，使得大量高、中、低相对分子质量的蛋白漏出，超过肾小管重吸收能力而出现在终尿中
	1. 选择性蛋白尿：以血清白蛋白为主，无大相对分子质量的蛋白
	2. 非选择性蛋白尿：尿中出现大相对分子质量的蛋白质，如免疫球蛋白、补体；肾小球毛细血管壁有严重损伤断裂，预后不良
2. 肾小管功能受损时，近端小管对蛋白质重吸收障碍所致
3. 血浆中蛋白异常增多超过肾小管重吸收能力
4. 肾髓袢升支及远曲小管起始部分分泌的T-H糖蛋白增加
###### 1. 生理性蛋白尿
1. 功能性：剧烈运动/受寒/交感兴奋等，肾血管痉挛或充血导致的肾小球毛细血管壁通透性增高
2. 体位性/直立性：脊柱压迫左肾静脉导致局部静脉压增高
###### 2. 病理性蛋白尿
![[Pasted image 20251208134006.png]]

├── *血浆蛋白来源* 
│   ├── 肾小球性
│   │   ├── 肾病综合征/ 肾病性蛋白尿
│   │   └── 原发、继发肾小球疾病

### 尿糖
- 原理：尿糖--葡萄糖氧化酶--过氧化氢--显色
- 肾糖阈：当血糖浓度超过 8. 88mmol/L 时，尿液中开始出现葡萄糖
	-  肾小球滤过率降低可导致肾糖阙增高
	- 肾小管重吸收率降低则可引起肾糖阑降低
1. ==血糖增高性糖尿==
	1. 代谢性：糖代谢紊乱【糖尿病】
	2. 应激性：颅脑外伤/脑血管意外/情绪激动，延髓血糖中枢受刺激，肾上腺素/胰高血糖素大量释出，出现暂时性高血糖和糖尿
	3. 摄入性：短时间摄入大量糖类/输注高渗葡萄糖
	4. 内分泌性：*生长激素/肾上腺素/糖皮质激素*分泌过多
2. ==血糖正常性糖尿==：
	- 血糖浓度正常，肾小管病变--重吸收葡萄糖能力降低--肾糖阈下降
	- 肾性糖尿：慢性肾炎、肾病综合征等
3. ==暂时性糖尿==：饮食性、精神性、妊娠期、应激性等
4. 其他
5. ==假性糖尿==：尿液中含有的某些还原性物质以及一些随尿液排出的药物
### 酮体
- 当肝脏内酮体产生的速度超过肝外组织利用的速度时，血液酮体浓度增高，称为酮血症。
- 过多的酮体从尿液排出形成酮尿 (ketonuria) 
- 主要用于 *糖代谢障碍和脂肪不完全氧化* 的判断与评价
1. ==糖尿病酮症酸中毒==
	- 葡萄糖利用减少，而分解脂肪产生酮体增加
	- 糖尿病酮症酸中毒病人伴有肾衰竭，而肾阙值增高时，尿液酮体亦可减少，甚至完全消失
2. ==非糖尿病性酮症==：感染性疾病（肺炎、伤寒、败血症、结核等）、严重呕吐、剧烈运动
3. ==中毒==
4. ==药物影响==：降糖药--抑制细胞呼吸

### 尿液胆红素与尿胆原
![[Pasted image 20251208140249.png|300]]
- 尿胆红素：
	- 方法：强酸的介质中直接胆红素与二氯苯胺重氮盐起偶联反应生成红色的复合物
	- 阳性/增高：
		1. 急性黄疸型肝炎、胆汁淤积性黄疸
		2. 门脉周围炎、纤维化/药物所致的胆汁淤积
		3. 先天性高胆红素血症
- 尿胆原：
	- 方法：在强酸性条件下，尿胆原与对-二甲氨基苯甲醛发生醛化反应，生成樱红色缩合物，模块发生从黄色到红色的颜色变化。在强酸条件下尿胆原和对-甲氧基重氮四氟化硼发生重氮盐偶联反应，生成胭脂红色化合物
	- 阳性/增高：肝细胞性黄疸、溶血性黄疸
	- 降低：胆汁淤积性黄疸、阻塞性黄疸

### 细胞
#### 红细胞
- *镜下血尿*：离心尿液中红细胞数量增多，超过 3 个/HPF, 且外观无血色的尿液*
- *红细胞淡影/影形红细胞*：低渗尿液中红细胞胀大，甚至使血红蛋白溢出，形成大小不等的空环
![[Pasted image 20251208143217.png]]
- 尿沉渣中不染色红细胞典型形状为浅黄色双凹盘状；碱性尿液中红细胞边缘不规则
	- 高渗：脱水皱缩，表面带刺、颜色较深的桑葚状
	- 低渗：吸水胀大甚至血红蛋白溢出，大小不等的空环形，红细胞淡影
- 玻片法正常为：0~3个/HP，定量检查0~5个/微升
- 多形红细胞大于80%→肾小球源性；小于50%→非肾小球源性

|          | 外观    | 是否均一 |
| -------- | ----- | ---- |
| 肾小管源性血尿  | 多形性改变 | 否    |
| 非肾小管源性血尿 | 双凹盘形  | 是    |
#### 白细胞&脓细胞
- 尿液中的白细胞主要是中性粒细胞
- *脓细胞*：炎症过程中破坏或死亡的白细胞
#### 上皮细胞
- 肾小管上皮细胞：肾小管病变
- 移行上皮：肾盂、输尿管、膀胱颈部炎症
- 扁平上皮细胞：大量且伴白细胞，见于泌尿生殖系统炎症

### 管型

- 蛋白质、细胞及其崩解产物在肾小管、集合管内凝固而成的圆柱形蛋白聚体，是尿沉渣中最具有诊断价值的成分。
- 管型是在肾小管管腔内形成的。当肾小球滤过膜通透性增强时，肾小管内大量的蛋白质不能完全重吸收--到达远端肾小管及集合管--凝固形成管型
- **主要成分**：肾小管分泌的Tamm-Horsfall蛋白(T-H蛋白)、血浆蛋白、各种细胞及其变性产物等
- **形成条件**：
	1. 尿中清蛋白、肾小管上皮细胞产生的T-H糖蛋白构成管型的**基质**；
	2. 肾小管仍有*浓缩和酸化尿液*的功能，前者使形成管型的蛋白等成分浓缩，后者促进蛋白变性聚集；
	3. 仍存在可交替使用的肾单位

| 管型     | 成分             | 意义                                                                  |
| ------ | -------------- | ------------------------------------------------------------------- |
| 透明管型   | T-H糖蛋白、清蛋白和氯化物 | 老年人清晨浓缩尿中可见到<br>- 运动/麻醉/利尿剂/发热-一过性增多<br>- 肾病综合征、慢性肾炎、恶性高血压和心力衰竭时可见； |
| 红细胞管型  | 基质+RBC         | 急性肾小球病变、肾小球出血                                                       |
| 白细胞管型  | 基质+WBC         | 肾脏感染性病变或免疫性反应                                                       |
| 上皮细胞管型 | 基质+肾小管上皮细胞     | 肾小管坏死                                                               |
| 颗粒管型   | 基质+变性细胞分解产物    | 粗颗粒：慢性肾炎/肾盂肾炎/肾小管损伤<br>细颗粒：慢性肾炎/急性肾小球肾炎后期                           |
| 蜡样管型   | 细颗粒管型演化        | 严重的肾小球变性坏死，预后不良                                                     |
| 混合管型   |                | 各种肾小球疾病                                                             |
| 脂肪管型   | 基质+脂肪滴         | 肾小管损伤 、 肾小管上皮细胞脂肪变性                                                 |
| 宽幅管型   |                | 慢性肾衰少尿期。肾功能不全管型                                                     |
| 细菌管型   |                | 感染性疾病                                                               |
| 结晶管型   | 盐类、药物等化学物质     |                                                                     |

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
