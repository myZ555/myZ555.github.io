## 2. 检查方法
### 1. 视诊
1. 要点：==呼吸频率、深度、运动、节律与幅度==
2. 病理情况
	1. 呼吸过速/过缓：
		- 正常：12-24次/分
		- **呼吸过速**：发热、疼痛、缺氧、呼吸衰竭、 心力衰竭、激动、贫血、甲亢……
		- **呼吸过缓**：麻醉剂或镇静剂过量、颅内压 增高、呼吸衰竭……
	2. 呼吸变浅/变深：
		- 呼吸变浅：见于肺部疾病、腹水等腹部疾 病、肥胖、呼吸中枢抑制或呼吸肌无力
		- 呼吸变深：剧烈运动、情绪激动、过度紧 张
	3. 呼吸模式
		- 胸式呼吸：呼吸以肋间肌运动为主。女性以胸式呼吸为主。
		- 腹式呼吸：正常成年男性和儿童，横膈运动为主，吸气时 上腹部膨隆较明显。正常成年男性和儿童的呼吸以腹式呼 吸为主。
		- 胸式呼吸减弱而腹式呼吸增强： 临床情况：见于肺炎、肺水肿、大量胸水、 气胸、肋间神经痛、肋骨骨折等
		- 腹式呼吸减弱而胸式呼吸增强： 临床情况：见于腹膜炎、大量腹水、肝脾肿 大、腹腔内巨大肿瘤，妊娠
		- 胸腹矛盾呼吸：正常人吸气时胸廓扩张伴 有腹壁膨隆。胸腹矛盾呼吸时吸气相胸廓 扩张而腹壁反而塌陷。
	4. 异常节律
		1. Kussmaul呼吸
			1. 糖尿病酮症酸中毒和 尿毒症酸中毒时，常见到呼吸加深。由于 体液PH降低，刺激呼吸中枢，使通气增加 所致。
			2. 缺氧的呼吸代偿
		2. 潮式呼吸：
			1. 浅慢到深快，再从深快到浅慢
			2. 呼吸中枢兴奋性降低，呼吸中枢对呼 吸节律的调节失常的表现。
		3. 间停呼吸
			1. 呼吸中枢抑制
		4. 叹息样呼吸
			2. 功能性改变。神经衰弱、精神紧张、抑郁症
	- ==三凹征==
		- 吸气时肋间隙回缩提示呼吸道阻塞使吸气 时气体不能顺利地进入肺内，常与胸骨上 窝和锁骨上窝同时发生凹陷，称为“三凹 征”
		- 常见于喉部、气管、大支气管的 狭窄和阻塞
### 2. 触诊
- 要点：胸廓扩张度，语音震颤，胸膜摩擦感
###### 1. 胸廓扩张度
1. 一侧胸廓扩张度增强：对侧肺扩张受限 
2. 一侧胸廓扩张度减弱：肺部疾病、胸膜病变、 肋骨病变、胸壁软组织病变、膈肌病变 
3. 两侧胸廓扩张度均增强：腹部疾病 
4. 两侧胸廓扩张度均减弱：神经病变，呼吸肌无 力、广泛肺部病变等
###### 2. 语音震颤（触觉语颤）
- 影响语颤的因素（声波传导的影响因素）
	- 发音的强弱
	- 音调的高低 
	- 胸壁的厚薄 
	- *支气管的通畅程度* 
	- *支气管到胸壁距离*
- **语颤减弱/消失**：
	- 肺泡内含气量过多 如肺气肿
	- 支气管阻塞 如 阻塞性肺不张
	- 大量胸腔积液或气胸
	- 胸壁水肿或皮下气肿
- **语颤增强**
	- 肺泡炎症浸润肺组织实变：如大叶性肺炎
	- 接近胸膜的肺内巨大空腔：如结核空洞
	- 压迫性肺不张：如胸腔积液上方
###### 3. 胸膜摩擦感
- 呼气.吸气均可触到，屏气消失
- 当各种原因引起胸膜炎症时，胸膜表面粗 糙，呼吸时两层胸膜互相摩擦，可触到摩 擦感
- 动度较大的前胸下前侧部或腋中线第5.6间 最易触及
### 3. 叩诊
1. 间接叩诊，对比检查
2. 叩诊音![[Pasted image 20250926132602.png|300]]
	1. 正常：清音
	2. 病理状态
		1. 异常浊音/实音：肺部病变、胸膜病变、 胸壁病变
		2. 过清音：肺气肿
		3. 鼓音：肺结核空洞，气胸
	3. 肺界叩诊（肺上界、肺下界、肺下界移动度的正常值及异常改变的意义）
		- 深吸气与深呼气时肺下界移动的范围
		- 肺下界移动减弱:肺气肿、肺不张、肺纤维化、 肺水肿、肺炎等，气胸、胸水、胸膜肥厚 或*膈肌麻痹*时肺下界移动度减少
### 4. 听诊
-  要点：呼吸音，附加音
	1. 呼吸运动引起气流进出呼吸道，并产生湍流 造成振动，经过肺和胸壁传到体表，借助听 诊器所听到的声音为肺部呼吸音。包括：*正常呼吸音、异常呼吸音、和附加音如啰音和 胸膜摩擦音。*
	2. 顺序：肺尖→上肺→下肺，前胸→侧胸→背部。 强调两侧对比听诊

##### 2. 附加音
呼吸音以外的附加音，正常人一般并无啰音存在。![[Pasted image 20250922144728.png]]
###### 1. 湿罗音
- 由于吸气时气体通过呼吸道内的 *稀薄分泌物*如渗出液、痰液、血液、粘液 和脓液等，使形成的水泡破裂所产生的声 音，故又称水泡音。【**气过水声**】
	- 连续而短暂，一次常连续多个出现
	- 吸气时或吸气终末时较为明显，有时也出现于呼气早期
	- 部位较恒定，性质不易变，中小水泡音可同时存在，咳嗽后可减轻或消失

|            |                                                                       |                                           |
| ---------- | --------------------------------------------------------------------- | ----------------------------------------- |
| 粗湿啰音（大水泡音） | 产生于气管，主支气管或空洞部位，多出现在吸气早期                                              | ![[Pasted image 20250922145101.png\|350]] |
| 中湿罗音（中水泡音） | 发生于中等大小的支气管，多见于吸气的中期                                                  | ![[Pasted image 20250922145144.png]]      |
| 细湿罗音（小水泡音） | 发生在细小支气管中，多在吸气后期出现                                                    | ![[Pasted image 20250922145207.png]]      |
| 捻发音        | 极细而均匀一致的湿啰音，颇似耳边 用手指捻发的声音。细支气管和肺泡壁因分泌 物存在而互相粘着陷闭，当吸气时被气流冲开 重新充气而产生的声音 | ![[Pasted image 20250922145226.png]]      |
###### 2. 干啰音
- 由于气管、支气管或细支气管*狭窄或部分阻塞*， 空气吸入或呼出时发生湍流所产生的声音![[Pasted image 20250922145338.png|500]]
- 特点：
	1. 持续时间较长
	2. 带乐音的呼吸附加音，音调较高
	3. 吸气及呼气均可闻及，尤以呼气时明显
	4. 强度和性质易改变，部位易改变
###### 3. 胸膜摩擦音
- 与心包摩擦音鉴别：屏住呼吸时声音消失，深呼吸时声 音增强。
- 与捻发音的鉴别：令病人掩鼻闭口并加强腹式运动，尽 管无气流进出气道，仍可闻及胸膜摩擦音。

# 呼吸系统常见疾病体征
#### 肺实变
- 定义：是指任何原因引起的以肺泡腔内积聚浆液、 纤维蛋白和细胞成分等，从而使肺泡含气 量减少、肺质地致密化的一种病变。肺体 积并不缩小，可不变或增大。
- 病因及发病机制
- 体征：

|     |                                  |
| --- | -------------------------------- |
| 视诊  | 胸廓对称、病侧呼吸运动减弱                    |
| 触诊  | 气管居中，病侧语音震颤增强                    |
| 叩诊  | 病变部位叩诊呈浊音。                       |
| 听诊  | 支气管呼吸音、湿罗音、语音共振 增强，累及胸膜者可闻及胸膜摩擦音 |
#### 肺气肿
- 定义：是指呼吸性细支气管远端（包括 肺泡管、肺泡囊和肺泡）过度膨胀、过度 充气和容积增大，往往不伴有明显纤维化。

|     |                                      |
| --- | ------------------------------------ |
| 视诊  | *桶状胸*、呼吸运动减弱、*肋间隙增宽*                 |
| 触诊  | 气管居中、*双侧*语音震颤减弱                      |
| 叩诊  | 两肺过清音，肺下界降低，肺下界 移动度减少，心浊音界缩小，肝浊音界下 移 |
| 听诊  | 肺泡呼吸音减弱、呼气延长、语音共振减弱，心音遥远             |

#### 肺不张
- 定义：是指肺充气减少，伴容积缩小的一 种病理改变。（区分肺实变：肋间隙变窄、胸廓局部塌陷）
- 阻塞性（主要）、压迫性

|     |                           |
| --- | ------------------------- |
| 视诊  | 病变部位*胸廓塌陷，肋间隙变窄*，呼吸运 动减弱  |
| 触诊  | *气管向患侧移位*。病变部位语音震颤减弱      |
| 叩诊  | 病变部位浊音或实音，*心脏向患侧移位*       |
| 听诊  | 病变部位呼吸音减弱或消失，*语音共振减 弱或消失* |
#### 胸腔积液
![[Pasted image 20250922152717.png|150]]

|     |                                 |
| --- | ------------------------------- |
| 视诊  | 患侧胸廓饱满、肋间隙增宽、呼吸 运动受限、心尖搏动向健侧移位。 |
| 触诊  | 气管移向健侧，患侧呼吸运动减弱， 语音震颤减弱或消失。     |
| 叩诊  | 积液区为浊音或实音                       |
| 听诊  | 积液区呼吸音减弱或消失，语音共 振减弱或消失。         |

#### 气胸
- 任何原因使胸膜破损，气体进入胸膜腔， 称为气胸。![[Pasted image 20250922152818.png|200]]

|     |                      |
| --- | -------------------- |
| 视诊  | 患侧胸廓饱满，肋间隙增宽，呼吸 运动减弱 |
| 触诊  | *气管向健侧*移位，*语音振颤消失*   |
| 叩诊  | 患侧呈*鼓音*              |
| 听诊  | 患侧呼吸音消失，*语音共振减弱/消失*  |
