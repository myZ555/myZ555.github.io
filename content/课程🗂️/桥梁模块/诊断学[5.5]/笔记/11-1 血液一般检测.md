# （一）红细胞数和血红蛋白浓度测定

|      | Hb(g/L) | RBC (×10^12/L) |
| ---- | ------- | -------------- |
| 成年男性 | 120-160 | 4.0-5.5        |
| 成年女性 | 110-150 | 3.5-5.0        |
| 新生儿  | 170-200 | 6.0-7.0        |
| 妊娠妇女 | >100    |                |

### 1. 红细胞及血红蛋白增多
##### 1. 相对性
- 由于各种原因（如剧烈呕吐、腹泻、大面积烧伤等）导致体液大量丢失，血液浓缩所致。
##### 2. 绝对性(红细胞增多症)
   1. 继发性：血中*红细胞生成素*增多。代偿性--慢性心肺疾病；非代偿性--某些肿瘤
   2. 原发性：==真性红细胞增多症 PV==，以红系增生为主的骨髓增生性疾病
### 2. 红细胞及血红蛋白减少(贫血)

   1. 生理性：婴幼儿、妊娠中晚期、部分老年人
   2. 病理性：
	   - 生成减少：造血干祖细胞异常、造血调节异常、造血物质缺乏或利用障碍
	   - 破坏增多：本身异常或外界因素导致溶血
	   - 意外丢失：急慢性失血
![500](Pasted%20image%2020251202141416.png)
### 3. 红细胞形态改变

- 正常形态：双凹圆盘状，直径6-9μm，中央淡染
- 周围血涂片检查
##### 1. 大小异常
- 小红细胞：<6μm
- 大红细胞：10-15μm
- 巨红细胞：>15μm
- 红细胞大小不均
##### 2. 形态异常

|                 | 特征                                | 见于                                                      |
| --------------- | --------------------------------- | ------------------------------------------------------- |
| **球形**          | 直径缩短，厚度增加，细胞中心区的蛋白比周围多，呈小球形       | 遗传形红细胞增多症、自身免疫性溶血性贫血、异常血红蛋白病（如Hb-S等）                    |
| **椭圆形**         | 呈椭圆形，横径缩短，长径增大，正常人椭圆形红细胞也可见(约占1%) | *遗传性椭圆形红细胞增多症*；<br>大红细胞性贫血、恶性贫血及严重缺铁性贫血、地中海贫血 及镰刀形贫血也可见 |
| **口形**          | 中心淡染区呈条状；正常<4%                    | 口细胞增多症；急性酒精中毒>5%                                        |
| **靶形**          | 红细胞中央色深，外周以苍白圈，在近红细胞边缘处又较深        | 黄疸、肝病、脾切除后，缺铁性贫血<br>*地中海贫血*                             |
| **镰状**          | *珠蛋白β链基因点突变引起*<br>GAG-谷氨酸→GTG-缬氨酸 | 镰状红细胞贫血                                                 |
| **泪滴形**         |                                   | 增生性贫血、骨髓纤维化、地中海贫血、脾功能亢进、肾病                              |
| **棘形**          | 带刺红细胞，刺呈针刺/尖刺状                    | 棘细胞增多症；肝病或制片不当                                          |
| **裂RBC**        | 红细胞碎片，正常人<2%                      | DIC，微血管病性溶血，心源性溶血性贫血等RBC破碎综合征；<br>化学中毒、肾衰、血栓性血小板减少性紫癜   |
| **红细胞异形症**      |                                   | HA， TTP                                                 |
| **红细胞缗(串)钱状形成** | 四个以上成熟红细胞如缗钱状堆叠                   | 高球蛋白血症，多发性骨髓瘤，巨球蛋白血症，感染/怀孕时的纤维蛋白原增高症                    |

##### 3. 染色异常
- 低色素性
- 高色素性
- 嗜多色性
##### 4. 细胞中出现异常结构
1. 嗜碱性点彩红细胞：RBC内有细小的蓝色点状物质；铅中毒筛查
2. 卡波氏环：成熟RBC内出现很细的淡紫红色线状体呈环形或8字形
3. 有核红细胞：新生儿正常，成年人病理。提示红细胞需求量、释放量明显增加或血髓屏障破坏
4. 何-乔小体/染色质小体（Howell-Jolly body）：RBC内圆形紫红色小体
# （二）白细胞检测
### 白细胞计数
- 外周血中白细胞总数
【参考值】
- WBC：4-10×10^9/L
- 分类计数（在WBC总数正常的前提下）
	- N：50-70%
	- L：20-40%
	- E：0.5-5%
	- B：0-1%
	- M：3-8%
- 正常**小儿**血象特点：
	- WBC分类变化：四六两交叉，七岁同成人![400](Pasted%20image%2020251202151627.png)
### 白细胞分类计数

#### 粒细胞
##### 中性粒细胞 N
![400](Pasted%20image%2020251202152242.png)

###### 1. 中性粒细胞增多
- 生理性：下午，饱餐，情绪激动，剧烈运动，高温，严寒，新生儿，月经期，妊娠5月以上
- 病理性：
	1. ==反应性==：应激反应，动员进入血循环。多为成熟的*分叶核*或较成熟的*杆状核*粒细胞
		1. 急性感染或炎症：化脓性球菌最明显、
		2. 广泛的组织损伤或坏死(12-36h内)
		3. 急性溶血
		4. 急性失血(2-5h)
		5. 急性中毒（外源性-生化；内源性-尿毒症，肝性脑病，糖尿病酮症酸中毒）
		6. 恶性肿瘤（*非造血系统*）（肿瘤组织坏死的分解产物刺激、粗转移破坏骨髓调控）
	2. ==异常增多==：造血干细胞克隆性疾病，主要是病理性粒细胞的*原始或幼稚细胞*大量增生
		1. 粒细胞白血病：急性/慢性
		2. 骨髓增殖性疾病：骨髓纤维化、原发性血小板增多症、真性红细胞增多症
###### 2. 中性粒细胞减少
- 粒细胞减少症：<1.5×10^9/L
- 粒细胞缺乏症：<0.5×10^9/L

1. 感染性疾病：病毒（流感/麻疹/病毒性肝炎）；细菌-*G-杆菌*（伤寒）
2. 血液系统疾病：再障、急性白血病、巨幼细胞贫血、骨髓增生异常综合征MDS、阵发性睡眠性血红蛋白尿PNH等
3. 物理化学因素
4. 单核-巨噬细胞系统功能亢进（脾亢、类脂质沉积病）
5. 自身免疫性疾病（SLE-自身抗体）
###### 3. 核象变化
==核象==：中性粒细胞的分叶状况，反映粒细胞的成熟程度
- 正常：外周血N核以分*3叶*居多，==杆状核:分叶核=1:13==
![500](Pasted%20image%2020251202153255.png)
- **核左移**：外周血中不分叶核粒细胞百分率超过5%
	- 常见于：*细菌性感染*，特别是急性化脓性感染、急性失血、急性中毒及急性溶血反应等
	- 细菌性感染相关的粒细胞==*类白血病反应*==(机体受某种刺激后出现的一种类似白血病的血象反应﹐表现为白细胞总数增高﹐周围血液中出现幼稚细胞)---极度核左移现象
- **核右移**：中性粒细胞核5叶以上者超过3%
	- 常伴有WBC减低
	- 缺乏造血物质、DNA减低、BM造血功能减退所致
	- 常见于：巨幼细胞贫血及造血功能减退，化疗后(应用抗代谢药物)
	- 炎症恢复期--一过性中性粒细胞核右移
	- 疾病进展期出现核右移--提示预后不良
###### 4. 形态异常
1. **中性粒细胞中毒性改变**
	1. *大小不均*：病程较长的化脓性炎症，细胞大小悬殊，内毒素作用下骨髓前期细胞发生顿挫不规则分裂所致。
	2. *中毒颗粒*：比正常中性颗粒粗大，大小不等，分布不均匀，染色较深，呈黑色或紫黑色。中毒颗粒为2～3个嗜天青颗粒的融合，可能为细胞摄取细菌或血浆蛋白后发生变性，也可能是自噬体。
	3. *空泡变性*：可为单个，常为多个。大小不等，亦可在核中出现。细胞脂肪变性或颗粒缺失。
	4. *杜勒小体*：球形包涵体；局部嗜碱性区域，界限不清，染成灰蓝色，直径约1～2μm。常见于严重感染。
	5. *核变性*：退行性变：胞体肿大、结构模糊、边缘不清晰、核固缩、核肿胀和 核溶解等。如胞质破裂后消失，只剩胞膜，则成裸核或篮状细胞。
2. **巨多分叶核中性粒细胞**
3. **与遗传相关的中性粒细胞形态异常**

==棒状小体==：紫红色细杆状物质，长约1～6μm，1 条或数条不 定。只出现在*非淋系急性白血病细胞中*，故见到棒状小体即可确诊为白血病。

##### 嗜酸性粒细胞 E

【生理功能】
	- 抑制嗜碱性粒细胞和肥大细胞合成与释放活性物质，吞噬其释出颗粒，并分泌组胺酶以破坏组胺，从而起到限制过敏反应的作用。
	- 胞核多为两叶，眼镜状
【参考值】0.5-5%
【临床意义】
###### 1. 嗜酸性粒细胞增多
1. 寄生虫病
2. 变态反应性疾病
3. 皮肤病
4. 血液病
5. 某些恶性肿瘤（某些上皮系如肺癌、部分淋巴瘤、多发性骨髓瘤）
6. 某些传染病（传染病急性期减少，恢复期增多）（*猩红热*-感染期增多）
7. 其他（风湿性疾病等）
###### 2. 嗜酸性粒细胞减少
- 伤寒、副伤寒初期
- 大手术、烧伤等应激状态
- 长期应用肾上腺皮质激素
- *临床意义不大*
##### 嗜碱性粒细胞 B

【生理功能】参与超敏反应
【参考值】0-1%
【临床意义】
###### 1. 嗜碱性粒细胞增多
1. 过敏性疾病
2. 血液病
3. 恶性肿瘤（转移癌）
4. 其他：糖尿病。传染病（水痘，天花，流感，结核病）
###### 2. 嗜碱性粒细胞减少
无临床意义

#### 淋巴细胞 L

【生理功能】
	- B淋巴细胞  15-30% 体液免疫
	- T淋巴细胞   50-70% 细胞免疫
	- NK细胞      15%       细胞介导细胞毒作用，分泌细胞因子
【参考值】20-40%
【形态】
	- 直径5-18μm，染色质致密块状，有*假核仁*；胞浆量少，蓝色，透亮。少许嗜天青颗粒
	- 大淋巴细胞-胞质丰富，天蓝色；小淋巴细胞-胞质很少，深蓝色
	- 异形淋巴细胞
【临床意义】
###### 1. 淋巴细胞增多
1. 生理性：儿童期（2个四六）
2. 病理性：
	1. 感染性疾病：*病毒感染*（风疹、流行性腮腺炎、传染性单核细胞增多症-EB）；*细菌感染*（百日咳、结核）
	2. 成熟淋巴细胞肿瘤
	3. *急性传染病恢复期*
	4. *移植排斥反应*
	5. 淋巴细胞比值相对增高的疾病：*再障*
###### 2. 淋巴细胞减少
1. 接触放射线
2. 应用肾上腺皮质激素或促肾上腺皮质激素
###### 3. 反应性淋巴细胞(异型淋巴细胞)
- 主要是CD8+的T淋巴细胞
	- ==Ⅰ型(泡沫型/浆细胞型)==
	- ==Ⅱ型(不规则型/单核细胞型)
	- ==Ⅲ型(幼稚型)==
![400](Pasted%20image%2020251202155615.png)
- 常见于：
	1. 感染性疾病（※：EB病毒-传染性单核细胞增多症核流行性出血热等）
	2. 药物过敏
	3. 输血、血液透析或体外循环
	4. 其他免疫性疾病、粒细胞缺乏症等

# （三）网织红细胞检测

> ==网织红细胞 Ret==：晚幼红细胞脱核后的红细胞阶段，胞质残存嗜碱性物质RNA等

【参考值】0.5-1.5%
【临床意义】反映骨髓造血功能的重要指标
1. 反映骨髓红系增生情况，间接反映骨髓造血功能
	1. 增高：溶血性贫血，急性失血等增生性贫血
	2. 减低：再生障碍性贫血，溶血性贫血再障危象等
2. 观察贫血疗效及确定诊断
	- 缺铁性贫血、巨幼红细胞性贫血治疗中，若明显增高--有效
# （四）血小板检测

【生理功能】粘附、聚集、释放--局部形成白色血栓机械填塞血管破损处--初步止血
【形态】
	呈圆盘状，直径约2-4um，生存期10天左右。星形、逗点状、不规则形， 直径2-5μm， 淡兰色，胞浆有淡紫色小颗粒，病态时有巨大血小板，直径 25-50μm（MDS、MF）
【参考值】100-300×10^9/L
【临床意义】
###### 1. 血小板减少
1. 生成障碍：造血功能障碍（再障，急性白血病，放射损伤）
2. 破坏过多：原发免疫性血小板减少症ITP，脾功能亢进
3. 消耗亢进：DIC
4. 分布异常：脾肿大、血液稀释
5. 假性血小板减少：EDTA抗凝
###### 2. 血小板增多
1. 原发性：骨髓增生性疾病
2. 反应性：急慢性炎症，急性失血/溶血，脾切除术后

# （五）红细胞沉降率测定

> ==红细胞沉降率 ESR==：简称血沉，指离体抗凝血静置后，红细胞在单位时间内沉降的速度
   影响因素：
	   1. 血浆中各种蛋白的比例
	   2. 红细胞数量和形状
	   3. 血沉管、管沉架、温度、标本放置时间等均可影响

【参考值】男性：0-15mm/第一小时末；女性：0-20mm/第一小时末
【临床意义】
1. 动态观察病情变化
2. 良恶性肿瘤鉴别的参考
3. 反映血浆中球蛋白增高
# （六）血小板比容测定和红细胞有关参数

> ==红细胞比容 Hct==：又称红细胞压积PCV，指一定量的抗凝全血经离心沉淀后，测得下沉的红细胞占全血的容积比。
> 	一种间接反映红细胞数量及体积的简单方法
> 	结合RBC计数和Hb含量可计算RBC平均值

【参考值】男：0.4-0.5；女：0.35-0.45
【临床意义】
###### 1. Hct增高
- 各种原因所致血液浓缩（腹泻、呕吐、大面积烧伤）
- 真性红细胞增多症
- 继发性红细胞增多症：体内氧供不足引起的代偿反应。新生儿、高原地区居住者、慢性心肺疾患
###### 2. Hct减少
- 贫血或血液稀释

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
