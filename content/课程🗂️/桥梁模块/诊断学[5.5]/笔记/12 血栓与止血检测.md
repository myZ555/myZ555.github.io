**血管壁**：
- 正常情况下，小血管的管壁结构包括：
	- 内膜层--内皮细胞、基底膜【内皮细胞表面光滑，带有负电荷(血小板表面负电荷)，防止血小板在其表面黏附聚集】【内皮细胞紧密连接完全覆盖在带正电荷的基底膜上】
	- 中层--平滑肌、弹力纤维、胶原
	- 外膜层--结缔组织
![Pasted image 20251201135301](Pasted%20image%2020251201135301.png)
**血小板**
- 圆盘状，直径2-4μm，电镜下分为三区：
	1. 外周区
	2. 溶胶-凝胶区
	3. 细胞器区
- 功能：黏附、聚集、释放。血管壁受损时，通过上述功能在局部形成白色血栓，机械性填塞血管破损处--初步止血【一期止血】
![Pasted image 20251201135312](Pasted%20image%2020251201135312.png)

**凝血系统**
- 凝血因子：1-13，无6.（6是5的激活状态）
- 除了4（钙离子）以外，均为蛋白质
- 1，2，5，7，9，10在肝内产生；2，7，9，10生成依赖VitK，故称维生素K依赖因子

## （一）血管壁检测
#### 1. 筛检试验
###### ★1. 出血时间 BT
- 参考值：Duke法：6.9±2.1min，> 9min为异常。
- 临床意义：（出血时间延长见于）
	1. 血小板明显减少（ITP）
	2. 血小板功能异常
	3. 严重缺乏某些凝血因子（VWD，DIC）
	4. 血管异常（遗传性出血性毛细血管扩张症）
	5. 药物影响
###### ★2. 束臂试验/毛细血管抵抗力试验 CRT
- 参考值：正常人<10个出血点（阴性）
- 临床意义
	1. 血管病变（结构和/或功能缺陷）
	2. 血小板数量和/或功能异常
	3. 血管性血友病（von Willebrand disease）
	4. 其他：高血压，糖尿病，败血症，维生素C缺乏症等
#### 2. 诊断试验
###### ★1. 血管性血友病因子抗原测定
- 参考值：Laurell免疫火箭电泳法：94.1%±32.5%；ELISA法：70%±150% 
- 临床意义：
	- ==vWF:Ag== 是血管内皮细胞的促凝指标之一。它由血管内皮细胞合成和分泌，参与血小板的黏附和聚集反应，起*促进凝血*的作用。
		- 减低：见于血管性血友病(vWD), 是诊断该病及其分型的指标之一。
###### ★2. 血管性血友病因子活性测定
- 参考值：O型血正常人为38-125.2%，其他血型为49.2-169.7%
- 临床意义（结合vWF、FⅧ:C 检测，主要用于vWD的分型诊断
	- 若vWF:Ag、vWF:A和FⅧ:C均正常：可排除血友病和vWD
	- 若vWF:Ag、vWF:A和FⅧ:C三项中有一项降低：计算彼此比值进一步明确诊断
###### 3. 6-酮-前列腺素F1α测定
###### 4. 血浆凝血酶调节蛋白抗原测定
## （二）血小板检测
#### 1. 筛检试验
###### ★1. 血小板计数 PC
- 参考值：(100-300)×10^9/L
- 临床意义：
	- 减少：
		1. 造血功能障碍
		2. 血小板破坏过多（ITP，脾功能亢进）
		3. 血小板消耗亢进（DIC）
	- 增多：
		1. 骨髓增生性疾病：CML、ET
		2. 反应性增多：急慢性炎症、急性失血或溶血、脾切除术后
###### ★2. 血块收缩试验 CRT
- 参考值：于凝固后2h开始退缩，于24h内退缩完全
- 临床意义：
	- 血小板减少
	- 血小板功能异常
	- 凝血因子异常
#### 2. 诊断试验
###### 1. 单克隆抗体血小板抗原固定试验 MAPIA
###### 2. 血小板黏附试验 PAdT
###### 3. 血小板聚集试验 PAgT
###### 4. 血小板P选择素测定
###### 5. 血小板促凝活性测定 PPA
###### 6. 血浆血栓烷B2测定
## （三）凝血因子检测
#### 1. 筛检试验
###### ★1. 活化的部分凝血活酶时间测定 APTT
- 参考值：30-45s，*正常对照组延长10s以上为异常*
- 临床意义：*内源性*凝血过程有无异常。延长见于： ^6d517c
	1. 先天性凝血因子异常
	2. 多种凝血因子缺乏（后天性）
	3. 循环抗凝血素增加
###### ★2. 凝血时间 CT
- 参考值：4-12min（试管法）
- 临床意义：主要反映内源性凝血第一期有无异常
	- 延长见于：
		1. 甲乙丙型血友病
		2. 凝血酶原、纤维蛋白原严重减少
		3. 应用肝素、双香豆素等抗凝药物
###### ★3. 血浆凝血酶原时间 PT（PPT）
- 参考值：16-18s，*较正常对照延长3s有意义*
- 临床意义：*外源性*凝血活性的综合性检查。延长见于：
	1. 先天性凝血因子异常（1，2，5，7，10）
	2. 后天性凝血因子异常（肝病、VitK缺乏、DIC）

**PT延长**：
- 严重肝脏损害、阻塞性黄疸
- 维生素K缺乏、纤维蛋白溶解亢进
- 应用华法林、双香豆素等抗凝药物
**PT缩短**：血液高凝状态（DIC早期、脑血栓形成）

==国际标准化比值 INR== = 患者凝血酶原时间/正常人平均凝血酶原时间
- 临床上：口服抗凝剂治疗INR为2-4是抗凝浓度的适当范围
- INR>4.5且纤维蛋白水平及血小板数目正常，提示抗凝过度

#### 2. 诊断试验
###### ★1. 血浆凝血因子8、9、11、12促凝活性测定
- 参考值：
	- FVIII:C 103%±25.7%
	- FIX:C 98.1%±30.4% 
	- FXI:C 100%±18.4%
	- FXII:C 92.4%±20.7%
- 临床意义：
	- 增高：血栓前状态、血栓性疾病
	- 减低：
		- FVIII:C减低：血友病A、血管性血友病、血中存在FVIII抗体
		- FIX:C减低：血友病B、肝脏病、维生素K缺乏症、DIC、口服抗凝药物等
###### ★2. 血浆因子2、5、7、10促凝活性测定
- 参考值：
	- FII:C 97.7%±16.7%
	- FV:C 102.4%±30.9%
	- FVII:C103%±17.3%
	- FX:C103%±19.0%
- 临床意义：
	- 增高：血栓前状态和血栓性疾病
	- 减低：先天性因子II、V、VII、X缺乏症、获得性见于肝病、DIC、口服抗凝剂， 维生素K缺乏症、新生儿出血症、肠道灭菌、吸收不良综合征
###### ★3. 血浆纤维蛋白原测定 Fg
- 参考值：2-4g/L （Clauss法）
- 增高：糖尿病、急性心梗、感染、血栓前状态等
- 减低：DIC、原发纤溶、重症肝炎、肝硬化
###### 4. 血浆因子13定性试验
###### 5. 可溶性纤维蛋白单体复合物测定
## （四）抗凝系统检测
#### 1. 病理性抗凝物质的筛检试验
###### ★1. 血浆凝血时间（凝血酶时间 TT）
- 参考值：正常人为16~18s，较正常对照延长3s以上有意义
- 临床意义：延长见于：
	1. 血循环中有抗凝物质增加（AT-III、肝素样物质、FDP）
	2. 纤维蛋白原明显减少或有结构异常
###### 2. 凝血酶时间的甲苯胺蓝纠正试验或血浆游离肝素时间
###### 3. APTT交叉纠正试验

###### 2. 抗心磷脂抗体测定
#### 3. 生理性抗凝因子检测
###### ★1. 血浆抗凝血酶活性测定（抗凝血酶III测定）
- 参考值：AT- Ⅲ活性：70%~130%；AT- Ⅲ的抗凝血酶活性（酶学法）：80%~120%
- 临床意义：
	- 减低：可导致血栓形成（先天性缺乏、后天性减低）
	- 增高：可导致出血
###### 2. 血浆蛋白C活性测定

## （五）纤溶活性检测
#### 1. 筛检试验
###### ★1. 血浆D-二聚体测定
- D二聚体是继发性纤溶的标志
- D二聚体增高是诊断DIC的辅助条件之一
###### ★2. 血浆纤维蛋白(原)降解产物测定 FDP测定
- 参考值：总FDP 血清0-8mg/L，尿液0-0.1mg/L
- 临床意义：
	- 总FDP增高是体内纤溶亢进的标志，但不能鉴别原发性纤溶和继发性纤溶
	- FDP增高主要见于容易引起DIC的基础疾患
###### 3. 优球蛋白溶解时间

###### 2. 血浆纤溶酶原活性测定

- 参考值：正常为阴性
- 临床意义：若阳性为*血管内纤维蛋白溶解*的标志

# 出血性疾病的检查要点：
- 出血性疾病的原因：
	1. 血管异常
	2. 血小板异常
	3. 凝血或纤溶异常
	4. 抗凝因子增多
- 出血性疾病的实验室检查要点：
	1. 过筛检查
	2. 特殊检查
==血栓与止血的筛选试验==
- **一期止血障碍：血小板和血管壁异常所致的止血障碍**
	- 毛细血管脆性试验（束臂实验）
	- 出血时间
	- 血小板计数
	- 血块退缩试验
- **二期止血障碍：凝血和抗凝血异常所致的止血障碍**
	- 凝血时间 CT
	- 凝血酶原时间 PT
	- 活性部分凝血活酶时间 APTT
	- 纤维蛋白原测定

// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===

    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));

            const li = document.createElement("li");
