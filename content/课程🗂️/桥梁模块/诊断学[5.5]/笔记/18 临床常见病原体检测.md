# 标本采集与处理

> 标本采集：
> 	- 应尽早在应用抗生素前采集
> 	- 应无菌操作及防止污染的原则下采集
> 标本运送：
> 	- 应尽快送至实验室
> 	- 相关法律法规要求进行运送处理（疑似1类病原体、危急症）
> 标本接收：
> 	- 信息齐全、有标签
> 	- 送检及时（细菌学-2h内；特殊标本立即送检
> 	- 送检容器完整、密封良好
> 	- 标本储存和运送方式满足检测要求
> 	- 标本量足够

#### 血液
- 采集指征：疑为菌血症、败血症和脓毒血症
- 采集时间：*发热初期、寒战时、发热高峰之前0.5-1h；应用抗生素之前*
- 采集方法：三步消毒；肘静脉穿刺采血；每次2-3次血培养；不同穿刺点
#### 尿液
- 采集指征：疑为尿路感染
- 采集方法：中段洁尿，最好采集晨尿；采集前清洗外阴；培养厌氧菌应采用耻骨上膀胱穿刺法收集尿液
#### 粪便
- 采集指征：疑为消化道感染
- 采集方法：取含脓/血/黏液的新鲜粪便1g；直肠拭子；感染性腹泻病人需非同一天送检3次粪便；感染性腹泻病人至少3次连续培养阴性才能出院
#### 呼吸道
- **支气管镜标本**：采集10-20ml支气管肺泡灌洗液(BALF)进行定量培养；支气管毛刷刷取标本应置于1ml生理盐水或乳酸钠林格液
- **痰液**：合格的痰液标本：*低倍镜下鳞状上皮细胞≤10个，白细胞≥25个*
- **鼻咽拭子**：去除鼻前孔内表面的分泌物，插入拭子至鼻咽部，到达后鼻咽部停留数秒吸取分泌物，置于转运培养基（用于病毒检测）
- **口咽拭子**：咽后壁及扁桃体隐窝、侧壁处反复擦拭3-5次，收集黏膜细胞，置于转运培养基（化脓性咽炎时查溶血性链球菌和溶血隐秘杆菌）
#### 体液
- **脑脊液**：
	- 腰椎穿刺术无菌操作；
	- 第3管用于核酸检测；
	- *最小标本量要求*：细菌≥1ml; 真菌/病毒≥2ml; 分枝杆菌≥5ml
	- CSF应立即保温送检或床边接种

- **胸/腹腔积液、心包液**
	- 无菌抽取，宜采集10ml标本送检
	- 标本可接种于血培养瓶
	- 腹膜透析液需采集≥50ml
# 常见病原体检测方法

> 初步诊断试验：
> 	- 直接染色镜检
> 	- 检测特异性抗原/病原体成分
> 	- 检测特异性IgG和/或IgM抗体（血清）
> 	- 检测病原体核酸
> 确定诊断试验：
> 	- 病原体分离培养、鉴定
> 	- 药敏试验


#### 直接显微镜检测

1. 涂片染色显微镜检查
	观察细菌形态、染色性或宿主细胞内包涵体特征
2. 涂片不染色显微镜检查
	观察病原菌生长、运动方式、螺旋体形态、真菌孢子和菌丝等
3. 荧光显微镜检查和免疫电镜检查
	标本经荧光染色后直接检查病原微生物如结核分枝杆菌、麻风分枝杆菌和白喉棒状杆菌（确诊价值）

#### 抗原检测

- 应用已知抗体检测病人血清或其他体液中存在的细菌、真菌等病原体抗原
- 简单快速、较高的敏感性，适用于多种感染性疾病的早期快速诊断
	- 细菌抗原：检测出特异性病原体抗原
	- 真菌抗原：用于辅助诊断
	- 病毒抗原：肝炎病毒抗原检测、人类免疫缺陷病毒p24抗原、CMV pp65抗原检测、呼吸道病毒抗原检测等。
	- 其他：依据立克次体、衣原体或活细胞内增殖的病毒的阳性结果可做出病原学诊断
#### 抗体检测
- 应用已知病原体的抗原检测病人血清中相应抗体的方法
- 用于现症诊断、疾病追溯性调查
	- 病程早期和晚期检测抗体，如抗体效价在病程4倍以上增长，提示现症感染
	- IgM可作为早期诊断、区分原发或复发性感染
- 抗体检测评价基于灵敏度、特异度和阴阳性预测值
#### 病原体核酸检测

- PCR技术
- 核酸探针杂交技术
- 实时荧光定量PCR技术（real-time PCR）
- 恒温扩增技术
- 基因芯片技术 （gene chip或DNA microarray）
- 数字PCR技术 （Digital PCR, dPCR）
- 核酸测序技术

#### 细菌毒素检测
- ==内毒素==：应用于革兰氏阴性菌感染的快速诊断
- ==外毒素==：
	 - 生物学方法：用于发现新的毒素时采用
	 - 免疫血清法：大量样品筛选
	 - 基因探针技术：检测单个菌落产生毒素的性质
	 - 自动化检测仪：通量高

#### 病原体分离培养和鉴定
- 细菌和真菌
	- 是感染性疾病诊断的重要依据，同时进行药敏试验
		- 菌种鉴定基本方法：观察菌落性状和菌体形态、染色性，选择合适的生化反应或飞行质谱进行菌种鉴定，检测白喉棒状杆菌毒力时需借助动物接种。
		- 质谱法：操作简单，可鉴定菌种范围广、准确率高、成本低。
- 其他病原体：
	- 病毒、衣原体等在人工培养基上不生长，需将标本接种于易感动物、鸡胚或合适的细胞。

# 病原体耐药性检测

==**耐药病原体**==：目前临床感染的致病菌以**G-杆菌为主**（~60%)，主要是大肠埃希菌、肺炎克雷伯菌、铜绿假单胞菌和鲍曼不动杆菌等。G+菌引起的感染约占30%，以葡萄球菌和肠球菌为主。

==**耐药机制**==：
- 对某种抗菌药物敏感的细菌变成对该药物耐受的变异称为*耐药性变异*
- 细菌耐药性的获得：细菌染色体耐药基因突变、耐药质粒的转移和转座子的插入，使细菌产生一些酶类或多肽类物质
	- 细菌水平和垂直传播耐药基因的整合子系统
	- 产生灭活抗生素的水解酶和钝化酶
	- 细菌抗生素作用靶点的改变
	- 细菌膜的改变和外排泵出系统
	- 细菌生物膜的形成

==**细菌耐药检测方法**==：
- 药物敏感试验
	- K-B纸片琼脂扩散法
	- 稀释法
	- 浓度梯度纸条扩散法
	- 耐药筛选试验
- 耐药基因产生：获得外源性基因、细菌自身基因突变
- 病原体耐药基因检测：
	- 可比药敏试验更早检测出病原菌的耐药性
	- 可作为考核其他耐药性检测方法的“金标准”
	

# 病毒性肝炎检测

> 7型：甲型(HA)、乙型(HB)、丙型(HC)、丁型(HD)、戊型(HE)、庚型 (HG)、输血传播病毒肝炎（TTV）
> 常用检测方法：
> 	- 针对抗原/抗体：酶联免疫法（ELA、ELISA）、化学发光法
> 	- 针对核酸：斑点杂交法、聚合酶链反应法（PCR）、实时荧光定量PCR

#### 甲型
- HBV属微小RNA病毒科，是一种无囊膜正20面体颗粒，直径27~32nm，内含一条线状单正股RNA 基因组，外由衣壳包封而成核壳体.
- 病毒标志物有**甲型肝炎病毒抗原HAVAg、抗体(IgM、IgA和IgG)及HAV-RNA**
【标本来源】非抗凝外周血，粪便、污染的水源或食物。
###### 抗原检测
【参考值】ELISA法检测血清HAV颗粒、RIA法或免疫电镜(IEM)检测粪便HAV颗粒为阴性
【临床意义】*HAVAg一般于发病前1~15天可从粪中排出*，发病第一周粪便的阳性率为42.9%，1~2周为18.3%，2周后消失，临床上不易捕捉到。*粪便中HAV或HAV抗原颗粒检测可作为甲肝急性感染的证据。*
###### 抗体检测
【参考值】HAV-IgM是病毒衣蛋白抗体，HAV-IgA是肠道黏膜分泌的局部抗体，HAV-IgG病愈后可长期存在
【临床意义】
1. 抗HAV-IgM 甲肝病人在发病后2周抗HAV-IgM的阳性率为100%，逐渐下降。阳性说明机体正在感染HAV，是早期诊断甲肝的特异性指标
2. 抗HAV-lgA 甲肝早期和急性期，由粪便中测得抗HAV-IgA阳性，是早期诊断甲肝的指标之一
3. 抗HAV-lgG 阳性出现于恢复期且持久存在，是获得免疫力的标志，提示既往感染，可作为流行病学调查指标
###### HAC-RNA测定
- 利用包被在PCR反应壁(微孔)上的HAV单克隆抗体，吸附样本中的HAV释放出病毒RNA，再进行RT-PCR, 可检测出样本中极微量的HAV。
【参考值】RT-PCR法为阴性
【临床意义】HAV-RNA阳性对诊断特别对早期诊断具有特异性。可检测粪便排毒情况和污染的水源与食物，有利于及时监测与预防甲性肝炎。可作基因分型研究。

#### 乙型
- HBV是一种嗜肝脱氧核糖核酸病毒，属于包膜病毒。HBV感染是我国肝硬化和肝细胞癌(hepatocellular carcinoma, HCC)的主要病因。
###### 乙肝六项检测
![[Pasted image 20251223084816.png|300]]
- 经典的病毒标志物包括血清学标志物:
	1. 乙型肝炎表面抗原(hepatitis B surface antigen, HBsAg) 及其抗体(抗 -HBs)
	2. 乙型肝炎e抗原(hepatitis B e antigen, HBeAg)及其抗体(抗-HBe)
	3. 乙型肝炎核心抗体(抗-HBc)
【临床意义】

| ----- | ------------------------------------------------------------------------------------------------------------- |
| HBsAg | **HBsAg 阳性表示HBV 感染**，其在感染后1~12 周出现，急性感染者持续存在5周~5月，若阳性超过6个月则为慢性感染，CHB和无症状携带者可持续存在多年甚至终身。                       |
| 抗-HBs | **抗-HBs为保护性抗体**，浓度≥10 mIU/ml 提示具备免疫力。接种疫苗产生的抗 -HBs对人群的保护可维持 30 年以上，一般人群不需要加强针 。 *约有 5%的感染者 HBsAg 和抗-HBs 同时阳性* |

###### 乙型肝炎病毒表面抗原蛋白前S1和前S1抗体测定

乙型肝炎病毒表面抗原蛋白前S1抗原位于病毒颗粒的表面，是乙肝病毒识别肝细胞表面特异性受体的主要成分，是**乙肝病毒复制和活动**的标志物。
【临床意义】
- 前S1抗原是非常重要的传染性指标，与病毒复制的关系密切。
- 较HBeAg敏感，避免由于HBeAg阴性造成的漏检。
- 前S1抗原阴转越早、前S1抗体阳转越早，病人病程越短、预后越好。
###### 乙型肝炎病毒表面抗原蛋白前S2和前S2抗体测定
乙型肝炎病毒表面抗原蛋白前S2(Pre-S2)是HBV表面蛋白成分，为HBV侵入肝细胞的主要结构成分；乙型肝炎病毒表面抗原蛋白前S2抗体(抗Pre-S2)是HBV的**中和抗体**。
【临床意义】
- Pre-S2 阳性提示HBV复制异常活跃，有传染性。
- 抗Pre-S2阳性见于乙肝急性期及恢复早期；提示HBV已被清除，预后较好。

###### 乙型肝炎病毒DNA测定
乙型肝炎病毒DNA(HBV-DNA)呈双股环形，是HBV的基因物质，也是乙型肝炎的直接诊断证据。HBVDNA 检测以**实时荧光定量 PCR 技术为主**。
【临床意义】
- **是病毒复制和具有传染性的直接标志**，也是抗病毒治疗适应证选择及疗效判断的最重要指标。
- 建议接受抗病毒治疗的CHB患者*每3~6个月检测1次 HBV DNA*
###### 乙型肝炎病毒YMDD变异测定
**YMDD(酪氨酸-蛋氨酸-天门冬氨酸-天门冬氨酸)位点是HBV反转录酶的活性部分**,属高度保守序列。在HBV的反转录过程中，YMDD位点中的YM能与模板核苷末端的糖基相作用，影响寡核苷酸与模板链的结合。
【临床意义】YMDD变异，与**拉米夫定(lamivudine)等耐药**相关。

#### 丙型
- 丙型肝炎病毒(hepatitis C virus,HCV)为**黄病毒属、单链正股RNA病毒**。其基因组为一线状正股RNA，全长9500bp，编码结构蛋白与核心蛋白。
- 临床上诊断HCV感染的主要标志物为**HCV-RNA、抗-HCV IgM和抗-HCV IgG**。
###### 丙型肝炎病毒RNA测定
【参考值】斑点杂交试验、RT-PCR法均为阴性。
【临床意义】**阳性提示HCV复制活跃，传染性强**；HCV-RNA转阴提示HCV复制受抑，预后较好。

###### 丙型肝炎病毒抗体测定
【临床意义】
1. **抗-HCV IgM抗体** 主要用于早期诊断，持续阳性常可作为转为慢性肝炎的指标，或是提示病毒持续存在并有复制。
2. **抗-HCV IgG抗体** 阳性表明已有HCV感染。输血后肝炎有80%~90%的病人抗-HCV IgG阳性。经常接受血制品(血浆、全血)治疗的病人可以合并HCV的感染，易使病变转为慢性、肝硬化或肝癌。

#### 丁、戊、庚、TTV

【丁型】
- *与乙肝重叠感染，易迅速发展成肝硬化或肝癌。*
- HDVAg出现早、持续时间短、以免疫复合物形式存在，难以检出。
- 抗-HDV IgG、抗-HDV IgM、HDV-RNA
【戊型】抗-HEV IgG、抗-HEV IgM、HEV-RNA
【庚型】抗-HGV、HGV-RNA、 TTV：TTV-DNA

            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}
