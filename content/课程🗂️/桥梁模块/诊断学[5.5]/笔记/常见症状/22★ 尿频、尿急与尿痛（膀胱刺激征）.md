#泌尿系统 
> [!NOTE] Title
> 掌握：概念与病因
> 熟悉：临床表现
> 了解：伴随症状

**尿频**：单位时间内排尿次数增多。正常：白天4-6次，夜间0-2次。
**尿急**：一有尿意迫不及待需要排尿，难以控制
**尿痛**：排尿时感觉耻骨上区、会阴部和尿道内疼痛或烧灼感

合称：**膀胱刺激征**

---
# 一、病因

1.尿频
	(1)生理性尿频 因饮水过多、精神紧张或气候寒冷时排尿次数增多,属正常现象。特点是每次尿量不少,也不伴随尿痛、尿急等其他症状。
	(2)病理性尿频
		1. 多尿性尿频：排尿次数增多而每次尿量不少,全日总尿量增多。*见于糖尿病、尿崩症、精神性多饮和急性肾功能衰竭的多尿期。*
		2. 炎症性尿频：尿频而每次尿量少,多伴有尿急和尿痛,尿液镜检可见炎性细胞。见于膀胱炎、尿道炎、前列腺炎和尿道旁腺炎等。
		3. 神经性尿频：尿频而每次尿量少，不伴尿急尿痛，尿液镜检无炎性细胞。见于中枢及周围神经病变,如癔症、神经源性膀胱
		4. 膀胱容量减少性尿频:表现为持续性尿频,药物治疗难以缓解,每次尿量少。见于膀胱占位性病变;妊娠子宫增大或卵巢囊肿等压迫膀胱;膀胱结核引起膀胱纤维性缩窄
		5. 尿道口周围病变:尿道口息肉,处女膜伞和尿道旁腺囊肿等刺激尿道口引起尿频
2. 尿急
	1. 炎症：急性膀胱炎、尿道炎,尤其是膀胱三角区和后尿道炎症,尿急症状特别明显;急性前列腺炎常有尿急,慢性前列腺炎因常伴有腺体增生肥大,故伴有排尿困难、尿线细和尿流中断等。
	2. 结石和异物
	3. 肿瘤（前列腺癌、膀胱癌）
	4. 神经源性（精神因素、神经源性膀胱）
	5. 高温环境下尿液高度浓缩，酸性高的尿可刺激膀胱/尿道黏膜产生尿急
3. 尿痛
	1. 引起尿急的原因几乎都可以引起尿痛
	2. 多于耻骨上区、会阴部和尿道内，尿痛性质可为灼痛或刺痛
	3. 尿道炎--排尿开始时疼痛
	4. 后尿道炎、膀胱炎、前列腺炎--终末性尿痛

# 四、伴随症状

|                     |                  |
| ------------------- | ---------------- |
| 尿频伴尿急+尿痛            | 膀胱炎、尿道炎          |
| 膀胱刺激征存在但不剧烈而伴有双侧腰痛  | 肾盂肾炎             |
| 伴会阴部、腹股沟和睾丸胀痛       | 急性前列腺炎           |
| 尿频+尿急伴血尿，午后低热，乏力，盗汗 | 膀胱结核             |
| 尿频伴多饮、多尿和口渴但不伴尿急/尿痛 | 精神性多饮、糖尿病、尿崩症    |
| 尿频+尿急伴无痛性血尿         | 膀胱癌              |
| 老年男性尿频伴尿线细、进行性排尿困难  | 前列腺增生            |
| 尿频、尿急、尿痛伴有尿流突然中断    | 膀胱结石堵住出口或后尿道结石嵌顿 |

# 五、问诊要点
- 起病情况 缓急、诱因(劳累、受凉、月经期)、时间。
- 主要症状特征 尿频、尿急程度,每次排尿量,尿痛部位和时间、发作频率。
- 伴随症状 泌尿系统和(或)局部症状,精神状态
- 既往史 慢性病史(结核、糖尿病、肾炎、尿路结石),尿路感染反复发作史(发作间隔、是否尿培养、细菌种类、药物使用史及疗程)
