#循环系统
> [!NOTE] 学习目标
> 掌握：心悸的概念、病因
> 熟悉：心悸的发生机制
> 了解：心悸的伴随症状

**概念**
- 心悸 palpitation 是一种自觉心脏跳动的不适感或心慌感，表现为心跳速度、力度或规律性的改变

**心脏工作原理**
- 心脏基本结构：4个腔室；瓣膜系统；血液单向流动
- 心脏跳动流程：窦房结--电信号--心房收缩--房室结--希氏束--心室--心室收缩--血液循环全身

**心悸的发生**
1. 心脏传导系统异常
2. 自主神经系统调控--交感神经兴奋时

**病因**
1. 心脏搏动增强
2. 心律失常
	1. 心动过速
	2. 心动过缓
	3. 其他
3. 心力衰竭
4. 心脏神经官能症
5. β-受体亢进综合征
6. 更年期综合征
7. 其他

**发生机制**
8. 血流动力学改变
9. 心律失常
10. 神经体液调节
11. 神经精神因素
---
**伴随症状**

|       |                               |
| ----- | ----------------------------- |
| 心前区疼痛 | 冠心病，心肌炎，心包炎，心脏神经官能症           |
| 发热    | 急性传染病，风湿热，心肌炎，心包炎，感染性心内膜炎     |
| 晕厥/抽搐 | 窦性停搏、高度房室传导阻滞、室性心动过速、病态窦房结综合征 |
| 贫血    | 急性失血、慢性贫血                     |
| 呼吸困难  | 急性心肌梗死、心包炎、心肌炎、心力衰竭、重症贫血      |
| 消瘦及出汗 | 甲状腺功能亢进症                      |
| 发绀    | 先天性心脏病、右心功能不全和休克              |
