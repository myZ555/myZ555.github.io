> 掌握：黄疸的概念、病因及鉴别诊断
> 熟悉：黄疸的发生机制
> 了解：黄疸的伴随症状

**黄疸 jaundice**：由于血清中胆红素升高致使皮肤、粘膜和巩膜发黄的症状和体征
- 正常胆红素最高为17.1μmol/l(1.0mg/d1)，其中结合胆红素 3.42μmol/l，非结合胆红素13.68μmol/l。
- 胆红素在17.1- 34.2μmol/l，临床不易察觉，称为隐性黄疸，**超过34.2μmol/l (2.0mg/d1)时出现黄疸**。

# 一、病因
1. 溶血性黄疸
2. 肝细胞性黄疸
3. 胆汁淤积性黄疸
4. 先天性非溶血性黄疸

==黄疸的分类==
- UCB升高为主：
	1. UCB生成过量
		1. 溶血
		2. 无效造血增多
	2. 肝细胞摄取的UCB减少
		1. 获得性（药物）
		2. 先天性
	3. UCB转化为CB 减少
		1. 新生儿黄疸
		2. 先天性
		3. 药物性
		4. 肝炎后性
- CB升高为主
	1. 肝内性胆汁淤积
	2. 肝外性胆道阻塞

# 二、发生机制
### 胆红素的正常代谢
1. 胆红素的来源
	1. 衰老红细胞（主要来源）【衰老RBC-脾脏-胆红素、铁和珠蛋白】
	2. 旁路胆红素
![[Pasted image 20251020165248.png]]
2. 胆红素在血液中的运输过程
	- 胆红素+白蛋白---胆红素-白蛋白复合物(非结合胆红素UCB)---运输至肝脏
3. 肝细胞对胆红素的摄取、结合和排泄
	1. 摄取：胆红素-白蛋白复合物-->白蛋白+胆红素，胆红素被肝细胞摄取
	2. 结合：UCB--CB 结合胆红素
	3. 排泄：CB从肝细胞—胆管—肠腔(经细菌脱氢作用)—尿胆原—大部分被氧化—粪胆原—随大便排出
4. 胆红素的肠肝循环
	- 小部分尿胆原肠内重吸收--肝门静脉回肝--再次转变为CB，随胆汁排入肠内


| 非结合性胆红素UCB       | 结合性胆红素CB             |
| ---------------- | -------------------- |
| 非极性              | 极性                   |
| 不溶于水，不能经尿排出      | 溶于水，可经尿排出            |
| 溶于脂质，可经过BBB和胎盘   | 不溶于脂质，不能经过BBB和胎盘     |
| 与血清蛋白可逆性结合，半衰期4h | 与血清蛋白结合较牢固，半衰期12-14d |

# 五、问诊要点
1. 起病经过（急慢性、病程长短、有无诱因、群体发病、外出旅游、药物使用情况、家族史）
2. 黄疸特点（波及范围、颜色特点、持续时间、波动情况）
3. 伴随症状
4. 诊疗经过（实验室检查-血胆红素/尿胆红素/尿胆原/肝功能、影像学检查）
