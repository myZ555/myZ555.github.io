#消化系统 
> [!NOTE] Title
> 掌握：临床表现及伴随症状
> 熟悉：病因
> 了解：发病机制

**便血 hematochezia**：消化道出血经肛门排出

---
# 一、病因
1. 下消化系统疾病
	1. 小肠疾病：小肠毛细血管畸形、麦克尔憩室、克罗恩病等
	2. 结肠疾病：结肠癌、溃疡性结肠炎、息肉出血、缺血性肠病
	3. 直肠肛管疾病：痔疮、肛裂、溃疡性直肠炎、直肠癌等
2. 上消化系统疾病
	- *各种引起上消化道出血的疾病均能诱发便血*
3. 全身性疾病
	1. 血小板减少等出血性疾病；Behcet等风湿性疾病；恶行组织细胞病；尿毒症性肠炎等
	2. 腹腔邻近脏器恶性肿瘤浸润或脓肿破裂侵入肠腔均可引起出血
4. 原因不明

# 三、临床表现
1. 便血多为下消化道出血，可表现为急性大出血、慢性少量出血及间歇性出血
2. 便血颜色因出血部位不同、出血量多少及血液在肠腔内停留时间长短而异
3. *隐血便*：消化道出血每日在5-10ml以内，无肉眼可见粪便颜色改变，隐血试验确定

| 性状           | 提示                                            |
| ------------ | --------------------------------------------- |
| 柏油样便         | 上消化道或小肠出血，血量较黑便多，肠中停留时间较长                     |
| 暗红色血便        | 下消化道出血，停留时间较长                                 |
| 鲜红色血便        | 下消化道出血，出血量较多                                  |
| 鲜血           | 不与粪便混合，仅黏附于粪便或于排便前后有鲜血滴出，提示肛门或肛管疾病，如痔、肛裂、直肠肿瘤 |
| 暗红色果酱样脓血便    | 阿米巴痢疾                                         |
| 粘液脓性鲜血便      | 急性细菌性痢疾                                       |
| 洗肉水样血便伴特殊腥臭味 | 急性出血性坏死性肠炎                                    |
| 米汤水样大便       | 霍乱                                            |

# 四、伴随症状

|               |                                                                                                                                            |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| 腹痛            | <1>慢性反复上腹痛，周期节律性，出血后疼痛减轻【消化性溃疡】<br><2>上腹绞痛或黄疸伴血便【肝/胆道出血】<br><3>腹痛时排血便或脓血便，便后疼痛减轻【细菌性痢疾/阿米巴痢疾/溃疡性结肠炎】<br><4>腹痛伴血便【急性出血坏死性肠炎，肠套叠，肠系膜血栓形成，膈疝】 |
| 里急后重 tenesmus | 肛门坠胀感；提示*肛门、直肠疾病*，见于*痢疾、直肠炎及直肠癌*                                                                                                           |
| 发热            | 急性传染病/恶性肿瘤<br>败血症、流行性出血热、钩端螺旋体病、肠道淋巴瘤、白血病等                                                                                                 |
| 全身出血倾向        | 急性传染性疾病及血液疾病<br>如重症肝炎、流行性出血热、白血病、过敏性紫癜、血友病等                                                                                                |
| 高血压、胸闷        | 缺血性肠炎                                                                                                                                      |
| 皮肤改变          | 蜘蛛痣、肝掌--肝硬化门脉高压；<br>成簇的毛细血管扩张--遗传性毛细血管扩张症等                                                                                                 |
| 腹部肿块          | 结肠癌、肠结核、肠道恶性淋巴瘤、肠套叠及克罗恩病等                                                                                                                  |

# 五、问诊要点
1. 是不是（鉴别诊断）
2. 多或少
3. 出不出
4. 什么原因

>==消化道出血病人要考虑的问题==
>*首先评估是否适合再问下去（生命体征、血常规、皮肤出血）*
>1. 造成出血的原因是什么？
>2. 出血相关的症状
	1. 是否是消化道来源？（排除口腔/鼻腔/咽喉来源，食物来源，咯血）
	2. 急性出血还是慢性出血？（出血多久？1天几次？有无血容量不足表现？-心悸/头晕/体力下降等）
	3. 出血量的大致判定
>3. 伴随 症状
>4. 病史

|      | 上消化道出血  | 下消化道出血         |
| ---- | ------- | -------------- |
| 出血前症 | 上腹不适、恶心 | 中下腹不适、便欲       |
| 出血方式 | 呕血、黑便   | 血便，无呕血         |
| 大便性状 | 黑便，没有血块 | 暗红色/鲜红色血便，常有血块 |
