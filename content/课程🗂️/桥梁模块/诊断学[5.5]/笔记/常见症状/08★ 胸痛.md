- 定义：胸痛主要由胸部疾病引起，少数由其他部位的病变所致。因痛閾的个体差异大，故 胸痛的程度与原发疾病的病情轻重并不完 全一致。

# 病因
1. 胸壁疾病：带状疱疹、肋间神经炎？*颈椎病和脊柱炎*
2. 心血管疾病：心绞痛？心肌梗塞？主动脉夹层？ **肺栓塞**？心肌炎？心包炎？
3. 呼吸系统疾病：肺炎？胸膜炎？气胸？肺癌？
4. 纵隔疾病：返流性食管炎？纵隔炎？纵隔气肿？ 纵隔肿瘤？ ^b96098
5. 其他（食管等）：腹部疾病：膈下脓肿？肝脓肿？……


# 发生机制

1. 各种刺激因子——> *胸部的感觉神经纤维产生痛觉冲动*——> 传至大脑皮层的痛觉中枢引起胸痛
	1. 肋间神经感觉纤维
	2. 支配*心脏和主动脉*的交感神经纤维
	3. 支配*气管与支气管*的迷走神经纤维
	4. 膈神经的感觉纤维
2. 放射痛或牵涉痛
	1. **放射痛/牵涉痛**：内脏病变与相应区域体表的传入神经进入脊髓同一节段并在后角发生联系；内脏的感觉冲动可直接激发脊髓体表感觉神经元，引起相应体表区域痛感

# 胸痛的特点问诊 PQRST
- 影响疼痛的因素：诱因，加重与缓解因素（P/Provocative-palliative factors）
- 性质：刀割样，烧灼痛，阵发性灼痛或刺痛，绞榨样痛，撕裂样疼痛，隐痛（Q/quality）
- 部位：胸壁疾病、心绞痛和心肌梗死、夹层动脉瘤、胸膜炎、食管及纵隔病变等（R/region）
- 程度（S/severity）
- 持续时间（T）

# 伴随症状

|                     |                         |
| ------------------- | ----------------------- |
| 吞咽困难、咽下痛            | 食管疾病（反流性食管炎）            |
| 咳嗽/咳痰/发热            | 气管-支气管-肺部疾病             |
| 咯血                  | 肺炎、肺栓塞、支气管肺癌            |
| 呼吸困难                | 较大范围病变（大叶性肺炎、自发性气胸、肺栓塞） |
| 脸色苍白、大汗、血压下降、休克、濒死感 | 心肌梗死、主动脉夹层水肿            |


#### 心源性疼痛

- 心肌梗死特征：持续性胸痛，疼痛剧烈且不易缓解，伴随出汗、恶心等
	- 提示心肌梗死可能性大：
		1. 放射痛
		2. 压榨感、濒死感
		3. 持续>30min
		4. 休息或含服硝酸甘油不缓解
		5. 伴有气急、大汗、虚弱、恶心、呕吐
- 心绞痛症状：常因体力活动诱发，短暂休息或服用硝酸甘油可缓解，常为压迫或紧缩感

#### 血管源性
- 主动脉夹层
	- 前胸突发撕裂样剧痛，同时有背部疼痛
	- 剧痛，难以忍受，持续性
- 肺栓塞
	- 呼吸困难伴胸痛
	- 心跳快，晕厥或咯血
	- 下肢静脉血栓及长期卧床病人

# 常见导致胸痛的疾病


### 胸膜炎
1. 深呼吸、咳嗽、喷嚏、躯干活动时加剧
2. 疼痛局限于病变部位（尖锐、定位准确）
3. 胸痛持续存在
4. 疼痛不能用硝酸酯缓解

### 心绞痛/心肌梗死

|      | 心绞痛         | 心肌梗死             |
| ---- | ----------- | ---------------- |
| 持续时间 | 5-15min     | >30min           |
| 发作频率 | 变化说明病情加剧    |                  |
| 诱因   | 劳力，天气，进食，自发 | 用力，寒冷，进食，自发      |
| 缓解因素 | 休息，硝酸甘油     | 镇痛药，溶栓药，PCI，CABG |
| 伴随症状 | 常无          | 显著乏力，出汗，呕吐，濒死感   |
- 疼痛部位并不确定，常位于胸骨后、心前区；可放射至左上臂，颈咽颌腭，牙齿，上腹部等
- 绝大多数都是由于冠心病引起的心肌缺血；老年病；男>55，女>65（雌激素可对抗动脉硬化，绝经前一般不会；吸烟/三高/家族史）
- **心绞痛**
	1. 劳力性心绞痛（典型）：胸骨后；压榨样；恶心呕吐、出汗；30min内，休息后缓解
	2. 自发性心绞痛：休息发生；持续时间长，不易缓解
- **心肌梗死**
	- 30min以上，休息或舌下含服硝酸甘油不能缓解
	- 心电图、血清酶谱、肌钙蛋白特征性的动态改变

### 主动脉夹层血肿
1. 突发剧痛，胸部/背部肩胛间，疼痛可沿脊柱下移
2. 严重休克貌（面色苍白、大汗淋漓），血压不低

### 肺栓塞

- 小的周围性肺血管栓塞或肺梗死可引起胸膜纤维性炎症，为胸 膜性疼痛，呼吸、咳嗽时加重
- 大血管栓塞引起肺动脉急性扩张及冠状动脉缺血，疼痛多位于 胸骨后，有夹板挤压感，似心绞痛
