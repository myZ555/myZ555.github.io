# 一、概念
- ==呼吸困难 dyspnea==
	- 主观症状：患者感到空气不足、呼吸费力
	- 客观体征：呼吸深度或频率改变，节律不规则，严重时可出现张口呼吸、鼻翼扇动、端坐呼吸、呼吸辅助肌参与呼吸运动

# 二、病因 
- 静息-呼吸顺畅？
	- 顺畅的气血通路
	- 通气、换气、V/Q比、血液运输、组织利用
1. **通气障碍**：中枢疾病、神经肌肉疾病、胸廓胸膜病、影响膈肌相关疾病、气道阻塞
2. **换气障碍**：肺实质疾病（肺部感染、肺水肿、肺不张、ARDS）、肺间质疾病
3. **心血管**：心衰、心瓣膜病、冠心病、心肌病、心包病、肺动脉栓塞、肺动脉高压
4. **血液病**
5. **中毒**
6. **神经精神病**

# 三、发生机制
1. 肺源性呼吸困难
	1. 吸气性：呼吸肌极度用力，胸腔负压增加【三凹征：胸骨上窝、锁骨上窝、肋间隙】*上呼吸道、较大气管阻塞*
	2. 呼气性：肺泡弹性减弱 and/or 小支气管痉挛或炎症。*下呼吸道阻塞*疾病。
	3. 混合性：肺/胸膜腔病变使呼吸面积减少--换气功能障碍
2. 心源性呼吸困难
	1. 左心衰：
		- 表现：混合型呼吸困难；端坐呼吸；急性-夜间阵发性呼吸困难
		- 机制：①肺淤血；②肺泡张力高；③肺泡弹性减退；④肺循环压力升高对呼吸中枢的反射性刺激
	2. 右心衰：
		- 机制：体循环淤血
3. 神经精神系统呼吸困难：颅内压增高抑制呼吸中枢
4. 血源性呼吸困难：RBC携氧减少，血氧含量降低
5. 中毒性呼吸困难
	1. 代谢性酸中毒：酸性代谢产物刺激外周化学感受器/直接刺激呼吸中枢。因各种*酸中毒*所致者多为深长规 则大呼吸（*Kussmaul呼吸*），频率或快或慢，据病 因不同呼出气可有尿（氨）味（见于尿毒症）、烂 苹果味（见于糖尿病酮症酸中毒）。
	2. 药物中毒：抑制呼吸中枢
	3. 化学毒物中毒：导致机体缺氧刺激或抑制呼吸中枢

# 四、临床表现
#### 肺源性

| 分类  | 发生机制                                   | 临床表现                                 |
| --- | -------------------------------------- | ------------------------------------ |
| 吸气性 | 呼吸肌极度用力，胸腔负压增加【上气道梗阻】                  | 吸气费力，“三凹征”，干咳及高调吸气性喉鸣【锁骨上窝，胸骨上窝，肋间隙】 |
| 呼气性 | 【下呼吸道阻塞 肺泡弹性减弱】肺泡弹性减弱 and/or 小支气管痉挛或炎症 | 呼气费力缓慢，呼吸时间延长，呼气期哮鸣音                 |
| 混合性 | 肺/胸膜腔病变使呼吸面积减少【换气功能障碍，肺部和胸膜病】          | 均感呼吸费力，呼吸频率增快，深度变浅，呼吸音异常             |

#### 心源性

##### 左心衰竭呼吸困难
- 机制
	1. 肺淤血：气体弥散功能↓，肺顺应性下降
	2. 肺泡张力↑：刺激感受器反射性兴奋呼吸中枢
	3. 肺循环压力↑：对呼吸中枢的反射性刺激
	4. 肺泡弹性↓：肺泡扩张，收缩能力↓，肺活量↓
- 表现
	1. 劳力性呼吸困难：活动时出现或加重， 休息时减轻或缓解
	2. 端坐呼吸：仰卧时加重，坐位时减轻
	3. 夜间阵发性呼吸困难：
		- 夜间睡眠中突感胸闷、憋气、 惊醒、被迫坐起、数分钟或 数十分钟缓解。重症伴有气 喘、发绀、粉红色泡沫痰， 称*心源性哮喘*。![[Pasted image 20250929144021.png]]
		- 机制：
			1. 迷走神经↑，冠状动脉收缩,心肌供血↓
			2. 回心血量↑,肺淤血加重 
			3. 膈肌上升,肺活量减少
			4. 中枢处于抑制状态，严重缺氧才能刺激中枢
##### 右心衰竭呼吸困难
- 机制：体循环淤血
	1. 左心房与上腔静脉压↑ 刺激压力感受器 反射性兴奋呼吸中枢
	2. 血氧含量↓,酸性产物堆积 刺激呼吸中枢
	3. 淤血性肝大、胸水、腹水 呼吸运动受限
- 常见：慢性肺心病、风湿性心脏病

#### 神经精神系统
1. 神经性：呼吸慢而深，呼吸节律改变（双吸气、呼吸遏制）
2. 精神性：呼吸快而浅，叹息样呼吸/出现手足搐搦

#### 血源性
1. 严重贫血,红细胞增多症, 输血反应,大出血休克,败血症 
2. 血氧含量降低,血压下降 
3. 刺激呼吸中枢 
4. 呼吸变快
呼吸浅，心率快

##### 中毒性

| 分类     | 发生机制                     | 临床表现              |
| ------ | ------------------------ | ----------------- |
| 代谢性酸中毒 | 酸性代谢产物刺激外周化学感受器/直接刺激呼吸中枢 | 酸中毒深大呼吸           |
| 药物中毒   | 抑制呼吸中枢                   | 呼吸缓慢变浅，伴有呼吸节律异常改变 |
| 化学毒物中毒 | 导致机体缺氧刺激或抑制呼吸中枢          | 呼吸缓慢变浅，伴有呼吸节律异常改变 |

# 五、伴随症状

|             |                                      |
| ----------- | ------------------------------------ |
| 发作性呼吸困难伴哮鸣音 | 支气管哮喘，心源性哮喘                          |
| 伴发热         | 肺炎、肺脓肿、肺结核、胸膜炎、急性心包炎                 |
| 伴一侧胸痛       | 大叶性肺炎、胸膜炎、肺栓塞、自发性气胸、急性心肌梗死、支气管肺癌     |
| 伴咳嗽咳痰       | 慢阻肺、肺炎、支扩、肺脓肿、有机磷农药中毒、急性左心衰          |
| 伴意识障碍       | 脑出血，脑膜炎，糖尿病酮症酸中毒，尿毒症，肺性脑病，急性中毒，休克型肺炎 |

# 六、问诊要点
1. 呼吸困难的特点
	1. 患者年龄性别
		1. 儿童：气道异物，哮喘，先心
		2. 老年：慢阻肺，心衰，肿瘤
		3. 孕妇：哮喘，心衰，产后羊水栓塞
	2. 呼吸困难诱因：活动后；剧烈咳嗽后；长期卧床后；吸入有毒气体；输液后；精神刺激
	3. 呼吸困难特点
		1. 吸入性--主气道狭窄
		2. 呼气性--慢阻肺及哮喘
		3. 混合性
	4. 呼吸困难与体位的关系
2. 伴随症状
3. 既往史
	1. 呼吸道感染，肺部肿瘤，心脏疾病，精神神经类疾病，血液系统疾病
	2. 手术：胸部手术史，气管镜手术史（气胸），放疗史
	3. 用药史：输液史（肺水肿），药物过敏（药物过敏（喉水肿或哮喘发作）
