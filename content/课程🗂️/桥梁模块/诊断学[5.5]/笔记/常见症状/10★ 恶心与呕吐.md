#消化系统 
> [!NOTE] Title
> 掌握：呕吐病因、临床表现及伴随症状
> 熟悉：呕吐的发生机制及神经反射通路
> 了解：恶心/干呕/呕吐的区别

**恶心 nausea**：上腹部不适和紧迫欲吐的感觉，可伴有迷走兴奋症状（皮肤苍白、出汗、流涎、血压降低及心动过缓等）
**呕吐 vomiting**：通过胃的强烈收缩迫使*胃或部分小肠内容物*经食管、口腔而排出体外

---
# 一、病因
1. 反射性呕吐
	1. 咽部刺激（吸烟、剧咳、鼻咽部炎症）
	2. 胃、十二指肠疾病（胃炎、消化性溃疡……）
	3. 肠道疾病（急性阑尾炎、各型肠梗阻……）
	4. 肝胆胰疾病（急性肝炎、肝硬化、急慢性胆囊炎……）
	5. 腹膜及肠系膜疾病
	6. 其他
2. 中枢性呕吐
	1. 神经系统疾病（颅内感染、脑血管疾病、颅脑损伤、癫痫-持续状态）
	2. 全身性疾病（尿毒症、肝昏迷、糖尿病酮症酸中毒、甲亢、肾上腺皮质功能不全）
	3. 药物
	4. 中毒
	5. 精神因素
3. 前庭障碍性呕吐
	- 凡呕吐伴有听力障碍 、 眩晕等症状者，需考虑前庭障碍性呕吐 。 
	- 常见疾病有迷路炎，是化脓性中耳炎的常见并发症；梅尼埃病，为突发性的旋转性眩 晕 伴恶 心 呕吐； 晕 动 病，一般在航空、乘船和乘车时发生 。
# 二、发生机制

> ==呕吐过程：恶心、干呕vomiturition、呕吐==
> 1. 恶心：胃张力和蠕动减弱，十二指肠增强，伴或不伴有十二指肠液反流
> 2. 干呕：胃上部放松而胃窦部短暂收缩
> 3. 呕吐：胃窦部持续收缩，贲门开放，腹肌收缩，腹压增加--胃内容物急速猛烈向上反流
> 
> ==呕吐中枢：延髓==
![[Pasted image 20251017111206.png|400]]
- 呕吐中枢：接受来自消化道、大脑皮质、内耳前庭、冠状动脉以及化学感受器触发带的冲动，直接支配呕吐动作
- **化学感受器触发带**：接受各种外来的化学物质或药物，与内生代谢产物的刺激，发出神经冲动，传至呕吐中枢引起呕吐



# 三、临床表现
1. 呕吐时间：
	1. 晨吐：
		1. 育龄妇女晨：早期妊娠；尿毒症、慢性酒精中毒、功能性消化不良
		2. 鼻窦炎病人：（脓液流出刺激咽部）
	2. 晚上/夜间：幽门梗阻
2. 呕吐与进食关系
	1. 进食过程中/餐后——即刻呕吐——幽门管溃疡/精神性呕吐
	2. 餐后1小时以上——延迟性呕吐——胃张力下降/胃排空延迟
	3. 餐后较久/数餐后——幽门梗阻——呕吐后可有隔夜宿食
	4. 餐后近期呕吐（集体发病）——食物中毒
3. 呕吐特点
	1. 【神经官能性呕吐】进食后立刻呕吐、恶心感轻/缺如，餐后又可进食，长期反复发作而营养状况不受影响
	2. 【喷射状呕吐】--颅内高压性疾病（CNS疾病/脑膜炎，不伴恶心或恶心很轻）
	3. 【反射性】先恶心，胃排空后仍呕
	4. 【中枢性】呈喷射状，剧烈、无恶心，伴有头痛和不同 程度意识障碍
4. 呕吐物性质
	1. 发酵、腐败气味【胃潴留】
	2. 粪臭味【低位小肠梗阻】
	3. 不含胆汁/含多量胆汁【梗阻平面在十二指肠乳头以上/以下】
	4. 含大量酸性液体【胃泌素瘤/十二指肠溃疡】
	5. 无酸味【贲门狭窄/贲门失迟缓症】
	6. 咖啡色样呕吐物【上消化道出血】
# 四、伴随症状

|                   |                                   |
| ----------------- | --------------------------------- |
| 腹痛、腹泻             | 急性胃肠炎、霍乱、副霍乱、细菌性食物中毒、其他原因引起急性食物中毒 |
| 右上腹痛及*发热*、寒战或有黄疸  | 【炎症-上消化道】急性胆囊炎、胆石症                |
| 头痛及喷射性呕吐          | 颅内高压症、青光眼                         |
| 眩晕、眼球震颤           | 前庭器官疾病                            |
| 应用阿司匹林、某些抗生素及抗癌药物 | 药物副作用                             |
| 已婚育龄妇女晨吐          | 早孕                                |

# 五、问诊要点
1. 起病经过：急慢性过程
2. 呕吐特点：发生时间、呕吐与进食关系、呕吐物性质
3. 伴随症状：腹痛、腹泻、发热、黄疸、吞咽困难
4. 既往史：慢性肝肾疾病、糖尿病、肿瘤；用药史；孕龄期女性患者关注月经史
