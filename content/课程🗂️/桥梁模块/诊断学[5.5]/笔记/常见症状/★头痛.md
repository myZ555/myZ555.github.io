**头痛 headache**：眉弓、耳廓上部、枕外隆凸连线以上部分的疼痛

---
# 一、病因
- 原发性头痛
- 继发性头痛【危险！】
	1. 颅脑病变
		1. *感染*：细菌/真菌/病毒/寄生虫等导致脑膜炎、脑膜脑炎、脑炎、脑脓肿等
		2. *血管病变*：蛛网膜下腔出血、脑出血、脑血栓形成、脑栓塞、高血压脑病、脑供血不足、脑血管畸形等
		3. 占位性病变：脑肿瘤、颅内转移瘤……
		4. 颅脑外伤：脑震荡、脑挫伤、硬膜下血肿、颅内血肿、脑外伤后遗症等
		5. 其他：腰椎穿刺后及腰椎麻醉后头痛
	2. 颅外病变
		1. 颅骨疾病：颅底凹陷症、颅骨肿瘤
		2. 颈部疾病：颈椎病及其他颈部疾病
		3. 神经痛：三叉神经、舌咽神经及枕神经痛
		4. 其他：眼、耳、鼻、齿等疾病（青光眼、牙痛、鼻窦炎）
	3. 全身性疾病
		1. 急性感染：流感、伤寒、肺炎等发热性疾病
		2. 心血管疾病：高血压病、心力衰竭
		3. 中毒：铅、酒精、一氧化碳、有机磷、药物中毒
		4. 其他：尿毒症、低血糖、贫血、肺性脑病、系统性红斑狼疮、中暑
	4. 精神心理因素
# 二、发生机制
- 颅外各层组织及毗邻组织对痛觉均敏感,颅内组织对痛觉敏感只限于一部分血管及软、硬脑膜
- 传导颅内外痛觉的神经主要是三叉神经、面神经、舌咽神经、迷走神经以及颈1~3神经
- 颅内外的痛敏结构受到各种病变损害时,可引起头痛的发生。

1. *血管因素*：各种原因引起的颅内外血管的收缩、扩张以及血管受牵引或伸展均可导致头痛。
2. *脑膜受刺激或牵拉*：颅内炎症或出血刺激脑膜,或因发生脑水肿而牵拉脑膜引起头痛。
3. *神经因素*：传导痛觉的脑神经和颈神经被刺激、挤压或牵拉均可引起头痛。
4. *肌肉因素*：头、颈部肌肉的收缩也可引起头痛。
5. *牵涉性因素*：眼、耳、鼻、鼻窦及牙齿等病变的疼痛,可扩散或反射到头部而引起疼痛。
6. *神经功能因素*：见于神经症和精神疾患

# 三、临床表现
1. 发病情况
	1. 急性起病+发热：感染性疾病
	2. 急剧头痛持续不减+意识障碍+无发热：颅内血管性疾病(e.g., 蛛网膜下腔出血)
	3. 长期反复发作性头痛：偏头痛、紧张型头痛、丛集性头痛
	4. 慢性进行性头痛+颅内压增高(呕吐、缓脉、视神经乳头水肿)：颅内占位性病变
2. 头痛部位
3. 头痛程度与性质
4. 出现时间和持续时间
5. 加重减轻因素
# 四、伴随症状

| 症状            | 常见疾病             |
| ------------- | ---------------- |
| 剧烈呕吐          | 颅内压增高；若呕吐后减轻-偏头痛 |
| 眩晕            | 小脑肿瘤，椎基底动脉夹层     |
| 发热            | 感染性疾病，颅内/全身性感染   |
| 慢性进行性头痛+精神症状  | 颅内肿瘤             |
| 慢性头痛突然加剧+意识障碍 | 脑疝               |
| 视力障碍          | 青光眼、脑肿瘤          |
| 脑膜刺激征         | 脑膜炎、蛛网膜下腔出血      |
| 癫痫发作          | 脑血管畸形、脑内寄生虫病、脑肿瘤 |

# 五、问诊要点
1. 头痛的特点
	1. 部位：单侧/双侧/前额/眼眶/枕部/全头部/部位不定
	2. 起病方式及病程：急性伴发热/慢性进行性加重/复发性
	3. 性质及程度：针刺样/搏动性/炸裂样
	4. 加重缓解因素：头位相关/经期相关
2. 伴随症状
3. 既往史、家族史
