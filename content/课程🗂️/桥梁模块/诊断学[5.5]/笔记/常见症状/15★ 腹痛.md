#消化系统 
> [!NOTE] Title
> 掌握：腹痛的概念、病因
> 熟悉：腹痛的发病机制
> 了解：腹痛的临床表现及伴随症状

**腹痛 abdominal pain**：多数由腹部脏器引起，腹腔外疾病及全身性疾病也可引起

---
# 一、病因
###### 1. 急性腹痛
1. 腹腔器官急性炎症
2. 空腔脏器阻塞或扩张
3. 脏器扭转或破裂
4. 腹膜炎症
5. 腹腔内血管阻塞
6. 腹壁疾病
7. 胸腔疾病所致的腹部牵涉性痛
8. 全身性疾病所致腹痛
###### 2. 慢性腹痛
1. 腹腔脏器慢性炎症
2. 消化道运动障碍
3. 胃、十二指肠溃疡
4. 腹腔脏器扭转或梗阻
5. 脏器包膜的牵张
6. 中毒与代谢障碍
7. 肿瘤压迫及浸润
###### 3. 急腹症

# 二、发生机制

| 内脏痛                          | 躯体痛                                              | 牵涉痛                               |
| ---------------------------- | ------------------------------------------------ | --------------------------------- |
| 定位不准确                        | 定位准确，锐痛                                          | 不同脏器不同牵涉部位                        |
| 疼痛模糊；钝痛、闷痛、不适感               | 疼痛剧烈而持续                                          | 程度剧烈，局部有压痛、肌紧张、感觉过敏               |
| 伴随植物神经症状（恶心/呕吐/出汗）           |                                                  |                                   |
| 痛觉信号来自腹腔脏器，经交感神经传入脊髓<br>有牵涉痛 | 痛觉信号来自腹膜壁层及腹壁，经体神经传至脊神经根，反映到相应脊髓节段所支配的皮肤<br>无牵涉痛 | 痛觉信号来自腹腔脏器，经内脏神经传入，影响相应脊髓节段而定位于体表 |


###### 1. 内脏性腹痛
- 腹内某一器官的痛觉信号由交感神经传入脊髓引起
- 特点：
	1. 疼痛部位不确切
	2. 疼痛感觉模糊
	3. 常伴恶心、呕吐、出汗等其他自主神经兴奋症状，如肝脏疼痛
###### 2. 躯体性腹痛
- 由来自腹膜壁层及腹壁的痛觉信号，经体神经传至脊神经根，反映到相应脊髓节段所支配的皮肤所引起
- 特点：
	1. 定位准确，可在腹部一侧
	2. 程度剧烈而持续
	3. 可有局部腹肌强直
	4. 腹痛可因咳嗽、体位变化而加重，如肋软骨炎
###### 3. 牵涉痛
- 内脏性疼痛牵涉到身体体表部位，即内脏痛觉信号传至相应脊髓节段，引起该节段支配的体表部位疼痛
- 特点：
	1. 定位明确
	2. 疼痛剧烈
	3. 有压痛、肌紧张及感觉过敏等，如胆囊炎

| 脏器   | 牵涉痛部位    |
| ---- | -------- |
| 胆囊   | 右肩       |
| 胰腺   | 左肩       |
| 食管   | 肩部及后背    |
| 胃    | 左肩       |
| 十二指肠 | 右肩下部     |
| 阑尾   | 脐周       |
| 输尿管  | 腹股沟      |
| 心脏   | 左肩，左臂，颈部 |

# 三、临床表现
###### 1. 疼痛部位
1. ==中上腹部==：【食管】疾病、【胃、十二指肠和胰腺】疾病、
2. ==右上腹部==：【肝脏、胆囊】胆囊炎、胆石症、肝脓肿
3. ==右下腹==McBurney点：急性阑尾炎疼痛
4. 脐部/脐周：小肠疾病
5. ==左下/下腹部==：【结肠】疾病
6. 下腹部：膀胱炎、盆腔炎及异位妊娠破裂
7. 弥漫性或部位不定：急性弥漫性腹膜炎、机械性肠梗阻、急性出血坏死性肠炎、血卟啉病、铅中毒、腹型过敏性紫癜等
###### 2. 诱发因素
1. 胆囊炎/胆石症：进油腻食物史
2. 急性胰腺炎：酗酒和/或暴饮暴食史
3. 机械性肠梗阻：腹部手术
4. 腹部受暴力作用引起的剧痛并伴有休克：肝脾破裂
###### 3. 腹痛性质和程度

| 腹痛特点                | 主要病因       |
| ------------------- | ---------- |
| 突发中上腹刀割样疼痛或烧灼样疼痛    | 胃十二指肠溃疡穿孔  |
| 中上腹持续性隐痛            | 慢性胃炎、消化性溃疡 |
| 上腹持续性钝痛或刀割样疼痛并阵发性加剧 | 急性胰腺炎      |
| 持续及广泛性的剧烈腹痛伴腹肌紧张    | 急性弥散性腹膜炎   |
| 阵发性绞痛               | 胆结石或泌尿系结石  |
| 剑突下阵发性钻顶样疼痛         | 胆道蛔虫症      |

**==三种绞痛鉴别表==**

| 疼痛类别 | 疼痛部位                    | 其他特点                   |
| ---- | ----------------------- | ---------------------- |
| 肠绞痛  | 多位于脐周、下腹部               | 常伴有恶心、呕吐、腹泻、便秘、肠鸣音增强等  |
| 胆绞痛  | 位于右上腹，放射至右背与右肩胛         | 常有黄疸、发热、肝可触及或Murphy征阳性 |
| 肾绞痛  | 位于腰部并向下放射至腹股沟、外生殖器及大腿内侧 | 常有尿频、尿急、尿含蛋白质、红细胞等     |
###### 4. 发作时间
1. 餐后疼痛：胆胰疾病、胃部肿瘤或消化不良
2. 周期性、节律性上腹痛：胃、十二指肠溃疡
3. 子宫内膜异位症：月经来潮
4. 卵泡破裂：月经间期
###### 5. 与体位关系
1. 胃粘膜脱垂：左侧卧位缓解
2. 十二指肠壅滞症：膝胸位或俯卧位缓解
3. 胰腺癌：仰卧位疼痛明显，前倾/俯卧位减轻
4. 反流性食管炎：烧灼痛在躯体前屈明显，直立位减轻

# 四、伴随症状

|         |                                                                   |
| ------- | ----------------------------------------------------------------- |
| 发热、寒战   | 提示炎症，见于急性胆道感染、胆囊炎、肝脓肿、腹腔脓肿，也可见于腹腔外感染性疾病                           |
| 黄疸      | 肝胆胰疾病；机型溶血性贫血也可出现腹痛与黄疸                                            |
| 休克同时有贫血 | 腹腔脏器破裂；无贫血者见于胃肠穿孔、绞窄性肠梗阻、肠扭转、急性出血坏死性胰腺炎等。腹腔外疾病如心肌梗死、大叶性肺炎也可有腹痛与休克 |
| 呕吐反酸    | 食管胃肠病变；呕吐量大提示胃肠道梗阻；伴泛酸嗳气提示胃、十二指肠溃疡或胃炎                             |
| 腹泻      | 消化吸收障碍或肠道炎症、溃疡或肿瘤                                                 |
| 血尿      | 泌尿系疾病如结石                                                          |

# 五、问诊要点
1. 腹痛的诱因（进食-饮酒、油腻、暴饮暴食……）
2. 腹痛特点（起病缓急、部位、性质、程度、发作时间、体位；排查外科急腹症）
	1. 什么样的疼痛？剧痛还是钝痛？
	2. 什么部位？（剑突下、上腹部、下腹部）
	3. 是否放射；放射到肩/背/腰/
	4. 与饮食有无关系
	5. 进食后疼痛能否减轻
	6. 食物种类对疼痛的影响
3. 伴随症状
4. 既往史及女性月经史
