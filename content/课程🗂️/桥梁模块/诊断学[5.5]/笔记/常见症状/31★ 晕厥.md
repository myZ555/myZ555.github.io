> [!NOTE] 学习目标
> 掌握：晕厥的概念、病因和临床表现





**晕厥 Syncope**：指一过性、广泛性脑供血不足 所致短暂的意识丧失状态；突然发作、倒地、迅速恢复、 很少后遗症

# 一、病因

1. 血管舒缩障碍：单纯性晕厥、体位性低血压、颈动脉窦综合征、排尿性晕厥、咳嗽性晕厥、疼痛性晕厥
2. 心源性晕厥：心脏结构、节律及收缩力改变，心排血量突然减少或心脏停搏，导致 脑组织缺氧而发生晕厥
3. 脑源性晕厥：脑部血管或主要供应脑部血液的血管发生循环障碍，导致一时性、广泛性脑供血不足所致
4. 血液成分异常：低血糖、通气过度综合征、哭泣性晕厥、重度贫血、高原晕厥

# 二、发生机制和临床表现
- 主要临床表现：短暂意识丧失，时间一般为数秒
1. 血管舒缩障碍
2. 心源性晕厥
3. 脑源性晕厥
4. 血液成分异常

# 三、伴随症状


|                |                |
| -------------- | -------------- |
| 伴明显的自主神经功能障碍   | 血管抑制性晕厥        |
| 伴面色苍白、发绀、呼吸困难  | 急性左心衰竭         |
| 伴心率和心律明显改变     | 心源性晕厥          |
| 伴抽搐            | 中枢神经系统疾病和心源性晕厥 |
| 伴头痛、呕吐、视听障碍    | 中枢神经系统疾病       |
| 伴发热、水肿、杵状指     | 心肺疾病           |
| 伴呼吸深而快、手足发麻、抽搐 | 通气过度综合征、癔症     |
| 伴心悸、乏力、出汗、饥饿感  | 低血糖性晕厥         |
