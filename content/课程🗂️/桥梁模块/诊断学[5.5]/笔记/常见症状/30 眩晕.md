**眩晕 vertigo**：病人感到自身或周围环境物体旋转或摇动的一种*主观感觉障碍*，常伴有*客观的平衡障碍*，一般无意识障碍
1. 前庭系统性眩晕：由前庭功能障碍引起，包括外周前庭与中枢前庭【真性眩晕】
2. 非前庭系统性眩晕：由全身疾病引起【一般性眩晕】

# 一、病因
1. 周围性眩晕（耳性眩晕）
	1. 梅尼埃Meniere病：机制不明确。内耳淋巴代谢失调，淋巴液分泌过多或吸收障碍引起内耳膜迷路积水所致。
	2. 迷路炎：中耳病变破坏迷路的骨壁引起
	3. 前庭神经元炎：前庭神经病毒感染
	4. 药物中毒：药物引起内耳前庭或耳蜗受损
	5. 位置性眩晕：椭圆囊耳石脱落至半规管引起
	6. 晕动病：乘坐车/船/飞机时，内耳迷路受刺激引起前庭功能紊乱
2. 中枢性眩晕（脑性眩晕）
	1. 颅内血管性疾病：动脉粥样硬化/椎基底动脉狭窄/锁骨下动脉盗血
	2. 颅内占位性病变：听神经瘤/小脑肿瘤
	3. 颅内感染性疾病：后颅窝感染，结核/隐球菌/梅毒/寄生虫
	4. 颅内脱髓鞘疾病及变性疾病
	5. 癫痫
	6. 其他：脑外伤、前庭性偏头痛
3. 全身疾病性眩晕
	1. 心血管疾病：高血压/低血压/心律失常/心瓣膜病
	2. 血液病：各原因所致贫血/出血
	3. 中毒性疾病：尿毒症/重症肝炎/糖尿病酮症/发热感染
4. 眼源性眩晕
	1. 眼病：屈光不正/眼肌麻痹
	2. 屏幕性眩晕
5. 神经精神性疾病

# 四、伴随症状

| 伴随症状   | 可能疾病                   |
| ------ | ---------------------- |
| 耳鸣听力下降 | 梅尼埃病、迷路炎               |
| 恶心呕吐   | 梅尼埃病、位置性眩晕、晕动病         |
| 共济失调   | 小脑、后颅窝病变               |
| 眼球震颤   | 真性眩晕（前庭性）通常伴有眼震，而非前庭性无 |
| 听力下降   | 药物中毒、听神经瘤              |
