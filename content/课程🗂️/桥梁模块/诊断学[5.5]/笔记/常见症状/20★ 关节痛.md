#运动系统

> [!NOTE] Title
> 掌握：关节痛的临床表现
> 熟悉：关节痛的伴随症状
> 了解：关节痛的病因和发生机制

**关节痛 arthralgia**：关节疾病最常见的症状，可由单纯的关节病变或全身性疾病所致
- 急性：关节及其周围组织的炎症反应为主
- 慢性：关节囊肥厚及骨质增生为主

---
# 一、病因
1. 外伤性
	1. 急性损伤：外力碰撞关节或是关节过度伸展扭曲，关节骨质、肌肉、韧带等急性损伤
	2. 慢性损伤：持续的慢性机械损伤，或急性外伤后关节面破损留下粗糙瘢痕或处理不当造成的慢性损伤
2. 感染性
	1. 细菌直接侵入关节内（外伤）/经血液到达关节内
	2. 关节邻近感染蔓延
	3. 关节穿刺时消毒不严
3. 变态反应和自身免疫性
	1. 变态反应关节炎--抗原抗体反应--风湿性关节炎、细菌性痢疾、过敏性紫癜、结核感染
	2. 自身免疫性关节炎--自身抗体形成--类风湿性关节炎、系统性红斑狼疮引起的关节炎等
4. 退行性关节痛
	- 关节炎症/慢性劳损/局部损伤引起关节面退行性变，软骨下骨板反应性增生形成骨刺
5. 代谢性骨痛
6. 骨关节肿瘤
	1. 良性肿瘤
	2. 恶性骨肿瘤

# 三、临床表现
1. 外伤性关节痛
	1. 急性：外伤后即出现疼痛、肿胀和功能障碍
	2. 慢性：明确外伤史，反复出现，过度活动/负重/气候寒冷等诱发
2. 化脓性关节炎
	1. 起病急，全身中毒反应明显
	2. 红肿热痛，功能障碍，各方向被动活动均引起剧烈疼痛
3. 结核性关节炎
	1. 儿童/青少年多见
	2. 负重大、活动多、肌肉不发达关节易患，脊柱>髋关节>膝关节
4. 风湿性关节炎
	1. 起病急剧，膝/踝/肩/髋关节多见
	2. *病变关节红肿热痛，游走性*，肿胀时间短，消失快，1-6周自然消肿，不留下关节僵直和畸形改变
5. 类风湿性关节炎
	1. 多由一个关节起病，以*手中指指间关节*首发疼痛
	2. 继而出现其他指间关节和腕关节的肿胀疼痛，也可累计踝、膝和髋关节，常对称
	3. 病变关节活动受限，晨僵
	4. 可伴有全身发热，晚期常因关节附近肌肉萎缩、关节软骨增生而出现畸形。
6. 退行性关节炎
	1. 早期表现为步行、久站和天气变化时病变关节疼痛，休息后缓解
	2. 晚期病变关节疼痛加剧，持续并向他处放射，关节有摩擦感，活动时有响声
	3. 关节周围肌肉挛缩常屈曲畸形，病人常有跛行
7. 痛风关节炎
	1. 饮酒、劳累或高嘌呤饮食后急起关节剧痛，局部关节红肿灼热
	2. 夜间痛醒；以第1跖趾关节、拇指关节多见。
	3. 踝、手、膝、腕和肘关节也可受累。
	4. 病变呈自限性,有时在1~2周内自行消退,但经常复发。
	5. 晚期可出现关节畸形,皮肤破溃,经久不愈,常有白色乳酪状分泌物流出。
# 四、伴随症状

|            |                   |
| ---------- | ----------------- |
| 化脓性关节炎     | 高热、畏寒、局部红肿热痛      |
| 结核性关节炎     | 低热、盗汗、乏力、消瘦、食欲下降  |
| 类风湿关节炎     | 晨僵、关节畸形           |
| 风湿热        | 心肌炎、舞蹈病           |
| 痛风         | 血尿酸升高、局部红肿热痛      |
| 系统性红斑狼疮    | 皮肤红斑、光过敏、低热、多器官损害 |
| 关节受累型过敏性紫癜 | 皮肤紫癜、腹痛、腹泻        |

# 五、问诊要点
1. 关节痛的特点:
	1. 起病诱因,有无外伤,急性起病还是慢性起病
	2. 部位是大关节还是小关节、多发还是单发,有无游走性、对称性、局部有无红肿及发热;
	3. 病程长短,关节痛是持续性还是间断性,与季节、气候的关系,有无活动障碍或变形
2. 伴随症状:有无发热,高热或低热,有无晨僵或关节畸形;有无尿酸升高局部红肿热痛;有无皮肤红斑、光过敏;有无皮肤出血点、腹痛腹泻等

3. 既往史
	1. 有无心脏、肝脏、肾脏、内分泌、结缔组织病等病史
	2. 有无结核、痛风等病史
