> [!NOTE] Title
> 掌握：概念、病因和临床表现
> 熟悉：发生机制
> 了解：伴随症状

**意识障碍**是指人对周围环境及自身状态的识别和觉察能力出现障碍；由高级神经中枢功能活动受损引起；表现为嗜睡/意识模糊/昏睡/谵妄/昏迷

---
# 一、病因
1. 重症急性感染：颅内感染或全身感染(败血症/感染性休克/重症肺炎等)
2. 颅脑非感染性疾病
	1. 脑血管疾病
	2. 脑占位性疾病
	3. 颅脑损伤
	4. 癫痫
3. 内分泌与代谢障碍
	1. 低血糖/高血糖
	2. 肝性脑病
	3. 肾性脑病
	4. 肺性脑病
	5. 桥本脑病
4. 心血管疾病(心梗/心衰/心律失常)
5. 水电解质平衡紊乱
6. 外源性中毒(CO/甲醇/有机溶剂/毒品/重金属)
7. 物理性及缺氧性损害(溺水窒息热射病等)
# 二、发生机制
- 由于脑缺血、缺氧、葡萄糖供给不足、酶代谢异常等因素可引起脑细胞代谢紊乱，从而导致网状结构功能损害和脑活动功能减退,均可产生意识障碍。
# 三、临床表现

^10d633

1. ==嗜睡 somnolence==：最轻的意识障碍，*可被唤醒*，唤醒后*能正确回答问题并作出各种反应*，停止刺激后很快又入睡
2. ==意识模糊 confusion==：意识障碍较嗜睡更深，意识水平轻度下降，*对时间地点人物的定向能力发生障碍*
3. ==昏睡 stupor==：只能在*强烈刺激下被唤醒*，唤醒后*不能执行指令或回答问题*
4. ==谵妄 delirium==：以兴奋性增高为主的高级神经中枢急性活动失调状态，表现为*意识模糊/定向丧失/精神错乱/躁动不安*，可好转或加重为昏迷
5. ==昏迷 coma==：严重的意识障碍，表现为意识持续的中断或完全丧失
	1. 轻度：对疼痛刺激出现痛苦表情或肢体退缩等防御反应，角膜反射/对光反射/眼球运动/吞咽咳嗽反射存在
	2. 中度：对剧烈疼痛刺激有防御反射，角膜反射/对光反射迟钝
	3. 深度：对各种刺激全无反应，深浅反射消失
# 四、伴随症状

| 症状        | 常见疾病                                                    |
| --------- | ------------------------------------------------------- |
| 发热        | 先发热然后有意识障碍：重症感染性疾病<br>先有意识障碍然后有发热：脑出血、蛛网膜下腔出血、巴比妥类药物中毒等 |
| 呼吸缓慢      | 吗啡、巴比妥类、有机磷杀虫药等中毒、银环蛇咬伤等                                |
| 心率/心律明显改变 | 见于心源性晕厥                                                 |
| 瞳孔散大      | 颠茄类、酒精、氰化物等中毒以及癫痫、低血糖状态等                                |
| 瞳孔缩小      | 吗啡类、巴比妥类、有机磷杀虫药等中毒                                      |
| 心动过缓      | 颅内高压症、房室传导阻滞以及吗啡类、毒蕈等中毒                                 |
| 高血压       | 高血压脑病、脑血管意外、肾炎尿毒症等                                      |
| 低血压       | 各种原因的休克                                                 |
| 皮肤黏膜改变    | 出血点、瘀斑和紫癜：严重感染和出血性疾病<br>口唇樱桃红色：一氧化碳中毒                   |
| 脑膜刺激征     | 脑膜炎、蛛网膜下腔出血                                             |
| 瘫痪        | 脑出血、脑梗死等                                                |

# 五、问诊要点
*问诊需通过第三者*
- 意识障碍发生时的情况:剧烈运动?高温作业?饮酒?情绪激动?家中有无毒物?
- 意识障碍起病方式:突发?逐渐加重?近期有无感染发热?有无新加药物?
- 既往史:高血压,糖尿病,肾病,肝病,肿瘤史,癫痫病史。
- 伴随症状:恶心呕吐,发热。抽搐,肢体偏瘫等
