# 一、概念
- 咳嗽cough：反射性防御动作，清除呼吸道分泌物及气道内异物
- 咳痰expectoration：痰液-气管/支气管分泌物或肺泡内渗出液，借助咳嗽排出

# 二、病因 
1. 呼吸道疾病
2. 胸膜疾病：胸膜炎、胸膜间皮瘤、自发性气胸
3. 心血管疾病：左心衰竭（二尖瓣狭窄……）、肺栓塞
4. 中枢神经因素：三叉神经支配的鼻黏膜及舌咽神经支配的咽峡部黏膜受刺激；脑炎；脑膜炎
5. 其他

# 三、发生机制
1. 延髓咳嗽中枢受刺激：喉返神经、膈神经和脊髓神经--咽肌、膈肌和其他呼吸肌的运动--咳嗽
2. 呼吸道炎症：黏膜充血水肿--黏液分泌增多--毛细血管壁通透性增加--浆液渗出
3. 肺淤血或肺水肿：肺泡和小支气管内浆液漏出

# 四、临床表现
1. 咳嗽的性质：干/湿性
2. 咳嗽的时间与规律：发作性、持续性；夜间咳嗽、慢性咳嗽
3. 咳嗽的音色
4. 痰的性状和痰量

# 五、伴随症状

|        |                                                       |
| ------ | ----------------------------------------------------- |
| 伴发热    | 急性上下呼吸道感染、肺结核、胸膜炎                                     |
| 伴胸痛    | 肺炎、胸膜炎、支气管肺癌、肺栓塞、自发性气胸                                |
| 伴呼吸困难  | 喉水肿、喉肿瘤、支气管哮喘、慢阻肺、重症肺炎、肺结核、大量胸腔积液、气胸、肺淤血、肺水肿、气管/支气管异物 |
| 伴咯血    | 支扩、肺结核、肺脓肿、支气管肺癌、二尖瓣狭窄                                |
| 伴浓痰    | 支扩、肺脓肿、肺囊肿合并感染、支气管胸膜瘘                                 |
| 伴哮鸣音   | 支气管哮喘、心源性哮喘、慢阻肺、气管/支气管异物                              |
| 伴杵状指/趾 | 支扩、慢性肺脓肿、支气管肺癌、脓胸                                     |
# 六、问诊要点
1. 咳嗽与咳痰的特点
	1. 咳嗽发生地点、诱因、时间及加重缓解因素
	2. 痰量、颜色、气味
	3. 咳嗽与体味变化或活动的关系
2. 伴随症状：发热、胸痛、呼吸困难、咯血、浓痰、哮鸣音、杵状指
3. 既往史
	1. 支气管哮喘、呼吸道感染、肺部肿瘤、鼻炎及鼻窦炎、胃食管反流、心脏、结缔组织病
	2. 青霉素、头孢菌素及大环内酯类药物过敏
	3. 用药史：ACEI及ARB类高血压药物、阿司匹林、胺碘酮及细胞毒性药物、靶向药物、免疫类药物等
4. 个人史：鸟类及宠物接触史；粉尘或有毒烟雾接触史
