 **发绀 Cyanosis**：血液中还原血红蛋白增多或存在异常血红蛋白衍生物，使皮肤、 粘膜呈现青紫色的现象


> [!NOTE] 学习目标
> 1. 掌握：发绀的概念、分类、病因及问诊要点
> 2. 熟悉：发绀的发生机制
> 3. 了解：发绀的伴随症状


# 病因

1. **血液中还原血红蛋白增加（真性发绀）**
	1. 中心性发绀 
		- 全身性、皮肤是温暖的
		1. 肺性发绀：呼吸功能不全，肺氧合作用不足
		2. 心肺混合性发绀：异常通道分流（部分静脉血未经氧合进入体循环动脉）
	2. 周围性发绀
		- 肢体末端与下垂部位、皮肤是冷的
		1. 淤血性：体循环淤血，周围血流缓慢（右心衰竭、下肢静脉曲张）
		2. 缺血性：心排出量减少和局部血流障碍（休克）
	3. 混合性发绀
		- 可见于心力衰竭
2. **血液中存在异常血红蛋白衍生物**
	1. 高铁血红蛋白血症
		1. 先天性：无心肺疾病等原因；家族史；身体一般状况较好
		2. 后天获得性：化学物质/药物中毒-->二价铁被三价铁取代
	2. 硫化血红蛋白血症

# 发生机制
- 血液中还原血红蛋白的绝对量增加

# 伴随症状

|        |                                    |
| ------ | ---------------------------------- |
| 伴呼吸困难  | 重症肺心疾病、急性呼吸道梗阻、大量气胸                |
| 伴杵状指/趾 | 发绀型先天性心脏病、某些慢性肺部疾病                 |
| 伴意识障碍  | 肺性脑病、某些药物或化学物质中毒/休克、急性肺部感染或急性心功能衰竭 |
