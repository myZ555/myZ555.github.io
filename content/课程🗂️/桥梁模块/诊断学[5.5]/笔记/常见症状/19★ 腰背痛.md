# 一、病因
> 腰背部组织层次：皮肤--皮下组织--肌肉--韧带--脊椎--肋骨--脊髓

==病因==：
1. 外伤性
	1. 急性损伤
	2. 慢性损伤
2. 炎症性
	1. 感染性（结核菌、化脓菌、伤寒菌……）
	2. 无菌性炎症
3. 退行性变
	1. 髓核突出和骨刺压迫或刺激神经
4. 先天性疾病
5. 肿瘤性疾病

==原发部位==：
1. 脊柱疾病（脊椎骨折、椎间盘突出、脊椎肿瘤、感染性脊柱炎）
2. 脊柱旁软组织疾病（腰肌劳损、风湿性多肌炎、腰肌纤维组织炎）
3. 脊神经根病变（脊髓压迫症、急性脊髓炎、腰骶神经炎、颈椎炎）
4. 内脏疾病（胸膜病变引起上背部疼痛、泌尿系疾病如结石、盆腔/直肠/前列腺/子宫附件区炎症--放射性腰背痛）

# 三、临床表现

==**主要按腰背痛的原发病部位**==

| **脊柱**    |         |                                                                                                          |
| --------- | ------- | -------------------------------------------------------------------------------------------------------- |
|           | 脊椎骨折    | 明显外伤史；骨折部压痛、叩痛，脊椎后突/侧突畸形，活动障碍                                                                            |
|           | 椎间盘突出   | 青壮年多见，L4-S1，腰痛、坐骨神经痛，咳嗽/喷嚏时加重，卧床休息缓解。下肢麻木、冷感或间歇性跛行                                                       |
|           | 增生性脊柱炎  | >50岁，晨起腰痛酸胀，僵直而活动不便，活动后好转。疼痛不剧烈，叩击腰部有舒适感，腰椎无明显压痛                                                         |
|           | 结核性脊柱炎  | 腰椎最易受累，其次未胸椎。背痛-首发症状，疼痛局限，夜间明显，活动加剧。伴有低热、盗汗、乏力                                                           |
|           | 化脓性脊柱炎  | 腰背痛剧烈，明显压痛及叩痛，伴畏寒、高热等全身中毒症状                                                                              |
|           | 脊椎肿瘤    | 转移性恶性肿瘤多见，顽固性腰背痛，剧烈而持续；放射性神经根痛                                                                           |
| **脊柱旁组织** | 腰肌劳损    | 腰骶酸/钝痛，休息缓解，劳累加重；伸腰或叩击腰部时缓解                                                                              |
|           | 腰肌纤维炎   | 腰背部弥漫性头痛，腰椎两旁肌肉及髂嵴上方为主，晨起加重，活动缓解；轻叩腰部缓解                                                                  |
| **脊神经根**  | 脊髓压迫症   | 神经根激惹征，疼痛剧烈，烧灼样/绞窄样痛，脊柱活动、咳嗽、喷嚏时加重，有一定定位性，可有感觉障碍                                                         |
|           | 蛛网膜下腔出血 | 出血刺激脊膜和脊神经后根时引起剧烈的腰背痛                                                                                    |
|           | 腰骶神经根炎  | 下背部和腰骶部疼痛，僵直感，疼痛向臀部及下肢放射，严重时节段性感觉障碍                                                                      |
| **内脏疾病**  | 泌尿系     | [肾炎]--腰肋三角区深部胀痛，轻微叩痛<br>[肾盂肾炎]--腰痛鲜明，叩痛明显<br>[肾脓肿]--单侧腰痛，局部肌紧张和压痛<br>[肾结石]--绞痛，叩痛剧烈<br>[肾肿瘤]--钝痛或胀痛，有时绞痛 |
|           | 盆腔器官    | [前列腺炎/癌]--下腰骶部疼痛，伴尿频尿急排尿困难<br>[女性慢性附件炎/宫颈炎/子宫脱垂/盆腔炎]--腰骶部疼痛，下腹坠胀感和盆腔压痛                                   |
|           | 消化系统    | [胃/十二指肠溃疡-后壁慢性穿孔]--上腹部疼痛+胸下腰椎区疼痛<br>[急性胰腺炎]--左侧腰背部放射痛<br>[胰腺癌]--前倾坐位缓解，仰卧位加重<br>[溃疡型结肠炎&克罗恩病]            |
|           | 呼吸系统    | [胸膜炎、肺结核、支气管肺癌]--后胸和侧胸肩胛部疼痛；<br>胸膜病变时随深呼吸加重<br>脊柱本身无病变、无压痛、运动不受限                                         |

# 四、伴随症状

|                  |                         |
| ---------------- | ----------------------- |
| 脊柱畸形             | 脊柱骨折、脊柱先天畸形、脊柱结核、强直性脊柱炎 |
| 活动受限             | 脊柱外伤、强直性脊柱炎、腰背部软组织急性扭挫伤 |
| 发热               | 脊柱结核、类风湿关节炎、化脓性脊柱炎、椎旁脓肿 |
| 尿频、尿急、排尿不尽       | 尿路感染、前列腺炎或前列腺肥大、肾或输尿管结石 |
| 嗳气、泛酸和上腹胀痛       | 胃十二指肠溃疡、胰腺病变            |
| 腹泻或便秘            | 溃疡性结肠炎、克罗恩病             |
| 下腰痛伴月经异常、痛经、白带过多 | 宫颈炎、盆腔炎、卵巢及附件炎症或肿瘤      |


# 五、问诊要点
1. 腰背痛的特点
	1. 诱因、起病时间、缓急、开始部位及发展顺序
	2. 性质、部位、程度、放射范围、加重环节因素
	3. 与体位变化或活动的关系
2. 伴随症状
3. 既往史（结核、肿瘤、消化系疾病、泌尿系结石、自身免疫性疾病、高血压）
