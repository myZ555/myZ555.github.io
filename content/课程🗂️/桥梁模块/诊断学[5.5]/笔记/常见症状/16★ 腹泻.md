#消化系统 
> [!NOTE] Title
> 掌握：腹泻的概念、分类、病因及问诊要点
> 熟悉：腹泻的发生机制
> 了解：腹泻的伴随症状

**腹泻 diarrhea**：排便次数增多，粪质稀薄，或带有黏液、脓血或未消化的食物。
- 液状便，*每日3次以上*，或每天粪便总量>200g，其中粪便含水量大于80%
- 急慢性（超过2个月）

---
# 一、病因
1. 急性腹泻
	1. 肠道疾病：各种原因感染引起肠炎及急性出血性坏死性肠炎、克罗恩病或溃疡性结肠炎急性发作、机型倡导缺血等。医院内感染、抗生素使用而发生的抗生素相关性肠炎。
	2. 急性中毒
	3. 全身性感染（败血症、伤寒、副伤寒、钩端螺旋体病）
	4. 其他：变态反应性肠炎、过敏性紫癜、服用药物
2. 慢性腹泻
	1. 消化系统疾病
		1. 胃部疾病：慢性萎缩性胃炎、胃大部切除后胃酸缺乏等
		2. 肠道疾病：
			1. 感染性：肠结核、慢性细菌性痢疾、慢性阿米巴性痢疾、钩虫病
			2. 非感染性：Crohn病、溃疡性结肠炎、结肠多发性息肉、吸收不良综合征
			3. 肿瘤：结肠绒毛状腺瘤，小肠、结肠恶性肿瘤等
		3. 胰腺疾病
		4. 肝胆疾病
	2. 全身性疾病
		1. 内分泌及代谢障碍疾病
		2. 其他系统疾病：SLE、硬皮病、尿毒症、放射性肠炎
		3. 药物副作用
		4. 神经功能紊乱：肠应激综合征
# 二、发生机制
![[Pasted image 20251019114645.png|400]]
==肠内水失平衡-腹泻（分泌>吸收）==

1. **分泌性腹泻**：胃肠黏膜分泌过多液体引起（霍乱弧菌外毒素-大量水样腹泻）
	- 大量水样便、无脓血、无腹痛
	- 禁食48h后腹泻不能缓解
	- 粪便含有大量电解质，渗透压与血浆接近
2. **渗透性腹泻**：肠内容物渗透压增高，阻碍肠内水分与电解质吸收
	- 禁食或停用高渗性药物后腹泻停止
	- 肠内渗透压超过血浆渗透压
	- 粪便中含有大量未经消化或吸收的食物或药物
3. **渗出性腹泻**：黏膜炎症/溃疡/浸润性病变，见于各类肠道炎症疾病
	- 粪便含有渗出液和血
	- 腹泻、全身症状与体征，严重程度取决于肠受累程度
4. **动力性腹泻**：肠蠕动亢进-肠内食糜停留时间缩短未充分吸收所致，肠炎、胃肠功能紊乱及甲亢
	- 粪便稀烂或水样，无渗出物
	- 腹泻伴肠鸣音亢进或腹痛
5. **吸收不良性腹泻**：肠黏膜吸收面积减少或吸收障碍，如小肠大部分切除
# 三、临床表现
1. 起病及病程
2. 腹泻次数及粪便性质
	1. 急性感染性腹泻：每天排便次数>10次；若为细菌感染常有粘液血便或脓血便
	2. 慢性腹泻：每天排便数次，可为稀便，可带黏液、脓血
3. 腹泻与腹痛的关系：急性腹泻常有，尤以感染性腹泻明显
	1. 小肠-脐周-便后缓解不明显
	2. 结肠-下腹-便后疼痛缓解
	3. 分泌性腹泻-无明显腹痛
# 四、伴随症状

|         |                                               |
| ------- | --------------------------------------------- |
| 发热      | 急性细菌性痢疾、伤寒或副伤寒、肠结核、肠道恶性淋巴瘤、Crohn病、溃疡性结肠炎急性发作期 |
| 里急后重    | 结肠直肠病变为主者，急性痢疾、直肠炎症或肿瘤                        |
| 明显消瘦    | 小肠病变为主，如胃肠道恶性肿瘤、肠结核及吸收不良综合征                   |
| 皮疹/皮下出血 | 败血症、伤寒/副伤寒、麻疹、过敏性紫癜、糙皮病                       |
| 腹部包块    | 胃肠恶性肿瘤、肠结核、Crohn病                             |
| 重度失水    | 分泌性腹泻，霍乱、细菌性食物中毒                              |
| 关节痛/肿胀  | Crohn病、溃疡性结肠炎、SLE、肠结核、Whipple病                |

# 五、问诊要点
1. 腹泻的特点：病程、大便性状、与腹痛的关系
	- 急性、慢性
	- 感染性、非感染性
	- 进展性、自限性
	- 右侧、左侧
		- 右侧（小肠）-量大|频率低|无里急后重|无脓血便
		- 左侧（大肠）-量小|频率高|有里急后重|有脓血便
