# 一、概念

**肥胖 obesity**：体内脂肪积聚过多而呈现的一种状态

- 分类：
	- 按病因：原发性（单纯性）；继发性
	- 按脂肪在身体分布：普遍型、腹型、臀型

# 二、肥胖的测量

1. 身高体重计算
	1. 超重：超过标准体重10%（排出肌肉发达或水分潴留因素）
	2. 肥胖：超过标准体重20%
2. 体重指数
	- BMI=体重/身高的平方（m）
	- 我国：24-27.9：超重；30：肥胖
3. 其他
	1. 肱三头肌皮褶厚度
	2. 腰围
# 三、病因与发生机制
1. 遗传因素
2. 内分泌因素
	- 下丘脑、垂体疾病、库欣综合征、甲减、性腺功能减退、多囊卵巢综合征
3. 生活方式
4. 药物因素
	- 糖皮质激素、氯丙嗪、胰岛素（医源性）
5. 脂肪细胞因子

# 四、临床表现

1. 单纯性肥胖
	- 家族史、营养过度史
	- 均匀性
	- 无内分泌代谢等疾病
2. 继发性肥胖

# 六、问诊要点

1. 身高，体重，BMI
2. 家族史，是否合并内分泌系统疾病，是否营养过度，是否使用激素等药物
3. 伴随症状：是否有饮水、进食、睡眠及智力精神异常，是否伴有食欲波动、血压易变、性功能减退及尿崩症者；是否半夜溢乳、闭经；是否有性功能丧失
