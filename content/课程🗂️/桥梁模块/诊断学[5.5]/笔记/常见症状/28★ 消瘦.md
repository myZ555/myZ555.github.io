> [!NOTE] 重点难点
> 掌握：消瘦的概念和诊断标准
> 熟悉：消瘦的病因，发病机制，临床表现
> 了解：消瘦的伴随症状


# 一、概念

- **消瘦 emaciation**：由于各种原因造成体重低于正常低限的一种状态
	- 标准：BMI<18.5kg/m^2

# 二、病因与发病机制

- 负氮平衡

1. 营养物质摄入不足
	1. 吞咽困难
		1. 口腔疾病
		2. 食管、贲门疾病
		3. 神经肌肉疾病（延髓性麻痹、重症肌无力）
	2. 进食减少
2. 营养物质消化吸收障碍
	1. 胃源性
	2. 肠源性
	3. 肝源性
	4. 胰源性
	5. 胆源性
3. 营养物质利用障碍（糖尿病）
4. 营养物质消耗增加
	1. 内分泌代谢性疾病
	2. 慢性消耗性疾病
	3. 大面积烧伤
	4. 高热
5. 减肥
6. 体质性消瘦

# 三、临床表现

1. 消化系统疾病
	- 食欲不振，恶心呕吐，腹胀，腹痛，腹泻
2. 神经系统疾病
3. 内分泌代谢疾病
	1. 甲状腺功能亢进症
	2. 肾上腺皮质功能减退（艾迪生病）
	3. 希恩综合征
	4. 1型糖尿病
4. 慢性消耗性疾病（结核病、肿瘤、慢性感染）
5. 神经精神疾病

# 五、问诊要点

1. 身高，体重，BMI是多少，是否达到消瘦诊断标准
2. 是否有吞咽困难和进食减少，胃肠肝胆胰是否有相关疾病
3. 是否有甲亢，糖尿病，结核，肿瘤
4. 是否在减肥或有情绪问题
