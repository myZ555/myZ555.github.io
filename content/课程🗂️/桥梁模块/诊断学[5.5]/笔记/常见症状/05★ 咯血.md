# 一、概念
- **咯血hemoptysis**：喉及喉以下的呼吸道及肺任何部位出血经口腔排出
- 鉴别：上呼吸道出血、牙病相关出血及消化道呕血![[Pasted image 20250925103401.png]]
- 呼吸系统疾病、循环系统疾病（二尖瓣狭窄、三尖瓣狭窄及急性左心功能不全）

# 二、病因
>  解剖学基础：两套血管：支气管循环、肺循环；大部分是支气管血管破裂（压力更高）
1. 支气管疾病：支扩、慢支、支气管结核及支气管肺癌
2. 肺部疾病：结核、肺炎、肺脓肿、肺真菌病、肺泡炎……
3. 心血管疾病：二尖瓣狭窄、先心所致肺动脉高压或原发性肺动脉高压、肺栓塞
4. 其他
	1. 血液病
	2. 急性传染病
	3. 风湿性疾病
	4. 气管、支气管子宫内膜异位症

# 三、发生机制
1. 支气管粘膜/毛细血管通透性增加
2. 肺泡壁或支气管粘膜毛细血管破裂
3. 小支气管动脉或动脉瘤破裂
4. 动静脉瘘破裂
5. 凝血障碍
6. 其他

# 四、临床表现
#### 1. 咯血量
1. 小量：100ml以内
2. 中量：100-500ml（肺癌、支扩、二尖瓣狭窄）
3. 大量：500ml以上（*肺结核空洞、支扩*、支气管动脉瘤、血管畸形）
#### 2. 咯血的颜色和性状 ^7af2c5
1. 鲜红色：肺结核、支扩、肺脓肿
2. 铁锈色：肺炎链球菌性肺炎；肺吸虫病和肺泡出血
3. 砖红色胶冻样：肺炎克雷伯杆菌
4. 浆液性粉红色泡沫痰：左心衰竭
5. 粘稠暗红色血痰：肺栓塞
6. 暗红色血痰：二尖瓣狭窄

#### 3. 年龄
- 青壮年：肺结核，支气管扩张症，二尖瓣狭窄
- 40岁以上，吸烟史：支气管肺癌
- 儿童慢性咳嗽伴少量咯血与低色素贫血：特发性含铁血黄素沉着
#### 4. 相关体格检查
1. 口腔、牙齿及鼻咽
2. 皮肤及粘膜出血
3. 是否杵状指
4. 二尖瓣面容
5. 肺部啰音（局部/弥漫性）
6. 颈部淋巴结

# 五、伴随症状

|         |                           |
| ------- | ------------------------- |
| 伴发热     | 肺结核、肺炎、肺脓肿、流行性出血热、支气管肺癌   |
| 伴胸痛     | 肺炎链球菌性肺炎、肺结核、肺栓塞、支气管肺癌    |
| 伴呛咳     | 支气管肺癌、支原体肺炎               |
| 伴浓痰     | 支扩、肺脓肿、空洞型肺结核继发细菌感染       |
| 伴皮肤黏膜出血 | 血液病、风湿病、肺出血型钩端螺旋体病、流行性出血热 |

# 六、问诊要点
1. 咯血特点
	1. 发生地点、诱因
	2. 咯血量、颜色、气味
	3. 咳嗽与体位变化或活动的关系
2. 伴随症状
3. 既往史
	1. 呼吸道感染、肺部肿瘤、心脏疾病、血液系统疾病、传染性疾病、风湿性疾病、消化道出血、肝胆疾病
	2. 肺栓塞高危因素
	3. 用药史：抗凝药物，抗血小板药物，抗肿瘤药物（抗血管生成）
