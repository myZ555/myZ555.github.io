#消化系统 
> [!NOTE] Title
> 掌握：呕血的临床表现及伴随症状
> 熟悉：呕血的病因
> 了解：呕血的发病机制

**呕血 hematemesis**：上消化道疾病（==屈氏韧带以上的消化道==，包括食管-胃-十二指肠-肝-胆-胰及胃空肠吻合术后的空肠上段疾病）或全身性疾病所致的上消化道出血，血液经口腔呕出

---
# 一、病因
1. 消化系统疾病
	1. 食管疾病：食管溃疡、食管癌、*食管静脉曲张出血*等
	2. 胃疾病：*胃溃疡、急性糜烂性胃炎、胃癌、胃底静脉曲张出血*、Dieulafoy病、贲门黏膜撕裂等
	3. 十二指肠疾病：*十二指肠溃疡*、十二指肠憩室、十二指肠乳头占位等
2. 上消化道邻近组织或器官疾病
	1. 肝胆疾病：肝胆恶性肿瘤、胆管出血等
	2. 胰腺疾病：胰腺肿瘤、胰管出血等
	3. 血管疾病：主动脉瘤累及十二指肠破裂出血等
3. 全身性疾病
	1. 血液系统疾病：过敏性紫癜、血小板减少性紫癜、血友病等
	2. 感染性疾病：流行性出血热、寄生虫感染等
	3. 结缔组织病：系统性红斑狼疮、结节性多动脉炎等
	4. 其他：尿毒症等

# 三、临床表现
1. **呕血与黑便**（上腹部不适和恶心-->呕吐血性胃内容物）
2. **失血性周围循环衰竭**【最有价值的判断】；动态观察血压和心率
	1. <循环血量10%：无明显临床表现
	2. 10-20%：头晕无力；无血压、脉搏等变化
	3. >20%：冷汗、四肢厥冷、心慌、脉搏增快等急性失血症状
	4. >30%：神志不清、面色苍白、心率加快、脉搏细弱、血压下降、呼吸急促等急性周围循环衰竭表现
3. **血液学改变**
4. **其他**
> **==出血严重度判断==：活动性出血的判断：内镜下Forrest分级**
> Ia：喷射样出血
> Ib：活动性渗血
> IIa：血管裸露
> IIb：血凝块附着
> IIc：黑色基底
> III：基底洁净
# 四、伴随症状

|             |                                                                                    |
| ----------- | ---------------------------------------------------------------------------------- |
| 上腹痛         | <1>消化性溃疡：中青年人，慢性反复发作，周期性/季节性，餐前/餐后痛<br><2>胃癌：中老年人，慢性，疼痛无明显规律性，伴厌食消瘦                |
| 肝脾肿大        | <1>肝硬化：脾大，蜘蛛痣，肝掌，腹壁静脉怒张，腹水，肝功障碍-肝硬化门脉高压<br><2>肝癌：肝区疼痛，肝大，质地坚硬，表面凹凸不平/有结节，甲胎蛋白AFP阳性 |
| 黄疸          | 胆道疾病、感染性疾病（败血症、钩端螺旋体病）                                                             |
