# 一、概念
**水肿 edema**：人体*组织间隙*有过多的液体积聚使组织肿胀
- 全身性水肿：液体在体内组织间隙呈弥漫型分布(常为凹陷性)
- 局部水肿：积聚在局部组织间隙
- 积液：发生于体腔内（e.g., 胸腔/腹腔/心包）
- *不包括*：内脏器官局部的水肿，如脑水肿、肺水肿

# 二、发生机制

#### 1. 毛细血管血流动力学改变
1. 毛细血管内静水压增加（容量负荷增加）
2. 血浆胶体渗透压降低（低蛋白血症）
3. *组织液胶体渗透压增高*
4. *组织间隙机械压力降低*
5. 毛细血管通透性增强
![[Pasted image 20250929111405.png|600]]
==细胞外液体积=通透系数（平均毛细血管内压力-平均组织间液压 力）-（血浆渗透压-间质渗透压）-淋巴回流量==
#### 2. 钠水潴留
1. 肾小球滤过功能降低
2. 肾小球对钠水的重吸收增加
#### 3. 静脉、淋巴回流障碍（多发生局部性水肿）
1. 淋巴管堵塞
2. 乳癌根治术

# 三、病因及临床表现

## 1. 全身性水肿
### 1. 心源性水肿
1. 病因：充血性心力衰竭、缩窄性心包炎
2. 特点：对称性、凹陷性
3. 首先出现于身体低垂部位
4. 有心脏病的病史、症状及特征
### 2. 肾源性水肿
1. 病因：各型肾炎和肾病综合征
2. 机制：钠水潴留
3. 特点：晨起有眼睑与颜面水肿，发生为全身水肿
4. 尿常规改变、高血压及肾功能损害的表现
### 3. 肝源性水肿
1. 病因：失代偿性肝硬化
2. 机制：门脉高压症、低蛋白血症、肝淋巴回流障碍、继发性醛固酮增多
3. 特点：腹腔积液
4. 常伴有肝硬化的病史、症状及体征
### 4. 内分泌性水肿
1. 见于甲减、甲亢、原发性醛固酮增多、库欣综合征等
2. **粘液性水肿**
	1. 病因：甲减
	2. 机制：组织间隙亲水物质增多
	3. 特点：非凹陷性，水肿不受体位影响；水肿部位皮肤增厚，粗糙，苍白，温度降低
### 5. 营养不良性水肿
1. 病因：慢性消耗性疾病长期营养缺乏，蛋白丢失性胃肠病，重度烧伤等所致低蛋白血症
2. 特点：从足部蔓延至全身，水肿发生前有体重减轻表现
### 6. 其他原因
1. 妊娠性水肿
2. 结缔组织疾病（系统性红斑狼疮）
3. 变态反应性水肿
4. 药物所致水肿
5. 经前期紧张
6. 特发性水肿
7. 功能性水肿（高温、肥胖、老年、旅行者……）

# 五、问诊要点
1. 水肿的特点 
	1. 水肿发生的时间、缓急、诱因、开始的部位及发展顺序
	2. 全身性/局部性，是否对称，是否为凹陷性
	3. 水肿与体位变化或活动的关系
2. 伴随症状：黄疸/呼吸困难/消瘦/心跳缓慢
3. 既往史
	1. 肝脏、心脏、肾脏、内分泌、结缔组织病
	2. 有无药物过敏
	3. 用药史
4. 月经史
