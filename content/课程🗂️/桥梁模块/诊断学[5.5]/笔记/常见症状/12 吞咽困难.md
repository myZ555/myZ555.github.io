#消化系统 
> [!NOTE] Title
> 掌握：吞咽困难的临床表现及伴随症状
> 熟悉：吞咽困难的病因、分类
> 了解：吞咽困难的发病机制

**吞咽困难 Dysphagia**：指食物从口腔至胃、贲门运送过程中受阻而产生咽部、胸骨后或剑突部位的梗阻停滞感觉。可伴有胸骨后疼痛。（注意区分假性吞咽困难）

---
# 一、病因
- **==机械性吞咽困难**==
	- 腔内因素
	- 管腔狭窄
		- 口咽部炎症
		- 食管良性狭窄
		- 食管炎症
		- 恶性肿瘤
		- 食管蹼
	- 外压性狭窄
- **==动力性吞咽困难**==
	- 吞咽启动困难
	- 咽、食管横纹肌功能障碍
	- 食管平滑肌功能障碍
	- 其他：感染、精神因素等

# 二、发生机制
1. **机械性吞咽困难（管腔狭窄）**
	1. 食管壁本身病变——整个管腔狭窄
	2. 外压性病变——偏心性狭窄
2. **运动性吞咽困难**
	1. 延髓麻痹
	2. 肌痉挛（e.g., 狂犬病）
	3. 肌丛内神经节调节减弱（e.g., 贲门失弛缓症）
	4. 全身性疾病（e.g., 系统性硬化症）
	5. 弥漫性食管痉挛
# 三、临床表现
1. 口咽性吞咽困难
	- 病因：吞咽中枢至控制口咽部横纹肌的运动神经节病变
	- 特点：食管由口腔进入食管受阻，阻滞于口腔及咽喉部
	- 常见：脑血管病变、PD、脑干肿瘤、脊髓灰质炎
2. 食管性吞咽困难
	- 病因：食管肿瘤、狭窄或痉挛引起
	- 特点：吞咽时食管阻滞于某一段，进食过程受阻
	- 常见：食管癌、食管良性肿瘤、反流性食管炎
3. 动力性吞咽困难
	- 特点：无液体、固体之分
4. 吞咽反射性动力障碍
	- 特点：吞咽液体比固体食物更加困难
5. 延髓麻痹
	- 特点：饮水由鼻孔反流伴以呛咳、呼吸困难等症状

> ==梗阻部位的定位诊断==
> 上：食管癌、胸骨后甲状腺肿、食管结核或肉芽肿、缺铁性吞咽困难等
> 中：食管癌、纵隔占位性病变、食管良性狭窄、食管粘膜下肿瘤等
> 下：食管癌、贲门失弛缓等
# 四、伴随症状

|            |                                                                    |
| ---------- | ------------------------------------------------------------------ |
| 伴声嘶        | 食管癌纵隔浸润、主动脉瘤、淋巴结肿大及肿瘤压迫喉返神经                                        |
| 伴呛咳        | 1）脑神经疾病、食管憩室和贲门失弛缓症导致潴留食物反流<br>2）食管癌致食管气管瘘<br>3）重症肌无力致咀嚼及吞咽困难及饮水呛咳 |
| 伴呃逆        | 病变多位于食管下端，见于贲门失弛缓症、膈疝等                                             |
| 伴吞咽疼痛      | 口咽炎或溃疡，如急性扁桃体炎、咽后壁脓肿、急性咽炎等                                         |
| 伴胸骨后疼痛     | 食管炎、食管溃疡、食管异物、晚期食管癌等                                               |
| 伴泛酸烧心      | 胃食管反流病                                                             |
| 伴哮喘和呼吸困难   | 纵隔肿物、大量心包积液压迫食管及大气管                                                |
| 伴未进食时咽部阻塞感 | 臆球症                                                                |

# 五、问诊要点
1. 起病经过：进行性吞咽困难&间歇发生
2. 伴随症状
3. 其他病史（吞入异物史/吞入强酸强碱等腐蚀剂史/特殊用药史/脑血管疾病史）
