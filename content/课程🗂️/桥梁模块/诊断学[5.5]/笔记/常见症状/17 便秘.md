#消化系统 
> [!NOTE] Title
> 掌握：便秘的概念、病因及问诊要点
> 熟悉：便秘的临床表现
> 了解：便秘的伴随症状

**便秘 diarrhea**：排便次数减少，一般每周小于3次，伴排便困难、粪便干结
1. 正常的排便形态改变
2. 排便次数减少
3. 排便不畅、排便困难
4. 粪便量减少
5. 排出过干过硬的粪便

---
# 一、病因
1. 功能性便秘
	1. 进食量少、食物缺乏纤维素或水分不足
	2. 精神紧张
	3. 结肠运动功能紊乱（肠应激综合征）
	4. 腹肌及盆底肌张力差，排便推动力不足
	5. 滥用泻药
2. 器质性便秘
	1. 直肠与肛门病变
	2. 局部病变导致排便无力（大量腹腔积液、系统性硬化症）
	3. 结肠梗阻（良恶性肿瘤、肠粘连、肠套叠）
	4. 腹腔或盆腔内肿瘤压迫（如子宫肌瘤）
	   全身性疾病致肠肌松弛、排便无力（尿毒症、糖尿病）
	5. 药物副作用（吗啡类、抗胆碱药、镇静剂、抗抑郁药及含钙、铝的制酸剂致肠肌松弛）
# 二、发生机制
==排便过程生理活动==
- 粪团在直肠内膨胀所致的机械性刺激，引起便意及排便反射和随后一系列肌肉活动
- 直肠平滑肌的推动性收缩——肛门内外括约肌的松弛

**常见的因素**：
- 摄入食物过少特别是纤维素和水分摄入不足
- 各种原因引起的肠肌张力减低和蠕动减弱
- 肠蠕动受阻致肠内容物滞留而不能下排
- 排便过程的神经及肌肉活动障碍


# 三、临床表现
1. 急性便秘：腹痛、腹胀、恶心、呕吐，多见于各种原因的肠梗阻
2. 慢性便秘：多无特殊临床表现
	1. 排便时下腹坠胀感或疼痛
	2. 肛门撕裂引起便血
# 四、伴随症状

|              |                   |
| ------------ | ----------------- |
| 呕吐、腹胀、肠绞痛    | 各种原因引起的肠梗阻        |
| 腹部包块         | 结肠肿瘤、肠结核、克罗恩病     |
| 便秘与腹泻交替      | 肠结核、溃疡性结肠炎、肠应激综合征 |
| 随生活环境改变、精神紧张 | 功能性便秘             |

# 五、问诊要点
1. 便秘的特点：病程和大便的性状
2. 伴随症状：是否伴有呕吐、肠绞痛、腹部包块、精神情绪等
