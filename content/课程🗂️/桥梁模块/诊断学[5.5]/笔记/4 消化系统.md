### 视诊
#### 1. 腹部外形
1. 腹部膨隆
	1. 全腹膨隆
		1. 腹腔积液-腹水 ascites
		2. 腹内积气
		3. 腹内巨大肿块
	2. 局部膨隆
2. 腹部凹陷
	1. 全腹凹陷-==舟状腹==：消瘦和脱水者 ^b9abc7
	2. 局部凹陷
#### 2. 腹壁皮肤
- 皮疹 rash
- 色素 pigment ^7db6b0
	- Gray-Turn二征：双侧或单侧腰部蓝色，见于重症胰腺炎
	- Cullen征：脐部或下腹部发蓝，见于腹腔大出血如宫外孕破裂
- 腹纹 ventral stripe
- 疤痕 scar
- 疝：为腹腔内容物经腹壁 或骨盆壁的间隙或薄弱部分向体表凸出而形成
- 脐部
- 腹部体毛
- 上腹部搏动
#### 3. 腹壁静脉
- 常见曲张原因：
	1. 门静脉高压
	2. 上腔静脉梗阻
	3. 下腔静脉梗阻（水母头状）
- 腹壁静脉血流方向的辨别：指压法
#### 4. 呼吸运动
1. 腹式呼吸减弱：腹膜炎症、腹水、急性腹痛、腹腔内巨大肿物、妊娠
2. 腹式呼吸消失：胃肠穿孔所致急性腹膜炎、隔肌麻痹
3. 腹式呼吸增强：癔症型呼吸、胸腔积液
#### 5. 胃肠型和蠕动波
- *胃肠道发生梗阻时*，梗阻近端的胃或肠段饱满而隆起，可显出各自的轮廓，称为胃型(gastral pattern) 或肠型 (intestinal pattern) 
- 蠕动波

#### 1. 肝脏和胆囊
- 肝相对浊音界-肝上界；肝绝对浊音界-肝下界
- ==肝浊音界异常==：
	- 扩大：肝癌、肝脓肿、肝炎、肝淤血、膈下脓肿
	- 缩小：急性肝坏死、肝硬化
	- 消失：胃肠道穿孔
	- 上移：右肺纤维化、右下肺不张
	- 下移：肺气肿、右侧张力性气胸
- ==肝区叩击痛==：肝脓疡、肝炎、肝癌

#### ※移动性浊音
- 定义：因体位不同而出现浊音区变动的现象

1. 腹壁紧张度增加

		1. 张力增加、无肌痉挛：腹壁内容物增加（肠胀气、腹水）

# 腹部常见病变的主要症状及体征
### 腹水
- 正常腹腔<200ml液体；移动性浊音阳性：>1000-1500ml腹水
###### 病因：
1. 心血管系统：心力衰竭
2. 肝脏及门脉系统：肝硬化、肝癌
3. 肾脏疾病：肾小球肾炎
4. 腹膜疾病：结核性腹膜炎
5. 营养缺乏：低蛋白水肿
6. 其他：淋巴系统、女性生殖系统、腹腔脏器破裂等
###### 发生机制：
同水肿[](03★%20水肿.md#二、发生机制)
1. 液体静水压增加
2. 血浆胶体渗透压下降
3. 淋巴循环受阻
###### 诊断与鉴别诊断：
1. 确定是否有腹水（详细查体、B超、必要时CT）
2. 需鉴别：肥胖者、胃肠胀气、巨大卵巢囊肿
###### 体征鉴别：
1. 结核性腹膜炎：腹部压痛，肌张力增高呈揉面感，反跳痛
2. 肝硬化：脾大，腹壁静脉怒张，移动性浊音阳性
3. 心源性水肿：发绀，颈静脉怒张，肝大，肝颈静脉反流阳性，下肢水肿
###### 实验室检查

|     | 外观            | 细胞数   | 比重     | SAAG   | 蛋白质定量  |
| --- | ------------- | ----- | ------ | ------ | ------ |
| 漏出液 | 澄清/微浑         | <1亿/L | <1.018 | >11g/L | <25g/L |
| 渗出液 | 多浑浊，乳糜性/血性/脓性 | >5亿/L | >1.018 | <11g/L | >30g/L |
- 感染性腹水：白细胞>5亿/L，中性粒细胞>50%
- 癌性腹水：腹水/血清LDH>1，肿瘤标记物测定，腹水细胞检查
- 结核性腹水：腹水腺苷脱氨酶ADA

### 腹部肿块
###### 病因：
- 炎症性、肿瘤性、梗阻性、先天性
###### 症状：
- 肿块生成速度（良性-慢）
- 活动幅度
- 伴随症状（黄疸、腹痛、呕吐等）
###### 体征：
1. 全身检查
2. 腹壁检查
	1. 视诊
	2. 触诊-①屏气起坐试验；②肘膝位检查法；③浮沉触诊法
	3. 叩诊
	4. 听诊
###### 诊断及鉴别诊断
- 肿块的病理类型
- 是否为肿块-->确定肿块的来源-->判断与内脏的关系-->确定病理类型及病变性质
### 肝肿大
- 正常肝脏：第5肋间隙；下缘-肋弓下<2cm、剑突下<3cm
###### 病因：
1. 感染：病毒性肝炎
2. 中毒性、药物性
3. 淤血：充血性心力衰竭
4. 淤胆
5. 代谢异常：脂肪肝
6. 肿瘤：肝细胞癌等
7. 血液病
8. 其他：免疫损伤
###### 诊断与鉴别诊断
- 病史
- 体征
	1. 肝脏肿大程度
		1. 轻度肿大：病毒性肝炎
		2. 中度肿大：细菌性肝脓肿
		3. 重度肿大：肝血吸虫病
		4. 进行性肿大：肿瘤
	2. 质地（[../../../疾病模块/消化与内分泌系统Ⅰ[2](../../../疾病模块/消化与内分泌系统Ⅰ[2)）
		1. 质软（如触口唇）：急性肝炎
		2. 质韧（如触鼻尖）：慢性肝炎
		3. 质硬（如触前额）：肝硬化、肝癌
- 伴随症状
- 实验室和特殊检查
	1. 白细胞计数
	2. 肝功能试验
	3. 病毒性肝炎抗原-抗体系统检测
	4. 各种病毒标记物检测
### 脾肿大
- 浊音区：左腋中线9-11肋 4-7cm，不超过腋前线；触诊-肋下未及

### 胃、十二指肠溃疡
[../../../疾病模块/消化与内分泌系统Ⅰ[2](../../../疾病模块/消化与内分泌系统Ⅰ[2)
- 腹痛特点：节律性、季节性、慢性反复发作
- 压痛点：胃溃疡、球部溃疡
### 急性腹膜炎
- *急性弥漫性*腹膜炎
	- 腹膜炎三联征阳性：*腹壁肌紧张-板状腹、腹部压痛、反跳痛*
	- 其他：腹式呼吸减弱或消失，腹膨隆，移动性浊音(+)，肝浊音界缩小或消失，肠鸣音减弱或消失
- *局限性*腹膜炎
	- 症状和体征局限于病灶周围，较急性弥漫性腹膜炎轻
### 肝硬化
- 症状：消化不良+全身症状+肝功能不全症状
- 主要体征：
	1. 面色晦暗，黄疸，蜘蛛痣，肝掌（[8 肝功能不全](疾病模块/消化与内分泌系统Ⅰ[2)）
	2. 肝脏：肿大/缩小，质地变硬，表面不光滑
	3. 门脉高压表现：腹水、静脉侧枝循环的建立和开放（三条）、脾肿大（[](消化系统病理.md#^783639)）
### 急性阑尾炎
- 症状：*转移性右下腹痛*+恶心、呕吐、发烧等
- 体征：
	- McBurney点（阑尾点）压痛和反跳痛
	- 洛（Rovsing）氏征阳性（结肠充气试验）
	- 腰大肌征阳性（盲肠后位阑尾炎）
### 肠梗阻
- 症状：痛、吐、胀、闭
- 体征：
	- 急性病容、脱水征、休克表现；
	- 腹膨隆，肠型及蠕动波
	- 腹壁紧张有压痛
	- 肠鸣音亢进【金属音调】；麻痹性肠梗阻时肠鸣音减弱或消失

// =====================================================
// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
