# 血管检查

### 脉搏
- **触诊部位**：桡动脉、肱动脉、股动脉、足背动脉 
- 两侧对比： 缩窄性大动脉炎或无脉症
- **触诊内容**： 脉率、脉律、紧张度和动脉壁弹性、强弱、 脉波
	- 窦性心律不齐：吸气时快，呼气时慢
	- 脱落脉：早搏或房室传导阻滞时，脉搏脱漏
	- *房颤*：脉搏*绝对不齐*，强弱不等，*脉率小于心率* **绌脉**

| 脉搏                        | 意义                                                        |
| ------------------------- | --------------------------------------------------------- |
| 水冲脉 water hammer pulse    | 前臂举过头顶可感觉到犹如水冲的脉搏冲击。由于高血流动力状态或存在返流所致，如甲亢、主动脉瓣关闭不全、动脉导管未闭。 |
| 交替脉 pulse alternans       | 节律规则而强弱交替的脉搏。左心衰竭                                         |
| 奇脉 paradoxical pulse（吸停脉） | 吸气时脉搏明显减弱/消失。心脏压塞或心脏缩窄+吸气时胸腔负压                            |
| 无脉 pulseless              |                                                           |

### 血压
- 双侧上下肢血压差别：下肢高20-40mmHg
	- *下肢血压<上肢*：主动脉缩窄、大动脉炎
- 脉压明显==增大==：甲亢，主动脉瓣关闭不全，动脉硬化
- 脉压明显==减小==：主动脉瓣狭窄，严重心衰，心包积液

### 血管杂音
- 静脉杂音（一般不出现）
	- *颈静脉*杂音：颈根部近锁骨处，无临床意义
	- 门脉高压*腹壁静脉*曲张杂音：脐周
	- 低调柔和连续营营声
- 动脉杂音（多见于周围动脉、肺动脉、冠状动脉）
	- *甲亢*：甲状腺侧叶的连续音
	- *多发性大动脉炎*：狭窄部位的收缩期杂音
	- *肾动脉狭窄*：上腹或腰背部的收缩期杂音
	- *肺内动静脉瘘*：胸部相应部位的连续性杂音
	- *冠状动脉瘘*：胸骨中下部的连续性或双期杂音

### 周围血管征
- 脉压显著**增大**所致一组体征

|                   |                                                   |
| ----------------- | ------------------------------------------------- |
| 枪击音               | 在股动脉等较大动脉表面，闻 及与心跳一致的短促如射枪的声音。                    |
| Duroziez <br>双重杂音 | 在股动脉闻及的双期杂音。                                      |
| 毛细血管搏动征           | 用手指轻压患者指 甲末端，使局部发白，当心脏收缩和舒张时则发白的局部边 缘发生有规则的红、白交替。 |
| 水冲脉               | 检查者握紧患者的手腕 掌面，将其前臂举过头顶，可感知犹如水冲的急促而有力的 脉搏冲击。       |

# 心血管常见疾病的症状和体征
### 二尖瓣狭窄
###### 病因
- 几乎均为风湿性， 2/3为女性患者
- 其他：
	- 先天性二狭
	- 退行性变
###### 病理生理
- 正常二尖瓣口面积为4～6cm^2
- 当各种因素造成二尖瓣口开放 受限，==面积小于2 cm^2==
	- 轻度：1.5-2
	- 中度：1.0-1.5
	- 重度：<1.0
- 临床分期：
	- 左房代偿
	- 左房失代偿
	- 右心衰竭
- 造成左心房排空障碍、左心室 灌注不足而产生一系列的血流动力学改变
###### 症状
1. 呼吸困难（最常见）
2. 咯血
3. 咳嗽
4. 声嘶
###### 体征
- **二尖瓣面容**：双颧呈绀红色
- 儿童/青少年可有心前区隆起伴抬举性搏动
- 心尖区可触及*舒张期震颤*
- 心界向左扩，**梨形心**
- ==心尖区舒张期杂音==是最特征性体征
	- 杂音常局限于心尖区，狭窄严重程度与杂音响度不一定成正比
	- *开瓣音*：在胸骨左缘第三，四肋间或心尖区的内上方 可闻及开瓣音 ( the opening snap )
	- 肺动脉瓣区P2亢进、分裂
- ==**Graham Steel杂音**==：肺动脉扩张，相对性肺动脉瓣关闭不全时，肺动脉 瓣区可闻及递减型、叹气样、 柔和、局限的舒张中期杂音。 见于二尖瓣狭窄伴明显 肺动脉高压。
---
### 二尖瓣关闭不全
###### 症状
- 劳累后呼吸困难
- 疲乏
- 心悸
###### 体征
- **心尖区全收缩期杂音，二闭最主要体征**
- ==二尖瓣脱垂综合征== 收缩中期*喀喇音*和收缩晚期杂音

---
### 主动脉瓣狭窄
- 成年人主动脉瓣口面积≥3.0cm^2
- 当瓣口面积减少到≤1.0cm^2时， 左室收缩压明显升高，跨主动脉瓣 压力阶差显著增加（>50mmHg）
- 导致*左室射血阻力增加，左室后负荷加重*等一系列血流动力学障碍
###### 病因
- 最常见的病因为风湿性，其次为 先天性畸形
- 在老年患者最主要病因则为老年 退行性改变
###### 症状
- 出汗和心悸
- 呼吸困难
- 心绞痛
- 晕厥
- 猝死
###### 体征
- 【**典型主动脉瓣狭窄的收缩期杂音**】典型主狭收缩期杂音在第一心音稍后，终止于第二心 音之前
- 粗糙、喷射性、递增递减型杂音伴震颤
### 主动脉瓣关闭不全
###### 体征
- ==**Austin-Flint 杂音**==：在心尖区可听到柔和 的舒张中期隆隆样杂音。 是主动脉瓣关闭不全时左 心室流出道返流血液使二 尖瓣开放受限，相对性二 尖瓣狭窄所致。
**周围血管体征**：4个
诊断：舒张期杂音+周围血管征


### 心包积液
###### 病理
- 纤维蛋白性心包炎和心包摩擦
- 渗液性心包炎和心脏压塞
###### 症状
- 心前区疼痛：多见于急性非特异性心包炎和感 染性心包炎，吸气、体位改变和咳嗽时加重。
- 呼吸困难：心包渗液时最突出的症状
- 其它：发热、干咳、嘶哑、吞咽困难
###### 体征
- *纤维蛋白性心包炎* ==心包摩擦音==是其最特异的征象
- *渗液性心包炎* 心界扩大、心尖搏动弱、 心音遥远、==Ewart征==、体循环回流障碍等
- *心脏压塞* 颈静脉怒张、血压下降、 奇脉 

### 心力衰竭
[../../../疾病模块/心血管、呼吸、血液与泌尿系统Ⅰ[3](../../../疾病模块/心血管、呼吸、血液与泌尿系统Ⅰ[3)
- 在静脉回流无器质性障碍的情 况下，通常是由于心肌收缩力下 降引起心排血量减少，不能满足 机体代谢需要的一种综合症。*临床上以肺和（或）体循环 淤血以及组织灌注不足为特征。*
###### 病因
1. 心肌本身病变：心肌缺血、心肌坏死、心肌炎症
2. 心室负荷过重： 
	1. 阻力负荷过重（高血压、主动脉狭窄） 
	2. 容量负荷过重（二尖瓣、主动脉瓣关闭不全）
###### 症状
- 左心衰竭：乏力、进行性劳力性呼 吸困难、夜间阵发性呼吸 困难、端坐呼吸、咳嗽、 粉红色泡沫痰、心源性哮 喘等。
- 右心衰竭：腹胀、少尿、食欲缺乏 甚至恶心呕吐。
