## 形成机制
![[Pasted image 20251222224054.png|250]]

|      | 正常        | 病理        |
| ---- | --------- | --------- |
|      | 少量液体，润滑作用 | **浆膜腔积液** |
| 胸腔   | <20ml     |           |
| 腹腔   | <50ml     |           |
| 心包   | 10-50ml   |           |
| 恶性积液 |           |           |

==分类==
**漏出液 transudate**：非炎性积液
	形成原因：
		1. 胶体渗透压↓
		2. 毛细血管内压↑
		3. 淋巴回流受阻
		4. 水钠潴留
**渗出液 exudate**：炎性积液
	形成原因：血管通透性增加，大分子及有形物质增加
		- 感染：结核、细菌、病毒
		- 非感染：外伤、化学性刺激、恶性肿瘤、风湿性疾病等

## 标本采集

**浆膜腔穿刺术**
- 适应症：
	- 新发生的浆膜腔积液
	- 已有浆膜腔积液且突然增多或伴有发热的患者
	- 需进行诊断性或治疗性穿刺的患者
## 一般性状检查
![[Pasted image 20251222224534.png|400]]

*常见渗出液的特点*：
- **脓性**：见于葡萄球菌、肺炎双球菌等化脓菌感染
- **血性**：见于创伤、恶性肿瘤、结核病变、出血性疾病
- **乳糜样**：见于纵隔肿瘤、淋巴结核、丝虫感染所致胸水
- **绿色**：铜绿假单胞菌感染、胆汁性
- **棕色**：阿米巴感染
- **黑色**：曲霉菌感染
- **草黄色**：尿毒症引起的心包积液
## 化学检查
#### 黏蛋白定型试验（Rivalta试验）
- 原理：浆膜上皮细胞受炎症刺激分泌黏蛋白增加，后者又称酸性糖蛋白，在稀醋酸溶液中析出而产生白色沉淀
- 临床意义：*漏出液(-)  渗出液(+)*

#### 蛋白定量试验
- 原理：双缩脲法--红色沉淀--光电比色
- 临床意义：*漏出液<25g/L，渗出液>30g/L*
==清蛋白梯度 AG== = 血清清蛋白-积液清蛋白（不受利尿剂和穿刺术的影响）
	- 腹水界值11g/L（高于--门脉高压；低于--腹膜转移癌、结核性腹膜炎）
	- 胸水界值12g/L（高于--漏出液；低于--渗出液）
#### 葡萄糖测定
- 原理：同血糖测定（邻甲苯胺法）[[16 临床常用生物化学检测#^4209d7]]
- 临床意义：漏出液含量与血糖相似，*渗出液常降低*（消耗）
#### 酶活性检测
==乳酸脱氢酶 LDH==：
- 漏出液--活性与正常血清相近
- 渗出液--增高：化脓性>癌性>结核性
==腺苷脱氨酶 ADA==： ^ead67a
- 增高：结核性、化脓性、风湿性
==淀粉酶==：
- 腹腔：急性胰腺炎、胰腺癌等
- 胸腔：食管穿孔、肺癌、胰腺外伤合并胸腔积液
## 显微镜检查
#### 细胞计数
- 漏出液：<100×10^6/L
- 渗出液：>500×10^6/L
#### 细胞分类
离心沉淀、瑞氏染色、油镜分类
**漏出液**：主要为淋巴和间皮细胞
**渗出液**：
	- 中性粒细胞为主：化脓性、结核早期
	- 淋巴细胞为主：结核性、癌性、病毒、结缔组织病
	- 嗜酸性粒细胞增多：过敏性、真菌、寄生虫病
**血性**：恶性肿瘤、结核、肺栓塞等

#### 脱落细胞检查
- 脱落细胞沉渣涂片染色找肿瘤细胞
- 肿瘤细胞的检出是诊断肿瘤的重要依据
#### 细菌学检查及寄生虫检验
- 无菌操作离心沉淀，取沉淀物
- 涂片作革兰染色或抗酸染色镜检，必要时进行细菌培养及药敏试验
- 显微镜下观察有无寄生虫及虫卵
