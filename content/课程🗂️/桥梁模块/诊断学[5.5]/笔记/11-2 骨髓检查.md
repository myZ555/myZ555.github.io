> [!NOTE] 骨髓
> - 人体的主要造血*器官*
> - 组成：由分布于骨小梁之间的基质、血管、神经、网状组织和发育过程中的血细胞组成
> - 流程：判断适应症--标本制作--镜检--分析

# 骨髓细胞学检查
### 1. 细胞形态学
#####  临床应用及其意义：
- 确诊是否造血系统疾病
- 辅助诊断某些造血系统疾病
- 作为鉴别诊断的应用（原因未明的发热、淋巴结、肝脾肿大、骨痛或关节痛）
#####  适应症及禁忌证
- 适应症
	- 不明原因外周血一系或多系异常
	- 不明原因发热、肝脾、淋巴结肿大
	- 不明原因骨痛、骨质破坏、黄疸、紫癜等
	- 某些遗传性、传染性疾病（如疟疾）或怀疑恶性肿瘤骨髓转移等
	- 助诊溶酶体贮积症
	- 进行血液病鉴别诊断和疗效观察
- 禁忌症
	- 凝血因子严重缺乏的出血性疾病，如*血友病*
	- 穿刺部位有炎症或畸形
	- 生命体征不稳定
	- 晚期妊娠妇女做髂骨穿刺需谨慎
##### 标本制作
- 部位：髂前上棘或髂后上棘穿刺。必要时行胸骨穿刺、胫骨（儿童）
- **骨髓穿刺成功的标志**：
	1. 抽吸骨髓时患者感锐痛
	2. 外观红色粘稠状，有骨髓小粒、油滴
	3. 显微镜下有骨髓特有成分（如：幼红细胞、幼粒细胞、巨核细胞、浆细胞、网状细胞等）
##### 骨髓标本的瑞氏染色
- 瑞氏染液：缓冲液1:2，时间1h，小水流冲洗，自然晾干后备检。
	- 含美蓝和易红两种成分
		- 美蓝--偏碱性--与酸性物质亲和(胞浆)--染成蓝色
		- 伊红--偏酸性--与碱性物质亲和(胞核)--染成紫红色![400](Pasted%20image%2020251201161459.png)
##### 显微镜检查
- ==低倍镜检查（10倍）==
	1. 制片质量（在涂尾部见到散在的约粟粒大小呈浅肉色半透明的骨髓小粒及少量脂肪小滴，在显微镜下见到较多骨髓特有细胞是*穿刺成功的标志*）
	2. 判断骨髓增生程度【五级法】
		- 通常以骨髓中有核细胞的数量反映![Pasted image 20251201162001](Pasted%20image%2020251201162001.png)
	3. 巨核细胞计数（数量、成熟程度、产血小板功能及其形态）
	4. 异常细胞筛查
	5. 选择油镜检查区域（涂膜体尾交界处）
![300](Pasted%20image%2020251201162239%201.png)
- ==油镜检查（100倍）==
	1. 有无造血岛
	2. 细胞形态观察
	3. 有核细胞分类计数
	4. 计算粒红比值
	5. 有无异常细胞及寄生虫

### 2. 细胞化学染色
#### 1. 髓过氧化物酶(MPO)染色 POX
- 过氧化物酶：存在于粒细胞和单核细胞。原粒大多呈阴性反应。==随细胞的发育成熟阳性成都逐渐递增，中性分叶核粒细胞呈强阳性反应==
- 淋巴细胞、幼红细胞、巨核细胞等均呈阴性反应
- *临床意义*：POX染色对急性粒细胞白血病与急性淋巴细胞白血病的鉴别最有价值
#### 2. 中性粒细胞碱性磷酸酶染色 NAP
- 碱性磷酸酶：主要存在于成熟阶段的中性粒细胞（分叶及杆状核），其他血细胞均呈阴性反应
- ![400](Pasted%20image%2020251201170055.png)
- 感染性疾病：细菌感染与病毒感染的鉴别
- 慢性粒细胞白血病：NAP活性明显减低，积分值常为0，类白血病反应的NAP活性极度增高
- 急性粒细胞白血病：NAP积分值减低；急性淋巴细胞白血病NAP积分值多增高；急性单核细胞白血病时一般正常或降低
- 再障：NAP活性增高
- PNH：活性减低
- 其他血液病：NAP活性中度增高；恶组NAP活性降低
#### 3. 氯化醋酸AS-D萘酚酯酶染色 SE
- 特异性酯酶：主要存在于粒细胞中。原粒(-)或(±)，早幼粒强阳性。
#### 4. α-醋酸萘酚酯染色 α-NAE, NSE
- 主要存在与单核细胞系细胞中
- 急单：阳性反应

### 3. 免疫表型分析
- 检测方法：
	- 免疫荧光法
	- 免疫酶标染色法
- **流式细胞术 FCM**
	- 通过对外周血细胞或骨髓细胞*表面抗原和DNA*的检测分析，……
	- 采用==各种抗血细胞表面分化抗原(CD)的单克隆抗体==，借助于各种荧光染料测定一个细胞的多种参数。
	- 通过对CD34, HLA-DR, CD33等细胞表面标志物测定，成为*干细胞移植*过程中的一项重要监测手段
- 细胞免疫分型的临床应用：
	1. 识别不同系的细胞（各系细胞对应抗体）
	2. 检测T淋巴细胞亚群
	3. 识别不同分化阶段的细胞
	4. 识别不同功能状态的细胞
	5. 各种白血病细胞的免疫表型分析
	6. 白血病微小残留病的检测

#### 正常骨髓象：
- 有核细胞增生活跃
- 粒-红比值为2-4:1，平均3:1
- 各系细胞比例、形态、染色大致正常
- 巨核细胞-多为产板型
- 核分裂细胞约为0.1%
#### 常见血液病的细胞学诊断

- 贫血
	- 再生障碍性贫血 aplastic anemia AA
	- 缺铁性贫血 IDA
	- 巨幼红细胞性贫血 MA
	- 溶血性贫血 HA
- 白血病
	- 急性白血病 AL
	- 慢性白血病 CL
- 骨髓增生异常综合征 MDS
- 多发性骨髓瘤 MM
- 特发性血小板减少性紫癜

###### Aplastic anemia AA
- 外周血：全血细胞减少，网织红细胞绝对数降低，淋巴细胞比例增高
- 骨髓片：
	- 骨髓多部位增生减低或重度减低
	- 造血细胞减少，非造血细胞比例增高
	- 巨核细胞数减少
	- 骨髓小粒空虚（骨髓活检可见造血组织均匀减少）![Pasted image 20251201185943](Pasted%20image%2020251201185943.png)
###### 缺铁性贫血 IDA
- 外周血：成熟红细胞常大小不等，中心明显淡染。*小细胞低色素性贫血*
- 骨髓片：
	- 幼红细胞体积小，胞质量减少，常伴有血红蛋白形成延缓或减少--胞质染色偏碱性，显示胞质发育落后于胞核（核老浆幼）
	- 铁染色阴性
###### 巨幼细胞贫血 MA
- 外周血：红细胞大小不均，较多体积大，椭圆形，着色深。*大细胞性贫血*
- 骨髓片：
	- 体积增大，胞质丰富，胞核也增大
	- 核染质细致疏松；胞质呈现早期血红蛋白特征，嗜碱程度浅
	- 核质发育不平衡，胞核落后于胞质（核幼浆老）
###### 溶血性贫血 HA
- 外周血：血红蛋白降低明显，网织红细胞计数增高
- 骨髓片：
	- 红系增生极度活跃，可见核分裂像，伴有核畸形，胞核哑铃状/菜花状/梅花瓣状/桑葚状/芽状突起。多出现在晚幼红细胞阶段。

###### 白血病-共同点：
- 血象：白细胞计数增高，红细胞、血红蛋白、血小板计数早期减少不明显（慢性早期血小板增高较常见）
- 骨髓：增生明显活跃或极度活跃。以较成熟的粒系（或淋巴系）显著增生为主

###### 骨髓增生异常综合征 MDS
- 外周血：全血细胞减少常见，常表现为大细胞贫血
- 骨髓：增生活跃，一系或多系细胞病态造血
###### 多发性骨髓瘤 MM
- 外周血：贫血常见，部分患者可表现为全血细胞减少
- 骨髓：增生活跃，出现典型的骨髓瘤细胞，其余正常系列受抑制

###### 免疫学血小板减少性紫癜 ITP
- 外周血：血小板减少。少数患者因出血明显可伴有贫血，也可伴有白细胞轻度下降
- 骨髓：增生明显活跃，红系和粒系细胞增生活跃，巨核细胞数量增多，但成熟障碍，产血小板能障碍。

// =====================================================
// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
