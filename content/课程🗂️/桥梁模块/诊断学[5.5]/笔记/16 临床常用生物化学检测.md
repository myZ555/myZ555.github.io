# 第一节 血糖及其代谢产物的检测

### （一）空腹血糖检测 FBG
- 是诊断糖代谢紊乱最常用和最重要的指标
- 以空腹血浆葡萄糖FPG检测较为方便，且结果也最可靠
- FBG易受多种因素影响；不同检测方法结果也不同
【参考值】葡萄糖氧化酶法：3.9~6.1mmol/L；邻甲苯胺法：3.9~6.4mmol/L ^4209d7
#### FBG增高
- 空腹血糖过高 IFG（6.1mmo/L <FBG < 7.0mmol/L）
- 高血糖症（FBG >7.0mmo/L）轻-中-重（7.9-8.4-10.1）
【临床意义】
1. 生理性：饱食餐后1-2h、高糖饮食、活动、激动等生理情况
2. 病理性：
	1. 影响血糖调节的激素
		1. 糖尿病
		2. 胰腺疾病（胰岛素）、肝脏疾病
		3. 内分泌疾病-甲亢、巨人症（甲状腺素、生长激素）
		4. 妊娠（雌孕激素对抗胰岛素）
		5. 应急性因素-烧伤等（肾上腺素）
		6. 药物影响（糖皮质激素）
	2. 其他：脱水/窒息/缺氧等
#### FBG减低
- 血糖减低 FBG < 3.9mmol/L【生理性；长期剧烈运动/饥饿】
- 低血糖症 FBG < 2.8mmol/L
	- 胰岛素用量过多、口服降糖药过量、胰岛B细胞瘤、对抗胰岛素的激素分泌不足（生长激素等缺乏）
	- 肝糖原储存缺乏（重症肝炎、肝硬化、肝癌）
	- 急性酒精中毒（糖异生受阻）
	- 先天性糖原代谢酶缺乏（糖原累积病）
	- 消耗性疾病
	- 药物影响（抗菌优、水杨酸等）
	- 特发性低血糖（迷走神经兴奋、餐后早发性低血糖）
### （二）口服葡萄糖耐量试验 OGTT
- 葡萄糖耐量试验GTT是检测葡萄糖代谢功能的试验，主要用于诊断症状不明显的可疑糖尿病
- GTT分为：静脉葡萄糖糖耐量试验、口服葡萄糖耐量试验
- OGTT方法：口服75g葡萄糖，分别检测FPG和服后30min、1h、2h、3h的血糖和尿糖
【参考值】
FPG：3.9-6.1mmol/L
30-60分钟达高峰 7.8-9.0mmol/L，峰值＜11.1mmol/L
2小时 ＜7.8mmol/L 不超
3小时 恢复至空腹水平 空腹水平
－－各检测时间点尿糖均为阴性
【适应证】
【临床意义】
1. 诊断糖尿病
2. 判断糖耐量异常
3. 鉴别低血糖
	1. 功能性低血糖：FPG正常，口服葡萄糖后出现高峰时间、峰值正常，2~3h后出现低血糖【特发性低血糖症】
	2. 肝源性低血糖：FPG低于正常，口服葡萄糖后提前出现高峰、峰值增高，2hPG处于高水平，尿糖阳性【广泛性肝损伤、病毒性肝炎】
4. 糖耐量曲线异常
	1. 平坦型
		- FPG降低，口服葡萄糖后血糖上升不明显、2hPG仍低
		- 见于胰岛B细胞瘤、肾上腺皮质功能减退症、腺垂体功能减退症，胃排空延迟、小肠吸收不良
	2. 储存延迟型
		- 口服葡萄糖后血糖急剧上升、提前出现峰值，大于11.1mmol/L，2hPG低于空腹
		- 见于胃切除、严重肝损伤
		- 血糖急剧升高，胰岛素分泌增高，肝外组织利用葡萄糖增多使2hPG降低

### （三）血清胰岛素和胰岛素释放试验
- 胰岛素释放试验：在进行OGTT同时检测血清胰岛素变化，*了解胰岛B细胞基础功能状态和储备功能状态*，间接了解血糖空置情况
【参考值】
- 血清胰岛素RIA测定参考值
	- 空腹血胰岛素： 10-20mU/L
- 胰岛素释放试验参考值
	- 空腹胰岛素：10-20mU/L
	- 30-60分钟达高峰值: 空腹胰岛素的5-10倍
	- 2小时：小于30mU/L
	- 3小时：恢复至空腹水平
【临床意义】
1. 糖尿病分型
	1. 一型：低平曲线
	2. 二型：空腹胰岛素可正常/稍高/降低，进糖后胰岛素延迟释放
2. 胰岛B细胞瘤：高水平曲线，出现高胰岛素血症
3. 其他：
	1. 增高：肥胖、肝损、肾功能不全、肢端肥大症
	2. 降低：腺垂体功能低下、肾上腺皮质功能不全、饥饿

### （四）血清C-肽检测
==C肽==
- 来源：
	- 胰岛素原在蛋白水解酶作用下裂解而成
	- 与胰岛素等分子的肽
- 特点：
	- 不被胰岛素酶灭活
	- 与外源性胰岛素无抗原交叉
	- 其生成量不受外源性胰岛素影响
	- *可真实反映胰岛素水平*
【参考值】
- 空腹：0.3-1.3nmol/L
- C肽释放试验：口服葡萄糖后30min-1h出现峰值，为空腹的5-6倍
- 可用于评价胰岛B细胞分泌功能和储备功能
【临床意义】**糖尿病分型、胰岛素调整**
1. C肽水平增高
	1. 胰岛B细胞瘤
	2. 肝硬化（C-肽/胰岛素降低（肝脏摄取和降解胰岛素减少））
2. C肽水平下降
	1. 空腹：糖尿病
	2. C肽释放试验：口服葡萄糖后1hC肽降低，提示胰岛B细胞储备功能不足；释放试验低平提示1型糖尿病；释放延迟或低水平见于2型糖尿病
	3. C肽不高，胰岛素增高：提示胰岛素用量过多
### （五）糖化血红蛋白检测 GHb
- RBC生存期间HbA与己糖（葡萄糖、果糖、磷酰葡萄糖）缓慢、连续的非酶促反应的产物。
- 其中与葡萄糖结合的组分为HbA1c，含量最高，占60-80%
- HbA1c--*可反映近2-3月的平均血糖水平*
【参考值】 HbA1c 4%-6%
【临床意义】
- 评价糖尿病控制程度
- 筛检糖尿病 ：大于9%预测糖尿病
- 预测血管并发症：大于10%提示并发症严重（HbA1c与氧的亲和力大可导致组织缺氧而发生血管并发症）
- 鉴别高血糖：应激性正常，糖尿病增高
### （六）糖化清蛋白检测 GA
- 是葡萄糖与清蛋白发生非酶促反应的产物
- GA--*可反映近2-3周的平均血糖水平*
- 采用GA/Alb%表示GA水平
【参考值】10.8%-17.1%
【临床意义】
- 评价短期糖尿病控制程度
- 筛检糖尿病 ：≥17.1%筛检出糖尿病
- 辅助鉴别应激性高血糖：与HbA1c联合测定有助于判断高血糖的持续时间，鉴别应激性或糖尿病
# 第二节 血清脂质和脂蛋白检测
## （一）血清脂质检测
#### 1. 总胆固醇测定 TC
- 胆固醇（CHO）是脂质的组成成分之一
- 在血浆中LCAT(卵磷酯胆固醇酰基转移酶)作用下生成胆固醇酯CE
- 70%胆固醇酯CE，30%游离胆固醇FC，称总胆固醇TC
【参考值】合适水平：<5.20mmol/L；边缘水平：5.23~5.69mmol/L；升高： >5.72mmol/L（不特异，不灵敏，作为危险因素）
【适应症】动脉粥样硬化危险的早期判断；降脂药物治疗的监测
#### 2. 三酰甘油测定 TG
- 甘油+3个脂肪酸
- 主要存在于前β脂蛋白和乳糜颗粒中
- 直接参与胆固醇及胆固醇酯的合成，为细胞提供能量和贮存能量
- *是动脉粥样硬化的重要因素之一*
- 血液中TG的半寿期仅为5-15分钟，进食高脂、高糖、高热量饮食后外源性TG增高，以乳糜颗粒的形式存在
- 健康人进食后12小时甘油三酯恢复至参考范围
【参考值】合适水平：0.56~1.70mmol/L；边缘水平： ≤2.30mmol/L；升高：＞2.30mmol/L
【适应症】
- 动脉粥样硬化危险性的早期判断
- 高脂蛋白血症的诊断与分型
- 降脂饮食与药物治疗的监测

## （二）血清脂蛋白检测
- 血脂在血液中存在、转运及代谢形式
- 脂类+载脂蛋白——脂蛋白

#### 1. 乳糜微粒测定 CM 
- 运输*外源性TG*的主要形式，餐后大量CM进入血液－乳糜血，正常进食12小时后阴性
#### 2. 高密度脂蛋白测定 HDL
① 颗粒最小，密度最大，主要由肝脏和小肠合成
② 蛋白质（ApoA1、ApoA2为主占90%）与脂质各占50%
③ HDL的功能之一是运输内源性胆固醇至肝脏处理，故有**抗动脉粥样硬化作用**
④ 通过HDL胆固醇（HDL-C）的含量间接反映HDL水平
#### 3. 低密度脂蛋白测定 LDL
① 是血清中携带**胆固醇**的主要颗粒
② 进入细胞内－ApoB100降解和胆固醇被水解－释放出游离TC－吞噬细胞摄取－泡沫细胞－胆固醇沉积－动脉粥样硬化斑块
③ 以LDL胆固醇（LDL-C）含量反映LDL水平
#### 4. 脂蛋白(a)测定
① 特殊的脂蛋白，结构类似LDL【促动脉粥样硬化】
② 与纤溶酶原有高度同源性【促血栓形成】
③ 是动脉粥样硬化和血栓形成的重要独立危险因子

## （三）血清载脂蛋白检测 Apo
血浆脂蛋白中的蛋白质部分
#### 1. 载脂蛋白AⅠ测定
- **HDL**的主要结构蛋白
- *Apo-AI 为临床常用检测指标*
- 可直接反映HDL的水平
- 与冠心病发病呈负相关，是诊断冠心病的一种较灵敏的指标
#### 2. 载脂蛋白B测定
- **LDL**中含量最多的蛋白质
- Apo-B可以直接反映LDL的水平，其增高与动脉粥样硬化、冠心病呈正相关，也是冠心病的危险因素
#### 3. 载脂蛋白AⅠ/载脂蛋白B比值测定
- 代替HDL/LDL比值作为判断动脉粥样硬化的指标
- 正常参考值：**1-2**，<1对诊断冠心病的价值较大


> [!NOTE] 临床意义综合分析
> - 血脂和脂蛋白含量与**动脉粥样硬化**的发生关系密切。TC、TG、LDL-C、VLDL-C、apo-B升高，可促进动脉粥样硬化，**与冠心病的发病率呈正相关**
> - **HDL-C、apoA**能促进胆固醇的清除，防止动脉粥样硬化，与冠心病的发病率呈负相关

# 第五节  心肌酶和心肌蛋白检测
## （一）心肌酶检测
#### 1. 肌酸激酶测定 CK
- 又称肌酸磷酸激酶CPK，主要存在于骨骼肌、心肌，其次存在于脑、平滑肌等细胞的胞浆和线粒体中
【参考值】
- 酶偶联法 37℃时：男性38~174U/L，女性26~140U/L
- 肌酸显色法：男性15~163U/L，女性3~135U/L
- 连续监测法：男性38~174U/L，女性26~140U/L
- 新生儿为成人的3~5倍，婴儿为3倍，儿童和青少年相当于成人的上限。
【临床意义】
- **增高**
	1. ==急性心肌梗塞（AMI）==：发病3-8小时增高，10-36小时达峰值，3-4天恢复正常，是*AMI早期诊断的敏感指标之一*
		① 如果在AMI病程中CK再次升高，提示心肌再次梗死
		② 发病8h内CK不增高，不可轻易排除AMI，应动态观察；发病24h的CK检测价值最大，此时CK应达峰值，如果CK小于参考值的上限，可排除AMI。但应除外CK基础值极低的病人和心肌梗死范围小及心内膜下心肌梗死等，此时即使心肌梗死，CK也可正常
	2. ==心肌炎和肌肉疾病==：如病毒性心肌炎、多发性肌炎、横纹肌溶解症等
	3. ==溶栓治疗==：有助于判断溶栓后的再灌注情况
		AMI溶栓治疗后出现再灌注，导致CK活性增高，使峰值时间提前。因此，CK水平有助于判断溶栓后的再灌注情况，但由于CK检测具有中度灵敏度，所以不能早期判断再灌注。如果发病后4h内CK即达峰值，提示冠状动脉的再通能力达40%~60%
	4. ==手术==
- **降低**：长期卧床、甲亢、激素治疗等
#### 2. 肌酸激酶同工酶测定
- CK分子含有两个亚单位M和B，即：CK1（CK-BB）、CK2（CK-MB）、CK3 （CK-MM）
	① CK-BB主要存在于脑、前列腺、肠和肺等组织；
	② CK-MB主要存在于心肌中；
	③ CK-MM主要存在于骨骼肌和心肌中。
- 正常人血清中以CK-MM为主，CK-MB少量（<总活性5%），CK-BB极微量。
【参考值】琼脂糖凝胶电泳法 CK-MM活性为94%-96%；CK-MB活性<5%；CK-BB极少或0
【临床意义】
① CK-MB主要存在于心肌中，其在AMI时的升高早于（3-6小时）总活力升高，且较灵敏，CK-MB活力≥3%即可诊断；*AMI时升高时间： CK-MB>AST>LDH1*
② CK-MM活性升高，是骨骼肌损伤的特异性指标；
③ CK-BB活性升高，见于缺氧性神经系统疾病（48~72小时脑脊液内升高）。
##### CK-MB
1. 急性心肌梗死CK-MB出现早，故对其早期诊断有重要价值，诊断的灵敏度高于总CK。
2. CK-MB对判断急性心肌梗死面积及预后也有帮助。CK-MB还可判断冠状动脉再通。
3. 心绞痛、心包炎、慢性心房纤颤、心脏手术、安装起搏器及冠状动脉造影时CK-MB水平升高。
4. 某些肌病和骨骼肌损伤时，如：肌营养不良，多发性肌炎，肌萎缩等血中CK-MB也可升高。

#### 4. 乳酸脱氢酶测定
- LDH由M和H两种亚基组成。有5种同工酶，广泛分布于许多组织 LD1-LD5
#### 5. 乳酸脱氢酶同工酶测定

	如果AMI病程中LDH持续增高或再次增高，提示梗死面积扩大或再次出现梗死。
2. 肝脏疾病，肺部疾病等
3. 肿瘤，白血病，恶性贫血等
4. 骨骼肌疾病
## （二）心肌蛋白检测

> 肌钙蛋白：三种亚单位：
> 	1. 肌钙蛋白T（TnT）
> 	2. 肌钙蛋白C（TnC）
> 	3. 肌钙蛋白I（TnI）
> 心肌肌钙蛋白T（cTnT），心肌肌钙蛋白I具有高度特异性；受损时从心肌中释放
#### 1. 心肌肌钙蛋白T测定
【参考值】cTnT：0.02~0.13 µg/L，诊断临界值>0.2 µg/L； >0.5 µg/L即可诊断
【临床意义】
1. 诊断AMI：**cTnT是诊断AMI的确定性标志物**
2. 判断微小心肌损伤：不稳定型心绞痛病人常发生微小心肌损伤，只有cTnT才能确诊
3. 预测血液透析病人心血管事件：cTnT增高提示预后不良或发生猝死的可能性增大
特征：灵敏度、特异性高于CK；窗口期较长
#### 2. 心肌肌钙蛋白I测定
【参考值】cTnI：<0.2 µg/L； 诊断临界值>1.5 µg/L
#### 3. 肌红蛋白测定 Mb
- 广泛存在于肌肉组织中，有贮氧输氧的功能。
- 当心肌、骨骼肌受损时，血中Mb升高。
【参考值】
- 定性：阴性
- 定量：ELISA法；50~85 µg/L，诊断临界值>75 µg/L
【临床意义】Mb是AMI发病后标志物
【指标评价】
- 优点：AMI最早的标志物
- 缺点：特异性较差
#### 4. 脂肪酸结合蛋白测定 FABP
- 它可结合长链未脂化的脂肪酸，故是一种细胞内重要的脂肪酸载体蛋白
- 在心肌和骨骼肌中含量最丰富
【参考值】<5 µg/L
【临床意义】
- 急性心肌梗塞：症状发作后3小时，FABP血浆浓度开始升高，但于12~24小时内恢复正常水平，**故与Mb都是AMI早期诊断的标志物**
- 其他：骨骼肌损伤和肾功能衰竭

## （一）甲状腺激素检测
#### 1. 血清游离T3, T4
- 不受血TBG变化的影响，直接反应甲状腺功能状态，敏感性、特异性高于总T3、T4
- T4在肝脏和肾脏中经过脱碘转变为T3
#### 2. 血清总T3, T4
- 易受血TBG变化的影响
- TBG又受妊娠、雌激素、病毒性肝炎等因素影响而升高；受雄激素、低蛋白血症、泼尼松等影响而下降
- 总T3：常与TT4的改变平行，甲亢初期及复发早期，TT3上升快于TT4，是早期GD、治疗中疗效观察及停药后复发的敏感指标
- 总T4：判定甲状腺功能最基本的筛选指标
#### 3. 反T3（rT3）
- rT3在血中含量甚微，主要来自于T4脱碘，生理活性很低
【参考值】0.2-0.8nmol/L
【临床意义】
1. 甲状腺疾病：甲亢 ：rT3升高；甲减： rT3降低；甲状腺激素替代治疗的用药观察
2. 非甲状腺疾病：心肌梗死、肝硬化、糖尿病等，rT3可升高，T3/ rT3比值降低。
#### 4. 血清甲状腺结合球蛋白 TBG
- TBG为血液中甲状腺素的主要结合蛋白，可与70%~75%的T3、T4结合，运输T3、T4到靶细胞
【参考值】15-34 mg/L
【临床意义】
- 增高：遗传性TBG增多症、甲减；
- 降低：遗传性TBG减少症、严重营养不良、甲亢等

// =====================================================
// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
