# 视诊
###### 1. 观察腹部外形和对称性（切线）
腹部外形平坦/膨隆/凹陷【舟状腹、蛙状腹】
###### 2. 测量腹围
###### 3. 观察呼吸运动（切线）
女-胸式；男/儿童-腹式
###### 4. 观察胃肠型和蠕动波（切线）
###### 5. 观察上腹部搏动（切线）
由腹主动脉搏动传导，正常人较瘦者可见
###### 6. 观察腹壁静脉（如静脉曲张，检查血液方向）
- 以一手食指、中指放于曲张静脉，一只手指向外滑动排空这段静脉。
	- 若迅速充盈，则血流方向从放松一侧向紧压一侧。
- 下腔静脉阻塞--腹壁浅静脉怒张--血流由下向上
- 上腔静脉阻塞
- 门脉高压--曲张静脉以脐为中心向四周伸展
###### 7. 观察腹壁其他情况
皮肤（皮疹、色素、腹纹、瘢痕）；脐（部位、形状、有无分泌物）
###### 8. 观察腹股沟区有无肿大/淋巴结
咳嗽--增加腹压

### 报告结果：
①腹部外形平坦/膨隆/凹陷，②呼吸运动以胸式/腹式呼吸为主，③有/无胃肠型和蠕动波，④有无上腹部搏动，⑤有无腹壁静脉曲张，⑥腹部皮肤……，脐部……，⑦腹股沟区有无肿大或肿大淋巴结，⑧腹围为__cm

# 听诊
###### 1. 听诊肠鸣音
- 置于脐旁稍向下压
	- 正常：每分钟4-5次
	- 活跃：>10次/分
	- 亢进：活跃+音调↑
	- 减弱：数分钟1次
	- 消失：3-5min无
###### 2. 听诊腹部血管杂音
静脉性杂音
腹主动脉（上腹部中线处，脐上5cm）
肾动脉（上腹部左右两侧）
髂动脉（下腹部左右两侧）
股动脉（双侧腹股沟中部）
###### 3. 听诊肝区和脾区摩擦音
肝区-右侧
脾区-左侧（左腋中线约7-11肋水平）
###### 4. 腹部振水音
- 听诊位置：上腹部
- 右手四指并拢于上腹部（胃）腹壁向下冲击触诊，气液撞击声

### 报告结果
①肠鸣音_次/分，是否正常，有无活跃/亢进/减弱/消失；②有无腹主动脉、肾动脉、髂动脉和股动脉杂音，有无静脉性杂音；③有无闻及肝区和脾区摩擦音；④有无腹部振水音

# 叩诊
###### 1. ★叩诊全腹
左下腹开始，逆时针至右下腹再到脐部；绝大部分为鼓音
###### 2. ★叩诊肝脏上下界
- 从*右锁骨中线* 肺野内清音区（*第2肋间*）开始向下叩诊，清音-->浊音：==肝上界==
- 从腹部鼓音区沿右锁骨中线向上叩诊，鼓音-->浊音：==肝下界==
- 测量并报告肝上下径（9-11cm）
###### 3. ★检查肝脏有无叩击痛
- 左手置于肝浊音区，右手*握拳*轻叩左手背。是否有退缩反应等
###### 4. 叩诊脾脏
- 左腋中线上
###### 5. 检查脾脏有无叩击痛
###### 6. 胃泡鼓音区叩诊
- *左锁骨中线*上，自上而下垂直叩诊，清音-->鼓音：==胃泡鼓音区上界==
- 水平方向，鼓音-->浊音：==左右界==
###### 7. ★★检查移动性浊音
- 自腹中部脐平面向受检者左侧叩诊，鼓音-->浊音时固定板指，嘱*受检者右侧卧位*，再叩诊：
	- 浊音变为鼓音：移动性浊音阳性
- 同样方法向右侧叩诊。
- 腹膜腔内游离液体>1000ml：可查出移动性浊音
###### 8. 检查肾区有无叩击痛-肋脊角
- 被检者坐位（抱膝），拳击叩诊
###### 9. 液波震颤
- 一手掌面贴于患者一侧腹壁，另一侧四指并拢屈曲，另一人手掌尺侧缘压于脐部腹中线上（目的：防止腹壁本身的震动传至对侧）
- 指端轻轻叩击对侧腹壁，如有大量液体存在，则贴于腹壁的手掌有被液体波动冲击的波动感
###### 10. 膀胱叩诊
- 腹中线上，自脐部开始从上往下叩至耻骨联合，至鼓音变为浊音-->充盈膀胱的上界
	- 板指与腹中线垂直
	- 膀胱充盈：凸面向上的半圆形浊音区
### 报告结果
①全腹叩诊大部分区域呈鼓音，②右锁骨中线上肝上下径的实测距离为9-11cm，③肝脏有无叩击痛，④左腋中线上脾脏长度为4-7cm，⑤胃泡鼓音区是否存在，有无明显缩小/消失，⑥移动性浊音阴/阳性，⑦肋脊角叩击痛阳性/阴性，⑧液波震颤阴/阳性，⑨耻骨联合上方叩诊无浊音/半圆形浊音区

# 触诊
被检者做平静腹式呼吸
自左下腹逆时针检查，左下--左上--上腹--右上--右下--下腹--脐部
###### 1. 浅触诊全腹部
- 四指并拢，右手全手掌平放于腹壁；掌面轻压，下压腹壁1cm，检查腹壁紧张度等
###### 2. 深触诊全腹部
- 掌指关节和远端手指掌面深压腹壁
- 下压腹壁2cm以上，可达4-5cm
- 检查*反跳痛*，出现压痛时，用并拢的三个手指压于原处稍停片刻，感觉趋于稳定后迅速抬起，是否疼痛骤然加重
- **麦氏点**检查：脐与右髂前上棘连线中外1/3交界处
---
==脏器触诊==：做较深的腹式呼吸
检查者右手四指并拢，掌指关节伸直，与*肋缘平行* 置于脐水平或右侧髂窝，用*食指桡侧*进行触诊
**快压慢弹**：
- 呼气-手指压向腹壁深部触诊肝脏边缘--在腹壁下陷前提前下压+上滑
- 吸气-手指缓慢抬起--手指上抬速度落后于腹壁的抬起

正常人深吸气：
- 右-肝脏肋下不超过1cm
- 左-剑突下不超过剑突基底部与脐连线上1/3

---
###### 3. 右锁骨中线上触诊肝脏右叶
###### 4. 前正中心线上触诊肝脏左叶
###### 5. 检查肝颈静脉回流征
- 仰卧位，张口平静呼吸
- 检查者右手掌紧贴于上腹部肝区，逐渐加压持续10s或更长，观察颈静脉怒张
- 右心衰：颈静脉持续而明显怒张，在停止压迫肝脏后颈静脉充盈最高点迅速下降大于4cm水柱
###### 6. 触诊脾脏
平卧-->右侧卧位；一般不触及
- 检查者左手绕过患者腹前方，手掌置于其左胸下部9-11肋，将脾脏从后向前托起并限制胸廓运动
- 右手掌平放于脐部，与左肋弓大致垂直

###### 7. 触诊胆囊
正常不触及
###### 8. 检查墨菲征 Murphy征
左手掌平放于右胸下部，拇指指腹勾压右肋下胆囊点，缓慢深吸气，吸气过程胆囊触痛--吸气中止
###### 9. 触诊肾脏
双手触诊；仰卧位/坐位；两腿屈曲+腹式呼吸
左手越过患者前方从后面托住左腰--右手掌横置于左腰部--双手触诊；夹持感
###### 10. 尿路压痛点检查
“前三后二”
腹部：三指
- 季肋点
- 上输尿管点
- 中输尿管点
背部：拇指
- 肋脊点
- 肋腰点
###### 11. 膀胱触诊
单手滑行触诊
###### 12. 触诊腹股沟
滑动触诊法

### 报告结果
①腹部是否柔软，有无腹壁紧张；②腹部有无压痛/反跳痛及其部位，有无触及腹部包块；③肝脏肋下及剑突下是否触及；④肝颈静脉回流征阴/阳性；⑤脾脏肋下是否触及；⑥胆囊是否触及；⑦Murphy征阴/阳性；⑧肾脏是否触及；⑨尿路压痛点-反应；⑩膀胱是否触及

// =====================================================
// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
