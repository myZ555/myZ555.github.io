### （二）评估胎儿健康的技术
1. 确定是否为高危儿
	1. 孕龄<37周或≥42周
	2. 出生体重<2500g或>4000g
	3. 小于孕龄儿或大于孕龄儿
	4. 出生1分钟内Apgar评分0~3分
	5. 产时感染
	6. 高危妊娠产妇的新生儿
	7. 手术产儿
	8. 新生儿的兄姐有严重的新生儿疾病史
2. 胎儿宫内状况的监测
	1. 早期：是否正常妊娠、能否妊娠、NT（胎儿颈项透明层）
	2. 中期：胎儿发育状况、是否畸形
	3. 晚期：胎动监测、电子胎心监护、预测胎儿宫内储备能力、NST判读
### （三）孕期营养与体重管理
母体：贫血、子痫前期、妊娠期糖尿病、产后出血
胎儿：流产、早产、难产、死胎、胎儿畸形、低出生体重儿、巨大儿
### （四）产科合理用药

【原则】
1. 必须有明确的指证，避免不必要的用药
2. 在医生指导下，选择对胎儿安全的药物
3. 单选，避免联合
4. 避免使用新药，对胎儿影响不明的药物
5. 严格剂量及用药时间；及时停药
6. 若病情许可，尽量推迟到孕中晚期用药

【药物的妊娠分类（FDA）】
1. A类：临床对照研究，未发现对妊娠各期胎儿有损害
2. B类：临床对照研究，对妊娠各期胎儿危害的证据不足或不能证实
3. C类：动物实验发现药物致胎仔畸形或死亡，无人类对照研究，不能排除危害，使用时需谨慎权衡
4. D类：对人类胎儿有危害，临床非常需要且无替代药，**应充分权衡利弊后用**
5. X类：动物和人类胎儿均证实有明显的致畸作用，**妊娠期禁用**

【暴露时间】
卵裂期：受精后2周内，药物对胚胎影响为“全或无”--致畸的相对不敏感期
胚胎期：受精后3-8周，致畸高度敏感期
胎儿期：受精后9周-足月，

主要致畸作用发生在受精后8周内的为胚胎病，发生在8周后为胎儿病
### （五）孕期常见症状及其处理
1、消化道症状
2、贫血
3、腰背痛
4、下肢及外阴静脉曲张
5、下肢肌肉痉挛
6、下肢水肿
7、痔疮、便秘，等

const cur = dv.current();
const filePath = cur?.file?.path ?? null;

function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

} catch(e) { outRaw = []; }

function createCard(title, paths, emptyMsg, icon){
