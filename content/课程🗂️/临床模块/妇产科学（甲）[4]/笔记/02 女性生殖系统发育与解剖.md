### （二）内外生殖器

#### 1. 外生殖器
1. 阴阜：耻骨联合前隆起的脂肪垫、青春期开始生长倒三角形阴毛
2. 大阴唇：外侧面为皮肤，大阴唇内侧湿润似黏膜；外伤后易形成血肿；未产女性两侧大阴唇自然合拢，产后分开，绝经后萎缩
3. 小阴唇：大阴唇内侧的皮肤皱襞，表面湿润，色褐无毛，富含神经末梢。前端融合形成前后两叶，前叶形成阴蒂包皮，后叶会合形成阴蒂系带
4. 阴蒂：小阴唇顶端下方、海绵体构成可勃起。分三部分：头体脚，阴蒂头富含神经末梢
5. 阴道前庭：菱形区域，前-阴蒂，后-阴唇系带，两侧-小阴唇。阴道口与阴唇系带间-舟状窝/阴道前庭窝，经产妇消失
	1. 前庭球：球海绵体，前庭两侧
	2. 前庭大腺：巴氏腺，大阴唇后部，左右各一；性兴奋时,分泌粘液起润滑作用。腺管细长易受感染影响。正常不能触及--前庭大腺脓肿
	3. 尿道外口：阴蒂头后下方、后壁有一对并列腺体称为尿道旁腺
	4. 阴道口和处女膜：阴道口位于尿道外口后方的前庭后部，周缘覆有处女膜
#### 2. 内生殖器
1. 阴道
	- 通道 ，上端包绕宫颈；子宫颈与阴道间的圆周状隐窝—阴道穹隆；后穹隆与子宫直肠陷凹相邻（重要临床意义-穿刺、引流或手术入路）
	- 粘膜为复层鳞状上皮，受性激素影响有周期性变化，阴道壁富有静脉。
	- 功能：性交器官，经血排出与胎儿娩出通道
2. 子宫
	- 空腔脏器， 胚胎、胎儿发育场所和产生月经器官
	- 分为宫体和宫颈，连接部为子宫峡部(isthmus)
		- 体/颈：青春期1:2，生育期2:1，绝经后1:1
		- 宫体
			- 浆膜层：脏层腹膜；膀胱子宫陷凹 直肠子宫陷凹
			- 肌层：外层纵行 中层交叉 内层环行
			- 内膜层：性激素最主要的靶组织，周期性变化
		- 子宫峡部
			- 解剖学内口 上端因解剖上狭窄
			- 组织学内口 宫腔内膜转为宫颈内膜
		- 宫颈：主要由结缔组织构成，宫颈管粘膜为单层高柱状上皮，受性激素影响有周期性变化。宫颈阴道部由复层鳞状上皮覆盖。
	- 子宫大小、宫体与宫颈比例受性激素影响。
	- *子宫韧带*：阔韧带、圆韧带、主韧带、宫骶韧带 
3. 输卵管
	- 空腔管道，受精部位、输送受精卵
	- 分为间质部、峡部、壶腹部、伞部（拾卵）【壶腹-宫外孕常见】
	- 由浆膜层、肌层、粘膜层组成，受性激素影响有周期性变化
4. 卵巢
	- 功能：【生殖】产生和排出卵子；【内分泌】合成和分泌甾体激素
	- 毗邻：以卵巢系膜与阔韧带后叶相连（卵巢门）；内侧以卵巢固有韧带与子宫相连；外侧以卵巢悬韧带（骨盆漏斗韧带）与盆壁相连
	- 自表面向深部依次为生发上皮、白膜、皮质（卵泡）、髓质

### （三）血管、淋巴及神经
1. 动脉
	1. 卵巢动脉：来自腹主动脉（左侧肾动脉）
	2. 子宫动脉：来自髂内动脉前干分支（与输尿管交叉）
	3. 阴道动脉：髂内动脉前干分支
	4. 阴部内动脉：髂内动脉前干终末支
2. 静脉：与动脉伴行，数目较多，常形成静脉丛
3. 淋巴
	1. 外生殖器：腹股沟前淋巴结、腹股沟深淋巴结
	2. 内生殖器：①髂淋巴组(髂内、髂外、髂总)；②骶前淋巴组；③腰淋巴组（腹主动脉旁)
4. 神经
	1. 外生殖器：主要来自阴部神经
	2. 内生殖器：交感-腹主动脉前神经从；副交感-骨盆神经丛
> 子宫平滑肌有自主节律活动，完全切除其神经后仍能有节律性收缩，能完成分娩
### （四）骨盆及骨盆底
1. 骨盆的组成
	1. 骨盆的骨骼：骶骨、尾骨、两块髋骨
	2. 骨盆的关节：耻骨联合、骶髂关节、骶尾关节
	3. 骨盆的韧带：骶结节韧带、骶棘韧带
2. 骨盆的分界：以耻骨联合上缘、髂耻线和骶岬上缘连线为界,分为:真骨盆和假骨盆；真骨盆是胎儿娩出的骨产道
3. 骨盆的类型：女型、扁平型、类人猿型、男型
4. 骨盆底：封闭骨盆出口的软组织，由多层肌肉和筋膜组成
	1. 外层：会阴浅筋膜、球海绵体肌、坐骨海绵体肌、会阴浅横肌和肛门外括约肌
	2. 中层（泌尿生殖膈）：上、下两层筋膜和一层肌肉（会阴深横肌和尿道括约肌）
	3. 内层（盆膈）：肛提肌（每侧包括耻尾肌、坐尾肌、髂尾肌）、上下两层筋膜
> 会阴: 阴道口与肛门之间的软组织；广义指封闭骨盆出口的所有软组织

const cur = dv.current();
const filePath = cur?.file?.path ?? null;

function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

const container = document.createElement("div");

function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);

                    // 回退尝试按显示名打开

            });

            li.appendChild(a);

        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}
