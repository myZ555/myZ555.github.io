# (二) 颞下颌关节紊乱病 TMD
- 口腔颌面部常见病，好发于20-40岁青壮年，女性多见
1. 病因不清，多因素共同作用 
2. 临床表现
	- 三个阶段：功能变化阶段--结构变化阶段--关节器质性破坏阶段
	- 三大症状：
		1. 下颌运动异常
			1. 开口度异常（正常3.5-4.2cm）
			2. 开口型异常
		2. 疼痛（开口运动和咀嚼时关节区或关节周围肌群疼痛弹响音）
		3. 弹响
3. 诊断依据
4. 临床分类分型
	1. 咀嚼肌紊乱疾病（咀嚼肌处持续性疼痛，开口困难）
	2. 关节结构紊乱疾病
	3. 炎性疾病类
	4. 骨关节病类
---
### 咀嚼肌紊乱疾病

- 病因：
	- 咀嚼肌直接损伤（外伤、微小创伤、精神紧张、寒冷刺激、紧咬牙、夜磨牙症）
	- 咀嚼肌过度活动（开口过大、长时间大张口）
	- 咀嚼肌过度伸展（不良修复体-颌间距增大）
	- 咀嚼肌过度收缩-肌疲劳（牙槽骨吸收/双侧后牙缺失）
- 临床表现：
	- 一处或多处咀嚼肌出现*局部持续性疼痛*，耳部或耳前区钝痛
	- 关节区无压痛，单纯的肌筋膜痛无关节弹响
	- 晨起时疼痛轻微，在一天中逐渐加重，咀嚼与大张口时疼痛加剧
	- 下颌运动受限，单侧肌筋膜痛，*开口型偏向患侧*
- 诊断及治疗原则：
	- 诊断性地封闭咀嚼肌神经和肌肉可使疼痛消失；颞下颌关节内无病理改变
	- 保守治疗为主
### 结构紊乱疾病
- 病因：
	- *关节盘移位*；损伤相关
- 临床表现：
	1. 可复性关节盘前移位：弹响、疼痛
	2. 不可复性：无弹响，急性期开口受限，慢性期缓解
	3. 关节半脱位：开口度过大，超过 45mm；大张口过程中有一个越过关节结节的跳跃，同时产生弹响，并出现短暂的下颌运动停顿
- 诊断及治疗原则：
	- 可复性盘前移位以保守治疗为主
	- 不可复性盘前移位早期可通过患者的下颌运动使关节盘复位，如不成功可用手法复位
	- 关节半脱位以保守治疗为主，限制大张口

### 炎性疾病
- 病因：*颞下颌关节滑膜以及关节囊的非感染性炎症反应*
	- 原发性-原因不明
	- 继发性-外伤/邻近组织蔓延/骨关节病/自身免疫反应
- 临床表现：
	- 关节区疼痛明显，下颌运动时疼痛加剧
	- 关节腔内有渗出物，可出现*波动性肿胀*
- 诊断及治疗原则：
	- 关节镜可见急性期滑膜发红，存在大量的血管，血管排列紊乱
	- 以保守治疗为主
### 骨关节病
- 病因：颞下颌关节组织发生磨损与变质并在关节表面形成新骨的*退行性病变*
	- 原发性-原因不明
	- 继发性-局部因素
- 临床表现：
	- 急性期可出现关节疼痛，与退行性改建和滑膜炎症有关
	- 慢性期可无明显关节疼痛，可出现下颌运动受限
- 诊断及治疗原则：
	- 以保守治疗为主

# (三) 颞下颌关节脱位

### 1. 颞下颌关节急性前脱位
- 病因：
	1. 大张口时，咀嚼肌紊乱或关节结构紊乱
	2. 解剖因素，关节结节过低或关节前斜面过陡
	3. 外力
- 临床表现：
	1. 下颌运动失常，呈开口状，流涎，语音不清，前牙呈反合，开合
	2. 下颌前伸，面颊变平，脸型相应变长
	3. 耳屏前方触诊有凹陷，在颧弓下方可能触及脱位的髁状突
- 治疗
	1. 原则：及时复位，限制下颌的运动
	2. 复位方法：手法复位
	3. 限制下和运动：颅颌绷带固定下颌运动20天左右，开口度不宜超过1cm
### 2. 颞下颌关节复发性脱位（习惯性脱位）
- 病因：
	1. 急性前脱位后未给予适当的治疗
	2. 长期的关节囊松脱，翼外肌功能亢进，各附着韧带松脱
	3. 老年人，肌张力失常，韧带松弛，关节结节低平
- 临床表现：
	- 反复出现颞下颌关节脱位
- 诊断及治疗：
	- 手法复位、颅颌绷带固定后效果不佳者，可进行关节囊内硬化剂治疗
	- 可行手术治疗，如关节囊及韧带加固术
### 3. 颞下颌关节陈旧性脱位
- 病因：
	- 急性前脱位或复发性脱位未及时治疗，关节长时间处于脱位状态
	- 脱位的髁突及关节盘周围纤维结缔组织增生，关节窝内也可出现纤维结缔组织增生
- 临床表现：
	- 颞下颌关节和咀嚼肌无明显疼痛，下颌有一定的活动度
- 诊断及治疗：
	- 在关节镜下行关节复位术
	- 或通过手术将髁突、关节结节之间的纤维结缔组织剥离，关节窝修整后撬动关节复位
# (四) 颞下颌关节强直
### 1. 颞下颌关节内强直
- 病因：
	1. 炎症，中耳炎，骨髓炎，关节周围炎等
	2. 血源性化脓性关节炎
	3. 外伤（目前多见）
	4. 类风湿性关节炎（少见）
- 临床表现：
	1. 开口困难（其程度因强直的性质而不同）
	2. 面下部发育障碍畸形（小颌畸形、偏颌畸形）
	3. 颌关系紊乱
	4. 髁状突活动减弱或消失
- 诊断及治疗原则：
	- X-ray检查：平片可见三种类型
		1. 【纤维性强直】关节间隙模糊，正常的关节结构消失，髁突、关节结节及关节窝骨密质有不规则破坏
		2. 【骨性强直】关节间隙消失，髁突与颞骨关节面融合成致密团块而呈骨球状。
		3. 髁突与颞骨关节面、乙状切迹、喙突、颧弓融合成骨球，下颌升支与颧弓完全融合而呈“T”形。波及下颌切迹，正常喙突，颧弓，下颌切迹影像消失
	- 以手术治疗为主
		- 术式：
			1. 髁状突切除术【纤维性强直】
			2. 关节成形术【骨性强直】
		- ==手术原则==
			1. 截开部位：尽可能在*下颌升支的高位*。有利于关节活动；常选择的截开部位有：
				1. 髁状突颈部截开形成小关节--适用于骨粘连范围较小且局限于髁突，下颌切迹尚存的病例
				2. 下颌切迹以下、下颌孔以上的部位截开--适用于骨粘连部位大，下颌切迹狭小或已消失的病例。若粘连范围更广，部位可再向下
			2. 骨断面处理：尽可能修成较小圆形骨突，形成点-面接触，利于下颌运动
			3. 保持截开的间隙：2类意见
				1. 广泛切除：1cm以上，避免再发
				2. 0.5-1cm：在间隙内将组织或替代品置入
			4. 双侧关节内强直的处理：先做较困难一侧；最好一次完成；间隔<2w
			5. 手术年龄问题：2类意见
				1. 儿童期患病早期手术，尽早恢复咀嚼功能，以利于下颌及面部发育
				2. 12-15岁以后手术，儿童成骨作用旺盛，术后不坚持开口训练易复发，复发后手术更难
### 2. 颌间挛缩（关节外强直）
- 病因：
	- 坏疽性口炎（走马疳）以前多见
	- 损伤（骨折/火器伤）目前多见
	- 颜面部重度烧伤
	- 口腔内手术处理不当形成瘢痕
	- 放射治疗后局部软组织广泛纤维变性
- 临床表现：
	1. 开口困难，其程度因关节外瘢痕粘连的程度不同
	2. 口腔或面部瘢痕挛缩或缺损畸形
	3. 髁状突活动减弱或消失
- 诊断及治疗原则：
	- X-ray检查：
		- 关节间隙和髁状突清楚，关节结构无明显破坏。有时可见上、下颌间隙变窄，有密度增高或骨性融合，骨性颌间挛缩。
	- 手术治疗： 
		- 切断和切出颌面挛缩的瘢痕，凿开颌间粘连的骨质，恢复开口度。
		1. 颌间瘢痕区较局限，可以从口内切除瘢痕，创面游离植皮。
		2. 颌间瘢痕广泛，口内手术困难，可以从颌下进路，行口内外贯通手术，彻底切除瘢痕。根据不同情况选用带蒂或不带蒂的皮瓣转移修复创面
		- 术中均需被动开口至最大位，并且术后保持在开口位，直到拆线，以防收缩

// =====================================================
// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
