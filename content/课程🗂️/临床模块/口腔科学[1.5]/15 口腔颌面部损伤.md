# ★（一）概述
==口腔颌面部损伤的特点==：
1. *口腔颌面部血供丰富*
	- 损伤后出血多--及时止血，防治失血性休克
	- 组织反应重且易形成血肿--可因肿胀、血肿压迫呼吸道而引起窒息
	- 组织抗感染能力强--彻底清创后，一期缝合时间可延长至48小时
	- 组织愈合能力强--清创时尽可能保留组织，避免缺损畸形
2. *口腔颌面部的腔、窦多*，如鼻腔、口腔、鼻窦等，腔、窦内常存在一定数量的病原菌--感染
	- 清创时尽早关闭与腔窦相通伤口，预防感染
3. 颌骨上有*牙*，颌骨骨折发生骨折段移位时，会引起*咬合关系错乱*和咀嚼功能障碍。【咬合关系错乱是诊断颌骨骨折的重要依据之一】
4. 口腔是*消化道的入口*，损伤后常妨碍正常进食，需选用正确的进食方法和食物
5. 口腔颌面部又是*呼吸道的上端*，损伤时最容易发生机械性阻塞
6. *多器官分布*；鼻部、唇部、舌、睑部、眶部和 颊部开放性损伤如处理不当，伤口愈合后可发生不同程度的*组织和器官移位*、变形以及瘢痕挛缩畸形
7. 颌面部有*腮腺、面神经和三叉神经*等组织
8. 严重的颌面部损伤常合并*颅脑损伤*
# （二）口腔颌面部损伤的急救处理
### 1. 解除窒息
###### 1. 分类
1. ==阻塞性窒息==：①异物阻塞；②组织移位；③气道狭窄；④活瓣样阻塞（受伤的黏膜/软组织盖住咽门）
2. ==吸入性窒息==：直接把血液、唾液、呕吐物或异物吸入气管/支气管/肺泡
###### 2. 临床表现
1. 前驱症状：烦躁不安、出汗、鼻翼扇动、吸气长于呼气、或出现喉鸣；
2. 严重：**三凹体征**（吸气时胸骨上窝、锁骨上窝、肋间隙深陷），呼吸急促而表现；
3. 脉弱、脉快、血压下降、瞳孔散大
###### ★3. 急救处理
1. 根据原因采取相应的措施
	- 尽早清除异物，建立通道
	- 采取侧卧或俯卧位
2. 咽部肿胀压迫呼吸道（紧急行环甲膜切开或气管切开术）
### ★2. 止血
> 口腔颌面部血供丰富，伤后出血较多
> - ==动脉出血==呈鲜红色，速度快，呈与心搏一致的间歇性喷射
> - ==静脉出血==呈暗红色，速度较慢，呈持续涌出状
> - ==毛细血管出血==也多呈鲜红色，由伤口缓缓渗出
> - ==深部出血==如果引流不畅，常在出血部位形成血肿


1. 指压止血：将主要动脉的近心端压迫于附近的骨骼上；![Pasted image 20251226101205](Pasted%20image%2020251226101205.png)
2. 包扎止血：头皮/颜面等处的毛细血管和小动静脉出血
3. 填塞止血：有组织缺损和洞穿性伤口者。敷料填塞于伤口内，并用绷带加压包扎
4. 结扎止血：【最为确切的止血方法】伤口内结扎出血血管或在远处结扎出血动脉的近心端
5. 药物止血：药物与出血创面直接接触，并用纱布加压包扎【局部+全身】
### 3. 伤口的包扎
- 作用：压迫止血、暂时性固定、保护创面、缩小创面、减少污染、减少唾液外流、镇痛
- 方法：三角巾风帽式包扎法、单眼包扎法、头颌绷带十字形包扎法、四尾带包扎法
### 4. 伤员的运送
### 5. 防治感染
1. 及时清创
2. 尽早包扎
3. 尽早使用抗生素
4. 破伤风抗毒素的使用

# （三）口腔颌面部软组织损伤

> 总体上讲，口腔颌面部血供丰富，具有伤口愈合快的有利条件，对有可能存活的软、硬组织，早期缝合、一期缝合的适应证更广

### ★1-闭合性损伤
###### 1. 擦伤
1. 表皮破损，并有*少量渗血和疼痛*，创面上常附有沙砾或其他异物
2. 治疗：主要是清洗创面和预防感染；可任创面暴露而无需包扎
###### 2. 挫伤
1. 没有皮肤开放伤口的软组织损伤，不仅是皮下组织，而且肌肉、骨膜和关节也可同时受伤。*局部肿胀或瘀斑*
2. 治疗：主要是止血、镇痛、预防感染、促进血肿吸收和恢复功能。*早期冷敷、局部加压，2天后热敷；并发感染则切开引流*
###### 3. 蛰伤
1. 蜂、蝎等昆虫所带毒刺所致的损伤。伤后局部红肿明显，疼痛剧烈
2. 治疗：先用镊子取出刺入皮内的毒刺，局部用 5% ~ 10% 氨水涂搽，以中和毒素
### ★2-开放性损伤
###### 1. 挫裂伤
1. 较大机械力量造成的钝器伤，伤口的特点是*创缘不整齐*，裂开较大，创缘周围的皮肤常有擦伤，并有发绀色坏死组织，还可伴发开放性骨折。
2. 治疗：清创时应刮除没有出血的坏死组织，修整创缘，彻底止血，常作减张缝合，充分引流
###### 2. 刺伤
1. 因尖锐的刀、锥、钉、笔尖、树枝等物的刺入而发生的，*伤口常为小入口，伤道深*，多呈盲管状，也可以是贯通伤
2. 治疗：清创时应彻底清除异物和止血，*应用抗生素*防治感染。
###### 3. 切割伤
1. 被锋利的刃器、玻璃片等所割，伤口特点是*边缘整齐*
2. 治疗：切割伤如无感染，缝合后可望一期愈合。有无血管、面神经及腮腺导管损伤
###### 4. 撕裂伤
1. 较大的机械力量造成的组织撕裂或撕脱。伤口特点是*边缘不整齐，出血多*，常*伴有肌肉、血管、神经和骨骼暴露，容易继发感染*。
2. 治疗：撕裂伤应及时清创、复位缝合。
###### 5. 砍伤
1. 较大机械力的利器如刀、斧等所致的损伤。伤口的特点是*创口较多，深浅不等*，多伴有挫伤、开放性粉碎性骨折等。
2. 治疗：处理方法是耐心地进行清创，探查神经、导管等重要结构的损伤，尽量保留可以保留的组织，复位缝合
###### 6. 咬伤
1. 被犬、鼠、猪等动物咬伤，被人和野生动物咬伤也不罕见。伤口特点是创缘常有咬痕，*组织常被撕裂，甚至撕脱*。犬咬伤可能导致*狂犬病*。
2. 治疗：彻底清洗创面，用含有抗生素的溶液湿敷，控制感染。
###### 7. 颜面部烧伤
- 特殊性：
	- ①头面部皮下组织疏松，血管、神经及淋巴管丰富，烧伤后组织反应大而快，水肿严重，渗出多。
	- ②颜面部凹凸不平，烧伤深度常不一致
	- ③在发生颜面部烧伤的同时，伤员常因热空气或烟雾吸入而发生呼吸道灼伤--呼吸困难/窒息危险
	- ④颜面部烧伤创面易受到口鼻腔分泌物或进食时的污染而感染，不易护理
	- ⑤伴有颈部烧伤时可出现颏、颈粘连以及颈部活动受限。
- 治疗：遵循全身与局部相结合的原则，并注意颜面部烧伤的特点

# （四）牙和牙槽突损伤
### 1. 牙挫伤
- 由直接或间接外力撞击所致，其主要特点是牙周膜和牙髓受损而产生充血、水肿
- 临床表现为受伤牙松动、疼痛、伸长，有牙周膜炎甚至牙髓炎的表现
- 治疗：对牙周膜损伤的牙，应作简单的结扎固定，并防止早接触。如牙髓受损，应作牙髓或根管治疗
### 2. 牙脱位
- 较大的暴力撞击下，牙部分或完全脱位，由于牙周膜撕裂，甚至从根尖孔进入牙髓的神经血管束也撕裂，临床上出现牙松动、倾斜、伸长和疼痛。完全脱位的牙脱离牙槽窝，或仅有软组织相连

# （五）颌骨骨折
### ★1. 上颌骨骨折
![Pasted image 20251226110101](Pasted%20image%2020251226110101.png)

|              |           |                                                      |
| ------------ | --------- | ---------------------------------------------------- |
| Le Fort Ⅰ型骨折 | 低位或水平骨折   | 骨折线从梨状孔下缘，经根尖下，水平向后过颧牙槽嵴，至上颌结节上方，延伸至两侧上颌骨翼上颌缝附近      |
| Le Fort Ⅱ型骨折 | 中位或锥形骨折   | 骨折线经过鼻骨、泪骨、眶底、颧上颌缝区达上颌骨翼上颌缝处                         |
| Le Fort Ⅲ型骨折 | 高位或颅面分离骨折 | 骨折线经过鼻骨，泪骨，眶内、下、外壁，颧额缝，颧颞缝，向后下止于上颌骨翼上颌缝，造成完全性颅与面骨的分离 |
#### 临床表现
1. **骨折段移位和咬合错乱**
	- 无论上颌骨为哪一型骨折，常同时伴有翼突骨折
	- 高位颅面分离的伤员，可见面部中段明显增长，同时由于眶底下陷，眼球移位，可出现复视
2. **眶区淤血、盘状脸**
	- 成为典型的“眼镜”症状
	- 眶下神经损伤时可出现上唇麻木症状
3. **颅脑损伤**
	- 颅脑损伤可出现意识模糊、肢体运动障碍等表现
	- 如发现鼻腔及外耳道出血，呈淡红色血水，应考虑可能发生脑脊液鼻漏或耳漏，是筛板骨折或合并颅前窝、颅中窝骨折的体征。

# （六）颧骨、颧弓骨折
### 临床表现与诊断
1. 骨折移位
2. 开口受限
3. 复视 （颧骨骨折移位后，眼内肌和外侧韧带也随之移位，或受骨折片的挤压，眼球失去支持而发生移位性复视）
4. 出血和淤血
5. 神经症状 
	1. 眶下神经：眶下区皮肤麻木
	2. 面神经颧支：患侧眼睑闭合不全

    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;

function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);

    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
