## 1. ★口腔单纯疱疹
- 自限性、易复发；单纯疱疹病毒HSV；出现簇集性小水疱
##### 1. 病因
- Herpes simplex virus, HSV
- 根据 HSV 的理化性状、生物学特征等不同，分为 1 型单纯疱疹病毒（HSV - 1）和 2 型单纯疱疹病毒（HSV - 2）两个血清型，引起口腔损害的主要为 *HSV - 1*
##### 2. 临床表现
###### 1. 原发性疱疹性龈口炎-儿童

| 病程  | 特征                                        |
| --- | ----------------------------------------- |
| 前驱期 | 疱疹患者接触史；潜伏期4-7d--经过1-3d，口腔黏膜、附着龈和龈缘广泛充血水肿 |
| 水疱期 | 口腔黏膜出现成簇小水疱-针头大小-壁薄透明，易溃破形成浅表溃疡           |
| 糜烂期 | 成簇小水疱溃破可引起大面积糜烂-造成继发感染，上覆假膜               |
| 愈合期 | 糜烂面缩小愈合                                   |

###### 2. 复发性疱疹性口炎-成人
- 复发部位在口唇附近
- 表现为成簇小水疱，痒、痛，破溃后结痂，有自限性，全身反应轻。
##### 3. 诊断
##### 4. 治疗
- 对症治疗疗法：抗病毒治疗【阿昔洛韦、伐昔洛韦】、清热解毒
- 抗菌消炎止痛，提高免疫功能【细胞免疫】
- *利巴韦林？*

## 2. ★口腔念珠菌病
##### 1. 病因
- 最常见口腔寄生菌，皮肤黏膜、消化道、阴道
- 条件致病菌-宿主或全身因素
- 器官移植、广谱抗生素、免疫抑制剂、激素、肿瘤放化疗、艾滋--发病率↑
- 全身系统病、肿瘤、黏膜慢性创伤
##### 2. 临床表现
1. **念珠菌性口炎**
	1. 急性假膜型
		- 新生儿婴儿、长期激素、HIV感染者、免疫缺陷者【新生儿鹅口疮/雪口病】
		- 口腔黏膜白色斑点斑片，剥离易出血
	2. 急性红斑型（萎缩型）
		- 大剂量抗生素（四环素、头孢类多见）
		- 舌背黏膜充血、乳头萎缩、灼痛。自觉症状为味觉异常或味觉丧失，口腔干燥，黏膜灼痛。
	3. 慢性红斑型（萎缩型）
		- 上颌托牙承托处黏膜（义齿上附着的念珠菌）
		- 充血水肿白色斑点斑膜，常合并口角炎
	4. 慢性增殖性（白念白斑）
		- 口角区颊粘膜，舌腹口底，舌背
		- 颗粒状增生或白色斑块
		- 病检-菌丝、上皮异常增生
2. **念珠菌性唇炎**
	- 糜烂型：下唇唇红中份长期存在鲜红色的糜烂面，周围有过角化现象，表面脱屑
	- 颗粒型：下唇肿胀，唇红皮肤交接处常有散在突出的小颗粒
3. **念珠菌性口角炎**（皲裂、充血、糜烂、渗出、结痂）

##### 3. 诊断
- 病史、临床特征、实验室检查
- 标本采集--黏膜拭子采样
- 涂片染色--方便快捷初筛方法可；阳性率低
- 培养定性：特异性强敏感性高，时间长
##### 4. 治疗
- 病因治疗（抗真菌药-氟康唑）
- 提高机体免疫功能
- 局部--保持口腔碱性环境
- 全身抗真菌药物

## 3. ★复发性阿弗他溃疡 RAU
- 又称复发性口腔溃疡。周期性、复发性、自限性。
##### 1. 病因
- 青壮年好发
- 复杂，个体差异
- 多因素（免疫、遗传、环境）
##### 2. 临床表现
- 青壮年好发，女性多于男性
- 周期性复发，具有自愈倾向
- 好发非角化区或角化不良区
- **分型**
	1. ==轻型==：初发时；好发于唇、舌、颊、软腭等无角化或角化差的黏膜；3-5个，散在分布，无明显全身症状与体征
	2. ==重型==：单个溃疡，大而深及黏膜下层直至肌层；边红肿隆起，基底较硬，但边缘整齐清晰，表面有灰黄色假膜或灰白色坏死组织；初始好发于口角；愈合后可留瘢痕
	3. ==疱疹样==：口炎型口疮，溃疡小，数目多，“满天星”；可融合成片，有全身症状
##### 3. 诊断
- 病史特点（复发性、局限性、自限性）+临床特征（红黄凹痛）
- 鉴别诊断：疱疹性口炎，创伤性溃疡，癌性溃疡（大而深且长期不愈）
- 活检？注意软腭复合体
##### 4. 治疗
- 目的：对症，减轻，减少，延长复发
- 原则：
	- 个体化治疗
	- 轻症，复发少--局部为主
	- 频繁，病情重--局部全身
	- 系统性疾病治疗，改善口腔状况

## 5. ★口腔白斑病 OLK
- 发生在口腔黏膜上以白色为主的损害，不能擦去，不能诊断为其他可定义的疾病；属于*癌前病变* 或*潜在恶性疾病范畴*。
##### 1. 病因
- 病因复杂；
- 吸烟--关系密切
- 局部刺激因素--刺伤口腔黏膜--形成口腔白斑病
- 白念珠菌是一个重要致病因素；白念白斑易发生癌变
##### 2. 临床表现
- 多见于中老年男性
- 发病部位以*颊粘膜* 最多，舌部次之
- 分型：*均质型*【斑块型/皱纹纸型】；*非均质型*【颗粒型/疣状型/溃疡型】
	1. 斑块型：口腔黏膜上出现白色或灰白色均质型较硬的斑块；边界清楚，触之柔软，不粗糙或略粗糙，周围黏膜多正常
	2. 皱纹纸型：口底及舌腹；病损呈灰白色或白垩色，边界清楚，表面粗糙
	3. 颗粒型：亦称颗粒-结节状白斑，颊黏膜口角区多见；颗粒状突起；多数可查到白念珠菌感染。
	4. 疣状型：灰白色；表面呈刺状或绒毛状突起，粗糙，质稍硬
	5. 溃疡型：增厚的白色斑块上有糜烂或溃疡
##### 3. 诊断
- 根据临床表现、病理检查，辅以脱落细胞检查及甲苯胺蓝染色，可对口腔黏膜白斑作出诊断。*注意癌变问题*
##### 4. 治疗
1. 去除刺激因素
2. 局部治疗
3. 药物治疗
4. 手术治疗
5. 物理治疗

## 6. 口腔扁平苔藓
- 常见的口腔黏膜慢性炎症性疾病
- 口腔黏膜病中仅次于复发性阿弗他溃疡的常见疾病
- 好发于中年女性
- 典型的病损特征为由*针头大小的小丘疹连成白色或灰白色细条纹*，类似皮肤损害的威克姆纹（Wickham 纹），条纹互相交织，呈网状、树枝状、环状、半环状或斑块状。
- 典型的皮肤或指（趾）甲损害可作为诊断依据

## 7. 性传播疾病的口腔表现
#### 1. ★艾滋病
1. 病因：人类免疫缺陷病毒（HIV）
2. 传播途径：AIDS 患者、HIV 携带者；**性接触传播、血液传播、母婴垂直传播**
3. 临床表现【口腔】
	1. ==口腔念珠菌病==：最常见；疾病早期；免疫抑制的早期征象
		- 发生于无任何诱因的健康成人。
		- 表现为红斑型或假膜型白念珠菌病，病情反复或逐渐加重。
		- *氟康唑*
	2. ==毛状白斑==：特殊口腔病损；不同寻常的病毒感染特征，抗生素治疗无效
		1. 双侧舌缘呈现白色或灰白色斑块，可蔓延至舌背和舌腹。
		2. 呈垂直皱褶，有的因过度增生而呈毛绒状，不能擦去。
		3. 毛状白斑的诊断除临床表现外，尚需检测证实病损内 EB 病毒的存在。
		- *维A酸和抗真菌药；阿昔洛韦*
	3. ==卡波西肉瘤==：罕见的血管恶性肿瘤
		1. 好发于腭部及牙龈，呈单个或多个紫红色、浅蓝色或褐色类似血管瘤的斑块或结节，病变初期较平，逐渐高出黏膜，可分叶、溃烂或出血。
		2. 病理特征为交织在一起的丛状梭形细胞，血管增生，淋巴细胞及浆细胞浸润。
		- 手术切除、烧灼刮治或冷冻治疗等
	4. ==口腔疱疹==：单纯疱疹为 HIV 感染者常见的疱疹病毒损害；口腔伴有生殖器疱疹；带状疱疹也为早期表现之一
		- 单纯疱疹：*阿昔洛韦*
	5. HIV 相关牙周病变
		- **牙龈线形红斑**：沿游离龈出现界限清楚的火红色线状充血，附着龈可有点状红斑。
		- 可有自发性出血或刷牙后出血
4. 治疗
	1. 提供健康教育和心理咨询
	2. 抗 HIV 治疗，坚持早期、持久、联合用药原则
	3. 免疫治疗与抗病毒治疗联合应用
	4. 针对机会性感染和肿瘤进行治疗
	5. 支持、对症治疗
#### 2. ★梅毒
1. 病因：**梅毒螺旋体**，人是梅毒的唯一传染源。
2. 传播途径：后天-性接触；先天-胎盘
3. 临床表现
	1. ==获得性梅毒【后天】==
		1. 一期梅毒：*硬下疳*和*淋巴结肿大*
			1. 唇硬下疳：常表现为巨唇，是一期梅毒常见的口腔损害，唇部下疳引起唇及周围组织肿胀，触之较硬，表面有黄色薄痂，可形成溃疡，下颌下淋巴结肿大。
			2. 舌硬下疳：表现为舌前部光滑，呈粉红色，覆盖灰白色假膜，触之稍硬，无痛，颏下及下颌下淋巴结肿大
		2. 二期梅毒：（螺旋体由淋巴系统进入血液循环）
			1. *梅毒性黏膜炎*
			2. *梅毒黏膜斑*：特征性损害；灰白色光亮微隆的斑块
		3. 三期梅毒
			1. 皮肤损害：结节性梅毒疹和树胶肿
			2. 口腔黏膜损害：三期梅毒*树胶肿*、*梅毒性舌炎*、*舌白斑*
	2. ==先天性梅毒【胎传】==
		- 经过与后天性梅毒相似，但不发生硬下疳
		- 标志性损害：
			1. 半月形切牙（哈钦森牙）：切牙的切缘比牙颈部狭窄，切缘中央有半月形缺陷，切牙之间有较大空隙。
			2. 桑葚牙：第一恒磨牙的牙尖皱缩，向中央偏斜，釉质呈多个小结节和坑窝凹陷
4. 治疗

// =====================================================
// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
