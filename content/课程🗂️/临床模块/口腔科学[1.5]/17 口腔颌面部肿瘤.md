> 掌握：口腔颌面部良恶性肿瘤的鉴别，恶性肿瘤的预防及治疗原则

# 概论

>- 牙源性和唾液腺源性肿瘤是口腔颌面部特有的肿瘤
>- 口腔颌面部肿瘤中，良性肿瘤比例高于恶性肿瘤
>	- 良性肿瘤：牙源性及上皮性肿瘤多见
>	- 恶性肿瘤：鳞状细胞癌常见
>	- 原发性口腔癌以舌癌最多

#### 诊断
#### 治疗
1. 手术治疗
	1. 手术仍是目前治疗口腔颌面部肿瘤最主要和有效的方法，适用于良性肿瘤，以及放疗及化疗不能治愈的恶性肿瘤
	2. 对可能有颈部淋巴结转移的恶性肿瘤，还应施行颈淋巴清扫术
2. 放射治疗
	1. 对多数口腔颌面癌瘤来说，放疗均为综合治疗的一部分
	2. 放疗前，应拔除口内病灶牙和肿瘤邻近的牙
	3. 如放疗后发生放射性颌骨坏死或骨髓炎，应进一步处理
3. 化学治疗
	1. 通常是先用化学药物治疗，使肿瘤缩小后再手术，可增加手术治疗的机会，称为*新辅助化疗或诱导化疗*
#### 预防
1.  消除或减少致癌因素
2.  及时处理癌前病损与癌前状态
	- 癌前病损是指“一种已有形态学上改变的组织，它较其相应的外观正常的组织具有更大的癌变可能
	- 癌前状态则是指“一种显著增加癌变危险的一般状态”
	1. 口腔颌面部常见的癌前病损有[]([](08%20口腔粘膜常见疾病.md#5.%20★口腔白斑病%20OLK)和红斑
	2. 口腔颌面部常见的癌前状态有[]([](08%20口腔粘膜常见疾病.md#6.%20口腔扁平苔藓)、口腔黏膜下纤维性变、盘状红斑狼疮、上皮过角化、先天性角化不良以及梅毒、着色性干皮病等。
3. 开展防癌普查及易感人群的监测

#### 牙源性囊肿
###### 根尖囊肿
- 炎症性囊肿，牙周膜内的上皮残余组织增生变性液化
###### 始基囊肿
###### 含牙囊肿
- 发育性囊肿；缩余釉上皮与牙冠之间出现液体渗出
- 好发于下颌第三磨牙、上颌尖牙
###### 角化囊肿
- 来源于牙胚或牙板残余，增殖的PCNA增强
- 好发于下颌第三磨牙及升支
- 多发性角化囊肿可伴发多发性基底细胞痣综合征
#### 非牙源性囊肿
> 胚胎发育过程中残留的上皮发展而来；与牙齿无关
###### 球上颌囊肿
- 侧切牙与尖牙之间，牙被移位
###### 鼻腭囊肿
- 切牙管扩大
###### 正中囊肿
- 切牙孔之后的腭中份任何位置
###### 鼻唇囊肿
- 位于上唇底与鼻前庭内，可能来源于鼻泪管上皮残余
#### 假性颌骨囊肿
###### 单纯性囊肿
- 出血性骨囊肿，孤立性囊肿
###### 静止性囊肿
- 发育过程中唾液腺和其他软组织增殖或迷入而引起的颌骨局限缺损

### （二）良性肿瘤
#### ★1. 成釉细胞瘤
- 常见的牙源性上皮肿瘤，肿瘤内主要含成釉器样结构
- 虽属良性，但生长具有局部侵袭性，术后复发率高
- 有恶变与远处转移的报道
- 下颌骨较上颌骨多见，*下颌多位于后部及下颌升支*
- 可有 *"蜂房样”“皂泡样”特殊的多囊结构*
【特征】
1. 颌骨膨隆，唇颊侧为主
2. *牙根呈锯齿状吸收*
3. 可致牙根间的牙槽骨吸收
4. *肿瘤边缘可有骨硬化增生，不整齐，呈半月形切迹*
5. 瘤区内可有缺牙或*牙移位*
6. 瘤内含牙（多见于单囊型）
7. 钙化或骨化少见
【鉴别诊断】牙源性钙化上皮瘤、牙源性腺样瘤、成釉细胞纤维瘤

# 口腔颌面部恶性肿瘤

> 口腔颌面部恶性肿瘤以癌最为常见，其中又以*鳞状细胞癌*最多，占 80% 以上
### （一）癌
1. 口腔癌（上皮源性）
	1. 舌癌
		- 最常见的口腔癌，多发生在舌中 1 / 3 侧缘
		- 常表现为溃疡及浸润同时存在，伴有自发痛和程度不同的舌运动障碍
		- 淋巴结转移率较高
	2. 牙龈癌
		- 溃疡型或外生型
		- 多源于龈乳头及龈缘区
		- 常发生继发感染，肿瘤伴以坏死组织，触之易出血
	3. 颊癌
		- 通常有溃疡形成，伴深部浸润
		- 明显疼痛，伴不同程度的开口受限，直至牙关紧闭
	4. 口底癌
		- 发生于口底黏膜的癌
		- 发生在舌系带两侧的前口底为常见，局部可出现溃疡或肿块
		- 易发生双侧颈淋巴结转移
	5. 腭癌
		- 仅指硬腭的原发性癌肿
		- 易早期侵犯骨质
2. 唇癌
3. 口咽癌
4. 上颌窦癌
5. 颜面部皮肤癌
### （二）软组织肉瘤
- 起源于间叶组织的恶性肿瘤
- 以青壮年多见
- 多表现为实质性（或有分叶）肿块，表面皮肤或黏膜血管扩张充血，并可出现溃疡或有溢液、出血
- 一般较少淋巴结转移，但常发生血行转移
### （三）骨源性肉瘤
- 起源于骨间质的恶性肿瘤
- 可发生于任何颌面骨，但以上、下颌骨最为常见
- 发病年龄轻，多见于青少年
### （四）恶性淋巴瘤
- 起源于淋巴系统的恶性肿瘤，在病理上可分为[]([](../../疾病模块/心血管、呼吸、血液与泌尿系统Ⅰ[3]/笔记/16%20淋巴造血系统疾病.md#HL(Hodgkin%20lymphoma霍奇金淋巴瘤)与[](16%20淋巴造血系统疾病.md#HL(Hodgkin%20lymphoma霍奇金淋巴瘤)]两大类
- 起源于淋巴结内者称结内型，以颈部淋巴结最为常见。
- 起源于淋巴结外者称结外型，可发生于牙龈、腭、颊、口咽、颌骨等部位。
- 恶性淋巴瘤对放疗及化疗都比较敏感，因此采用以放疗或化疗为主的综合治疗
- 恶性淋巴瘤中 HL 的预后较 NHL 好，但总的来说，预后不够理想

// =====================================================
// 链接关系仪表盘（稳定版） — 纯 DOM 渲染，避免 Dataview 元素类型问题
// =====================================================
const cur = dv.current();
const filePath = cur?.file?.path ?? null;

// === 辅助函数 ===
function extractPath(item){
    if(!item) return null;
    if(typeof item === "string") return item;
    if(item.path) return item.path;
    if(item.file && item.file.path) return item.file.path;
    return null;
}
function isValidNote(path){
    if(!path) return false;
    const lower = path.toLowerCase();
    return !lower.match(/\.(png|jpg|jpeg|gif|svg|webp|mp4|mov|zip|json|csv|xlsx|docx)$/);
}
function displayNameFor(path){
    const pg = dv.page(path);
    return pg?.file?.name ?? path.split('/').pop();
}

// === 获取出链（多重兼容） ===
let outRaw = [];
try {
    if(typeof dv.outgoing === "function" && filePath) outRaw = dv.outgoing(filePath);
    else if (cur?.outlinks) outRaw = cur.outlinks;
    else if (cur?.file?.outlinks) outRaw = cur.file.outlinks;
} catch(e) { outRaw = []; }
const outPaths = [...new Set((outRaw || []).map(extractPath).filter(isValidNote))];

// === 获取反链（优先 dv.incoming，否则全库扫描） ===
let inRaw = [];
try {
    if(typeof dv.incoming === "function" && filePath) inRaw = dv.incoming(filePath);
    else {
        inRaw = dv.pages().where(p => {
            try {
                const ol = p.file?.outlinks ?? p.outlinks ?? [];
                return ol.some(l => extractPath(l) === filePath);
            } catch(e){ return false; }
        }).array();
    }
} catch(e) { inRaw = []; }
const inPaths = [...new Set((inRaw || []).map(extractPath).filter(isValidNote))];

// === 使用纯 DOM 构建视图 ===
const container = document.createElement("div");
container.className = "link-dashboard";

// 卡片构造器（返回 DOM 节点）
function createCard(title, paths, emptyMsg, icon){
    const card = document.createElement("div");
    card.className = "link-card";

    const h3 = document.createElement("h3");
    h3.textContent = `${icon} ${title} (${paths.length})`;
    card.appendChild(h3);

    if(paths.length){
        // 排序
        paths.sort((a,b) => (displayNameFor(a) ?? a).localeCompare(displayNameFor(b) ?? b));
        const ol = document.createElement("ol");
        for(const p of paths){
            const li = document.createElement("li");

            const a = document.createElement("a");
            a.href = "#";
            a.textContent = displayNameFor(p);
            a.style.cursor = "pointer";

            // 点击通过 Obsidian API 打开链接（在新 leaf）
            a.addEventListener("click", (e) => {
                e.preventDefault();
                try {
                    app.workspace.openLinkText(p, "", true);
                } catch(err) {
                    // 回退尝试按显示名打开
                    try { app.workspace.openLinkText(a.textContent, "", true); } catch(e2){ console.error(e2); }
                }
            });

            li.appendChild(a);
            ol.appendChild(li);
        }
        card.appendChild(ol);
    } else {
        const p = document.createElement("p");
        p.textContent = emptyMsg;
        p.style.opacity = "0.75";
        card.appendChild(p);
    }

    return card;
}

// 插入卡片
container.appendChild(createCard("本文引用的笔记", outPaths, "📭 本文未引用其他笔记。", "📓"));
container.appendChild(createCard("引用了本文的笔记", inPaths, "📪 目前没有笔记引用本文。", "📑"));

// 将纯 DOM 容器挂到 Dataview 输出区域（稳健方式）
dv.container.appendChild(container);
