# 概述

**良性前列腺增生（benign prostatic hyperplasia，BPH）**：是引起中老年男性排尿障碍最为常见的一种良性疾病。
主要表现为组织学上的*前列腺间质和腺体成分的增生*、解剖学上的前列腺增大、尿流动力学上的*膀胱出口梗阻*和以*下尿路症状*（lower urinary tract symptoms，LUTS）为主的临床症状

【流行病学】BPH 的发病率随年龄的增长而增加
【病因】==年龄的增长和有功能的睾丸是 BPH 两个必备条件==
	1. BPH 的发病率随年龄的增大而增加
	2. 前列腺的正常发育有赖于雄激素
【病理】
	- 前列腺腺体增生开始于围绕尿道的腺体，这部分腺体称为**移行带**
	- 前列腺其余腺体由中央带（占 25%）和外周带（占 70%）组成[^1]
	- **中央带**似楔形并包绕 射精管
	- **外周带**组成前列腺的背侧及外侧部分，*是前列腺癌最常发生的部位*
	- BPH 主要发生于前列腺尿道周围**移行带**，增生组织呈多发结节，并逐渐增大
	- 增生腺体突向*后尿道*，使前列腺部尿道伸长、弯曲、受压变窄，尿道阻力增加，引起排尿困难
	- 此外，前列腺内尤其是围绕膀胱颈部的平滑肌内含有丰富的α肾上腺素受体，这些受体的激活使该处平滑肌收缩，可明显增加前列腺尿道的阻力
	- *BPH 及α肾上腺素受体兴奋*致后尿道平滑肌收缩，造成*膀胱出口梗阻*
	- 为了克服排尿阻力，逼尿肌增强其收缩能力，逐渐<u>代偿性肥大</u>，肌束形成粗糙的网状结构，加上长期膀胱内高压，膀胱壁出现小梁、小室或假性憩室
	- 由于逼尿肌退变、顺应性差，出现*逼尿肌不稳定收缩*，病人有明显**尿频、尿急和急迫性尿失禁**，可造成输尿管尿液排出阻力增大，引起*上尿路扩张积水*
	- 如梗阻长期未能解除，*逼尿肌萎缩，失去代偿能力*，收缩力减弱，导致膀胱不能完全排空而出现残余尿。随着残余尿量增加，膀胱壁变薄，膀胱腔扩大，可出现**慢性尿潴留及充溢性尿失禁**，尿液反流引起**上尿路积水及肾功能损害**。梗阻引起尿潴留，还可继发**感染和结石**形成

# 诊断
##### 【临床表现】
- BPH 多在 50 岁以后出现症状，60 岁左右症状更加明显
- 主要表现为**下尿路症状**
	- 储尿期：尿频、尿急、尿失禁、夜尿增多
	- 排尿期：排尿踌躇、排尿困难、排尿中断
	- 排尿后：尿不尽、尿后滴沥
- **尿频**是BPH最常见的早期症状、夜间更为明显
	- 早期：增生的前列腺充血刺激
	- 进展：梗阻加重、残余尿量增多、膀胱有效容量减少、尿频加重
	- 此外，梗阻诱发逼尿肌功能改变，膀胱顺应性降低或逼尿肌不稳定，尿频更为明显，并出现急迫性尿失禁等症状
- **排尿困难**是BPH最重要的症状，病情发展缓慢。典型表现是*排尿迟缓、中断、尿流细而无力、射程短、终末滴沥、排尿时间延长*
	- 如梗阻严重，残余尿量较多时，常需要用力并增加腹压以帮助排尿，排尿后常有尿不尽感。
	- 当梗阻加重达一定程度时，残余尿进一步增加，继而发生<u>慢性尿潴留或充溢性尿失禁</u>。
- 合并**感染或结石**时可出现明显尿频、尿急、尿痛症状
	- 增生腺体表面黏膜的血管破裂时，亦可发生不同程度的<u>无痛性肉眼血尿</u>，应与泌尿系肿瘤引起的血尿鉴别
	- 梗阻引起严重肾积水、肾功能损害时，可出现慢性肾功能不全的表现，如食欲缺乏、恶心、呕吐、贫血、乏力等症状。
	- 长期排尿困难导致腹压增高，还可引起[^2]腹股沟疝、[^3]内痔与脱肛等
##### 【诊断】
50 岁以上男性出现 LUTS 等临床表现，须考虑有 BPH 的可能
1. 病史问诊
2. 症状评估（*国际前列腺症状评分IPSS*）[^4]
3. 直肠指诊（最重要的检查）
4. 超声检查
5. $尿流率检查$[^6]
6. $血清前列腺特异性抗原PSA测定$：[^5]
##### 【鉴别诊断】
1. *前列腺癌*（前列腺有结节、质地硬、或血清PSA升高）
2. *膀胱颈挛缩*（慢性炎症、结核或手术后瘢痕；发病年龄较轻；前列腺不大）
3. *尿道狭窄*
4. *神经源性膀胱*
	- 动力性梗阻、前列腺体积不大
	- 常有中枢或周围神经系统损害的病史和体征
	- 静脉尿路造影常显示上尿路有扩张积水，<u>膀胱常呈“圣诞树”形改变</u>。尿流动力学检查可帮助诊断

# 治疗

==原则==：改善下尿路症状、恢复正常排尿功能，提升生活质量，延缓疾病进展，预防并发症发生

1. 观察等待
2. 药物治疗
	1. $α受体拮抗剂$
		- α1受体：前列腺基质平滑肌、对排尿影响较大
		- 拮抗α1 受体能有效地降低膀胱颈及前列腺的平滑肌张力，减小尿道阻力，改善排尿功能
		- 特拉唑嗪、多沙唑嗪等[^7]
	2. $5α还原酶抑制剂$
		- 原理：在前列腺内阻止睾酮转变为*有活性的双氢睾酮*，进而使前列腺体积部分缩小， 改善排尿症状
		- 药物：非那雄胺
3. 手术治疗
	- 外科治疗指征：：①反复尿 潴留；②反复血尿；③反复泌尿系感染；④膀胱结石；⑤继发性上尿路积水；⑥合并腹股沟疝、严重的痔 或脱肛，临床判断不解除下尿路梗阻难以达到治疗效果者
	- 手术方式：
		1. 开放性前列腺摘除术OP[^8]
		2. 经尿道前列腺切除或切开术
		3. 经尿道前列腺剜除术
		4. ……
