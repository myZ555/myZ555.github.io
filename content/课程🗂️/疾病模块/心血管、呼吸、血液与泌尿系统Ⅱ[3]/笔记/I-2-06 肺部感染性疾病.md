## （一）肺炎概述
**肺炎 pneumonia**：指终末气道、肺泡和肺间质的炎症，可由病原微生物、理化因素、免疫损伤、过敏及药物所致

#### 【分类】
###### 【解剖分类】

| 大叶性(肺泡性)肺炎                                                                             | 小叶性(支气管性)肺炎                                                                  | 间质性肺炎                                                           |
| -------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------- | --------------------------------------------------------------- |
| 病原体*先在肺泡*引起炎症，经肺泡间孔（Cohn 孔）向其他肺泡扩散，致使部分肺段或整个肺段、肺叶发生炎症。<br>典型者表现为*肺实质炎症*，通常并不累及支气管。[^1] | 病原体*经支气管入侵*，引起细支气管、终末细支气管及肺泡的炎症。<br>常继发于其他疾病，如支气管炎、支气管扩张、上呼吸道病毒感染以及长期卧床的危重病人 | 以肺间质受累为主的炎症，累及支气管壁、支气管周围组织及肺泡壁<br>因病变仅在肺间质，故呼吸道症状较轻，病变广泛则呼吸困难明显 |
| 肺炎链球菌                                                                                  | 肺炎链球菌、葡萄球菌、病毒、肺炎支原体以及军团菌等                                                    | 细菌、支原体、衣原体、病毒或肺孢子菌                                              |
| 肺叶/肺段的实变阴影                                                                             | 沿着*肺纹理分布的不规则斑片状阴影*，边缘密度浅而模糊，无实变征象，肺下叶常受累                                     | 一侧或双侧肺下部不规则阴影，可呈磨玻璃状、网格状，其间可有小片肺不张阴影                            |
###### 【病因分类】
1. 细菌性肺炎 
2. 非典型病原体所致肺炎 （军团菌、支原体和衣原体）
3. 病毒性肺炎 
4. 肺真菌病 （念珠菌、曲霉、隐球菌、肺孢子菌、毛霉）
5. 其他病原体所致肺炎 （立克次体（如 Q 热立克次体）、弓形体（如鼠弓形体）、寄生虫（如肺包虫、肺吸虫、肺血吸虫） 
6. 理化因素所致肺炎（放射性损伤，胃酸吸入，类脂性肺炎）
###### 【患病环境】
==社区获得性肺炎 CAP==
	- 指在医院外罹患的感染性肺实质（含肺泡壁，即广义上的肺间质）炎症，包括*具有明确潜伏期*的病原体感染在入院后于潜伏期内发病的肺炎。
	- CAP 常见病原体为==肺炎链球菌==、支原体、衣原体、流感嗜血杆菌和呼吸道病毒（甲、乙型流感病毒，腺病毒，呼吸道合胞病毒和副流感病毒）等
==医院获得性肺炎 HAP==
	- ★指病人住院期间没有接受有创机械通气，未处于病原感染的潜伏期，且入院≥48 小时后在医院内新发生的肺炎
	- *呼吸机相关性肺炎 VAP*：指气管插管或气管切开病人，接受机械通气 48 小时后发生的肺炎及机械通气撤机、拔管后 48 小时内出现的肺炎。
	- 我国 HAP/VAP 常见病原菌包括 鲍曼不动杆菌、铜绿假单胞菌、肺炎克雷伯菌、大肠埃希菌、金黄色葡萄球菌等
#### 【临床表现】
- 常见症状为咳嗽、咳痰，或原有呼吸道症状加重，并出现脓性痰或血痰，伴或不伴胸痛
- 病变范围大者可有呼吸困难、呼吸窘迫。大多数病人有发热。
- 早期肺部体征无明显异常，重症者可有呼吸频率增快、鼻翼扇动、发绀。 
- 肺实变时有典型的体征，如叩诊呈浊音、语颤增强和支气管呼吸音等，也可闻及湿啰音。
#### 【诊断】
1. 确定肺炎诊断【鉴别诊断】
	1. 急性上呼吸道感染与气管支气管炎（无肺实质浸润）
	2. 类似肺炎的疾病
		1) 肺结核（全身中毒症状；结核分枝杆菌）[^2]
		2) 肺癌（可伴发阻塞性肺炎）[^3]
		3) 肺栓塞（静脉血栓危险因素）[^4]
		4) 非感染性肺部浸润
2. 评估严重程度
	*肺部局部炎症程度、肺部炎症的播散和 全身炎症反应程度*
3. 确定病原体[^6]
	1) 痰（最常用的下呼吸道病原学标本）[^5]
	2) 经支气管镜或人工气道吸引
	3) ……
#### 【治疗】
- 青壮年和无基础疾病的 CAP 病人，常用青霉素类、第一代头孢菌素等。
- 重症肺炎首先应选择广谱的强力抗菌药物，并应足量、联合用药。
- 重症 CAP 常用β- 内酰胺类联合大环内酯类或氟喹诺酮类药物；青霉素过敏者用呼吸氟喹诺酮类和氨曲南。
- 抗菌药物治疗应尽早进行，一旦怀疑为肺炎即应马上给予首剂抗菌药物，越早治疗预后越好。
## （二）细菌性肺炎

|      | 肺炎链球菌肺炎             | 葡萄球菌肺炎                     | 肺炎克雷伯菌肺炎               |
| ---- | ------------------- | -------------------------- | ---------------------- |
| 易感人群 | 既往健康的青壮年、受凉/劳累后（淋雨） | 免疫力低下、流感并发症、院内感染           | 老年人、酗酒者、糖尿病患 者（基础疾病）   |
| 影像特征 | 大叶性肺炎（实变影伴 支气管充气征）  | 小叶性肺炎，极易形成多发性肺脓肿/空洞（常伴气囊肿） | 肺叶实变，叶间裂下坠（因炎性渗出液黏稠重坠） |
| 痰特征  | 铁锈色痰                | 大量黄脓痰                      | 砖红色胶冻样痰                |
| 首选治疗 | 青霉素类（首选青霉素G）、头孢类    | 苯唑西林；MRSA首选万古霉素/利奈唑胺       | 三代/四代头孢、碳青霉烯类[^7]      |
###### 肺炎链球菌肺炎
- SP 为革兰氏阳性球菌，多成双排列或短链排列。有荚膜；SP不产生毒素，不引起组织坏死或形成空洞。其致病力是由于高分子多糖体的荚膜对组织的侵袭作用
- 病理改变有充血期、红肝变期、灰肝变期及消散期。
- 冬季与初春多见，常与呼吸道病毒感染相伴行
- 病人呈急性热病容，面颊绯红，鼻翼扇动，皮肤灼热、干燥，口角及鼻周有单纯疱疹
###### 葡萄球菌肺炎
- 多急骤起病，表现为高热、寒战、胸痛、脓性痰， 可早期出现循环衰竭。
- 胸部影像学表现为坏死性肺炎，如肺脓肿、肺气囊肿和脓胸。
- 葡萄球菌为革兰氏阳性球菌，可分为凝固酶阳性的葡萄球菌（主要为金黄色 葡萄球菌，简称金葡菌）及凝固酶阴性的葡萄球菌（如表皮葡萄球菌和腐生葡萄球菌等）。
- 其致病物质主要是毒素与酶，如溶血毒素、杀白细胞素、肠毒素等，具有溶血、坏死、杀白细胞及血管痉挛等作用。 
- 葡萄球菌致病力可用血浆凝固酶来测定，阳性者致病力较强。
- 部分病人因 感染产杀白细胞毒素的金葡菌，出现白细胞计数降低
- 细菌学检查是确诊的依据
## （三）病毒性肺炎
**病毒性肺炎 viral pneumonia** 是由病毒侵入呼吸道上皮及肺泡上皮细胞引起的肺间质及实质性炎症。免疫功能正常或抑制的个体均可罹患。大多发生于*冬春*季节，暴发或散发流行。病毒是成人社区获得性肺炎除细菌外第二大常见病原体，*大多可自愈*
###### 【流行病学】
冬春季好发，成人以甲/乙型流感病毒最常见[^8]
近年需关注新冠 (COVID-19)、腺病毒。
###### 【临床表现与诊断】
- 全身中毒症状重： 高热、剧烈乏力、全身酸痛明显。 
- 实验室检查： WBC计数正常、稍高或偏低；PCT（降钙素原）通常不升高
- 影像学： 双肺弥漫性磨玻璃影 (GGO)、间质浸润，严重者呈“白肺”
###### 【靶向治疗】
- 流感病毒： 奥司他韦 (Oseltamivir)、玛巴洛沙韦 
- 巨细胞病毒 (CMV)： 更昔洛韦 
- 单纯疱疹/水痘： 阿昔洛韦
[^9]
## （四）非典型病原体肺炎
- 包括：支原体、衣原体、军团菌 
- 起病隐袭：多见于青少年或聚集性发病。 
- 症状与体征不符： 显著的刺激性阵发性干咳，但肺部听诊多无明显湿啰音（即“症状重、体征轻”）。 
- 肺外表现： 可伴有头痛、咽痛、肌痛、腹泻（尤其是军团菌）。 
- 治疗原则（重点）： 
	- 禁用： 青霉素、头孢类（此类病原体无细胞壁，内酰胺类无效）
	- 首选： 大环内酯类（阿奇霉素）、四环素 类（多西环素）、呼吸喹诺酮类（左氧氟沙星、莫西沙星）。 

| 肺炎支原体肺炎                                                         | 衣原体肺炎             | 肺军团病                          |
| --------------------------------------------------------------- | ----------------- | ----------------------------- |
| 最常见<br>持久的阵发性剧咳为支原体肺炎较为典型的表现<br>影像学呈间质性改变或节段性浸润（小叶性肺炎）；冷凝集试验（+） | 专性细胞内细菌样寄生物；始体-原体 | 病情最重，常伴有高热、 相对缓脉、低钠血症及多 器官损伤。 |

## （五）真菌性肺炎
- 肺真菌病是最常见的深部真菌病
- 真菌多在土壤中生长，孢子广泛存在于空气中，被吸入到肺部可引起肺真菌病（外源性）
- 病理改变有过敏、化脓性炎症或形成慢性肉芽肿。
- X 线影像表现无特征性
- 病理学诊断是肺真菌病的“金标准”
【宿主因素】（除孤立隐球菌肺炎外）
	免疫受损人群：长期广谱抗生素/激素/免疫抑制剂使用者、血液病、糖尿病、HIV
