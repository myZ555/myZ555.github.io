> - 记忆：列举颅内肿瘤的常见类型（胶质瘤、脑膜瘤、垂体瘤、听神经瘤）及其好发部位
> - 理解：能够解释颅内压增高“三主征”在肿瘤患者中的表现差异，区分原发性与转移性脑肿瘤的病理特征
> - 应用：识别不同部位肿瘤的影像学特征（如脑膜瘤的“硬脑膜尾征”、垂体瘤的鞍区占位），制定胶质瘤的综合治疗方案（手术+放化疗）
> - 分析：能够比较不同病理类型肿瘤（髓母细胞瘤vs室管膜瘤、听神经瘤vs脑膜瘤）的临床特点与预后差异

##### 【临床表现】
1. 颅内压增高[^1]
2. 神经功能定位症状
	1) 破坏性症状：肿瘤侵及脑组织
	2) 压迫症状（鞍区肿瘤->视力、视野障碍；海绵窦区->3/4/5/6脑神经）
3. 癫痫
4. 老年人和儿童颅内肿瘤特点
	- 老年人脑萎缩，颅内空间相对增大，发生颅脑肿瘤时颅内压增高不明显，易误诊。老年人以幕上脑膜瘤和转移瘤多见
	- 儿童以发生于中线区肿瘤多见，幕下以髓母细胞瘤和室管膜瘤为常见，幕上以颅咽管瘤为多；常出现脑积水症状而掩盖肿瘤定位体征，易误诊为胃肠道疾病

| （肿瘤）部位 | 表现特点                                                                                             |
| ------ | ------------------------------------------------------------------------------------------------ |
| 大脑半球   | 精神症状：额叶<br>癫痫发作：额叶、颞叶、顶叶<br>感觉障碍：顶叶<br>运动障碍<br>失语症<br>视野损害                                       |
| 鞍区肿瘤   | 1)视力视野改变<br>2)眼底视乳头萎缩<br>3)内分泌功能紊乱                                                               |
| 松果体区肿瘤 | 颅内高压出现早；双眼上视困难；儿童性早熟                                                                             |
| 后颅窝肿瘤  | 1)小脑半球肿瘤：同侧肢体共济障碍，眼球震颤<br>2)小脑蚓部肿瘤：步态不稳、行走不能<br>3)桥小脑角肿瘤：眩晕、耳鸣、进行性听力下降、V、VII颅神经麻痹，后期IX、XI后组颅神经麻痹 |
##### 【诊断】
1. *病史*：详细询问症状发生时间、进展速度及演变过程，包括头痛性质与规 律、癫痫发作类型、局灶性神经功能缺损等 
2. *体检*：神经系统专科查体，重点评估意识状态、颅神经功能、运动感觉系 统、病理反射及眼底检查 
3. *影像学检查*：头颅CT（初步筛查、钙化显示）、MRI平扫+增强（定位定性首选， 含fMRI、DWI）、PET-CT（代谢活性评估、肿瘤分级、复发鉴别） 
	- 直接征象：肿瘤组织形成的异常密度/信号
	- 间接征象：脑室或脑池的变形移位
	- 增强扫描：静脉注射造影剂提高组织的对比
4. *病理确诊*：立体定向活检（深部/功能区病变）、术中快速冰冻切片（指导手术 范围）、术后石蜡病理+免疫组化（确定组织学类型与分子分型）
##### 【治疗】
1. 内科治疗：降颅压+抗癫痫药物治疗
2. 手术治疗：主要方法
	1. 良性：尽可能全切除
	2. 恶性：充分减压，为放化疗打下基础；如有脑积水则行分流术
## 神经上皮组织肿瘤

| 肿瘤类型       | 恶性程度 | 好发部位             | 好发人群    | 临床特点                        | 治疗原则                |
| ---------- | ---- | ---------------- | ------- | --------------------------- | ------------------- |
| ==星形细胞瘤==  | 较低   | CNS任何部位，多见于大脑半球  | 中青年     | 最常见神经上皮性肿瘤                  | 手术切除；根据分级辅助放化疗      |
| ==胶质母细胞瘤== | 高度恶性 | 额、顶、颞叶           | 中老年     | 可由星形细胞瘤恶变而来；起病急、病程短、进展快     | 首选手术，术后辅助放化疗        |
| ==少突胶质瘤==  | 低-中度 | 大脑半球白质           | 成人      | 生长缓慢，瘤内常见钙化，多以癫痫为首发症状       | 手术切除后辅助放化疗，预后较理想    |
| ==室管膜瘤==   | 中度   | 侧脑室、四脑室、三脑室      | 儿童、青年   | 可经脑脊液播散；颅内型常伴脑积水，椎管内型常伴脊髓空洞 | 手术全切+放疗，播散者行全脑全脊髓照射 |
| ==髓母细胞瘤==  | 高度恶性 | 颅后窝中线小脑蚓部，突入第四脑室 | 2-10岁儿童 | 易引起梗阻性脑积水                   | 手术切除+术后全脊髓放化疗       |
## 听神经鞘瘤
- 听神经鞘瘤(acoustic neuroma)：位于**CPA**内，为第八脑神经前庭支生长的**良性肿瘤**，约占颅内肿瘤的8%-10%，占CPA肿瘤的65%-72.2%。
- *临床表现*：
	- 患侧神经性耳鸣、耳聋，面神经（周围性面瘫）及后组脑神经受累症状（声音嘶哑、吞咽困难）
	- 同侧小脑症状，眼球震颤，步态不稳，闭目难立，同侧肢体共济失调。颅内压增高症状。
- *治疗*：以手术为主，<u>全切可根治</u>，注意保留面听神经功能。术后残留或直径<3cm，可行γ刀治疗

## 脑膜瘤
- 脑膜瘤（meningioma）发病率仅次于胶质瘤，约占颅内肿瘤20%
- 多为良性，病程长；
- 女:男为2:3，发病高峰30-50岁
- 多见于*矢状窦旁和大脑凸面*
- MRI上常见 *“硬脑膜尾”征*
- 脑膜瘤包膜完整，双重供血， 术中出血多，手术切尽预后好 。＜3cm可行X刀或γ刀治疗。
## 原发性中枢神经系统淋巴瘤
- 75%与室管膜或脑膜相连，10%-30%为多发。
- CT可见*中央灰质或胼胝体处*均匀一致的*高密度灶*，水肿明显，可强化。 
- 治疗：明确肿瘤性质后，放疗和化疗。 
## 生殖细胞肿瘤
- 多见于儿童
- *松果体区和鞍上区*最多见。易发生脑脊液播散转移
- 松果体区—压迫中脑顶盖—眼球上视困难
- 鞍上区—压迫视神经、下丘脑、垂体柄—视力视野障碍，尿崩，垂体功能减退
- *放疗敏感*
- 合并脑积水颅内压增高时，可行分流手术后再予放疗
## 表皮样囊肿和皮样囊肿
- *均起源于神经管闭合期间残留的上皮细胞*
- **表皮样囊肿**：
	- 先天性良性肿瘤 
	- 鳞状上皮层状排列 
	- 好发于桥脑小脑角 
	- 胆脂瘤，珍珠瘤
- **皮样囊肿**：
	- 先天性良性肿瘤
	- 含皮肤附属器官
	- 最常见于颅后窝

|         | 表皮样囊肿           | 皮样囊肿             |
| ------- | --------------- | ---------------- |
| 成分      | 鳞状上皮、脱落的角蛋白、胆固醇 | 鳞状上皮、毛发、汗腺、皮脂腺成分 |
| 发病位置    | 脑池内             | 颅后窝、鞍上池          |
| 影像学特征   | 类似*脑脊液*         | 类似*脂肪瘤*          |
| 治疗      | 手术切除            | 手术切除             |
| 未完全切除   | 会复发             | 会复发              |
| 恶性转化    | 鳞状细胞癌           | 鳞状细胞癌            |
| 自发破裂    | 不会              | *破裂后致命*          |
| 无菌性蛛网膜炎 | 术后容易发生          | 不会发生             |

## 垂体腺瘤
- *最常见的鞍区肿瘤*
- 来源于*垂体前叶的良性肿瘤*
- 症状：包括内分泌功能紊乱、视力视野改变和颅内压增高
	- 视力减退
	- 视野缺损（<u>双颞侧偏盲</u>）
	- 原发性视神经萎缩
- 分类：
	- <u>根据内分泌功能分类</u>：
		- **泌乳素腺瘤（PRL瘤）**：因PRL分泌过多，女性闭经、泌乳、 不育；男性性功能减退
		- **生长激素腺瘤（GH瘤）**：巨人症和肢端肥大症
		- **促肾上腺皮质激素瘤（ACTH瘤）**：因ACTH分泌过多，出现皮质醇增多症， 如向心性肥胖、高血压、性功能减退
		- 混合型腺瘤
		- 无功能性腺瘤等
	- <u>根据肿瘤大小分类</u>：
		- 直径≤1cm，限于鞍内生长称微腺瘤，确诊靠MRI及血清激素水平测定
		- 直径>1cm，突破鞍隔为大腺瘤
		- 直径>3cm，为巨大腺瘤
- 治疗：
	- *以手术为主*
	- 入路：经蝶、经颅
	- 伽玛刀治疗：微腺瘤、术后残留 
	- 嗅隐亭可用于泌乳素腺瘤的治疗，可使肿瘤减小和减低泌乳素水平
## 颅咽管瘤
- 颅咽管瘤(craniopharyngioma)： 发自颅咽管残余在垂体柄结节部鳞状上皮细胞的先天性肿瘤 
- 约占颅内肿瘤的5% 
- 多见于儿童、少年，男性多于女性 
- 大多位于鞍隔上，肿瘤多为囊性， 钙化多见 
- 表现为视力障碍，视野缺损，尿崩 、肥胖、发育迟缓、性功能减退。
- *手术治疗为主*，注意围手术期皮质功能低 下，尿崩症控制及水电解质平衡
## 脊索瘤
- 来源于胚胎残留脊索组织，一般为良性
- *好发中线的骨性结构*，如颅底斜坡、蝶鞍、岩尖、骶尾
- 20-40岁男性多见
- 治疗：手术+放疗
## 脑转移瘤
- 多继发于肺癌、乳腺癌、胃癌
- 单发或多发，小结节，大水肿
- 多位于大脑中动脉分布区，灰白质交界处
- 治疗：治疗原发灶、个体化治疗
## 血管网状细胞瘤
- 约占颅内肿瘤1.3-2.4%，又称*血管母细胞瘤*
- *多发小脑半球*，偶见脑干
- 20-40岁成人为多，男多于女，有家族遗传倾向
- 有时颅外视网膜血管瘤，肝肾胰血管瘤伴发
- 肿瘤多为*囊性*，*囊内有血管丰富的瘤结节*
