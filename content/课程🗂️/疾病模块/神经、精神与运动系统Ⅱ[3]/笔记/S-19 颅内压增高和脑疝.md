>[!Note] 大纲
> 1. 颅内高压的典型临床表现（头痛、呕吐、视神经乳头水肿）及诊断标准。
> 2. 颅内高压与颅脑损伤继发颅内压增高的鉴别诊断。
> - 记忆 ：能够准确描述颅内压正常值范围（成人/儿童）及颅内压增高的三联征临床表现
> - 理解：能够阐述Monroe-Kellie原理的生理意义，解释颅内容物代偿 调节的限度
> - 应用：能够识别两种常见脑疝（小脑幕切迹疝/枕骨大孔疝）的临床差异，制定颅内压增高的紧急处理流程
> - 分析：能够比较不同病因（创伤、肿瘤、血管性）导致颅内压增高的病理生理机制差异
# 概述
- 颅内压的形成与正常值【颅内压 ICP】
	- *颅腔内脑组织*、*脑脊液*和*运行于脑血管内的血液*是颅内压形成的物 质基础。
	- 成人正常颅内压（侧卧位）为 70～200mm$H_2O$
	- 儿童（侧卧位）为 50～100mm$H_2O$
- 颅内压的调节与代偿
	- 生理状态下，血压和呼吸可引起颅内压小范围的波动
	- 颅内压增高后的调节主要依靠*调节脑血流量*以及*脑脊液的分布和分泌*来实现
- 颅内压增高的原因
	1. 颅内占位性病变挤占了颅内空间
	2. 脑组织体积增大
	3. 脑脊液循环和/或吸收障碍
	4. 脑血流过度灌注或静脉回流受阻
	5. 先天性畸形使颅腔容积变小
- 颅内压增高的病理生理
	- 影响因素
		1) 年龄：婴幼儿-颅缝未闭合；老年人-脑萎缩
		2) 病变扩张速度：颅内病变体积扩增与颅内压上升呈现指数曲线
		3) 病变部位：
			- 颅脑中线或颅后窝的占位性病变容易导致梗阻性脑积水
			- 静脉窦受累可引起颅内静脉回流障碍或脑脊液吸收障碍
		4) 伴发脑水肿程度
		5) 全身系统性疾病
	- 后果
		1) 脑血流量的降低：$脑血流量（CBF）= 脑灌注压（CPP）/ 脑血管阻力（CVR）$
			- 如果颅内压不断增高使脑灌注压低于 5.3kPa（40mmHg），自动调节功能失效，脑血管不能再作进一步扩张以减少血管阻力，脑血流量随之急剧下降，造成脑缺血。
			- 当颅内压升至接近平均动脉压水平时，颅 内血流几乎完全停止，甚至出现脑死亡
		2) 脑移位和脑疝
		3) 脑水肿
			1) 血管源性：水积聚在细胞外间隙
			2) 细胞毒性：水积聚在细胞膜内
		4) *库欣反应*[^1]
		5) 胃肠功能紊乱及消化道出血（与颅内压增高引起下丘脑自主神经中枢缺血有关）
		6) 神经源性肺水肿

# 颅内压增高

| 弥漫性颅内压增高                                                    | 局灶性颅内压增高                                                                                           |
| ----------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| 由颅腔狭小或脑实质体积增大引起，其特点是颅腔内各部位及各分腔之间压力均匀升高，不存在明显的压力差，因此脑组织无明显移位 | 因颅内有局限的扩张性病变，病变部位压力首先增高，使附近的脑组织 受到挤压而发生移位，并把压力传向远处，造成颅内各腔隙间压力差。<br>此压力差增大时可导致脑室、脑干及中线结构移位，*更易形成脑疝* |
| 弥漫性脑膜脑炎、弥漫性脑水肿、交通性脑积水、静脉窦血栓等                                |                                                                                                    |
- 脑疝发生后，移位脑组织被挤进小脑幕切迹裂孔、硬脑膜裂隙或枕骨大孔中，压迫脑干，产生一系列危急症状。
- 脑疝的发生加剧了脑脊液和血液循环障碍，使颅内压力进一步增高，从而形成恶性循环，最终导致死亡。
##### 【引起颅内压增高的常见疾病】
1. 颅脑损伤
2. 颅内肿瘤
3. 颅内感染
4. 脑血管疾病
5. 脑寄生虫病（局部肉芽肿性占位+炎性粘连影响脑脊液循环吸收+伴发脑水肿）
6. 颅脑先天性疾病
7. 良性颅内压增高/假脑瘤综合征
8. 脑缺氧
##### 【临床表现】
1. **头痛**：最常见；早晨或夜间较重，多在额部及颞部
2. **呕吐**：喷射性
3. **视乳头水肿**：颅内压增高重要客观体征；表现为<u>视乳头充血，边缘模糊不清，中央凹陷消失，视乳头隆起，静脉怒张</u>。若颅内压增高长期存在，则<u>视乳头颜色苍白，视力减退，视野向心性缩小--视乳头继发性萎缩</u>。
4. 意识障碍及生命体征变化
5. 其他症状和体征（头颅叩诊 <u>破罐音（MacEwen征）</u>）
##### 【诊断】
- 当发现有“三主征”（头痛+呕吐+视乳头水肿）时颅内压增高可确诊。
- 小儿反复呕吐及头围迅速增大，成人进行性的剧烈头痛、进行性瘫痪及视力进行性减退等，都应考虑到有颅内病变可能。
1. *头部CT*（快速、精确、无创伤；颅内病变初诊首选）
2. 头部MRI
3. 头部数字减影血管造影DSA
4. X线平片
5. 腰椎穿刺
6. 颅内压监测
##### 【治疗原则】
1. 一般处理
2. 病因治疗
3. 药物治疗（20%甘露醇、呋塞米、高渗盐水）
4. 糖皮质激素（可减轻脑水肿、缓解颅内压增高）
5. 巴比妥治疗（大剂量异戊巴比妥钠或硫喷妥钠注射可<u>降低脑的代谢、减少脑血流，减少氧耗及增加脑对缺氧的耐受力</u>）
6.  过度换气（动脉血的 CO2 分压每下降 1mmHg，可使脑血流量递减 2%，从而使颅内压相应下降）
7. 对症治疗
# 脑疝
##### 【病因】
1. 各种颅内血肿
2. 大面积脑梗死
3. 颅内肿瘤
4. 颅内脓肿、颅内寄生虫病及各种肉芽肿性病变
5. 医源性因素

| 脑疝               | 说明                    |
| ---------------- | --------------------- |
| 颞叶钩回疝/==小脑幕切迹疝== | 颞叶海马回、钩回通过小脑幕切迹被推移至幕下 |
| 小脑扁桃体疝/==枕骨大孔疝== | 小脑扁桃体及延髓经枕骨 大孔被推挤向椎管内 |
| 扣带回疝/大脑镰下疝       | 一侧半球 的扣带回经镰下孔被挤入对侧    |
##### 【病理生理】
1. ==小脑幕切迹疝==：
	- 移位的脑组织疝入小脑幕切迹下方，脑干受压移位。
	- 同侧的*大脑脚*受到挤压而造成*病变对侧偏瘫*，同侧*动眼神经*受压可产生*先刺激、后麻痹*的症状。
	- 移位的钩回、海马回可将*大脑后动脉*挤压于小脑幕切迹缘，导致*枕叶皮质*缺血坏死。
2. ==枕骨大孔疝==：
	- *延髓*直接受压，病人可迅速出现呼吸骤停。
	- 脑疝发生后，脑脊液循环通路进一步受阻，加剧了颅内压增高，形成恶性循环，使病情迅速恶化。
##### 【临床表现】
###### 小脑幕切迹疝
1. 颅内压增高症状：表现为剧烈头痛，与进食无关的频繁呕吐
2. 瞳孔改变：
	1) 初期由于病侧<u>动眼神经</u>受刺激，病侧瞳孔变小，对光反射迟钝。
	2) 随病情进展，动眼神经则麻痹，瞳孔逐渐散大，直接和间接对光反射均消失，并有病侧上睑下垂、眼球外斜。
	3) 如果脑疝进行性恶化，影响脑干血供时，脑干内动眼神经核功能丧失可致双侧瞳孔散大，对光反射消失，此时病人多已处于濒死状态。
3. 运动障碍：病变<u>对侧肢体</u>的肌力减弱或瘫痪，病理征阳性。
4. 意识改变：脑干内网状上行激活系统受累->嗜睡、浅昏迷、深昏迷
5. 生命体征紊乱：脑干内生命中枢功能紊乱或衰竭，表现为心率减慢或心律不规则，血压忽高 忽低，呼吸不规则，大汗淋漓或汗闭，面色潮红或苍白。病人最终 因呼吸循环衰竭而出现呼吸停止，血压下降，心脏停搏
###### 枕骨大孔疝
- 由于颅内压急速增高，病人剧烈头痛，频繁呕吐，颈项强直，强迫头位。
- <u>生命体征紊乱出现较早，意识障碍出现较晚</u>。
- 因脑干缺氧，瞳孔可忽大忽小。
- 由于位于延髓的呼吸中枢受损严重，病人早期可突发呼吸骤停而死亡。
##### 【治疗】
- 一旦确诊应按颅内压增高的处理原则快速静脉输注高渗降颅内压药物
- 病因明确者，应尽快去除病因，如清除颅内血肿或切除脑肿瘤等。
- 如难以确诊或病因难以去除，可选用下列手术，以降低颅内高压和抢救脑疝
	1. *侧脑室外引流术*：经额、枕部快速钻孔或锥颅，穿刺侧脑室并安置引流管，引流脑脊液，以迅速降低颅内压
	2. *脑脊液分流术*：脑积水病人可施行脑室-腹腔分流术V-P shunt、 腰大池-腹腔分流术或侧脑室-心房分流术等。导水管梗阻或狭窄者，可选用神经内镜下第三脑室底造瘘术ETV。
	3. *减压术*
		- 外减压术：
			- 小脑幕切迹疝时可采用颞肌下减压术
			- 枕骨大孔疝时可采用枕下减压术
			- 大面积脑梗死、重度颅脑损伤致严重脑水肿而使颅内压增高时，可采用去骨瓣减压术
		- 内减压术：开颅术中出现脑组织肿胀膨出，在排除颅内血肿的前提下，可切除失活组织或部分非功能区脑组织，以达到减压目的。

[^1]: 库欣反应：颅内压急剧增高时，病人出现心率变慢、呼吸减慢、血压升高（又称“两慢一高”）， 称为库欣反应。多见于急性颅内压增高病例，慢性者则不明显。
