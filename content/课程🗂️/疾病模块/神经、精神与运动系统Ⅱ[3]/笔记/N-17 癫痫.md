# 概论
>[!Important] 定义
>- **癫痫发作**：是由不同病因引起的脑部神经元高度同步化异常放电所导致的 脑功能失调，具有发作性、短暂性、刻板性、重复性的特点
>- **癫痫**：一种有着不同的病因基础、临床表现各异但以反复癫痫发作为共同特征的慢性脑部疾病状态
>- **癫痫综合征**：由特定病因所致，具有特征性临床和脑电图表现的癫痫
##### 【病因】
1. 结构性
	1. 皮质发育不良
	2. 肿瘤
	3. 头外伤
	4. 脑血管疾病
2. 感染性
3. 遗传性
4. 代谢性
5. 免疫性
6. 不明原因
##### 【影响发作的因素】
1. 年龄
2. 遗传因素
3. 睡眠
4. 内环境改变
##### 【发病机制】
1. **离子通道异常**：<u>神经元高度同步化异常放电</u>是产生癫痫的电生理基础，而异常放电的原因是<u>离子异常跨膜运动</u>，后者的发生则与<u>离子通道结构和功能异常</u>有关。
2. **异常网络重组**：
	- 各种病因引起脑损伤及神经元坏死，坏死后病灶内 残存的神经元、新生神经元及增生的胶质细胞将<u>形成新的神经网络</u>。
	- 当这种异常网络有利于异常电活动形成并传播时就会导致癫痫的发生，而多次癫痫发作可能<u>引起新的神经元坏死</u>，坏死区域残存神经元、新生神经元及胶质细胞又会形成新的网络，加剧癫痫的发生，形成导致癫痫反复发作的恶性循环
3. 脑电图上癫痫样放电与临床发作
	- 当电流增加到足以冲破脑部的 抑制功能，或脑内对其抑制作用减弱时，就会引起临床上的癫痫发作。
	- 癫痫样放电是以**兴奋性谷氨酸（glutamate，Glu）** 为代表的脑内兴奋功能增强的结果
	- 还与**γ- 氨基丁酸（gamma aminobutyric acid，GABA）** 等物质的脑内抑制功能绝对或相对减弱有关
4. 不同类型癫痫发作的可能机制
	- 癫痫样放电被局限在<u>一侧脑部网络内</u>，临床上就表现为**局灶性起源发作**
	- 癫痫样放电在<u>双侧脑部网络内</u>快速扩布则出现**全面性起源癫痫**
	- 异常放电在<u>边缘系统扩散</u>可引起传统分类中的**复杂部分性发作**
	- 异常放电扩散至<u>丘脑和脑干网状结构上行激活系统</u>可出现**失神发作**

# 癫痫发作和癫痫综合征的临床表现
##### 【共同特征】
1. ==发作性==：即症状突然发生，持续一段时间后迅速恢复，间歇期正常；
2. ==短暂性==：即发作持续时间非常短，通常为数秒钟或数分钟，除癫痫持续状态外，很少超过 5～15 分钟；
3. ==重复性==：即第一次发作后，经过不同间隔时间会有第二次或更多次的发作；
4. ==刻板性==：指每次发作的临床表现几乎一致
##### 【全面性起源的癫痫发作】
1. 有明显运动症状的癫痫发作（==运动性==）：主要表现为**强直、阵挛、 强直 - 阵挛及肌阵挛发作**。**负性肌阵挛及失张力发作**则是特殊现象
2. 无明显运动症状的癫痫发作（==非运动性==）：即为==失神发作==，突然发生和迅速终止的意识丧失

# 癫痫的诊断
- 原则：首先明确是否为癫痫发作，随后明确其发作类型，最后区分其是否 为特定的癫痫综合征，同时还要考虑可能的病因和共患病
- 标准：<u>临床出现 2 次（间隔至少 24 小时）非诱发性癫痫发作时就可诊断癫痫</u>
1. **病史与体格检查**
2. **辅助检查**
	1. 脑电图EEG[^1]
	2. 神经影像学检查：MRI较敏感
	3. 其他
##### 【鉴别诊断】
1. *假性癫痫发作*：又称 癔 症 样 发 作、心因性非癫痫性发作PNES，是一种非癫痫性的发 作性疾病，是由心理障碍而非脑电紊乱引起的脑部功能异常。可有类似癫痫发作的运动、感觉症状（如倒下等），难以区分。<u>发作时脑电图上无相应的癫痫样放电和抗癫痫发作治疗无效</u>是鉴别的关键
2. *晕厥*：为弥漫性脑部短暂性缺血、缺氧所致。常有意识丧失、跌倒[^2]。
3. *偏头痛*[^3]：
	1) 癫痫头痛程度较轻，多在发作前后出现，偏头痛则以偏侧或双侧剧烈头痛为主要症状；
	2) 癫痫脑电图为阵发性棘波或棘 - 慢复合波，偏头痛主要为局灶性慢波或正常； 
	3) 简单视幻觉二者均可出现，但复杂视幻觉以癫痫常见；
	4) 癫痫的意识障碍发生突然，很快终止，程度重，而基底动脉型偏头痛的意识障碍发生较缓慢，易唤醒
4. *短暂性脑缺血发作TIA*
	1) TIA 多见于老年人， 常有动脉硬化、冠心病、高血压、糖尿病等病史，持续时间从数分钟到数小时不等，而癫痫可见于任何年龄，以青少年多见，前述的危险因素不突出，发作时间多为数分钟，极少超过半小时；
	2) TIA 的临床症状多为缺失而非刺激，因而感觉丧失或减退比感觉异常多，肢体的瘫痪比抽搐多；
	3) TIA 患者脑电图上无明显的癫痫样放电
5. *低血糖症*：血糖水平低于<u> 2mmol/L</u> 时可产生<u>局部阵挛样抽动或四肢强直发作</u>，伴意识丧失， 患者可有呼吸表浅、双侧瞳孔过大等，常见于[^4]胰岛细胞瘤或长期服降糖药的2型糖尿病患者，病史及发作时的血糖检测有助于诊断

|           | 癫痫发作                       | 假性癫痫发作                                     |
| --------- | -------------------------- | ------------------------------------------ |
