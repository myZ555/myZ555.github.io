>[!Note] 大纲
>1. **重点：**
> 	1. 运动障碍疾病的分类与基本概念：明确区分运动减少综合征（hypokinetic disorders）与运动过多综合征（hyperkinetic disorders）。
> 	2. 帕金森病的核心运动症状与非运动症状。运动症状：静止性震颤、运动迟缓、肌强直、姿势步态障碍。非运动症状：嗅觉下降、便秘、抑郁、睡眠障碍、自主神经功能异常、认知障碍等。
> 	3. 帕金森病的诊断标准及临床诊断思路。 
> 	4. 帕金森病的综合治疗原则，包括药物治疗和非药物治疗，强调“个体化、长期管理”的治疗理念。
> 
> 2. **难点：**
> 	1. 基底节-丘脑-皮层环路的功能与失调机制。直接通路、间接通路的结构与功能。多巴胺在两条通路中的双向调节作用。多巴胺缺乏如何导致运动迟缓、震颤及姿势障碍。 
> 	2. 帕金森病诊断标准的临床化应用。如何在病例中识别帕金森症（运动迟缓 + 震颤或肌强直）。如何判断警示或排除项（如：早期跌倒、早期自主神经功能障碍、眼球活动障碍、共济失调等）。
> 	3. PD 与非典型帕金森综合征（MSA、PSP、DLB）的鉴别。MSA：早期自主神经功能障碍，小脑体征。PSP：垂直眼动障碍、轴性肌张力增高、早期跌倒。DLB：早期认知波动、幻觉。


- 运动系统：下运动神经元、上运动神经元/锥体系统、锥体外系、前庭/小脑系统
- 锥体外系疾病![[锥体外系图示.png]]
- **锥体外系**是指锥体系以外负责协调躯体==随意运动==、控制躯体习惯性和规律性运动的所有神经传导通路
- 锥体外系调节肌张力、协调肌肉的运动、维持体态姿势和产生习惯性动作
- *锥体系产生骨骼肌的随意运动，锥体外系对躯体运动进行调节*，确保锥体系能够进行精确的随意运动。
- 锥体外系各结构之间存在广泛的联系，主要构成 3 个重要的神经环路：
	1. 皮质-纹状体-背侧丘脑-皮质环路
	2. 黑质-纹状体环路
	3. 纹状体-苍白球环路
- 背侧丘脑对皮质的运动功能起**易化**作用。
- 纹状体对背侧丘脑起**抑制**作用。
- 黑质对纹状体的输出功能也起**抑制**作用。
[^1]

| 通路                                   | 作用                       | 主要环路                                                                                                                                                                                                               | 神经递质调控                                                          |
| ------------------------------------ | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------- |
| **直接通路**                             | **促进运动** (Go Pathway)    | 1. **大脑皮层**（兴奋性，Glutamate） → **纹状体（尾状核+壳核）**（抑制，GABA）  <br>2. **内侧苍白球（GPi）& 网状部黑质（SNr）**（抑制，GABA）    <br>3. **丘脑（VL/VA核）**（兴奋性，Glutamate） → **大脑皮层运动区**（促进运动）                                                      | - **多巴胺（D1受体）**（来自黑质致密部，兴奋纹状体，增强运动）                             |
| **间接通路**                             | **抑制运动** (No-Go Pathway) | 1. **大脑皮层**（兴奋性，Glutamate） → **纹状体**（抑制，GABA）  <br>2. **外侧苍白球（GPe）**（抑制，GABA）  <br>3. **丘脑底核（STN）**（兴奋性，Glutamate） → **内侧苍白球（GPi）& 网状部黑质（SNr）**（抑制，GABA）  <br>4. **丘脑（VL/VA核）**（兴奋性，Glutamate） → **大脑皮层运动区**（抑制运动） | - **多巴胺（D2受体）**（来自黑质致密部，抑制纹状体，减少运动抑制，间接促进运动）                    |
| **黑质-纹状体通路** (Nigrostriatal Pathway) | 调节直接和间接通路的平衡，最终**促进运动**  | **黑质致密部（SNc） → 纹状体（释放多巴胺）**                                                                                                                                                                                        | - **D1受体（兴奋直接通路）** → 促进运动  <br>- **D2受体（抑制间接通路）** → 解除运动抑制，促进运动 |
- **直接通路 → 促进运动**
- **间接通路 → 抑制运动**
- **黑质-纹状体通路** 通过释放**多巴胺**调节两者的平衡，最终**促进运动**
    - **帕金森病**时，黑质-纹状体通路受损，多巴胺减少，导致运动减少
    - **亨廷顿病**时，间接通路受损，运动抑制减弱，导致不自主运动

## 帕金森病 PD
##### 【病因】
[^2]
1. 遗传因素
2. 环境因素（毒物：1-甲基-4-苯基-1,2,3,6-四氢吡啶 MPTP）
3. 神经系统老化
4. 多因素交互作用
##### 【发病机制】
- 黑质多巴胺能神经元通过黑质 - 纹状体通路将神经递质多巴胺输送到*纹状体*，参与基底节的运动调节。
- 帕金森病患者的黑质多巴胺能神经元显著变性丢失，黑质 - 纹状体多巴胺能通路变性，*纹状体多巴胺递质水平显著降低*，降至 70% 以上时则出现临床症状。
- <u>多巴胺递质降低的程度与患者的症状严重度呈正相关</u>。 
- 纹状体中*多巴胺与乙酰胆碱（ACh）* 两大递质系统的功能相互拮抗，两者之间的平衡对基底节运动功能起着重要调节作用。
- PD 患者纹状体多巴胺水平显著降低，造成乙酰胆碱系统功能相对亢进。 这种递质失衡及*皮质-基底节-丘脑-皮质环路活动紊乱*与 PD 患者肌张力增高、动作减少等运动症状的产生密切有关。
- 中脑-边缘系统和中脑-皮质系统的多巴胺水平的显著降低是智能减退、情感障碍等高级神经活动异常的生化基础。
##### 【病理】
1. 基本病变[^1]
	1. **黑质致密区多巴胺能神经元及其他含色素的神经元大量变性丢失**
	2. 在残留的神经元胞质内出现**嗜酸性包涵体-路易体 Lewy body**，是由胞质蛋白组成的玻璃样团块，中央有致密的核心、周围有细丝状晕圈。<u>α-突触核蛋白、泛素、热休克蛋白</u>是形成路易小体的重要成分。
2. 新进展学说……
##### 【临床表现】
1. *运动症状*：常始于一侧上肢，逐渐累及同侧下肢，再波及对侧上肢及下肢
	1. **静止性震颤**：常为首发症状，静止位时出现或增强，随意运动时减轻或停止。典型表现是拇指与屈曲的示指间呈==“搓丸样”==（pill-rolling）动作。
	2. **肌强直**：患者处于放松体位时，屈肌与伸肌张力同时增高。四肢及颈部主要关 节的被动运动缓慢。关节被动运动时始终保持阻力增高，似弯曲软铅管，称为==铅管样强直==（lead-pipe rigidity）；如患者伴有震颤，检查者感觉在均匀阻力中出现断续停顿，如同转动齿轮，称为==齿轮样强直==（cogwheel rigidity）。
	3. **运动迟缓**：查体见面容呆板、双眼凝视、 瞬目减少，酷似==“面具脸”==（masked face）；口、咽、腭肌运动迟缓时，表现为语速变慢、语音低沉、吐字欠清；书写字体越写越小，呈现==“小字征”==（micrographia）；做快速重复性动作如拇指、示指对指时表现为运动速度缓慢和幅度减小
	4. **姿势步态与平衡障碍**：在疾病早期，表现为走路时患侧上肢摆臂幅度减小或消失，下肢拖曳。有时行走中步伐突然变小或停止，好似双脚突然粘在地上，不能动弹，称为==“冻结”==（freezing）现象。有时迈步后，以极小的步伐越走越快，不能及时止步，称为==前冲步态==（propulsion）或==慌张步态==（festination）。
2. *非运动症状*
	1. 感觉障碍：早期即可出现*嗅觉减退*，中、晚期常有肢体麻木、疼痛。
	2. 睡眠障碍：以*快速眼动睡眠行为障碍 RBD*常见
	3. 自主神经功能障碍：临床常见，如便秘、多汗、溢脂性皮炎（油脂面）等
	4. 精神和认知障碍
##### 【辅助检查】
1. 实验室检查
2. 嗅觉测试（减退）
3. 影像学检查（正电子发射断层扫描PET或单光子发射计算机断层扫描SPECT检查在疾病早期甚至临床前驱期即能显示异常）
4. 病理（外周组织如胃窦部和结肠黏膜、下颌下腺、皮肤等部位可以检见<u>α- 突触核蛋白异常聚积</u>）
##### 【诊断】
![[PD诊断流程.png]]
1. 首先明确帕金森病综合征
	- 帕金森综合征诊断的确立是诊断帕金森病的先决条件[^3]
	- 帕金森综合征的诊断条件： **运动迟缓**，合并*静止性震颤*或（和）*肌强直*
2. 再明确帕金森综合征是否属于PD
	1. 至少存在2条支持标准（4条）[^5]
	2. 不存在绝对排除标准（9条）[^4]
	3. 没有警示征象（10条）[^6]
3. PD的诊断
	1. 临床确诊PD：不符合绝对排除标准+至少2条支持性标准+无警示征象
	2. 很可能的PD：不符合绝对排除标准+若出现警示征象需通过支持性标准抵消
		- 1条警示征象：≥1条支持性标准
		- 2条警示征象：≥2条支持性标准
##### 【鉴别诊断】
1. *继发性帕金森综合征*（相关病史是关键）
2. *其他神经变性疾病的帕金森综合征（帕金森叠加综合征）*：所伴发的帕金森症状常以强直、少动为主，震颤少见，对左旋多巴治疗不敏感
3. **特发性震颤 ET**：特征是姿势性或运动性震颤；发病年龄较早、1/3患者有家族史；饮酒或服心得安后震颤显著减轻；无肌强直和运动迟缓；姿势性震颤 
4. **多系统萎缩 MSA**：累及锥体外系、锥体系、小脑和自主神经系统等多部位
	- MSA-P型：以帕金森综合征为突出表现
	- MSA-C型：以小脑性共济失调为突出表现 
5. **进行性核上性麻痹 PSP**：轴性肌张力增高，早期就有姿势平衡障碍；震颤不明显；核上性眼肌麻痹，垂直凝视不能（尤其是向下视）；对左旋多巴反应差；MRI可见脑干和中脑萎缩 
6. **肝豆状核变性 Wilson病**：遗传性铜代谢障碍疾病；铜在肝、脑、肾、角膜等处沉积；发病年龄小；其他类型的不自主运动，如舞蹈样动作；肝脏损害；*角膜K-F环 (Kayser-Flersher ring)*；血清铜蓝蛋白降低、尿铜增加 
##### 【治疗】
==原则：药物治疗首选、主要治疗手段==
- 总体目标：有效改善症状，提高工作能力和生活质量；早期诊断，早期治疗
- 治疗方案个体化
- 避免、推迟或减少药物的副作用和运动并发症
	- 剂量滴定 
	- 尽可能以小剂量达到较满意临床效果
- 药物治疗特点：只能改善症状，不能完全阻止病情的发展

###### 外科治疗
- 原理：纠正基底节过高的抑制性输出、仅能改善症状
- 适应症：药物治疗效果减退、不能耐受或出现运动障碍的患者
- 方法：苍白球、丘脑毁损术；深部脑刺激术DBS

## 小舞蹈症
- **小舞蹈症（chorea minor）** 又称西德纳姆舞蹈症（Sydenham chorea）、风湿性舞蹈症，是==风湿热==在神经系统的常见表现
- 多见于儿童和青少年
- 临床特征为<u>舞蹈样动作、肌张力降低、肌力减退和 / 或精神症状。</u>
- 小舞蹈症发病与 *A 组乙型溶血性链球菌感染*有关
- 病理改变主要为黑质、纹状体、丘脑底核、小脑齿状核及大脑皮质充血，水肿，炎性细胞 浸润及神经细胞弥漫性变性。
- 临床表现：
	1. *舞蹈症*：可以是全身性，也可以是一侧较重，主要累及面部和肢体远端
	2. *肌张力低下和肌无力*：可有明显的肌张力低下和肌无力
	3. *精神障碍*
	4. *其他*
## 亨廷顿病 HD
- **常染色体显性遗传**，多在中年起病，缓慢进展，主要症状为舞蹈样动作、精神异常和进行性认知障碍
- 本病的致病基因 **IT15**（interesting transcript 15）位于第 4 号染色体 **4p16.3**， 编码一个含 3144 个氨基酸的多肽，命名为 Huntingtin。
- 在 IT15 基因 **5’端编码区**内存在一个**三核苷酸（CAG）重复序列**，该重复序列异常扩增拷贝数超出一定范围可导致疾病的发生，且==拷贝数越多，发病年龄越早，临床症状越重==。
- *病理变化*：皮质 - 基底节 - 丘脑 - 皮质环路是主要受累区域，表现为不同程度的神经元丢失和 胶质细胞增生，并以尾状核的改变最为显著。
- *生化改变*：纹状体传出神经元中γ- 氨基丁酸、乙酰胆碱及其合成酶明显减少，多巴胺浓度正常或略增加；
- 临床表现：患者后代常有发病提前倾向，称为**遗传早现**，其中父系遗传的遗传早现现象更明显。
	1. <u>锥体外系症状</u>：舞蹈样不自主运动最常见、最具特征性。首发症状多始于颜面部及上肢。
	2. <u>精神障碍及痴呆</u>
- *基因检测*：遗传学检测 IT15 基因 CAG 重复序列异常扩增拷贝数，当 CAG 重复次数为 36～ 39 时，呈不完全外显，部分携带者可不发病；<u>如 CAG 拷贝数≥40 次，呈完全外显，所有携带者均发病</u>

	5. 可导致帕金森综合征或疑似与患者症状相关的其他疾病， 或者基于整体诊断学评估，专家感觉可能为其他综合征， 而不是PD (继发性帕金森综合征）
	6. 采用多巴胺受体阻滞剂或多巴胺耗竭剂治疗，且剂量和时间过程与药物诱导的帕金森综合征一致 （药物性帕金森综合征）
	7. 发病超过3年仍局限在下肢的帕金森综合征的表现 （血管性帕金森综合征）
	8. 突触前多巴胺能系统功能神经影像学检查正常 （分子影像）
	9. 尽管病情至少为中等严重程度，但对高剂量的左旋多巴治疗 缺乏可观察到的治疗应答

[^5]: 支持性标准：
	1. 多巴胺能药物治疗具有明确且显著的有效应答
	2. 出现左旋多巴诱导的异动症
	3. 临床观察到单个肢体静止性震颤（既往或本次检查）
	4. 辅助测试结果阳性

[^6]: 警示征象：
	1. 发病5年内出现快速进展的步态障碍，且需要经常使用轮椅
	2. 发病3年内由于平衡损害导致的反复摔倒
	3. 发病5年内出现[^7]球麻痹症状
	4. 发病5年内出现吸气性呼吸功能障碍：白天或夜间吸气性喘鸣或频繁的吸气性叹息
	5. 发病5年内出现严重的[^8]自主神经功能障碍
	6. 发病10年内出现不成比例的颈部前倾（肌张力障碍）或手足挛缩
	7. 发病≥5年，运动症状或体征完全没有进展；除非这种稳定是与治疗相关的
	8. 病程超过5年无任何一种常见的非运动症状
	9. 其他原因不能解释的锥体束征
	10. 双侧对称性的帕金森综合征，没有任何侧别优势，且客观体检也没有观察到明显的侧别性

[^7]: 球麻痹症状：表现为严重的发音困难、构音障碍或吞咽困难（须进食较软的 食物，或通过鼻胃管、胃造瘘进食）

[^8]: 自主神经功能障碍：包括：
	1. 直立性 低血压，即在站起后 3 分钟内，收缩压下降至少 30mmHg 或舒张压下降至少 20mmHg，并排除脱水、药 物或其他可能解释自主神经功能障碍的疾病；
	2. 发病后 5 年内出现严重的尿潴留或尿失禁（不包括女 性长期存在的低容量压力性尿失禁），且不是简单的功能性尿失禁（如不能及时如厕），对于男性患者 来说，尿潴留必须不是由前列腺疾病引起的，且伴发勃起障碍

[^9]: [[21 抗帕金森病药]]

[^10]: ==疗效减退（wearing-off）/剂末现象==：每次用药有效作用时间缩短，症状随血药浓度发生波动。处理：
	1. 增加每日服药次数
	2. 增加每次服药剂量
	3. 改用缓释剂
	4. 加用其他辅助药物

[^11]: ==“开-关”现象（on-off phenomenon）==：症状在突然缓解（开期）与加重（关期）之间波动。特点：与服药时间、血药浓度无关，发生机制不详；处理：可用DA受体激动剂、左旋多巴微泵持续输注

[^12]: ==异动症==：
	1. *剂峰异动症*：出现在血药浓度高峰期，与用药过量或纹状体DA受体超敏有关。处理：减少复方左旋多巴单次剂量、可加用DA受体激动剂
	2. *双相异动症*：剂峰与剂末均可出现。处理：治疗困难，可尝试增加复方左旋多巴单次剂量及服药次数；加用DA受体激动剂
	3. *肌张力障碍*：足或小腿痛性痉挛，多发生在清晨服药前。处理：睡前服用复方左旋多巴控释剂或长效DA受体激动剂；起床前服用复方左旋多巴
