# 肠结核
**肠结核**：是==结核分枝杆菌==引起的==肠道慢性特异性感染性疾病==，常继发于肺结核
> [!quote] 
> 以腹痛、排便异常、腹部包块和全身中毒症状为主要临床表现
> 多见于中青年，女性稍多于男性，近年本病发病率有所增加
##### 【病因和发病机制】
1. **呼吸道结核感染而吞下含菌痰液**或与开放性肺结核病人共餐被感染（人型）[^1]
2. 饮用带菌牛奶/乳制品（牛型）
3. 血行播散（粟粒性肺结核）；**腹盆腔内结核病灶直接蔓延**
##### 【病理】
<u>最常见于回盲部</u>
> [!important] 回盲部是肠结核好发部位的原因
> 1. 含结核分枝杆菌的肠道内容物在回盲部停留时间较长，增加了肠黏膜的感染机会
> 2. 回盲部淋巴组织丰富，而结核分枝杆菌易侵犯淋巴组织

| 溃疡型                                                                                             | 增生型                                          | 混合型                 |
| ----------------------------------------------------------------------------------------------- | -------------------------------------------- | ------------------- |
| 肠壁的集合淋巴组织和孤立淋巴滤泡首先受累-充血、水肿->==干酪样坏死==->边缘不规则、深浅不一的溃疡<br>病变修复过程中-纤维组织增生+瘢痕形成->肠管狭窄<br>较少发生穿孔/大出血 | 黏膜下层及浆膜层可有大量结核肉芽肿和纤维组织增生->局部肠壁增厚、僵硬/瘤样肿块突入肠腔 | 兼有两种改变；临床及影像表现可交替出现 |
> [!note] 病理学
> - 特征形态：**干酪样坏死性上皮样肉芽肿**，肉芽肿体积较大且相互融合
> - 结肠活检可见*肉芽肿位于黏膜下层*，表面黏膜结构改变轻微
> - 鉴别：==克罗恩病肉芽肿体积小、数量少==
> - 因病变肠段常与周围组织发生粘连，==多不发生急性穿孔==，慢性穿孔形成腹腔脓肿或肠瘘亦少见
> - 因溃疡基底常有==闭塞性动脉内膜炎==，较少发生大出血
##### 【临床表现】
1. 腹痛（==右下腹/脐周==；间歇发作；餐后加重）
2. 大便习惯改变（溃疡型-糊样、不伴里急后重）
3. 腹部肿块（==右下腹固定包块==）
4. 全身症状和肠外结核表现（结核毒血症-溃疡型）==低热、盗汗、消瘦==
##### 【辅助检查】
1. 实验室检查（==血沉增快==、大便脓细胞与红细胞、结核菌素试验）
2. 影像学
	- <u>溃疡型</u>：X 线钡剂灌肠可见钡剂于病变肠段排空增快，充盈不佳，而在病变的上、下肠段钡剂充盈良好，称为 ==**X 线钡剂激惹征**==。
	- <u>增生型</u>：肠黏膜呈结节状改变，肠腔变窄、肠段缩短变形、回肠盲肠正常角度消失
3. CT/MRI：可见腹腔淋巴结肿样坏死或钙化
4. ==结肠镜==
	- 内镜下见病灶处黏膜充血水肿、==环形溃疡==、==回盲瓣固定开放==、==溃疡瘢痕伴假息肉/炎性息肉形成==及肠腔狭窄等表现
	- 病灶处活检，发现肉芽肿、干酪样坏死或抗酸杆菌，可以确诊。
	- 黏膜组织聚合酶链式反应 （TB-qPCR）阳性可协助诊断
##### 【鉴别诊断】
1. *克罗恩病*[^2]
2. *原发性肠道淋巴瘤*（腹膜后和肠系膜淋巴结显著肿大，免疫组织化学检查可协助诊断）
3. *阿米巴病/血吸虫病性肉芽肿*（既往有感染史，脓血便常见；大便果酱色；病原体）
##### 【治疗】
1. ==抗结核化学药物治疗（关键）==[^3]
2. 对症治疗：腹痛-抗胆碱药
3. 手术治疗（注意指征）
4. 病人教育
# 结核性腹膜炎
**结核性腹膜炎**：结核分枝杆菌引起的慢性弥漫性腹膜感染
##### 【病因】
- 多继发于肺结核/体内其他部位结核病
- 主要感染途径：腹腔内结核病灶直接蔓延
##### 【病理】
1. **渗出型**：腹膜*充血、水肿*，表面覆有==纤维蛋白渗出物==，可伴黄（灰）白色细小、可融合结节。腹腔积液通常呈*草黄色或淡血性*，偶为乳糜性。
2. **粘连型**：==大量纤维组织增生和蛋白沉积==使*腹膜、肠系膜明显增厚*。肠袢相互粘连时可发生肠梗阻。
3. **干酪型**：多由渗出型或粘连型演变而来，可兼具上述两型病理特点，并发症常见。==以*干酪坏死病变*为主==，坏死的肠系膜淋巴结参与其中，形成*结核性脓肿*。病灶可累及邻近器官，如穿透空腔脏器或腹壁可形成*窦道或瘘管*。
##### 【临床表现】
1. 全身症状：*结核毒血症*常见；==弛张热/稽留热==、盗汗
2. *腹痛*：脐周/下腹/全腹；持续/阵发性隐痛
3. 腹胀
4. *腹壁柔韧感*：常有<u>揉面感</u>（并非特征性体征）；如压痛明显且有反跳痛--*干酪型结核性腹膜炎*
5. 腹部肿块（多由增厚的大网膜、肿大的肠系膜淋巴结、粘连成团的肠袢或干酪样坏死脓性物积聚而成）
##### 【辅助检查】
1. 血液检查（轻度-中度贫血；==血沉增快==）
2. 结核菌素试验及γ- 干扰素释放试验 ==PDD试验强阳性==
3. 腹腔积液（==草黄色渗出液==、ADA活性增高）
4. X线（腹部平片有时可见钙化影-==肠系膜淋巴结钙化===）
5. B超/CT/MRI（==网膜污垢、增厚==，网格状；肠系膜僵硬）
6. 腹腔镜（腹膜/网膜/内脏表面有散在或集聚的==灰白色结节==，浆膜失去正常光泽、浑浊粗糙）
> [!important] 结核性腹膜炎的腹腔积液
> - **草黄色渗出液**为主，少数为淡血性，偶见乳糜性
> - ==比重>1.018,蛋白质>30g/L，白细胞计数>500 * 10E6/L,白细胞分类以淋巴细胞为主==
> - 腺苷脱氨酶（ADA）增高， SAAG小于11g/L
##### 【鉴别诊断】
（诊断标准[^4]）
1. 以*腹腔积液*为主要表现
	1. 腹腔恶性肿瘤
	2. 肝硬化腹腔积液（漏出液）
2. 以*腹部包块*为主要表现【腹部肿瘤、克罗恩病】
3. 以发热为主要表现【伤寒、淋巴瘤】
4. 以急性腹痛为主要表现【外科急腹症】
##### 【治疗】
1. 营养支持
2. 抗结核化学药物[^5]
3. 腹腔穿刺
4. 手术治疗-适应证
5. 病人教育

> [!question] Anki
> 1. 渗出性结核性腹膜炎：四联；==异烟肼+利福平+吡嗪酰胺+链霉素==
> 2. 乙胺丁醇的副反应：==球后视神经炎==
> 3. 链霉素：==耳毒性==、前庭功能损害和肾毒性
> 4. 结核性腹膜炎抗结核药物的疗效一般比溃疡型肠结核==略差==
> 5. **溃疡型肠结核**：常伴腹泻/腹泻与便秘交替出现；有全身症状
> 6. **增生型肠结核**：==便秘==；无明显结核毒血症状
> 7. 肠结核的主要感染途径是==胃肠道感染==
> 8. 肠结核一般好发于回盲部，体表位置为==右下腹或脐周==
> 9. 肠结核最有诊断价值的检查时*结肠镜*下活检找到==干酪性上皮样肉芽肿==，而不是大便中查到结核感觉（仅表明肠道内有结核杆菌、可能是咽下引起）
> 10. 结核性腹膜炎的发热特点：主要==低热/中等热==，约1/3患者呈弛张热，少数病人呈稽留热
> 11. 结核性腹膜炎-腹胀、==麻痹性肠梗阻==
> 12. ==腹膜活检==对其结核性腹膜炎有确诊价；*血培养*很难培养出结核分枝杆菌，且周期长，不用于确诊结核性腹膜炎
> 13. 腹水结核杆菌培养阳性率很低，腹水浓缩结核杆菌培养阳性率可增高，但==腹水中多不能找到结核杆菌==


[^1]: 该菌为抗酸菌，很少受胃酸影响，进入肠道后多在回盲部引起病变。这是因为：
	1. 含结核分枝杆菌的肠内容物在回盲部停留，增加局部黏膜的感染机会；
	2. 回盲部富有淋巴组织，细菌易侵犯淋巴组织。

[^2]: 肠结核与克罗恩病的鉴别

|              | 肠结核    | 克罗恩病        |
| ------------ | ------ | ----------- |
| 肠外结核         | 多见     | 一般无         |
| 病程           | 复发不多   | 病程长、缓解-复发交替 |
| 瘘管/腹腔脓肿/肛周病变 | 少见     | 可见          |
| 病变节段性分布      | 常无     | 多节段         |
| 溃疡形状         | 环形、不规则 | 纵行、裂沟状      |
| 结核菌素试验       | 强阳性    | 阴性/阳性       |
| 抗结核治疗        | 好转     | 无好转         |
| 抗酸杆菌染色       | 可阳性    | 阴性          |
| 干酪性肉芽肿       | 可有     | 无           |

[^3]: [肺结核-化学治疗](../../心血管、呼吸、血液与泌尿系统Ⅱ[3]/笔记/I-2-08%20肺结核.md#【化学治疗】)

[^4]: 以下情况考虑本病：
	1. 中青年病人，有结核感染史或接触史，伴有其他器官结核 病证据；
	2. 长期发热原因不明，伴有腹痛、腹胀、腹腔积液、腹壁柔韧感或腹部包块；
	3. 腹腔积液为渗出液，以淋巴细胞为主，普通细菌培养阴性，ADA（尤其是 ADA2）明显增高；
	4. 影像学检查发现肠粘连、 腹膜增厚、肠梗阻或散在钙化点；
	5. 结核菌素试验强阳性或γ- 干扰素释放试验阳性。
