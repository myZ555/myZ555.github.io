# 小肠解剖与生理
###### 分段与结构
- 十二指肠：20-25cm；C形包绕胰头；胆汁与胰液汇入的关键部位
- 空肠：小肠上段2/5；管壁厚、血供丰富、环形皱襞密集，主要吸收段
- 回肠：小肠下段3/5；管壁薄、集合淋巴小结丰富，胆盐与维生素B12吸收的关键部位
###### 组织学
1. **黏膜层**：含环形皱襞、肠绒毛及微绒毛，极大增加吸收面积
2. **黏膜下层**：含较大血管、淋巴管及Meissner神经丛，调控黏膜分泌与血流
3. **肌层**：内环外纵，协调产生分节运动与蠕动，推进食糜
4. **浆膜层**：即腹膜脏层，光滑的浆膜面减少器官间摩擦，维持腹腔稳态
###### 血管与神经
- 血管丰富、<u>其动脉弓结构是外科手术中判断血供和切除范围的重要解剖标志</u>
- ==动脉供血==：主要来自**肠系膜上动脉SMA**，分支形成动脉弓；空肠1-2级、回肠3-4级。*分支越多表明末梢血供越需要侧枝代偿*
- 静脉回流：汇入肠系膜上静脉SMV，最终经*脾静脉*汇入门静脉系统，进入肝脏代谢处理
- 神经支配：交感神经、副交感神经及*肠神经系统ENS* 三重调控[^5]
###### 淋巴引流
*乳糜管*：小肠绒毛内的淋巴毛细管因含有脂肪微粒而呈乳白色
- 绒毛乳糜管：吸收脂肪形成乳糜
- 肠壁淋巴结：汇集来自绒毛淋巴
- 肠旁淋巴结：初级过滤与免疫反应
- 肠系膜淋巴结：向干道汇流
###### 生理功能
> 每日分泌消化液总量：6-8L；约80%在小肠内重吸收

1. 消化吸收
2. 分泌功能
3. 免疫防御
4. 运动推进

# 短肠综合症
短肠综合征（short bowel syndrome，SBS）是指由各种原因造成广泛小肠切除或旷置后，**肠道有效吸收面积减少，残存的功能性肠管不能维持机体营养吸收**，出现*营养不良、腹泻、水电解质紊乱及代谢障碍*为主的症候群。

##### 【病因】
- *血管性*：肠系膜血管栓塞或血栓形成，导致肠管 急性缺血坏死，常需紧急大段切除
- *机械性*：肠扭转、腹内外疝绞窄、腹部外伤等导致肠管血运障碍，继发肠坏死
- *炎症与手术*：克罗恩病反复手术、放射性肠炎及不恰当的外科扩大切除
##### 【病理生理】
1. 有效吸收面积减少
	1. 轻度：切除小肠达 50%～70% 可引起吸收不良的表现
	2. 严重→SBS：**==残存小肠少于 75cm（有完整结肠），或丧失回盲瓣、残存小肠少于 100cm==**
2. 再吸收障碍：*胆盐与维生素B₁₂*吸收障碍，引发脂肪泻与巨幼细胞性贫血
3. 胃肠激素紊乱：胃泌素水平升高→胃酸分泌增多→胃排空加速→消化性 溃疡与腹泻加重
4. 回盲瓣影响："回肠闸"消失，肠内容物排空加快，结肠细菌逆行移 居小肠，加重吸收不良
##### 【临床表现】
- 早期：严重腹泻(每日多次水样便)、水电解质紊乱(低钾、低钠、低镁)、代谢性酸中毒
- 后期：体重下降肌肉萎缩、贫血、低蛋白血症、代谢性骨病、肝功能异常、维生素及微量元素缺乏
##### 【分期治疗】
- 急性期：0-2m；稳定生命体征、纠正电解质紊乱
	1. 维持内环境稳定、*尽早开始全[^2]肠外营养*；
	2. 减少肠道分泌，[^1]应用质子泵抑制剂PPI、H2受体拮抗剂及*奥曲肽*，减少胃肠液分泌量
	3. 促进代偿启动 当腹泻量降至*2 L/d以下*时，给予少量等渗[^3]肠内营养，刺激肠黏膜代偿性增生
- 代偿期：2m-2y；逐步恢复肠功能、过渡营养支持（促代偿药物：*谷氨酰胺*-肠黏膜细胞主要能量来源；*生长激素*-促进肠黏膜细胞增殖；*GLP-2类似物*（替度鲁肽）-特异性促进肠道适应）
- 维持期：>2y；长期营养管理、预防并发症
##### 【外科干预】
1. 减缓肠道运行，如建立小肠瓣和括约肌、逆蠕动肠段、结肠间置等；
2. 增加肠表面积，包括肠变细增长术、小肠移植等。
# 肠息肉与肠息肉病
> 肠息肉及肠息肉病是一类从黏膜表面突出到肠腔内的**隆起状病变**的临床诊断，可分为以下几类：
> *息肉数目 在 100 枚以上者称为息肉病，反之则称为散发性息肉。*

##### 【分类】
1. ==普通型腺瘤性息肉==：成人最常见类型，直径 >2 cm者约半数 癌变，**绒毛状腺瘤**癌变率更高，是**癌前病变**的典型代表
2. ==炎性息肉==：炎症反应刺激肠上皮引起，可继发于任何一种炎症反应或感染性疾病，*一般没有恶变倾向*
3. 锯齿状息肉
	1. <u>增生性息肉HP</u>：结直肠最常见的非肿瘤性息肉，常多发，直径多 <5 mm，一般不需特殊治疗
	2. <u>无蒂锯齿状腺瘤/息肉SSA/P</u>：与*BRAF基因*突变相关结直肠癌的前驱病变，好发于右半结肠
	3. <u>传统锯齿状腺瘤TSA</u>：与*KRAS基因*突变相关结直肠癌关系更为密切
4. ==错构瘤性息肉==：幼年性息肉及色素沉着息肉综合征；儿童多见，*大多发生于10岁以下*
5. 其他：化生性息肉及黏膜肥大赘生物。
##### 【治疗】
- 有蒂或直径＜2cm 的无蒂腺瘤性息肉可内镜下切除
- 更大的无蒂 / 宽基底息肉应鉴别恶性肿瘤，必要时行手术治疗
- 恶变或高度可疑恶变者，按结直肠癌治疗原则行根治性手术[^6]
##### 【肠息肉病】
1.  ==色素沉着息肉综合征（Peutz-Jeghers 综合征）==：**错构瘤**一类；青少年多见；家族史。多发息肉见于全 消化道（小肠占64%），口唇、颊部及四肢 末端可见特征性色素沉着斑。
2.  ==家族性肠息肉病FAP==：*5号染色体长臂APC基因突变所致显性遗传病*，青年发病，表现为结直肠满布腺瘤。若不治疗几乎所有FAP病人都将发展为结直肠癌
3.  ==肠息肉病合并多发性骨瘤和多发性软组织瘤（Gardner 综合征）==：可有肠外表现（多发性骨瘤和软组织瘤）；*癌变倾向明显*
# 小肠肿瘤
##### 【分类】
- 良性：*腺瘤、平滑肌瘤*；脂肪瘤、纤维瘤、血管瘤等
- 恶性：*小肠间质瘤GIST、神经内分泌肿瘤NET*、腺癌、恶性淋巴瘤、平滑肌肉瘤、部分转移性肿瘤
##### 【临床表现】
1. **腹痛**：最常见；可为隐痛、胀痛，当并发肠梗阻时常为剧烈疼痛。
2. **肠道出血**：常为间歇性排柏油样便、血便，少数可出现大出血，有的因长期隐性出血表现为慢性贫血。
3. **肠梗阻**：引起急性肠梗阻最常见的原因是继发性肠套叠，肿瘤引起的肠腔狭窄和压迫邻近肠管也是发生肠梗阻的原因，少数可诱发肠扭转。
4. **腹内肿块**：活动度大、位置不固定，触诊可扪及。 
5. **肠穿孔**：多见于小肠恶性肿瘤，急性穿孔导致腹膜炎，慢性穿孔则形成肠瘘。 
6. **类癌综合征**：多见于伴有肝转移的*神经内分泌肿瘤*，最常见的症状是腹泻、支气管痉挛、皮肤潮红。
##### 【诊断】
①影像学 ②内镜检查 ③探查手术
##### 【手术策略】
- 良性：小/带蒂肿瘤-*局部切除*；
- 较大/局部多发-肠段切除吻合术；
- 恶性-连同肠系膜及区域淋巴结行*根治性切除*；
- 无法切除者-*短路手术*
# 肠梗阻
##### 【病因】
1. 梗阻原因
	1. ==机械性==：肠腔狭小或不通。①肠外因素 ②肠壁因素，如肠套叠等 ③肠腔内因素，如蛔虫梗阻、异物、粪块等
	2. ==动力性==：神经抑制或毒素刺激，*肠壁肌运动紊乱*， 使肠蠕动丧失或肠管痉挛 
		1) **麻痹性**-常见；*腹腔手术后、腹部创伤或弥漫性腹膜炎病人*
		2) **痉挛性**-少见；急性肠炎、肠道功能紊乱或慢性铅中毒
	3. ==血运性==：肠系膜血管栓塞或血栓形成造成*肠管血运障碍*，肠失去蠕动能力。> 可迅速继发肠坏死
2. 肠壁血运有无障碍：
	1. 单纯性：仅肠内容物通过受阻、无血运障碍
	2. **绞窄性：相应肠段血运障碍，可引起肠坏死、穿孔**
3. 梗阻部位：
	1. 高位(空肠)-呕吐早切频
	2. 低位(回肠)-腹胀显著
	3. 结肠/[^4]==闭袢性梗阻==(回盲瓣-肠内容物只能从小肠进入结肠)
4. 程度：
	1. 完全性
	2. 不完全性 
##### 【病理生理】
==局部==：机械性肠梗阻发生后，梗阻以上肠管蠕动增强、逐渐膨胀，而梗阻以下肠管则瘪陷。*两者交界处即为梗阻部位*-术中定位
	- **肠腔压力升高**（肠内容物及气体积聚）
	- **静脉回流受阻**（肠壁充血水肿、液体外渗至腹腔和肠腔）
	- **缺血坏死穿孔**（闭袢性肠梗阻肠内压更高，可致肠管破溃穿孔及腹膜炎）
==全身==
1. *水电解质和酸碱失衡*：胃肠道分泌的液体不能被吸收返回全身循环+肠壁继续有液体向肠腔内渗出-->体液在第三间隙的丢失
	- 高位-不能进食、呕吐-脱水-**代谢性碱中毒**
	- 低位-丢失*碱性消化液*、组织灌注不良酸性代谢产物剧增-**代谢性酸中毒**
2. *血容量下降*
3. *休克*：最后可引起严重的**低血容量性休克**和**中毒性休克**
4. *呼吸和心脏功能障碍*
##### 【症状】
**痛、吐、胀、闭**
1. ==腹痛==
	- 机械性肠梗阻：*阵发性绞痛*，间歇期缩短转为持续性腹痛需警惕绞窄性
	- 麻痹性肠梗阻：持续性胀痛
2. ==呕吐==
	- 高位：呕吐早且频
	- 低位：呕吐晚、后期为粪样物
	- 呕吐物*棕褐色或血性*提示肠管血运障碍
3. ==腹胀==
	- 高位：腹胀不明显
	- 低位及麻痹性：腹胀显著
	- *闭袢性肠梗阻*：腹部隆起不对称
4. ==排气排便停止==
	- 完全性：梗阻远端肠管排空
	- 绞窄性如*肠套叠*可排*血性黏液性粪便*

| 项目  | 表现                          |
| --- | --------------------------- |
| 视诊  | 可见肠型和蠕动波，提示*机械性肠梗阻*         |
| 触诊  | 单纯性有轻度压痛；绞窄性有*固定压痛及肿块*      |
| 叩诊  | 绞窄性可出现*移动性浊音*（腹腔积液）         |
| 听诊  | 机械性肠鸣音亢进、可闻气过水声；麻痹性肠鸣音减弱或消失 |
##### 【辅助检查】
1. 早期无明显异常、后期出现白细胞、血红蛋白及尿比重升高
2. 血气分析及电解质检测了解内环境状况
3. 呕吐物、粪便隐血阳性提示*肠管血运障碍*
4. X线检查：气液平面
5. ==CT检查-重要==；显示梗阻部位、原因及肠壁血运情况
	- 胆石性肠梗阻：CT见*胆管积气、肠腔内异位胆石及小肠扩张-Rigler三联征*
	- 粪石性肠梗阻：肠腔内可见高密度或混杂密度粪石影，多发生于老年卧床患者
	- 肠粘连：CT可见*小肠扩张与塌陷的移行带*，粘连处肠壁增厚或成角
	- **肠套叠**：CT显示*同心圆征*及肠管嵌套表现
	- 腹内疝：小肠疝入异常间隙、*肠系膜血管聚拢*
> [!Important] 诊断要点：是否为肠梗阻？
> - 四大症状：痛吐胀闭
> - 特征性体征：肠型、蠕动波、肠鸣音亢进
> - 辅助检查-腹部X线：液气平面、扩张肠袢
> - 机械性vs动力性
> 	- *机械性*：表现典型，梗阻以上肠管扩张、以下塌陷，有明确的移行带
> 	- *动力性*：多继发于腹部手术或炎症，腹胀显著，X线见*大小肠均匀扩张*
##### 【治疗】
###### 非手术治疗
1. **胃肠减压**
	- 减少胃肠道积气积液，减轻肠腔膨胀与腹内压，利于肠壁血运恢复。
	- 低位肠梗阻可用*小肠减压管*
	- 结直肠支架置入
2. **纠正紊乱**（先补充平衡盐溶液）
3. **防治感染**（应用广谱抗生素）
4. **其他措施**（吸氧-改善肺功能；生长抑素-减少胃肠液分泌）
###### 手术治疗
1. *单纯解除梗阻*（粘连松解、肠切开取异物、肠套叠或扭转复位等）
2. *肠切除肠吻合*（肿瘤/炎症狭窄/失活坏死肠管）
3. *肠短路吻合*（切除困难时，分离远近端作短路吻合，避免旷置肠管过长引发盲袢综合征）
4. *肠造口或肠外置*（后期二期手术重建）
# 特殊类型肠梗阻
#### 粘连性肠梗阻
- 术后最常见的肠梗阻类型
- 术中预防措施：
	- 减少异物刺激
	- 精细操作
	- 无菌与引流
	- 术后早期活动
- 手术方式
	- 粘连松解
	- 肠切除吻合
	- 短路吻合
	- 肠排列术
#### 腹茧症
- 一种罕见的腹膜增厚包裹小肠的疾病；全部或部分小肠被纤维包膜包裹如蚕茧状，可导致反复肠梗阻发作
- 手术治疗：切开包膜、松解粘连、恢复肠管通畅，避免损伤肠管浆膜面
#### 肠扭转
- 指肠管沿系膜长轴扭转360-720度形成的*闭袢性绞窄性肠梗阻*
==小肠扭转==
	- 发病急，进展快，剧烈腹痛无间歇
	- 早期可致休克，痛可放射至腰背部
	- X线：绞窄性梗阻征象及空回肠换位
==乙状结肠扭转==[^7]
	- 老年人多见、常有便秘史
	- X线：圆顶向上双腔充气肠袢、*双液平*
	- 钡剂灌肠：尖端*鸟嘴形*-特征性征象
#### 肠套叠
- 肠的一段套入其相连的肠管腔内；多见于*幼儿*
- 三大典型症状：*腹痛、果酱样血便、腊肠形腹块*
- 影像学表现：超声同心圆征、钡灌肠杯口状征、双肠轨道征/腊肠征
- 治疗：
	- 非手术：空气或钡剂灌肠复位（早期回盲型/结肠型）
	- 手术：肠壁损伤严重或肠坏死时切除，必要时造口
#### 肠系膜血管缺血性疾病
- 肠系膜血管急性血液循环障碍致肠管短时间缺血坏死，形成*血运性肠梗阻*。
- *死亡率高、易漏诊*，是外科急腹症中最凶险的疾病之一
1. **动脉栓塞**：发病急骤，突发难缓解剧痛，*症状与体征不符为特征*。白细胞早期显著升高。早期无腹 膜炎可行血管内介入取栓。
2. **动脉血栓形成**：先有慢性缺血（餐后腹痛、食物恐惧、消瘦 ），急性阻塞时表现似动脉栓塞。全程需抗凝治疗。
3. **静脉血栓形成**：症状发展慢、不典型，有前驱症状。CT见 SMV增粗、充盈缺损。有腹膜炎时尽早剖腹 ，术后继续抗凝。
#### 非闭塞性肠系膜缺血NOMI
- 指*肠系膜血管功能性收缩，导致肠道血流灌注不足*，但无实际血管阻塞。
- 常发生于危重症患者，且诊断困难，预后较差
【特点】无明确血管阻塞（栓塞/血栓形成），主要由于肠系膜血管痉挛导致血流不足
【病因】低血容量、休克、心力衰竭、脓毒症、应用血管收缩剂（如 洋地黄、升压药）等，引起肠系膜动脉持续收缩。
【临床表现】*腹痛症状与体征不符（腹痛剧烈但腹部体征轻微）*，晚期可 出现肠坏死及腹膜炎表现
【诊断】*DSA（数字减影血管造影）*可见肠系膜动脉弥漫性狭窄和分支稀疏*
【治疗】纠正原发病，局部血管扩张剂灌注（如罂粟碱），必要时手术切除坏死肠段
