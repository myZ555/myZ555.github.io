## 炎症性肠病 IBD
- 慢性、非特异性肠道炎症性疾病
- 病因不明确，与*免疫*、遗传、环境等有关

|      | 溃疡性结肠炎 UC                                 | 克罗恩病 CD                                   |
| ---- | ----------------------------------------- | ----------------------------------------- |
| 症状   | 脓血便多见                                     | 有腹泻但脓血便较少见                                |
| 病变分布 | 连续                                        | 节段性                                       |
| 直肠受累 | 绝大多数受累                                    | 少见                                        |
| 肠腔狭窄 | 少见，中心性                                    | 多见，偏心性                                    |
| 内镜表现 | 溃疡浅，黏膜弥漫性充血水肿，颗粒状，脆性增加                    | 纵行溃疡，卵石样外观，病变间黏膜外观正常（非弥漫性）                |
| 活检特征 | 固有膜全层弥漫性炎症，隐窝脓肿，隐窝结构明显异常，杯状细胞减少           | 裂隙状溃疡，非干酪性肉芽肿，黏膜下层淋巴细胞聚集                  |
