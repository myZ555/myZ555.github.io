#内分泌系统  #病理生理

**学习目标**：
1. 复述内分泌功能紊乱的定义、分类和常见病因
2. 说出典型内分泌疾病中激素分泌异常的调控机制、解释临床表现
3. 根据上述概念解释内分泌功能紊乱的病理生理学治疗基础

---
**内分泌系统**：内分泌腺+内分泌组织/细胞群
- 内分泌腺：垂体、甲状腺、肾上腺等 
- 内分泌组织：胰岛 
- 内分泌细胞：存在于几乎所有组织器官中具有神经 内分泌功能的细胞。
![400](../Attachments/Pasted%20image%2020250422094914.png)
==内分泌系统调控==
**Nerve System**：神经纤维调控激素的合成和释放
[](../课件/3%20内分泌紊乱病理生理%202025%20任俊明.pdf#page=5&annotation=688R)
- 内分泌系统受到中枢神经系统的调控，如神经递质可影响激素的分泌，激素的变化也可影响神经系统的功能。
- *下丘脑*是联系神经系统和内分泌系统的中枢，下丘脑含有重要的神经核，同时可以合成并释放激素。下丘脑、垂体与靶腺之间存在着反馈调节。
**Immuno System**：产生细胞因子等免疫活性物质调节内分泌系统的功能
- 机体中的递质或激素与淋巴细胞表面受体结合起到免疫应答的过程。
- 同时免疫系统可以通过细胞因子对神经内分泌系统发生影响，起到反向调节作用。
- 内分泌、免疫、神经三个系统共同协作调控机体的生理功能。
Endocrine System：内分泌系统分泌的激素可以影响其他内分泌腺的功能和激素分泌，形成复杂的内分泌网络

==**Hormones**==
[](../课件/3%20内分泌紊乱病理生理%202025%20任俊明.pdf#page=6&annotation=690R)
- 是细胞分泌的微量活性物质，由血液输送到远处组织并通过受体而发挥调节作用的化学信使物质。现代内分泌学中已把激素范围扩大到具有局部调节作用的旁分泌和自身调节作用的自分泌物质。
- 自分泌、旁分泌、内分泌
- 固醇类、多肽/蛋白质类、氨基酸类（甲状腺激素）
- 大部分激素与其特异性的转运载体结合，只有少部分以游历的形式存在。一般，肽类激素的半衰期较短，而类固醇类较长。
**两种主要的激素激活机制：**
1. 肽类或蛋白类激素：能与其膜受体形成复合物，进而改变受体构象，产生第二信使，如cAMP，cGMP，ca，等，激活酪氨酸激酶或其他磷酸酶，（IP3，DAG，PKC）磷酸化活化相关信号转录因子蛋白，激活核内的遗传物质的转录/表达，从而合成新的蛋白质来行使其生物学功能。
2. 类固醇类/甲状腺激素：具有较高的脂溶性，能通过细胞膜与其受体形结合，形成活化的聚合物，促使dna的基因转录以及mrna的翻译，从而合成行使其生物学功能的新蛋白。
- 受体与激素结合的特点，*特异性，高亲和性，可逆性和饱和性*。

==内分泌调节轴==
- 调控下丘脑、垂体与靶腺的重要方式。其中靶腺调节下丘脑和垂体的激素合成和分泌称为长反馈调节轴。 
- 其中较为典型的feedback调节轴为，甲状腺，肾上腺，性腺。

## 内分泌系统功能紊乱

**功能低减**：先天性腺体发育异常，合成激素酶缺陷，炎症，肿瘤，手术切除 等疾病导致的激素作用障碍, 激素受体缺陷或受体后缺陷。可采用激素代替治疗，药物治疗，移植等。
**功能亢进**：激素分泌过度或激素作用过度。可采用手术治疗，药物治疗，放射性治疗等。

*原发性*：内分泌腺自身功能异常而引起的激素分泌异常
*继发性*：垂体功能异常而引起的激素分泌调节失常
*三发性*：下丘脑功能异常引起

### （一）肾上腺功能紊乱
- 皮质激素分泌调节：
	- 下丘脑调节肽促肾上腺皮质激素释放素CRH——*刺激*——垂体激素促肾上腺皮质激素ACTH释放——激活cAMP促进肾上腺糖皮质激素GC释放（CRH-ACTH-GC）

| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 脂肪    | **向心性肥胖**，面如**满月**，红润多脂，颈部**水牛背**，锁骨上窝脂肪垫，**悬垂腹**，四肢相对瘦细<br>*皮质醇促进脂肪分解，抑制脂肪合成*——四肢(对皮质醇敏感)脂肪分解，皮下脂肪减少，加之肌肉萎缩—— 四肢瘦细；<br>*皮质醇抑制葡萄糖利用*，促进糖异生使血糖增高——胰岛素分泌增加——促进脂肪合成——面部和躯干(对胰岛素敏感)脂肪合成占优，形成肥胖。 |
| 糖     | 拮抗胰岛素，抑制糖利用，促进糖异生，使血糖升高，半数患者糖耐量减低，20％的患者 **类固醇性糖尿病**。                                                                                                                                       |

| 心血管系统 | RAS系统过度激活，血管舒张系统抑制。*高血压，左心室肥大，心力衰竭，脑血管意外*。凝血功能异常、脂代谢紊乱--易发生动静脉血栓                                                                                                                            |
| 免疫系统  | 免疫力低下，易发生各种感染（皮肤毛囊炎、牙周炎、泌尿系感染等）；感染不易控制，易发展为败血症和毒血症                                                                                                                                          |
| 神经系统  | **肌无力**，不同程度的精神、情绪变化                                                                                                                                                                        |
| 性腺功能  | 高皮质醇血症直接影响性腺，抑制**下丘脑**分泌促性腺素释放素和**垂体**分泌黄体生成素，性功能减退。网状带分泌的**雄激素增加**，在妇女可导致闭经、男性化和多毛症。                                                                                                       |

#### 功能减退--==Addison's Disease==
- 由于双侧肾上腺的绝大部分被毁所致。继发性者由下丘脑-垂体病变引起。
- 病因：感染（肾上腺结核常见）、自身免疫性肾上腺炎、其他（肿瘤转移、先天性发育不良）
- 主要症状：

| 部位    | 表现                                              |
| ----- | ----------------------------------------------- |
| 皮肤    | **黏膜色素沉着**，垂体负反馈减弱，ACTH，MSH，LPH分泌增多，全身皮肤颜色加深    |
| 神经/精神 | 乏力，淡漠，疲劳，重者嗜睡，意识模糊，可出现精神失常                      |
| 胃肠道   | 食欲减退，*嗜咸食*，胃酸过少，消化不良；有恶心，呕吐，腹泻                  |
| 心血管   | *血压降低，心脏缩小，心音低钝*；可有头昏眼花，直立性昏厥                   |
| 代谢    | 糖异生作用减弱，肝糖原损耗，可发生**低血糖**症状                      |
| 泌尿    | 排水能力减弱，大量饮水后出现*稀释性低钠血症*；糖皮质激素缺乏/血容量不足-抗利尿激素释放增多 |
| 生殖    | 女性阴毛、腋毛减少/脱落，月经失调/闭经；男性性功能减退                    |
| 免疫    | 对感染/外伤等应激的抵抗力减弱，可出现**肾上腺危象**                    |

### （二）生长激素功能紊乱
![200](../Attachments/Pasted%20image%2020250422102011.png)
- GH抑制葡萄糖摄取和利用，增加氨基酸摄取和蛋白质合成
#### 功能亢进--==巨人症 Gigantism==
- 青春期前体内GH过度分泌，即使过了生长发育阶段仍然继续生长。男>2.0m，女>1.85m
- 临床表现：
	- 单纯少见，成年后半数以上继发肢端肥大症
	- 始于幼年，持续长高至骨骼闭合，肌肉发达，臂力过人，性器官发育较早，*血糖偏高，少数患者有垂体性糖尿病*
	- 患者生长至最高峰后，逐渐开始衰退，精神不振、四肢无力、肌肉松弛、毛发脱落、性欲减退、外生殖器萎缩、体温下降、心率减慢、血糖降低
	- 多数可因心血管疾病而死亡
#### 功能亢进--==肢端肥大症 Acromegaly==
- 主要是平骨的过度生长。比例为百万分之3~4。95%为生长激素细胞腺瘤引起。
- 眉导，颧骨，下巴突出，鼻大，唇厚，舌肥厚，头大，手足宽厚如铁铲。
- 病因：
	1. **垂体肿瘤压迫**：头痛、视物模糊/视野缺损/复视、腺垂体功能减退、高PRL血症(→闭经、溢乳)、下丘脑功能障碍(食欲亢进/肥胖/睡眠障碍/尿崩症.etc)
	2. **GH过度分泌**：骨、软骨和软组织生长过度，特殊面貌(头围增大，眉弓和颧骨过长，唇肥厚，鼻宽舌大，下颌增大前突，牙齿稀疏，咬合困难)
#### 功能减退--==侏儒症 Nanism==

- 生长激素缺乏或分泌不足，导致身高不能正常生长（即通常所指的侏儒症），如果不进行生长激素替代治疗，最终身高只能达到130厘米左右。

### （四）甲状腺功能紊乱

**下丘脑-垂体-甲状腺轴负反馈机制**
下丘脑--*TRH 促甲状腺激素释放激素*--垂体--*TSH 促甲状腺激素*--甲状腺--*T3 三碘甲状原氨酸+T4 甲状腺素*

**甲状腺激素THs功能**
1. 启动和维持机体细胞的分化和生长，尤其在儿童生长、神经发育和成熟以及保障中枢神经系统的功能至关 重要。
2. 刺激线粒体用氧、线粒体蛋白合成和线粒体生成
3. 影响心血管血流动力学和心脏的收缩和舒张功能
4. 影响生殖系统功能

#### Type I Diabetes

[[2 糖尿病病理#1. 胰岛素依赖性糖尿病 IDDM（1型、幼年型，10%）|T1DM]]
- 一型糖尿病是指胰岛素B细胞破坏导致**胰岛素绝对缺乏**所引起的糖尿病。可分为两类： 
	- 自身免疫性疾病(细胞+体液免疫)
	- 特发性糖尿病（原因未明）

#### Type II Diabetes

[[2 糖尿病病理#2. 非胰岛素依赖性糖尿病 NIDDM（2型、成年型，90%）|T2DM]]
- 指胰岛素抵抗为主并伴有胰岛素相对不足到胰岛素分泌不足为主伴胰岛素抵抗所致的糖尿病。因此**胰岛素抵抗**和**B细胞功能缺陷**为2型糖尿病的基本特征。
- 伴有骨骼肌摄取葡萄糖；肝糖异生上调
- 40-90% 2型病患中胰岛中存在淀粉样沉积，损伤β细胞，降低胰岛素分泌。  
- GCk葡萄糖激酶，是葡萄糖感受器的重要组成部分，调节胰岛素的分泌。 突变-B细胞缺陷
- Glut2能引起干胰岛素抵抗，降低b细胞胰岛素分泌。
