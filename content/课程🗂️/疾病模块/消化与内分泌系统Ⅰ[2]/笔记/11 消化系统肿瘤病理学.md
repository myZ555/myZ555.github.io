# 二、胃癌
>胃肠道癌的发生：正常黏膜——异常增生IEN——癌
>==胃癌的科迪亚级联反应 Correa's Cascade==
>>**慢性胃炎--萎缩性胃炎和肠化生--异常增生--胃癌**

- 是中国消化系统恶性肿瘤中最常见的死亡原因
- 通常出现在50~60
- 男女比例约2-3:1
## 常见病因
- 幽门螺旋杆菌 HP
	- 慢性胃炎、胃溃疡、MALT型边缘带淋巴瘤、胃癌
	- 通常由父母传染给子女：口-口或粪-口
	- 附着于 *胃型黏液表面细胞*；不存在于肠型细胞或鳞状细胞
	- 终生生活在宿主体内
- 吸烟
- X射线
- 石棉；日本-滑石粉
- 硝酸盐或亚硝酸盐-亚硝化作用
- 腌菜
- 咸鱼
## 胃良性上皮性息肉
aka 腺瘤性息肉/胃腺瘤，胃癌的癌前病变

# 三、结直肠癌
![Pasted image 20250521230520](../Attachments/Pasted%20image%2020250521230520.png)
### 腺瘤
>**肠息肉 polyp**：临床诊断用语；肉眼所见的隆起于黏膜表面的境界清楚的病灶，传统统称为息肉，病变分为腺瘤和息肉两大类
>**腺瘤 adenoma**：上皮增生活跃，总是伴有*上皮内瘤变（异型增生）*，可*恶变成腺癌*。息肉一般不恶变，但如伴有IEN也可恶变，如幼年性息肉如合并有上皮内瘤变则可恶变。
>**息肉病 polyposis**：大多有遗传缺陷，癌发生率明显增高
##### 定义：
- 经典：以异型增生上皮构成的局灶性良性肿瘤，有肿物且界限清楚
- WHO肿瘤分类：无明确原因的、组织学上有上皮内瘤变存在的病灶。
	1. 无论病灶是否息肉状，而以组织学改变为标准；
	2. 炎症等明确原因引起的IEN不列入腺瘤的范畴
###### 数目：
- 可单发或多发，一般很少超过50-60个
- 若超过100个，称为腺瘤病
###### 类型：
- 隆起 elevated
	- 有蒂 pedunculated

### 结直肠癌

>==**结直肠癌的Vogelstein模型**==
>>正常黏膜--IEN--癌
- 定义：结直肠黏膜上皮来源，具有恶性形态学特征的细胞和/或结构，**穿过黏膜肌层**，浸润黏膜下层或以上
- 好发部位：直肠50%，乙状结肠20%，盲肠+升结肠16%
##### 结直肠上皮内瘤变 IEN
- 根据黏膜上皮不同程度的细胞和结构异型性，分为低级别和高级别
##### 结直肠黏膜内瘤变/黏膜内癌 intramucosal neoplasia/carcinoma
- 高级别异型增生细胞*进入黏膜固有层，但未穿透黏膜肌层*

# 四、原发性肝癌
## (一) 肝细胞性肝癌
##### 病因和发病机制
- 肝硬化
- HBV, HCV
- 酒精
- 黄曲霉素
- 遗传或代谢疾病
##### 病理变化

| 大体                                                                         | 光镜                                                                                                                                                |
| -------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| 可表现为**巨块型、多发结节型或弥漫型**。一般质软，常有出血坏死，偶尔可有淤胆而呈绿色。有的肿瘤可有包膜。大小差异很大，一般小于3cm称为小肝癌。 | ==高分化==：细胞多排列成**细小梁状**并常有假腺样或腺泡样结构，常有脂肪变。瘤细胞间有**丰富的血窦样腔隙--毛细血管化**。==低分化==：以实性生长类型为主，其间很少血窦样腔隙，仅见裂隙样血管。癌细胞明显异型性，瘤巨细胞。一般肿瘤间质稀少，偶见间质丰富者，称为硬化性肝细胞性肝癌。 |

- 肿瘤常侵入门脉系统形成门脉瘤栓。晚期病例几乎均有门静脉瘤栓

- 发烧、疼痛、血清甲胎蛋白**AFP+**
## (二) 胆管细胞癌
- ==大体==：肿瘤通常灰白、实性、硬韧，大多数表现为肝内灰白色结节或融合的结节；结节中常见坏死和瘢痕。累及肝门者主要表现为肝脏明显淤胆、胆汁性肝硬化和继发性胆道感染。
- ==光镜==：大多为不同分化程度的**腺癌**。发生于较大胆管者，可形成乳头状结构。肿瘤常有丰富的间质反应，可见局部钙化

## (三) 混合型肝癌
- 肝细胞癌和胆管细胞癌两种成分同时存在
- 预后不良

# 五、胆道癌
## (一) 胆囊癌
- ==大体==：巨大息肉样肿块，充填胆囊腔内，或呈结节状，或弥漫浸润使胆囊壁明显增厚。偶尔可成环形浸润使胆囊形成哑铃状。常发生于胆囊底部。
- ==光镜==：80%以上为不同分化程度的腺癌，腺体间可有大量间质。
- 大多数胆囊癌在发现时已经侵入肝脏或已累及胆囊管、肝门的淋巴结，有时可转移至腹膜后、胃肠道和肺。
## (二) 肝外胆道癌
>肝外胆管：左右肝管、总肝管、胆囊管、总胆管

==肝内外的胆管细胞引起的腺癌==
肝内：肝脏肿块和非特异性症状（体重减轻、疼痛）
肝外：黄疸，大便脱色，恶心呕吐

- ==大体==：胆管壁内呈灰色结节状，质地变硬，呈砂砾状，弥漫性浸润性变，胆管壁增厚，呈乳头状，息肉样病变
- ==光镜==：腺管结构+丰富的纤维间质

# 六、胰腺癌
**"癌王**
- 根据部位可分为胰头癌、胰体癌、胰尾癌和全胰癌
- 临床上胰头癌大多数因累及总胆管而表现为进行性阻塞性黄疸
![Pasted image 20250524161539](../Attachments/Pasted%20image%2020250524161539.png)
- ==大体==：大多为一质地硬韧、与周围组织界限不清的肿块；切面灰白色或黄白色，有时因出血、囊性变和脂肪坏死而夹杂有红褐色条纹或斑点。原有胰腺结构消失。
- ==光镜==：大多数为**导管腺癌**，主要由异型细胞形成不规则、有时不完整的管状或腺样结构，伴有丰富的纤维间质；高分化导管腺癌主要由分化好的导管样结构构成，内衬高柱状上皮细胞。中分化者由不同分化程度的导管样结构组成，有的和高分化腺癌相似。低分化导管腺癌仅见少许不规则腺腔样结构。70%的胰腺癌可侵袭周围神经丛。

	- 上皮样细胞，梭形细胞，或混合

# 总结
1. 消化系统癌占了新发癌和癌死亡的一半
2. 食管癌、胃癌和肝癌发病率在下降，结直肠癌在上升；结直肠癌已是第二位新发癌，癌死亡第4、5位
3. 消化道早癌可在内镜下治疗（切除）
4. 术后化疗有较好的疗效；抗肿瘤免疫治疗是近年的热点
5. 基于分子改变的分层是个体化治疗的基础
6. 胆道癌发生率在增加
7. 胰腺癌是“癌王”
# 基于LLM思考题
##### 1. 消化道不同部位早癌定义有何不同、为什么？
- 主要体现在**侵犯深度的标准**和**是否伴随淋巴结转移的容许程度**上。这些差异是由各部位**解剖结构、生理功能、组织学特征**以及**癌症进展规律**决定的，目的是为临床提供更合理的治疗决策依据。

| 部位  | 早癌定义                    | 原因分析                                |
| --- | ----------------------- | ----------------------------------- |
| 食管  | 癌细胞未突破黏膜下层，无论有无淋巴结转移    | 食管黏膜下层血管丰富，淋巴管密集，哪怕是黏膜内膜癌也可能早期转移    |
| 胃   | 癌局限于粘膜层或黏膜下层，无论是否有淋巴结转移 | 胃黏膜较厚，黏膜下层淋巴管分布相对均匀，早期胃癌虽可能N+，但预后较好 |
| 结直肠 | 癌局限于黏膜层或黏膜下层，要求无淋巴结转移   | 黏膜下层结构复杂，癌细胞一旦突破黏膜，转移风险显著增加         |

##### 2. 结直肠癌定义与食管癌和胃癌有何不同，其临床意义是什么？
- 关键：要求无淋巴结转移
- 临床意义：
	- 结直肠黏膜下层结构转移，一旦突破黏膜，转移风险显著增加；
	- 食管癌和胃癌早期定义较宽，是为了扩大内镜切除的适应症范围；结直肠癌早期定义较窄，是因为一旦有转移必须追加根治性手术，内镜治疗对于侵犯黏膜下层的癌灶风险评估标准不完善
##### 3. 简述结直肠不同上皮性病变的治疗方法
| 病变类型          | 主要治疗方法      | 是否需追加治疗     | 术后随访策略      |
| ------------- | ----------- | ----------- | ----------- |
| 低级别上皮性病变LGIEN | 内镜切除（EMR）   | 无           | 1年内镜复查      |
| 高级别上皮性病变HGIEN | ESD或整块EMR   | 依据病理评估是否追加  | 6-12月复查     |

| T2及以上（进展期癌）   | 根治术+辅助治疗    | 是           | 长期肿瘤随访（>5年） |
