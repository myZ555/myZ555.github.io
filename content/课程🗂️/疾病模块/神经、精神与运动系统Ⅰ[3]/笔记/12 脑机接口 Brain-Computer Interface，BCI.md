# 基本概念、术语

- ==脑机接口==：在大脑与外部设备之间建立的一条直接通路，通过双向交互、协同工作来实现脑机混合智能，也是实现人工智能的新途径
	- 大脑--*绕过外周神经与运动系统*-外部世界
- 脑机接口的基本原理：思维意图--神经活动--采集的神经信号--解码后的思维形态
- 脑机接口的基本框架：![[Pasted image 20250313133654.png|500]]
- 脑机接口的分类：

| **根据交互方式分类**   |                                                                             |
| -------------- | --------------------------------------------------------------------------- |
| 主动式 BCI        | 输出的控制信号反映大脑的随意性活动，不依赖于外部事件                                                  |
| 反应式 BCI        | 由外部刺激引发出大脑活动，再根据大脑做出的反应产生输出                                                 |
| 被动式BCI         | 不以控制为目的，在人机交互过程中为计算机提供大脑的隐形状态，从而利于计算机及时作出调整，实现个性化的交互                        |

# 非侵入式脑机接口 BCI

![[../课件/12 脑机融合-姚林.pdf#page=18&selection=0,0,0,7&color=red]]

| 时间   | 脑机接口的发展        | 特点                                                           |
| ---- | -------------- | ------------------------------------------------------------ |
| 1973 | concept of BCI |                                                              |
| 1988 | P300 BCI       | 事件相关电位；P300峰值                                                |
| 1991 | SMR BCI        | 运动感觉节律（SMR: sensory motor rhythm）可以通过脑电波<br>控制机械手抓握/在二维平面上活动 |
| 1993 | MI BCI         | “想象运动” 实现脑控 MI：motor imagery（瘫痪--实现自理）                       |
| 1999 | SCP BCI        | 3Hz以下的慢波节律也可受到调节<br>SCP：slow cortical potential (<3Hz)       |
| 2002 | SSVEP BCI      | Steady-state visual evoked potential 5Hz                     |
| 2006 | SSSEP BCI      | Steady-state somatosensory evoked potential 26/31Hz          |
| 2008 | m-VEP BCI      | motion-onset visual evoked potential                         |
| 2011 | MRCP BCI       | movement related cortical potential (0.05-3Hz)<br>运动相关皮层电位   |

- 总结：非侵入式脑机接口
	- 优势：无创，方便，易开展研究
	- 不足：
		1. **信噪比低**：脑电图到达头皮传感器之前经过骨骼和皮肤，无法提供颅内记录中与任务相关的精确神经元信号【信号大量衰减】
		2. **部分依赖外部刺激**：外界视觉刺激（计算机屏幕等）
		3. **主动式需训练**：想象运动等主动式BCI
		4. **只能完成简单任务**：三维任意控制 ×


# 侵入式脑机接口 BMI

- 脑信号获取手段的进展决定了脑机接口的起源和发展
- 早期：1950-1998
	- 1950年：第一个概念性BMI
	- 1970年代末：第一个猴子脑控光标BMI
	- *早期最成功的产品*
		- 人工耳蜗：最成功的早期产品
		- 视觉皮层假体：通过表面电极网格将电流施加到失明患者的视觉皮层，识别简单视觉对象
- 兴起：1999- **原因：多通道神经元记录取得突破**
	- *微丝 microwires*  新型多电极可记录大脑多个单神经元
	- 一系列成果：
		1. 人脑控二维电脑光标
		2. 猴子脑控三位机械臂实现自主进食（三维运动+一维抓握的机械臂控制）
		3. 人脑控三维机械臂自主喝咖啡
		4. 信息输入：皮层微电刺激重建触觉
		5. 脑控外骨骼
		6. 脑控机械臂喝啤酒：植入后顶叶皮层PPC区电极，实现意念控制机械臂
		7. 脑控双机械臂完成复杂任务（吃蛋糕）
		8. 语音解析：从脑信号合成语音（脑活动--嘴--语音）
		9. 不得不提马斯克：小型可植入神经芯片和超细电极线--植入猪脑--实现实时信号读取

# 脑机融合/智能

- 脑与机的双向信息通道
	- 脑→机：电，光，磁
	- 机→脑：光，超声，电
	- *人工视网膜*
- 脑机融合→混合智能
	- 概念：以脑机混成系统为载体，通过神经连接通道，实现生物感认知基本单元与机器计算功能单元之间的层次化多尺度的深度融合，达到感知、认知、运动等功能的增强、替代、补偿等目的
	- *神经智能假肢，记忆假体，意念控制器*
	- 三种增强类别：
		1. 感知增强混合智能：视觉增强、触觉恢复&增强
		2. 认知增强混合智能：虚假记忆植入、学习能力增强
		3. 行为增强混合智能：神经接口、机械手控制
