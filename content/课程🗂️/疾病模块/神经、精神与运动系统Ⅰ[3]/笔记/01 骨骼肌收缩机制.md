# （一）骨骼肌细胞的微细结构

- **肌原纤维 myofibril**：横纹肌细胞内含大量平行排列的肌原纤维，纵贯肌细胞全长
- **肌节 sarcomere**：横纹肌细胞收缩和舒张的基本功能单位；肌节=1/2明+暗+1/2明=2条Z线间的区域
- ![[肌原纤维结构模式图.png|400]]
- 肌原纤维明暗带相间？2套不同的肌丝及不同程度的重叠
	- **粗肌丝**：两端游离，中间固定于M线
	- **细肌丝**：一端固定于Z线，一端游离
	- 安静情况下：M线两侧无细肌丝插入-->明亮H带；H两侧暗带-粗细肌丝重叠区
	- 细肌丝是粗肌丝数量的2倍

### 粗肌丝 thick filament
- **肌球蛋白myosin （长杆状）**
- 2条重链、2条碱性轻链、2条调节轻链
- 尾部、铰链部、横桥
- ![[肌球蛋白结构.png|300]]
- ==横桥== cross-bridge
	- 具有ATP结合位点及ATPase，结合并分解ATP，提供横桥扭动（肌丝滑行）和做功的能量
	- 与细肌丝肌动蛋白的横桥结合位点发生可逆性结合，势能转换为动能，拉动细肌丝滑行
### 细肌丝 thin filament

| 成分               | 功能                                                 |
| ---------------- | -------------------------------------------------- |
| 肌动蛋白 actin       | 表面有横桥结合位点，与肌丝滑行直接相关；静息时该位点被原肌球蛋白覆盖<br>与肌球蛋白合称为收缩蛋白 |
| 原肌球蛋白 tropomysin | 肌动蛋白与横桥之间；遮盖结合位点，位阻作用。调节蛋白                         |
| 肌钙蛋白 troponin    | 3个亚单位<br>与钙离子结合变构，使原肌球蛋白位移，暴露结合位点。调节蛋白<br>         |
![[细肌丝结构.png|400]]
- ==肌管系统==muscular tubular system
	- 肌管系统是横纹肌细胞实现兴奋-收缩耦联的关键结构
	- *def*，包绕在每条肌原纤维周围的囊管状结构，两组来源和功能不同的独立管道系统
	- **横管系统 transverse**	T管，肌膜内凹形成；内含细胞外液，肌膜AP沿T管传导
	- **纵管系统 longitudinal**	L管，肌质网SR，钙离子储存释放。*终池*：肌节两端的SR形成膨大，富含钙离子（胞内钙库）
- 三联管 triad：T管+终池*2

# （二）骨骼肌收缩分子机制（肌丝滑行理论）
![[肌丝滑行理论.png|300]]![[骨骼肌收缩分子机制.png|200]]

- **肌肉收缩：水解ATP释放的化学能--机械能**
- ==横桥周期==：横桥与肌动蛋白结合、扭动、解离、复位与再结合的过程
	1. 横桥（高势能状态）与细肌丝肌动蛋白结合（[Ca2+]i!）
	2. 横桥扭动（构象改变，势能转换为肌节长度的缩短和/或克服负荷的张力）（ADP+P解离）
	3. 横桥与细肌丝解离（ATP结合）
	4. 横桥水解ATP获能并复位
# （三）骨骼肌的兴奋-收缩耦联
- *def*, 肌细胞的电兴奋和肌细胞的机械收缩连接起来的中介过程，excitation-contraction coupling
- 过程：
	1. AP扩布到横管系统
	2. 三联管处的信息传递
	3. 终池膜上钙通道开放，触发肌丝滑行
- 最关键的因素是细胞质内钙离子浓度升高（**钙瞬变 calcium transient**）
- 胞内钙瞬变的结构基础：
	- 骨骼肌 RyR1 电-机械耦联
	- 心肌 RyR2 钙诱导钙释放
