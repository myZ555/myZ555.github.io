#### 1. 神经纤维的兴奋传导和突触传递
###### 1. 神经纤维传导兴奋的特征和影响因素
【特征】
	1. 完整性
	2. 绝缘性
	3. 双向性
	4. 相对不疲劳性
【影响因素】
- 神经纤维直径：直径越大、传导越快
- 有无髓鞘：有髓>无髓（郎飞结-跳跃式传导）
- 髓鞘厚度
- 体温（体温在一定范围内升高、传导加快）
###### 2. 化学性突触传递的过程及其影响因素
【过程】
	1. 突触前膜钙通道开放
	2. 突触囊泡递质释放
	3. 递质抵达突触后膜
	4. 突触后电位形成
【影响因素】
1. 释放：进入神经末梢的钙离子量
2. 清除
3. 突触后膜反应性
###### 3. 突触后电位、突触后神经元动作电位的产生
1. 兴奋性突触后电位 EPSP：突触后膜在某种神经递质作用下产生的局部去极化点位；兴奋性神经递质--后膜对钠、钾通透性增大，且钠内流>钾外流--净内向电流
2. 抑制性突触后电位 IPSP：氯离子通道开放--内流--突触后膜超极化
![[突触后电位.png]]
###### 4. 中枢兴奋传播的特征
1. 单向传播：突触前末梢--突触后神经元；感觉传入神经--运动传出神经
2. 中枢延搁：兴奋在中枢传播时，比在相同长度的神经纤维上传导时所额外花费的时间，*本质上是指在反射过程中花费在反射中枢的所有化学突触传递上的时间*
3. 兴奋的总和：多个神经元的 EPSP 在时间和空间上的总和达到突触后神经元的阈电位
4. 兴奋节律改变：运动传出神经的放电频率与感觉传入神经的放电频率也往往不同
5. 后发放与反馈：效应器所引起的变化可再次作为刺激因素被感受器感受并引起反射效应
6. 对内环境变化敏感和易疲劳
#### 2. 外周神经递质和受体
###### 1. 乙酰胆碱及其受体
- 由胆碱能神经元、胆碱能受体以及表达胆碱能受体的神经元或效应细胞一起构成的胆碱能系统，是体内分布和涉及作用最广的神经信号传递系统。
**乙酰胆碱 ACh**：胆碱和乙酰辅酶 A 在胆碱乙酰转移酶催化下于胞质中合成，然后被输送到轴突末梢储存于小而清亮透明的突触囊泡内
**毒蕈碱受体 M受体**： G 蛋白耦联受体
	- M 样作用可被 M 受体拮抗剂阿托品阻断
**烟碱受体 N受体**：促离子型受体，具有递质门控特性，也称 N 型 ACh 门控通道
	- N1/Nn受体：中枢神经系统和自主神经节后神经元
	- N2/Nm受体：骨骼肌神经肌肉接头处的终板膜上
		- N2 受体因受自身免疫抗体攻击而功能异常是重症肌无力的重要发病机制
	- 烟碱样作用，简称 N 样作用，可被筒箭毒碱阻断

###### 2. 去甲肾上腺素及其受体
- 属儿茶酚胺类物质；以酪氨酸为合成原料
- 能与 NE 和肾上腺素结合的受体称为肾上腺素能受体；均属 G 蛋白耦联受体，可分为 α 型和 β 型两类以及多种亚型
	- 心肌主要表达 β 受体
	- 血管平滑肌表达 α 和 β 两种受体；皮肤、肾、胃肠的血管平滑肌上以 α 受体为主，而在骨骼肌和肝脏的血管则以 β 受体为主

#### 3. 反射
###### 1. 非条件反射和条件反射
【非条件反射】指生来就有、数量有限、比较固定和形式低级的反射活动，如屈肌反射、瞬目反射、性反射等；建立无须大脑皮层的参与，通过皮层下各级中枢就能形成
【条件反射】是在非条件反射的基础上，通过后天学习和训练而形成的反射，是反射活动的高级形式；与其形成有关的主要中枢部位在大脑皮层
###### 2. 反射的中枢整合、中枢神经元之间的联系方式
【单突触反射】传入神经元和传出神经元之间，在中枢只经过一次突触传递的反射
【多突触反射】在中枢经过多次突触传递的反射
*反射的整合*同时涉及低级和高级中枢，使反射活动变得更复杂和更具适应性

==中枢神经元之间的联系方式==
1. 单线式联系：一个突触前神经元仅与一个突触后神经元发生突触联系。视网膜视锥系统
2. 辐散和聚合式联系：
	1. 辐散式联系：一个神经元通过其轴突侧支或末梢分支与多个神经元形成突触联系，传入通路中较多见
	2. 聚合式联系：一个神经元可接受来自许多神经元轴突末梢的投射而建立突触联系；传出通路中较多见
3. 环式或回返式联系：后一级的神经元会通过其侧支再次与前一级神经元发生突触联系，从而在结构和功能联系上都形成闭合的环路。可因负反馈而使活动及时终止，也可因正反馈而使兴奋增强和延续。即使最初的刺激已经停止，传出通路上的冲动发放仍能继续一段时间，这种现象称为【后发放或后放电】

#### 4. 神经系统的感觉功能
###### 1. 感受器的一般生理特性
1. 感受器的适宜刺激：一种感受器通常只对某种特定形式的刺激最敏感
2. 感受器的换能作用：将作用于它们的特定形式的刺激能量转换为相应的感受器电位或发生器电位，并最终引起传入神经的动作电位
3. 感受器的编码功能：感觉系统将刺激信号转变为可识别的感觉信号，主要包括刺激的类型、部位、强度和持续时间四种基本属性
4. 感受器的适应现象：某一恒定强度的刺激持续作用于一个感受器时，其传入神经纤维上动作电位的频率会逐渐降低
###### 2. 感觉通路中的信息编码和处理
1. 感觉通路对刺激类型的编码：不同类型感觉的引起，除了与不同的刺激类型及其相对应的感受器有关外，还取决于传入冲动所经过的专用通路及其最终到达的大脑皮层的特定部位。
2. 感觉通路中的感受野：指由所有能影响某中枢感觉神经元活动的感受器所组成的空间范围。
3. 感觉通路对刺激强度的编码：当刺激较弱时，阈值较低的感受器首先兴奋；当刺激强度增加时，阈值较高的感受器也参与反应，感受野将扩大
4. 感觉通路中的侧向抑制：与来自刺激中心区感觉神经元的信息相比，来自刺激周边区的信息则是抑制的。能加大刺激中心区和周边区之间神经元兴奋程度的差别、增强感觉系统的分辨能力。是空间（两点）辨别的基础

###### 4. 痛觉
- 丘脑是重要的皮层下痛觉整合中枢，参与痛觉的感觉分辨和情绪反应特性的产生
【躯体痛】大脑皮层对来自躯体浅表和深部的各种伤害性信息进行整合，形成躯体痛，包括体表痛和深部痛。
【内脏痛】是临床上的常见症状，通常由机械性牵拉、痉挛、缺血或炎症等刺激所引起。
	1. 定位不准确，这是内脏痛最主要的特点
	2. 发生缓慢，持续时间较长，常呈渐进性增强，但有时也可迅速转为剧烈疼痛
	3. 中空内脏器官对扩张性刺激和牵拉性刺激十分敏感
	4. 常伴有情绪和自主神经活动的改变
#### 5. 神经系统对姿势和躯体运动的调节
###### 1. 脊髓休克及其发生和恢复的意义
【发生】常在第 5 颈髓水平以下切断脊髓；与高位中枢离断后，暂时丧失了反射活动能力而进入无反应状态，这种现象称为脊髓休克
	- 断面以下的所有反射暂时消失，表现为骨骼肌的紧张性降低甚至消失，血管舒张，血压降低，发汗反射以及排尿、排便反射消失。
【恢复】简单、原始的反射先恢复，如屈肌反射、腱反射等；较复杂的反射后恢复，如对侧伸肌反射、搔爬反射等
==意义==：脊髓具有完成某些简单反射的能力，但这些反射平时受高位中枢的控制而不易表现出来，如脊休克恢复后，通常是伸肌反射减弱而屈肌反射增强，说明高位中枢平时具有易化伸肌反射和抑制屈肌反射的作用。
###### 2. 脊髓对姿势反射的调节
【屈肌反射】
	- 当脊动物一侧肢体的皮肤受到伤害性刺激时，可反射性引起受刺激侧肢体关节的屈肌收缩而伸肌舒张，使肢体屈曲
	- 躲避伤害的保护意义，但不属于姿势反射
	- 肢体屈曲程度与刺激强度有关
【对侧伸肌反射】
	- （屈肌反射中）如果刺激强度进一步加大，除引起同侧肢体屈曲外，还可引起对侧肢体的伸展
	- 是一种姿势反射，在保持身体平衡中具有重要意义
【骨骼肌牵张反射】
	- 指有完整神经支配的骨骼肌在受外力牵拉伸长时引起的被牵拉的同一肌肉发生收缩的反射。
	- *感受器*：肌梭
	1. **腱反射**：指快速牵拉肌腱时发生的牵张反射（股四头肌收缩-膝反射；小腿腓肠肌收缩-跟腱反射）。效应器--快肌纤维。单突触反射
	2. **肌紧张**：指缓慢持续牵拉肌腱时发生的牵张反射。肌紧张是维持身体姿势最基本的反射活动，也是随意运动的基础。效应器-慢肌纤维。多突触反射。常表现为同一肌肉的不同运动单位交替进行收缩，故能持久进行而不易疲劳。
		![[腱反射与肌紧张的对比.png]]
【反牵张反射】
	- 由腱器官兴奋引起的牵张反射抑制，称为反牵张反射。当牵拉力量加大时，腱器官可因受牵拉张力的增加而兴奋，其反射效应是抑制牵张反射。
	- 可防止牵张反射过强而拉伤肌肉，因此具有保护意义
###### 3. 低位脑干对肌紧张的调节
1. 脑干网状结构抑制区和易化区
	- 易化区的活动较强，在肌紧张的平衡调节中略占优势
2. 去大脑僵直：在麻醉动物中脑上、下丘之间切断脑干，肌紧张出现明显亢进，表现为四肢伸直，坚硬如柱，头尾昂起，脊柱挺硬，呈角弓反张状态
	- 发生机制：抗重力肌（伸肌）紧张增强的表现 
	- γ僵直：高位中枢的下行作用首先加强脊髓 γ 运动神经元的活动，使肌梭的敏感性提高，肌梭传入冲动增多，转而使 α 运动神经元兴奋，导致肌紧张增强而出现僵直
	- α僵直：高位中枢的下行作用也可直接或通过脊髓中间神经元间接使 α 运动神经元活动增强，引起肌紧张增强而出现僵直
###### 4. 小脑的运动调节功能
1. 维持身体平衡
2. 调节肌张力

	2. 步态失常（共济失调步态-蹒跚步态）
3. 协调随意运动（辩距不良、快速轮替运动障碍）
4. 参与*运动性*学习记忆（体育运动）
5. 参与*感觉性*学习记忆

###### 6. 大脑皮层的运动调节功能
【大脑皮层运动区】
	- 初级运动皮层
	- 运动前区/次级运动皮层
【运动传出通路】
	1. 皮层脊髓束：由皮层发出，经内囊、脑干下行，到达脊髓前角运动神经元的传导束
	2. 皮层脑干束：由皮层发出，经内囊到达脑干内各脑神经运动神经元的传导束
==损伤表现==
1. 轻瘫：因单纯的运动传出通路损伤而引起的运动能力减弱，常伴有肌张力下降，但没有腱反射和肌紧张亢进的表现
2. 痉挛性瘫痪：患肢肌张力增高，无明显的肌萎缩，同时腱反射亢进，巴宾斯基征阳性。常见于中枢性损伤，如内囊出血引起的卒中。
3. 迟缓性瘫痪：表现为牵张反射（包括腱反射和肌紧张）减弱或消失，肌肉松弛，并逐渐出现肌肉萎缩，巴宾斯基征阴性，见于脊髓运动神经元损伤，如脊髓灰质炎
#### 6. 神经系统对内脏活动的调节
###### 1. 自主神经系统的功能及其特征
**自主神经系统**（植物神经系统、内脏神经系统）：主要功能是调节内脏活动
【结构特征】交感+副交感
	- 自主神经系统由节前神经元和节后神经元组成。
	- 节前神经元胞体位于脊髓和低位脑干内，发出的节前纤维（均有薄层髓鞘包裹）抵达效应器官前进入神经节内换元，由节内神经元发出节后纤维（均为无髓神经纤维）支配效应器官。
	- 交感神经节位于椎旁节和椎前节内：节前纤维短、节后纤维长
	- 副交感神经节通常位于效应器官壁内：节前纤维长、节后纤维短
	*- 交感神经兴奋时产生的效应较广泛，而副交感神经兴奋时的效应则相对局限。*
【功能】主要功能是调节心肌、平滑肌和腺体（消化腺、汗腺、部分内分泌腺）的活动，以维持内环境的稳态
【基本特征】
1. 紧张性活动：安静状态下，自主神经系统持续发放一定频率的冲动，使所支配的器官处于一定程度的活动状态
2. 对同一效应器的影响常相互拮抗 
3. 受效应器所处功能状态的影响 
4. 作用范围和生理意义不同（交感神经系统的活动一般比较广泛-促使机体适应环境的急剧变化；副交感神经系统的活动相对局限）
###### 2. 脊髓、低位脑干和下丘脑对内脏活动的调节
【脊髓】脊髓是内脏反射的基本中枢
【脑干】
	- 脑干网状结构中存在许多与内脏功能活动有关的神经元，其下行纤维支配并调节脊髓水平的自主神经功能
	- 许多基本生命现象（如循环、呼吸等）的反射调节在延髓水平已基本完成，故延髓有“生命中枢”之称
	- 中脑是瞳孔对光反射的中枢，中脑和脑桥对心血管、呼吸、排尿等内脏活动也有调节作用。
【下丘脑】调节内脏活动的较高级中枢
	1. 自主神经系统活动调节：改变自主神经系统节前神经元的紧张性，调控内脏功能
	2. 体温调节：视前区 - 下丘脑前部（PO/AH）是体温调节的基本中枢
	3. 水平衡调节：控制视上核和室旁核合成和释放抗利尿激素调节肾脏对水的排出
	4. 对垂体激素分泌的调节：垂体门脉系统、下丘脑垂体束
	5. 生物节律控制：下丘脑视交叉上核SCN是哺乳动物控制日节律的关键部位
#### 7. 脑电活动以及睡眠和觉醒

- 脑电活动包括自发脑电活动和皮层诱发电位两种不同形式，是大脑皮层许多神经元群集电活动的结果
- *自发脑电活动*：在无明显刺激的情况下，大脑皮层自发产生的节律性电位变化
- *脑电图 EEG*：利用脑电图仪在头皮表面记录到的自发脑电活动

###### 2. 睡眠和觉醒
==睡眠的两种状态及生理意义==
- 非快速眼动睡眠  NREM
	- 于感觉迟钝或缺乏导致感觉传入冲动减少，大脑皮层神经元活动趋向步调一致，脑电波以频率逐渐减慢、幅度逐渐增高、δ 波所占比例逐渐增多为特征，表现出同步化趋势，故NREM 睡眠又称同步化睡眠。
	- 脑的能量消耗亦处于一天中的最低点
	- 腺垂体分泌生长激素则明显增多
	- NREM 睡眠有利于体力恢复和促进生长发育
- 快速眼动睡眠 REM
	- 脑电波呈现与觉醒相似的低幅高频的去同步化快波，但在行为上却表现为睡眠状态--*快波睡眠*
	- 其睡眠深度要比慢波睡眠更深
	- REM 睡眠与幼儿神经系统的成熟和建立新的突触联系密切相关，因而能促进学习与记忆以及精力的恢复
==与觉醒有关的脑区 ==
- 觉醒状态的维持与感觉的非特异投射系统有关。
- 觉醒的产生与脑干网状结构的活动有关，故称其为网状结构上行激动系统 ARAS
- 网状结构是个多递质系统，已知网状结构中大多数神经元上行和下行纤维的递质是谷氨酸
==与睡眠有关的脑区==
- NREM：腹外侧视前区VLPO；释放 GABA 抑制促觉醒脑区的活动，促进觉醒向睡眠转化
- REM：位于脑桥头端被盖外侧区的胆碱能神经元-- REM 睡眠启动（REM-on）神经元
#### 8. 脑的高级功能
###### 1. 大脑皮层的语言中枢
- Broca区：中央前回底部；说话有关
- Wernicke区：颞上回后部，理解语言有关，语言感受中枢。

颞上回后部-**感觉失语症**-能讲话及书写，也能看懂文字，能听到别人的发音，但听不懂别人谈话的含义，因此不能回答别人的问题
Broca 区---**运动失语症**-能看懂文字，也能听懂别人的说话，但自己却不会讲话，失去词语的组织搭配能力，不能用词语进行口头表达，但与发音有关的肌肉并不麻痹
角回受损----**失读症**------看不懂文字，但其视觉并无损害，其他语言功能活动均健全
左侧颞叶后部或 Wernicke 区--**流畅失语症**-说话正常，有时说话过度，但言不达意，言语中充满杂乱语和自创词，对别人说的话和文字的理解能力也有明显缺陷
额中回后部接近中央前回手部代表区-**失写症**-虽能听懂别人的说话，能看懂文字，自己也会讲话，但不会书写；然而，其手部的其他运动功能并无缺陷
