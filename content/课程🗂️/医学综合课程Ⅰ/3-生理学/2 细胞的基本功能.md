### 1. 细胞膜的物质转运功能
1. **单纯扩散**
	- 物质从质膜的高浓度一侧通过脂质分子间隙向低浓度一侧进行的跨膜扩散。被动过程，无代谢耗能
	- 对象：脂溶性/非极性物质或少数不带电荷的极性小分子
	- 速率：浓度差越大、通透性越高、所在溶液温度越高、膜有效面积越大，转运越快
2. **易化扩散**
	- 指非脂溶性的小分子物质或带电离子在跨膜蛋白帮助下，顺浓度梯度和/或电位梯度进行的跨膜转运。被动转运，不消耗ATP
	- *经通道*/离子通道：几乎均为离子；关闭-开放；高速度；中央亲水性孔道。①离子选择性；②门控特性（电压门控、化学门控/配体门控、机械门控）
	- *经载体*/转运体：水溶性小分子；整合膜蛋白；底物结合-构象变化-解离；速度较慢。①结构特异性；②饱和现象；③竞争性抑制
3. **主动转运**
	- 指某些物质在膜蛋白的帮助下，由细胞代谢提供能量而进行的逆浓度梯度和 / 或电位梯度的跨膜转运。
	- *原发性*：细胞直接利用代谢产生的能量将物质逆浓度梯度和 / 或电位梯度转运的过程。底物-带电离子；膜蛋白/载体-离子泵；本质-ATP酶。钠钾泵、钙泵、质子泵
	- *继发性*：间接利用 ATP 能量的主动转运过程，利用原发性主动转运机制建立起的 Na+ 或 H+ 的浓度梯度，在 Na+ 或 H+ 离子顺浓度梯度扩散的同时使其他物质逆浓度梯度和 / 或电位梯度跨膜转运。①同向转运（葡萄糖在小肠黏膜上皮的吸收）②反向转运/交换体（心肌细胞钠-钙转运体、肾小管近端上皮钠-氢转运体）
4. **膜泡运输**
	- 大分子和颗粒物质进出细胞并不直接穿过细胞膜，而是由膜包围形成囊泡，通过膜包裹、融合和离断等一系列过程完成转运。bulk批量运输
	- *出胞*：外分泌腺细胞排放酶原颗粒和黏液、内分泌腺细胞分泌激素、神经纤维末梢释放神经递质。几乎所有的分泌物都是通过内质网 - 高尔基体系统形成和处理的。①持续性出胞②调节性出胞
	- *入胞*/内化：①吞噬（固态）②吞饮（液态）
### 2. 细胞的电活动
1. 静息电位和动作电位及其产生机制
	1. 静息电位：静息状态下存在于细胞膜内外两侧的电位差。
		【产生机制】细胞在静息状态即存在的带电离子的跨膜转运
			1. 细胞膜两侧离子的浓度差是引起离子跨膜扩散的直接动力、由膜离子泵（钠泵、钙泵）的活动形成和维持
			2. 静息时细胞膜对离子的通透性--取决于离子的通透性
			3. 钠泵的生电作用
		【影响静息电位水平的因素】①细胞外液K浓度；②膜对K和Na的相对通透性；③钠泵活动水平
	2. 动作电位：细胞在静息电位基础上接受有效刺激后产生的一个快速并可向远处传播的膜电位波动
		【特点】①全或无；②不衰减传播；③脉冲式发放
		【产生机制】①离子的电化学驱动力；②细胞膜对离子的通透性
2. 兴奋性及其周期性变化、局部电位
	- **兴奋性**：指机体的组织或细胞接受刺激发生反应的能力或特性，是生命活动的基本特征之一；兴奋性又可定义为细胞接受刺激后产生动作电位的能力或特性。
	- 兴奋性的周期性变化：
		1. *绝对不应期ARP*：相当于动作电位的锋电位或其主体部分。无论施加多强的刺激也不能使细胞再次兴奋，此时细胞的兴奋性为零，其原因是引起动作电位的电压门控钠（或钙）通道已经激活或进入失活状态，无法再次接受刺激而激活开放，产生新的动作电位
		2. *相对不应期RRP*：绝对不应期之后的一段时间，给予足够强的阈上刺激可以引起新的动作电位发生；这段时间是细胞的兴奋性从零开始逐渐恢复到接近正常的时期，通常相当于复极化晚期和后超极化的时期
	- **局部电位**：细胞受到刺激后，由膜的主动特性参与的，即部分离子通道开放形成的、不能向远距离传播的膜电位改变称为局部电位（local potential）
		【特征】①等级性电位，幅度与刺激强度相关；②衰减性传导；③无不应期，可叠加总和
3. 动作电位在同一细胞上的传导
	![[动作电位在同一细胞上的传导.png|300]]
	【局部电流学说】，在动作电位的发生部位即兴奋区，膜电位呈内正外负的反极化状态，而相邻的未兴奋区仍处于内负外正的极化状态。因此，兴奋区与邻近未兴奋区之间存在电位差，并产生由正电位区流向负电位区的电流。局部电流可使邻近未兴奋区的膜电位减小，产生去极化，若去极化超过阈电位时即可触发该区爆发动作电位，使它成为新的兴奋区。因此，动作电位在同一细胞上传导的实质是细胞膜依次产生动作电位的过程。
	- 无髓神经纤维：顺序发生的，即整个细胞膜都依次发生动作电位
	- 有髓神经纤维：有髓纤维上只有郎飞结处能够产生动作电位，局部电流也仅在兴奋区的郎飞结与相邻静息区的郎飞结之间发生--跳跃式传导
### 3. 骨骼肌的收缩功能
1. 骨骼肌神经肌接头处的兴奋传递及其影响因素
	- 【电 - 化学 - 电传递】由运动神经纤维传到轴突末梢的动作电位（电信号）触发接头前膜 Ca2+依赖性突触囊泡出胞，释放 ACh 至接头间隙（化学信号），再由 ACh 激活终板膜中 N2 型 ACh 受体阳离子通道而产生膜电位变化（电信号）。
	![[骨骼肌神经肌肉接头处的兴奋传递.png]]
	- 影响因素：
		1. 使用性增强现象：强直后增强（post-tetanic potentiation，PTP)
		2. 药物和病理因素影响：如筒箭毒碱和 α- 银环蛇毒可特异性阻滞终板膜中的 N2 型 ACh 受体阳离子通道而松弛肌肉；机体产生自身抗体破坏 N2 型 ACh 受体阳离子通道可导致重症肌无力，新斯的明可抑制 AChE 而改善肌无力患者的症状；有机磷农药中毒因 AChE 被磷酸化丧失活性而引起中毒症状等。
2. 骨骼肌的兴奋-收缩偶联和收缩机制
	【兴奋-收缩偶联】
		1. T 管膜的动作电位传导：肌膜上的动作电位沿 T 管膜传至肌细胞内部，并激活 T 管膜和肌膜中的 L 型钙通道。
		2. JSR 内 Ca2+ 的释放：肌膜的去极化，在骨骼肌 RyR1 可通过构象变化触发钙释放机制，在心肌 RyR2 则通过钙致钙释放（calcium-induced calcium release，CICR）机制使 JSR 内的 Ca2+ 顺浓度差释放到胞质中，胞质中的 Ca2+ 浓度由静息时的 0.1μmol/L 水平迅速升高百倍以上。
		3. Ca2+ 触发肌丝滑行：胞质内 Ca2+ 浓度的升高促使 Ca2+ 与 TnC 结合而触发肌肉收缩。
		4. JSR 回摄 Ca2+：骨骼肌--胞质内增加的Ca2+激活JSR膜中的钙泵回摄；心肌--经LSR膜中钙泵活动被回收

3. 影响骨骼肌收缩的因素：前负荷、后负荷和肌肉收缩能力
	1. 前负荷/初长度：肌肉收缩存在一个最适初长度
	2. 后负荷：后负荷增大时肌肉收缩张力和速度呈反变关系；后负荷在理论上为零时肌肉缩短速度最大，称为最大缩短速度（Vmax），表现为等张收缩；随着后负荷的增大，表现为先等长收缩后等张收缩；当后负荷增加到使肌肉不能缩短时，肌肉产生的张力达到最大，称为最大收缩张力（P0），表现为等长收缩。
	3. 肌肉收缩能力/收缩性：是指与负荷无关又能影响肌肉收缩效能的肌肉内在特性。
