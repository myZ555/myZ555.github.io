### （七）缺血-再灌注损伤
##### 1. 概述
###### 缺血性损伤 ischemic injury
- *组织血液灌注减少而使细胞发生损伤*
	- 损伤机制：缺氧 hypoxia、ATP生成减少、细胞酸中毒
	- 缺血后果重于缺氧（缺氧+营养物质供应减少+代谢产物堆积）
###### 缺血-再灌注损伤 IRI 
- *恢复某些缺血组织器官的血液灌注反而加重组织损伤，甚至发生不可逆损伤的现象*
	- 可继发于多种病理过程：心梗、缺血性卒中、急性肾损伤、创伤
	- 再灌注组织器官发生：氧反常 oxygen paradox，钙反常 calcium paradox，pH反常（基于在体和离体细胞实验）
##### 2. 发病机制
###### （1）自由基
==导致自由基增多的机制==
1. 线粒体损伤
2. 吞噬细胞聚集及激活（再灌注时，吞噬细胞激活伴随耗氧量迅速增加、产生大量氧自由基的现象，称为呼吸爆发 respiratory burst 或 氧爆发）
3. 毛细血管内皮细胞生成黄嘌呤氧化酶增多，生成ROS增多
4. 儿茶酚胺自身氧化增加

==自由基增多引起损伤的机制==
1. 膜脂质过氧化
2. 蛋白质功能抑制（变性、降解）
3. 核酸破坏与DNA断裂
###### （2）钙超载
- 细胞外游离$Ca^{2+}$浓度约为1.0mmol/L，细胞内游离$Ca^{2+}$浓度约为0.1μmol/L（膜对$Ca^{2+}$低通透），胞外$Ca^{2+}$浓度是胞内的1万倍
- 线粒体内$Ca^{2+}$含量约为胞质的500倍【钙库】
==导致钙超载的机制==
1. $Na^{+}-$$Ca^{2+}$ 交换异常
2. 肾上腺素能受体激活
3. 生物膜损伤

==钙超载引起损伤的机制==
1. 促进ROS产生
2. 能量代谢障碍
3. 细胞膜及结构蛋白分解
4. 加重胞内酸中毒
###### （3）炎症反应过度激活
> 缺血期间，中性粒细胞在缺血区边缘逐渐聚集，再灌注则加速/加重其聚集，则发生无菌性炎症反应

==导致炎症反应过度激活的机制==
1. 细胞黏附分子生成增多（缺血损伤引起内皮细胞表面CAM↑--中性粒细胞与内皮细胞发生异常力学互作*滚，集，黏*）
2. 趋化因子等生成增多

==炎症反应过度激活引起损伤的机制==
1. 微血管损伤（血液流变学改变【轴流--边流--[^1]无复流】+微血管通透性增高）
2. 微血管舒缩功能失调
3. 周围组织细胞损伤
##### 3. 功能代谢变化
###### 心肌缺血-再灌注损伤
1. **再灌注心律失常**
	- 常发生在再灌注早期，发生率较高
	- 发生率可恢复心肌细胞数量、缺血心肌数量、缺血程度、再灌注速度呈正相关
	- 以*室性心律失常*（室性心动过速和心室颤动）居多
	- 可能机制
		- 再灌注心肌之间动作电位时程的不均一性，增强心肌兴奋折返
		- *延迟后除极（动作电位平台期Ca2+内流、在AP后形成短暂去极化），减缓传导，主因之一*
		- 儿茶酚胺除极α受体，自律性↑
2. **心肌舒缩功能障碍**
	- *再灌注心肌顿抑 myocardial stunning* 缺血心肌恢复血供后，其舒缩功能要经过数日至数周后才能恢复，为可逆性心肌功能障碍
	- 微血管阻塞，内皮肿胀，无复流，ATP↓，能量依赖性舒缩功能↓
3. **心肌结构变化**
	- 肌原纤维出现严重收缩带，肌丝断裂溶解，线粒体肿胀、嵴断裂，甚至出血坏死
### （八）休克
[[../../疾病模块/心血管、呼吸、血液与泌尿系统Ⅰ[3]]
##### 1. 概念、病因和分类
- **概念**：机体在严重失血失液、感染、创伤等强烈致病因素下，==有效循环血量急剧减少，组织血液灌流量严重不足==，引起细胞缺血、缺氧，以致重要生命器官的功能、代谢障碍或结构损害的全身性危重病理过程
- **病因**：失血性、烧伤性、创伤性、脓毒性、过敏性、心源性、神经源性

| 分类      | 定义                                                                   | 包括                                   |
| ------- | -------------------------------------------------------------------- | ------------------------------------ |
| 低血容量性休克 | 机体血容量减少所致的休克                                                         | 失血失液性休克、创伤性休克、烧伤性休克                  |
| 血管源性休克  | 由于外周血管扩张、血管床容量增加，大量血液瘀滞在扩张的小血管内，使有效循环血量减少而引起的休克。<br>**分布性休克、低阻力性休克** | 感染性休克、过敏性休克、神经源性休克                   |
| 心源性休克   | 由于心泵功能障碍或压力/阻塞性病因，心排血量急剧减少， 使有效循环血量显著下降所引起的休克                        | 心肌源性休克，非心肌源性休克 （又：obstructive shock） |

##### 3. 功能与代谢改变
###### （1）代谢紊乱
1. 脓毒性自身==分解代谢==：脓毒休克时氧耗↓，糖酵解↑，糖原、脂肪和蛋白分解代谢↑，合成代谢↓
2. 休克病因引发==应激==：一过性高血糖和糖尿，脂肪和蛋白分解代谢↑，尿氮增多、负氮平衡
3. ==氧债==（oxygen debt = 需求氧耗量 - 实测氧耗量）增大，原因：组织供应、利用氧↓，能量生成↓
4. 代谢性酸中毒==：乳酸↑，肝代谢乳酸↓，肾排泄乳酸↓ → 高乳酸血症（心肌、血管）
5. ==呼吸性碱中毒==：早期诊断指标之一，休克病因导致过度通气（PaCO2↓）。后期因通换 气障碍，可出现呼吸性酸中毒，导致混合性酸碱失衡
6. ==高钾血症==：Na+-K+-ATP酶/钠泵障碍、H+-K+交换↑，影响心肌兴奋、自律、传导和收缩性
###### （2）器官功能障碍
多器官功能障碍综合症（multiple organ dysfunction syndrome, **MODS**）
##### 4. 几种常见休克的特点
###### （1）失血性休克
- 失血是否引起休克，取决于失血的量和速度
- 15-20分钟内失血少于总血量的10-15%，机体可经代偿维持血压；15分钟内快速、大量失血超过总血量的20%（~1000 ml），超出机体代偿 能力，引起失血性休克；失血量超过总血量的45-50%，会很快导致死亡
- 分期明显、临床症状典型，易并发急性肾衰（休克肾）和肠源性内毒素血症
###### （2）感染性休克
|          | 高动力型休克 hyperdynamic shock，先                                                           | 低动力型休克 hypodynamic shock，后                                                       |
| -------- | ------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| 定义       | 病原体或其毒素侵入机体引起**高代谢和高动力循环状态**，出现发热、心排血量增加、 外周阻力降低、脉压增大等，又称为**高排低阻型休克或暖休克（warm shock）** | 心排血量减少、外周阻力增高、脉压显著缩小等，又称**低排高阻型休克或称冷休克 （cold shock）**，表现为皮肤苍白、四肢湿冷、尿量减少、血压下降、酸中毒 |
| 机制       | β受体效应（心肌收缩力↑；动-静脉短路开放→回心血量↑→心排出量↑）<br>外周阻力下降（NO、PGE2、PGI2、BK、TNF-α、IL-1等扩血管物质增多）      | 毒素、酸中毒抑制心泵功能，交感-肾上腺髓质系统兴奋释放缩血管物质                                                 |
| 弊端       | 动-静脉短路开放，真毛细血管物质交换少，组织缺血缺氧                                                            |                                                                                  |
| **心输出量** | 高                                                                                     | 低                                                                                |
| **外周阻力** | 低                                                                                     | 高                                                                                |
###### （3）过敏性休克
- 过敏原首次暴露 → IgE持久吸附于肥大细胞、嗜碱性粒细胞、血小板，致敏；
- 过敏原再次暴露 → 过敏原-IgE复合物引起脱颗粒反应，释放组胺、激肽、补体等活性物质
- 过敏性休克又称变应性休克，属**I型变态反应即速发型超敏反应**，常伴有 皮肤（荨麻疹）、呼吸道和消化道的过敏症状，发病急骤
- **机制**
	1. 活性物质扩血管，血管床容量增大，血容量相对减少
	2. cap通透性增高，血浆外渗，血容量绝对减少
###### （4）心源性休克
- 因心泵功能障碍，心排血量迅速↓，血压在休克早期即下降，死亡率80%
1. **低排高阻型**：多见；表现为外周阻力增高，与血压下降、减压反射抑制→交感-肾 上腺髓质系统兴奋、外周小动脉收缩
2. **低排低阻型**：少见；表现为外周阻力降低，心室充盈异常↑→牵张感受器兴奋→交 感-肾上腺髓质系统抑制。亦见于其他类型休克的晚期阶段，为休克失代偿表现。
### （（九）凝血与抗凝血平衡紊乱
[[凝血与抗凝血平衡紊乱]]
##### 弥散性血管内凝血（DIC）
###### （1）概念、病因和发病机制
- **概念**：继发性的，以广泛微血栓形成并出现止、凝血功能障碍为病理特征的临床综合征
- **病因**：感染性疾病、恶性肿瘤、妇产科疾病、手术及创伤……
- **发病机制**：高凝状态→微血栓→低凝状态→多发性出血
	1. 凝血系统的激活
	2. 纤溶系统功能失调
	3. 血管运动活性和血液流动性改变
###### （2）影响因素
1. 单核-巨噬细胞功能受损
2. 严重肝功能障碍
3. 血液的高凝状态
###### （3）功能与代谢改变及其机制
1. 出血
2. 多器官功能障碍
3. 微循环障碍-休克
4. 贫血

[^1]: *无复流 no-reflow*：恢复血液灌注后，缺血区依然得不到充分的血液灌注

[^2]: 微循环：微动脉与微静脉之间的微血管内的血液循环，是血液与组织进行物质交换的*基本结构* 和*最小功能单位*
