### （二）水、电解质代谢紊乱
#### 1. 水、钠代谢紊乱
**正常水、钠平衡**：低渗--130mmol/L--等渗--150mmol/L--高渗
##### 脱水
|      | 等渗性         | 低渗性             | 高渗性             |
| ---- | ----------- | --------------- | --------------- |
| 失水钠  | 成比例         | 失钠              | 失水              |
| 体液丢失 | 细胞外液↓，细胞内液- | 细胞外液↓，**细胞内液↑** | 细胞外液↓，**细胞内液↓** |
| 病因   | 消化液、腹水丢失    | 消化液丢失           | 高热、大汗、烧伤        |
| 尿量   | ↓           | ↓/-             | ↓↓              |
| 尿钠   | ↓           | ↓↓              | -               |
| 血压   | ↓/-         | ↓↓              | ↓               |
| 治疗   | 等渗盐水/平衡盐    | 生理盐水/高渗盐水       | 低渗盐水            |
###### 低渗性脱水（低容量性低钠血症）
【特征】
	- 失钠多于失水
	- 脱水的特征：血钠<135mmol/L，血浆渗透压<280 mOsm/L 
	- ==细胞外液↓，细胞内液↑==
【病因】
	- 肾脏丢失
		- 长期连续应用排钠利尿剂
		- 肾上腺皮质功能减退（醛固酮↓，钠重吸收↓）
		- 肾实质疾病
		- 肾小管性酸中毒（集合管分泌氢）
	- 肾外丢失
		- 大量消化液（属于细胞外液）丢失而只补充水/葡萄糖溶液（渗透压=0）
		- 液体在第三间隙集聚
		- 经皮肤丢失（大面积出汗/烧伤）
【临床表现】脱水征、血压降低、休克、细胞水肿
【治疗】
	- 对病因治疗
	- 轻症补充*等渗*盐水，重症补充*高渗*盐水
	- 迅速纠正严重低钠血症可能导致*中央脑桥脱髓鞘*改变

| 轻度(<135mmol/L)     | 中度(<130mmol/L)                          | 重度(<120mmol/L)               |
| ------------------ | --------------------------------------- | ---------------------------- |
| 疲乏，头晕，手足麻木<br>尿钠减少 | 恶心呕吐，脉搏细速，血压不稳定/下降<br>视物模糊<br>尿小，几乎不含钠氯 | 神志不清，肌痉挛，腱反射减弱/消失<br>呼吸困难，昏迷 |
######  等渗性脱水
【特征】
	- 水、钠等比例丧失
	- 脱水的特征是：血钠正常范围 135-150mmol/L；血浆渗透压 280-310 mOsm/L ；
	- ==细胞外液↓，细胞内液-==；*血容量*减少
【病因】
	- 任何等渗体液丢失，*短期内*均属等渗性脱水
	- 等渗性脱水--(不处理)---[高渗性脱水]
				  --(只补水)---[低渗性脱水]--大量补水--水中毒
【临床表现】口渴、尿少、脱水征、休克
【治疗】积极处理病因；补液：平衡盐或等渗盐水
###### 高渗性脱水（低容量性高钠血症）
【特征】
	- 失水多于失钠
	- 脱水的特征是：血钠>150mmol/L，血浆渗透压>310mOsm/L 
	- ==细胞内液和细胞外液量都减少==
【病因】
	- 摄入水分不足
	- 水丧失过多
	- 呕吐、腹泻及消化道引流等导致等渗或含钠低的消化液丢失
	- 经肾排出大量低渗性尿液
	- 任何原因引起的过度通气
【临床表现】
	- 轻度：2-4%；口渴
	- 中度：4-6%；乏力，尿少，皮肤失去弹性，烦躁不安，肌张力增高
	- 重度：>6%；中毒症状+：躁狂/幻觉/谵妄/昏迷，高热，死亡
【治疗】快速纠正细胞外液缺乏以改善组织灌注；纠正脱水的同时适当补钠
##### 水中毒（高容量性低钠血症）
【定义】水潴留导致液体量明显增多，细胞外液低渗，细胞内液↑（细胞水肿）；钠总量正常或增多
【病因】
	 - 急性肾衰竭，各种原因所致的==抗利尿激素==分泌过多
	 - 继续性大量饮水或精神性饮水过量，静脉输入不含盐或含盐量少液体过多过快，超过肾脏排泄能力
【临床表现】
	 - 急性水中毒：脑细胞肿胀可造成颅内压增高，引起一系列神经、精神症状。
	 - 慢性水中毒：可被原发疾病的症状所掩盖，可有软弱无力、恶心、呕吐、嗜睡等。体重明显增 加，皮肤苍白而湿润
##### 水肿
【定义】组织间隙或体腔内积聚过多液体
【病因】
	- 全身性水肿：多见于充血性心力衰竭、肾病综合征和肾炎、肝脏疾病。也见于营养不良和某些 内分泌疾病。
	- 局限性水肿：常见于器官组织局部炎症，静脉或淋巴管阻塞等情况
#### 2. 钾代谢紊乱
##### 正常钾平衡
正常：3.5-5.5mmol/L
1. 成人体内正常含钾量：50-55mmol/kg 98%在细胞内液（140-160mmol/L），2%在细胞外液(3.5-5.5mmol/L)
2. 肾脏是排钾主要器官，多吃多排，少吃少排，不吃也排（20-40mmol/天）
==钾平衡的调节==
	1. 钾的跨细胞转移：胰岛素、儿茶酚胺、血钾浓度、酸碱平衡状态、物质代谢状况
	2. 肾脏对钾的调节：醛固酮、远曲小管&集合管内原尿流速、跨膜电位
##### 钾代谢紊乱
###### 低钾血症
【病因】
	1. 摄入不足
	2. 丢失过多：消化道（呕吐/腹泻/胃肠引流）、肾（袢利尿剂/噻嗪类/醛固 酮等）
	3. 钾向胞内转移：胰岛素+葡萄糖、碱中毒等
【临床表现】最早是肌无力：先是四肢肌无力，可延及 躯干和呼吸肌
	*神经肌肉*：肌无力、甚至软瘫；胃肠道平滑肌活动减弱-麻痹性肠梗阻
	*心脏*：心肌细胞兴奋性增高、传导性降低、自律性增高、收缩性先增后减--心律失常；
	*心电图*：ST段压低、T波低平、T波后出现U波
	*肾脏*：髓质集合管、小管上皮细胞肿胀、空泡变性等；间质肾炎；ADH反应性减弱-尿浓缩功能减弱；==多尿、低比重尿==
	*酸碱平衡*：代谢性碱中毒、反常性酸性尿
【治疗】
	- 见尿补钾
	- 补钾浓度：外周输液<0.3%，补钾速度应<1～2g/h
	- 若严重累及心脏，可更高浓度、更快，通过中心静脉补钾，严密监测
	- 同时纠正*碱中毒*，**补镁**
###### 高钾血症
【病因】
	1. 摄入过多
	2. 排出减少
	3. 胞内钾外移
【临床表现】
	*神经肌肉*：轻度-肌肉兴奋性增高、感觉异常、肌束震颤；重度-四肢软弱无力-迟缓性麻痹
	*心肌*：兴奋性先增后减、传导性降低、自律性降低、收缩性降低；房室传导阻滞、室颤、心脏骤停
	*心电图*：T波高尖，P波和QRS波变低增宽
	*酸碱*：代谢性酸中毒、反常性碱性尿
【治疗】
	- 促使钾转入细胞内
		- 钙剂：保护心脏（葡萄糖酸钙）
		- 高糖+胰岛素
		- 碳酸氢钠
	- 利尿剂：袢利尿剂
	若高于7.0：上述之外马上血透
### （三）酸碱平衡和酸碱平衡紊乱

#### 1. 酸碱平衡及其调节
（1）概念：是指体内氢离子浓度的平衡，维持pH在7.35-7.45之间。
（2）调节：缓冲系统、呼吸系统和肾脏
（3）常用指标

| 指标              | 意义   | 正常值       | ↓       | ↑       |
| --------------- | ---- | --------- | ------- | ------- |
| pH              |      | 7.35~7.45 |         |         |
| 动脉血二氧化碳分压 PaCO2 | 呼吸因素 | 35~45mmHg | 呼碱/代酸代偿 | 呼酸/代碱代偿 |
| 碳酸氢根浓度 HCO3-    | 代谢因素 | 22~27mmHg | 代酸/呼碱代偿 | 代碱/呼酸代偿 |
| 剩余碱 BE          | 代谢因素 | -3~+3mmHg | 负值代酸    | 正值代碱    |
#### 2. 酸碱平衡紊乱
| 失衡类型                         | 病因                                                                                                                                     | 临床表现                               | 治疗                   |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------- | -------------------- |
| ==代谢性酸中毒==<br>H+ ↑<br>HCO3-↓ | 1. 酸性物质产生过多(缺氧和组织低灌注)<br>2. 外源固定酸摄入过多(阿司匹林、氯化铵…)<br>3. H+排出减少（肾脏排酸保碱×)<br>4. HCO3-排出过多(碱性物质丢失过多-腹泻/肠瘘…)<br>5. 高钾血症(细胞外液K+与H+交换-胞外H+增加) | Kussmual呼吸-深大呼吸<br>脑-心脏-肺-胃肠道      | “宁酸勿碱”<br>碳酸氢钠-严重酸中毒 |
| ==代谢性碱中毒==<br>H+↓<br>HCO3-↑  | 1. 酸性物质丢失(呕吐-丢H+；腹泻-丢HCO3-)<br>2. 碱性物质产生过多<br>3. H+向胞内移动--*反常性酸性尿*(H+-K+交换↑)                                                           | 呼吸浅慢，换气减少<br>神经肌肉症状<br>心脏传导阻滞、心搏骤停 | 纠正低钾血症               |

| ==呼吸性碱中毒==<br>PaCO2↓         | 1. 过度通气                                                                                                                                | 神经肌肉兴奋性↑<br>神经系统功能障碍               | 纸袋罩住口鼻               |
### （四）缺氧
#### 1. 概述
| 指标                                | 意义                          |
| --------------------------------- | --------------------------- |
| Partial pressure of O2 (PO2) 血氧分压 | 溶解在血液中的氧所产生的张力              |
| O2 capacity (CO2 max) 血氧容量        | 100ml血液中的Hb被氧充分饱和时的最大携氧量    |
| O2 content (CO2) 血氧含量             | 100ml Hb的实际携氧量              |
| CaO2-CvO2 动静脉血氧含量差                | 100ml血液流经组织时所提供的氧量或组织细胞利用氧量 |
| O2 saturation (SO2) 血氧饱和度         | 反映Hb与氧的结合程度                 |
| $P_{50}$: 50% SO2 时的PO2           | 显示血红蛋白氧亲和力。右移-亲和力减小-易释放氧    |
#### 2. 类型
###### （1）低张性缺氧
- 特征：
	- 动脉血分压降低，PaO2<60mmHg, CO2↓, SO2↓, oxygen delivery↓
	- 最常见的缺氧形式
- 原因
	- 大气氧分压降低（高原 海拔3000为界、密闭空间）
	- 外呼吸功能↓
	- 大血管动脉向静脉分流（先天性心脏病）【法洛四联症】
- **发绀 cyanosis
	- 脱氧血红蛋白≥5g/dl
	- 缺氧不一定有发绀、发绀不一定有缺氧
###### （2）血液性缺氧
- 特征：
	- PaO2 正常
	- Isotonic hypoxemia
- 原因：
	- 贫血 anemia
	- ==CO中毒【樱桃红色】==（治疗：立即从CO源移除+高压氧舱）
	- ==高铁血红蛋白血症 MHb==   “肠源性发绀”，血液颜色较深（深咖啡色/巧克力色）
- 皮肤颜色：苍白贫血貌
###### （3）循环性缺氧
- 特征：
	- 低动力性缺氧（循环血量不足）
- 原因：
	- 全身问题：休克、心衰
	- 局部问题：AS，血栓
- 分类：
	1. 缺血性缺氧：动脉灌注障碍
	2. 淤血性缺氧：静脉回流障碍
###### （4）组织性缺氧
- 特征：氧利用障碍性缺氧 dysoxidative hypoxia
- 原因：
	- 抑制细胞氧化磷酸化
	- 线粒体损伤
	- 呼吸酶缺乏
#### 3. 功能与代谢改变
###### （1）呼吸系统
1. 代偿：频率↑，深度↑
2. 失代偿：频率↓，深度↓
	- PaO2 < 30mmHg→中枢能量代谢障碍→呼吸中枢抑制
	- 呼吸肌疲劳、能量不足
###### （2）循环系统
1. 代偿：
	- CO↑--氧的运输↑
		- 心率↑
		- 心肌收缩性↑
		- 静脉回流↑（呼吸↑）
	- Hypoxic Pulmonary Vasoconstriction (HPV) 缺氧性肺血管收缩
		- 调整肺内不同区域通气/血流比→氧的摄取↑
	- 肺水肿 lung edema（高原肺水肿，high-altitude pulmonary edema，HAPE）
	- Blood redistribution: 保持重要器官的氧供
		1. 皮肤、腹腔脏器血流↓（交感兴奋，α受体密度高）
		2. 心脑血流↑（局部代谢产物、扩血管）
		3. 心脑血管平滑肌细胞膜的Kca和K ATP 开放，钾外向电流↑，超极化，钙离子内流
	- capillary proliferation
2. 失代偿

###### （3）血液系统
1. 代偿
	1. RBC↑（EPO↑）
	2. 氧离曲线右移
2. 失代偿
	1. PO2<60mmHg, Hb与O2结合受阻，SO2↓，失去代偿意义
	2. RBC过度↑，血液粘滞度，促发心衰DIC
	3. 脑耗氧高，对缺氧耐受性差，完全缺氧5-8min会造成不可逆损伤
