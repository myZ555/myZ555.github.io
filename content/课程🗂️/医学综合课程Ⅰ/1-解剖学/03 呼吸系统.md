### 1. 鼻
1. **鼻腔：外侧壁，易出血Little区，嗅区，蝶筛隐窝**	
	- 外侧壁：上中下鼻甲、上中下鼻道、鼻旁窦和鼻泪管开口
	- Little区：鼻中隔前下部易出血区
	- 嗅区：上鼻甲内侧面及与其相对的鼻中隔以及二者上方鼻腔顶部区域的鼻黏膜
	- 蝶筛隐窝：最上鼻甲或上鼻甲后上方与蝶骨体之间的凹陷
2. **鼻旁窦的位置及开口**
	![[鼻旁窦的位置及开口.png|300]] ^a072ff
	1. 额窦
		- 额骨额鳞的下部内，左、右各一，呈三棱锥体形。底向下，尖向上
		- 在窦底部通筛漏斗，开口于中鼻道
	2. 筛窦
		- 位于筛骨迷路内的海绵状小气房；前筛窦、中筛窦和后筛窦
		- 前、中-开口于中鼻道；后-开口于上鼻道
		- 后筛窦的感染向周围蔓延可引起视神经炎
	3. 蝶窦
		- 蝶骨体内的含气空腔，位于鼻腔上部的后方，与后筛窦毗邻
		- 左右两腔分别开口于左、右蝶筛隐窝
	4. 上颌窦
		- 位于上颌骨体内，呈三角锥体形
		- 上颌窦开口于中鼻道的半月裂孔
### 2. 喉
1. **喉的位置，喉软骨及其连结，弹性圆锥，环甲肌**
	- 位置：上中下鼻甲、上中下鼻道、鼻旁窦和鼻泪管开
	- 喉软骨：
		- 甲状软骨	最大。喉的前壁和外侧壁。喉结。
		- 环状软骨	喉软骨中唯一完整的软骨环。支撑呼吸道并保持其畅通
		- 会厌软骨	
		- 杓状软骨	左右各一。声带突-声韧带附着；肌突-喉肌附着
	- 喉软骨的连结：
		- 环杓关节：杓状软骨底+环状软骨板上缘的杓关节面；旋转和滑行运动，声门开大/缩小
		- 环甲关节：甲状软骨下角+环状软骨两侧甲关节面；甲状软骨在冠状轴上作前倾和复位运动，使声带紧张/松弛
		- 甲状舌骨膜：连于甲状软骨上缘与舌骨之间
		- 环气管韧带：连接环状软骨下缘和第1气管软骨环之间
		- 方形膜：会厌软骨两侧缘和甲状软骨前角的后面，向后连于杓状软骨的前内侧缘。下端游离-前庭韧带
		- *弹性圆锥：环甲膜。起自甲状软骨前角后面扇形向下附着于环状软骨上缘大部分，至环杓关节处与杓状软骨声带突相连。***声韧带***位于弹性圆锥上缘
		- 环甲正中韧带：急性喉阻塞是切开的部位
	- 环甲肌：起于环状软骨弓前外侧面，止于甲状软骨下角和下缘；==紧张声带==
2. **喉腔的分部和结构**
	- 喉前庭：喉口与前庭襞之间，上宽下窄呈漏斗状
	- 喉中间腔：声襞与前庭襞之间的部分，向两侧经前庭襞与声襞间的裂隙至喉室
		- *声门裂*：两侧声襞与杓状软骨底和声带突之间的裂隙，较前庭裂长而窄，是喉腔最狭窄之处
		- 膜间部-声门裂前 2/3 位于两侧声带之间；软骨间部-后 1/3 位于两侧杓状软骨底和声带突之间
		- 声带和声门裂合称为声门
	- 声门下腔：声襞与环状软骨下缘之间的部分，其黏膜下组织疏松，炎症时易发生喉水肿，尤以婴幼儿更易发生急性喉水肿而致喉阻塞，造成呼吸困难
### 3. 气管与支气管
1. **气管的位置，结构和毗邻**
	1. 位置：位于喉与气管杈之间；起自环状软骨下缘（约平C6），向下至胸骨角平面（约平T4椎体下缘），分叉形成左、右主支气管，分叉处称气管杈 
	2. 结构：气管由黏膜、气管软骨、平滑肌和结缔组织构成
		1. 气管软骨： 14～17 个呈 C 形缺口向后的透明软骨环构成；缺口-膜壁
		2. 气管肌：膜壁由弹性纤维和平滑肌构成的平滑肌纤维
	3. 毗邻：
2. **左右主支气管的形态和特点**
	==左主支气管==：管细而长，嵴下角大，斜行，通常有 7～8 个软骨环
	==右主支气管==：短而粗，嵴下角小，走行较陡直，通常有 3～4 个软骨环；经气管坠入的异物多进入右主支气管
### 4. 肺
1. **肺的位置**：位于胸腔内，纵隔的两侧
2. **形态**：右肺宽而短，左肺狭而长。肺呈圆锥形，包括一尖、一底、三面、三缘
	1. 肺尖：钝圆，经胸廓上口突入颈根部，达锁骨内侧 1/3 段上方 2～3cm
	2. 肺底：肺的下面，与膈相贴，受膈压迫肺底呈半月形凹陷
	3. 肋面：外侧面，与胸廓的侧壁和前、后壁相邻
	4. 纵膈面：内侧面，与纵隔相邻，其中央的椭圆形凹陷称肺门/第一肺门
	5. 膈面：肺底，与膈相邻。前缘是肋面与纵隔面在前方的移行处，较锐利，左肺前缘下部有心切迹，切迹下方有一突起称左肺小舌
	6. 后缘：肋面与纵隔面在后方的移行处，位于脊柱两侧的肺沟内
	7. 下缘：肋面与膈面和膈面与纵隔面的移行处，位置随呼吸运动而变化

### 5. 胸膜
1. **胸膜的分部和体表投影**
	1. 胸膜顶：覆盖在肺尖上方的部分，越过胸廓上口到达颈根部。最高点为锁骨内侧1/3处高出锁骨上缘2-3cm。损伤-气胸。
	2. 肋胸膜：紧贴胸壁内面，前缘位于胸骨后方，后缘至脊柱两侧，上缘与胸膜顶相移行，下缘与膈胸膜相连
	3. 膈胸膜：覆盖在膈上面
	4. 纵隔胸膜：衬覆在纵隔两侧，中部包被肺根并移行于脏胸膜。
2. **胸膜腔**：脏、壁两层胸膜共同围成的密闭的潜在性腔隙。脏壁胸膜在肺根处相互移行，在肺根下方移行的胸膜前后重叠形成肺韧带-连于肺与纵隔之间，对肺有固定作用。
3. **肋膈隐窝**：肋胸膜和膈胸膜折返形成、呈半环状的间隙，容积较大，左右各一。胸膜腔的最低部位，胸膜腔积液时首先积聚于此处。*胸膜腔穿刺：腋中线以后沿下一肋上缘*

### 6. 纵隔
1. **纵隔的概念**：两侧纵隔胸膜间全部器官、结构和结缔组织的总称
2. **分部和各部内的主要结构**
	1. 上纵隔
		1. 胸腺
		2. 左右头臂静脉和上腔静脉
		3. 左右膈神经和迷走神经，左喉返神经
		4. 主动脉弓及其分支
		5. 气管、食管胸部、胸导管
	2. 前纵隔：胸腺或胸腺遗迹、结缔组织、淋巴结
	3. 中纵隔：心包、心和出入心的大血管根部
	4. 后纵隔
		1. 胸主动脉
		2. 奇静脉、半奇静脉、副半奇静脉
		3. 食管、主支气管
		4. 迷走神经、胸交感干
		5. 胸导管、胸淋巴结
