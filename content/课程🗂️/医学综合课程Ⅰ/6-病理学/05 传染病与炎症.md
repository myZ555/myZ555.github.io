### （十二）常见传染病及寄生虫病
#### 1. 结核病
[[13 下消化道炎症和寄生虫病病理#3. 肠结核病]]
###### （1）概述及基本病理变化
【概述】
	- 由结核杆菌引起的一种慢性肉芽肿性炎症
	- 典型病变为结核结节形成和干酪样坏死
【基本病理变化】
	1. 渗出为主
		- 病变主要表现为浆液性或浆液纤维素性炎。
		- 早期病灶内有中性粒细胞浸润，但很快被巨噬细胞所取代。
		- 在渗出液和巨噬细胞内可查见病菌
	2. 增生为主
		- 形成具有诊断价值的结核结节，又称结核性肉芽肿【上皮样细胞+朗格汉斯巨细胞+外部聚集的淋巴细胞和少量反应性增生的纤维母细胞】
		- 典型的结核结节中央可出现干酪样坏死
	3. 坏死为主
		- 结核坏死灶由于含脂质较多（来自破坏的结核杆菌和脂肪变性的单核细胞）,病变呈淡黄色、均匀细腻，质地较实，状似奶酪
		- 镜下为红染无结构的细颗粒状物
###### （2）原发性肺结核病的病理变化和结局
- 机体第一次感染结核杆菌所引起的肺结核病称为原发性肺结核病
**病理变化**：
	1. 结核杆菌被吸人肺泡后,最先引起的病变称为原发灶或称为Chon灶；以右肺多见，通常只有一个，常位于通气较好的上叶下部或下叶上部靠近胸膜处，形成直径1-1.5cm的灰白色圆形炎性实变病灶，病变以*结核性肉芽肿形成*为特征，病灶中央多呈现*干酪样坏死*
	2. 原发灶的细菌游离或被巨噬细胞吞噬，很快侵入淋巴管，循淋巴液引流到所属肺门淋巴结，引起相应结核性淋巴管盐和淋巴结炎。表现为*淋巴结肿大和干酪样坏死*
	3. 肺的原发灶、淋巴管炎和肺门淋巴结结核三者合称==原发综合征==
**结局**：
1. 支气管播散：肺原发灶的干酪样坏死范围扩大，侵及相连的支气管，液化的坏死物质通过支气管排出后形成空洞--引起邻近或远隔的肺组织发生*小叶性干酪样肺炎*。原发性肺结核病形成空洞和支气管播散者较少见
2. 淋巴道播散：病变淋巴结肿大，出现干酪样坏死，并可互相粘连形成肿块。
3. 血道播散：可引起血源性结核病
###### （3）继发性肺结核病的类型、病理变化和结局
- 指再次感染结核杆菌所引起的肺结核病，多见于成人，故又称成人型肺结核病
==比较==
	![[原发性和继发性肺结核病比较.png]]
【特征】
	![[原发性和继发性肺结核病比较.png]]
【分类】
1. 局灶型肺结核：病变多以增生为主，中央为干酪样坏死，周围有纤维组织包裹。
2. 浸润型肺结核：最常见类型，属活动性肺结核。
	- 病变以渗出为主，中央有干酪样坏死，伴有病灶周围炎
	- 结局：
		- 完全/部分吸收；纤维化、包裹、钙化而痊愈
		- 渗出性病变和干酪样坏死区不断扩大--急性空洞--干酪样肺炎；
		- 自发性气胸、结核性脓胸
		- 迁延不愈-慢性纤维空洞型结核
3. 慢性纤维空洞型肺结核：
	1. 肺内有一个或多个厚壁空洞形成、内悬有因血栓形成机化而闭塞的血管；洞壁分三层：内层为干酪样坏死物,其中含有大量病菌；中层为结核性肉芽组织;外层为纤维结缔组织
	2. 同侧和对侧肺组织，特别是肺下叶可见新旧不一、病变类型不同的病灶
	3. 后期肺组织严重破坏，广泛纤维化，最终肺体积缩小，变形、变硬--硬化型肺结核
	- 由于空洞与支气管相通，成为结核病的重要传染源--开放性肺结核
	- 肺广泛纤维化导致肺动脉高压，引起肺源性心脏病
4. 干酪样肺炎
	- 肉眼：肺叶肿大实变，切面黄色干酪样，坏死物液化排出后可有急性空洞形成
	- 镜下：肺内广泛的干酪样坏死,周围肺泡腔内有大量浆液纤维素性渗出物，内含以巨噬细胞为主的炎细胞。
5. 结核球/结核瘤：指孤立的有纤维包裹的境界分明的球形干酪样坏死
6. 结核性胸膜炎
	1. 干性：增生性结核性胸膜炎；病变以增生为主，很少有胸腔积液。
	2. 湿性：渗出性结核性胸膜炎；病变主要为浆液纤维素性炎。浆液渗出量多时可引起胸腔积液
###### （4）肺外结核病的病理变化
==肠结核病==：好发于回盲部
- **溃疡型肠结核**（多见）：黏膜面可见**环形溃疡**，溃疡和肠管长轴垂直，较浅，底部干酪样坏死物，下为结核性肉芽肿。愈合后由于**纤维化**易导致**肠狭窄**。穿孔、出血少见
- **增生型肠结核**（较少见）：肠壁内有大量结核性肉芽肿和纤维组织增生，肠壁高度肥厚，肠腔狭窄，黏膜面可有浅溃疡或息肉形成，易导致慢性不完全低位**肠梗阻**或被误诊为结肠癌
#### 2. 细菌性痢疾
[[13 下消化道炎症和寄生虫病病理#5. 细菌性痢疾 Bacillary dysentery]]
###### （1）概述及病理变化
- 一种由**志贺氏菌**引起的**急性假膜性（pseudomembranous）炎症**， 是一种常见的**肠道传染病**（渗出性纤维素性炎）
- 四种：宋内、福氏、鲍氏、志贺菌
- 传染源：患者、健康带菌者
- 感染途径：非典型粪口传播。通过被志贺氏菌污染的食物和水传播
- **蝇类**是重要的传播媒介
- 发病环节：
	1. 微生物直接侵入肠黏膜
	2. 在固有层繁殖并释放毒素
	3. 毒血症
	4. 产生急性炎症和组织损伤

==急性细菌性痢疾==
- 最初的**急性卡他性炎症**
	- 产生急性炎症，伴有 弥漫性充血、水肿、点状出血、炎症细胞浸润（中性粒细胞）、粘液分泌过多和粘膜 形成浅表糜烂——
	- **急性假膜性炎症** （纤维蛋白、粘液和炎症碎片构成的假膜）
	- 假膜本质上是**纤维化脓性渗出物**（fibrinsuppurative exudate） 假膜先在患病结肠粘膜表面斑片状分布，然后弥漫性覆盖粘膜，形成脏的膜状物质
	- 镜下，可见假膜由纤维蛋白、大量中性粒细胞混合红细胞、细胞和细菌组成(粘液 脓血便)。假膜脱落时，出现不规则浅表溃疡（**地图样溃疡**）
	- 还有许多其他病原体可形成**假膜性肠炎**；滥用抗生素、形成耐药菌（e.g.,艰难梭菌）
	- 肠道菌群失调的大部分炎症是假膜性炎
==慢性细菌性痢疾==
	- 病程已超过**2个月**
	- 这类慢性病例的特点是**反复损伤和修复，溃疡和愈合**，并可导致新老病变共存， 慢性溃疡形成，**息肉样粘膜不规则伴纤维瘢痕形成，随后肠狭窄**（stenosis）
	- 临床表现： 腹痛，慢性腹泻，肠梗阻
==中毒型细菌性痢疾==
	- **儿童常见**
	- 病原体**毒力较低**，疾病的发作是突然的
	- 小肠病变表现为**卡他性肠炎或滤泡性肠炎**（catarrh enteritis or follicular enteritis）
	- 有严重的**全身中毒症状（高热惊厥）**；发病数小时后，迅速出现中毒性休克、昏迷或脑水肿和呼吸衰竭。注意并发症
###### （2）病理临床联系
- 毒血症—发热，外周血白细胞增多
- 炎症— 痉挛(abdominal cramping)，刺激性黏液腹泻，粘液脓血便 (blood and pus in the stool) **里急后重**
- 并发症：出血 穿孔 (不常见). 脱水 电解质紊乱（严重时）
- 结局: 治愈（大多数情况下），在某些情况下，延长并演变成慢性疾病
	- 大部分病例在急性期治愈，部分慢性病例可反复发作或成为慢性带菌者；
	- 少数患者可出现肠出血、肠穿孔、肠狭窄及支气管炎等并发症；
	- 中毒型菌痢的病死率较高。
#### 3. 伤寒
[[13 下消化道炎症和寄生虫病病理#4. 肠伤寒 Typhoid fever]]
###### （1）概述及病理变化
- 伤寒是由**伤寒沙门氏菌**引起的一种**急性传染病**，以**全身单核巨噬细胞系统**(mononuclear phagocyte system，MPS) （肠和肠系膜淋巴 结、肝脏、脾脏和骨髓等）网状内皮中**单核-巨噬细胞增生**为特征， 具有相应的临床表现。
- 病原体：**伤寒杆菌只感染人类**。 ==内毒素==(致病性强)，没有外毒素。
- 抗原：
	- 菌体：O抗原
	- 鞭毛抗原：H抗原--血清凝集
	- 表面抗原：Vi抗原
- 肥达反应检测
- 传染源：患者及健康携带者
- 感染途径：**粪口传播**
	- 由于有症状的伤寒病例或有症状带菌者的粪便污染了食物和水
	- 苍蝇是重要载体
---
- **急性增生性炎症**（Acute hyperplastic inflammation，**单核巨噬细胞增生**）
- 增生性巨噬细胞吞食细菌、红细胞和核碎片，往往在淋巴组织内形成小结节，这 种结节即为**伤寒肉芽肿**（typhoid granuloma），这种巨噬细胞称为**伤寒细胞** （typhoid cells）。
- 部位：**回肠末端**，肠内淋巴组织——通常在回肠远端集合淋巴结（派伊尔结， **Peyer’s patches**）和盲肠孤立淋巴滤泡中最明显。

| w1-髓质肿胀期                              | w2-坏死期                                      | w3-溃疡期                                                      | w4-愈合期                           |
| ------------------------------------- | ------------------------------------------- | ----------------------------------------------------------- | -------------------------------- |
| 肉眼：淋巴样斑最初表现为**炎性肿胀**，肿大的形状和程度与淋巴样斑一致。 | **坏死深度延伸**至肌固有层甚至浆膜。局灶性坏死的发展可导致淋 巴样斑块软化和破裂。 | 粘膜溃疡的形状和范围往往与下面的淋巴样斑块相似（**圆形或椭圆形**），通常为**回肠Peyer斑块上的纵向溃疡**。 | 这些溃烂的病变愈合后留下光滑的疤痕，一般不会发生肠腔**狭窄** |
| 镜下：淋巴斑充血、水肿，**巨噬细胞增生**明显，形成**伤寒肉芽肿**。 | 临床：5/6持续高热                                  |                                                             |                                  |
| 临床：**菌血症伴进行性发热**，有时呈阶梯状上升。            | **心动过缓**，肝脾肿大，皮肤**玫瑰疹**，**白细胞减少**，腹泻        | 临床：患者一般症状缓解，但小心**穿孔**                                       | 临床：体温呈阶梯状下降，恢复正常。                |

###### （2）病理临床联系
伤寒与多种广泛分布的病变有关，这些病变可能在疾病过程中或之后出现：它们是由**内毒素血症和菌血症**引起的。
1. ==胆囊==：伤寒杆菌易在胆囊内大量繁殖，病愈后仍可由*胆汁长期排菌*，成为带菌者。
2. ==心肌==：中毒性心肌炎：心肌纤维浊肿，间质充血、水肿、单核细胞浸润。
3. ==横纹肌==：主要是腹直肌和大腿内收肌常发生**凝固性坏死**（亦称蜡样变性/坏死） 
#### 4. 流行性脑脊髓膜炎
[[14 神经系统病理#Epidemic Cerebrospinal Meningitis 流行性脑脊髓膜炎]]
###### （1）概述及病理变化
- 飞沫--上呼吸道感染--鼻咽部繁殖；（大部分引起局部炎症或称为带菌者）
- 菌血症/败血症
- 通过BBB--化脓性脑膜炎
- 易感人群：6m-2y儿童；冬春季流行
- 分期：
	1. **上呼吸道感染期**：粘膜充血水肿，炎细胞浸润，分泌物增多
	2. **败血症期**：皮肤粘膜瘀斑和全身中毒症状（高热、寒战）
	3. **脑膜炎症期**：脑脊髓膜的急性化脓性炎症
==病理变化==
- 大体：
	1. 脑脊髓膜血管扩张充血
	2. 脓性渗出物沿血管分布——蛛网膜下腔充满灰黄色脓性渗出物（覆盖脑沟脑回、导致结构不清）【弥漫性变化；脑组织体积增大】
	3. 渗出物阻塞——脑室扩张
- 镜下：
	1. 蛛网膜血管高度扩张充血
	2. 蛛网膜下腔增宽，有脓性渗出，大量中性粒细胞、纤维蛋白和少量单核细胞、淋巴细胞浸润，可找到细菌
	3. 脑实质一般不受累或轻度水肿；靠近软脑膜的脑组织疏松、染色较浅（水肿）
	4. （脑脊液中可查见脑膜炎双球菌）
###### （2）病理临床联系
1. **脑膜刺激征**：炎症累及脊髓神经根周围的蛛网膜、软脑膜及软脊膜--神经根通过椎间孔处受压
	1. Kernig's sign(+) 【克氏征】屈髋伸膝证 
	2. Brudzinski's sign(+) 【布氏征】颈部疼痛
	3. stiff neck 颈项强直
	4. opisthotonos 角弓反张
2. **颅内压升高征**
	- 原因：脑膜血管充血、蛛网膜下腔渗出物堆积脑脊液吸收受阻；脑水肿。
	- 表现：三联征 (头痛、喷射性呕吐、视神经乳头水肿)、 小儿前囟饱满。
3. **脑脊液变化**：浑浊不清，大量脓细胞；蛋白升高，糖减少；致病菌阳性（涂片、培养）
4. **脑神经麻痹**：炎症累及脑神经（III、IV、V、VI、VII）
#### 5. 流行性乙型脑炎
[[14 神经系统病理#Epidemic encephalitis B 流行性乙型脑炎]]
###### （1）概述及病理变化
==概述==
- 病原体：*乙型脑炎病毒* (RNA病毒, Japanese B Encephalitis virus)。
- 蚊虫叮咬
- 单核细胞系增生
- 病毒血症
- 夏秋季节
- 2-7岁儿童
==病理变化==
- **Grossly**：
	1. 灰质内有针尖大小的半透明软化灶【实质：液化性梗死】 
	2. 脑沟脑回无明显变化，脑组织体积无明显变化
- **镜下**：
	1. 神经元变性坏死（无特异性）：嗜神经现象、卫星现象、尼氏小体消失
	2. 软化灶形成（诊断意义）：灶性神经组织坏死，液化形成筛状软化灶 softening foci （边界清楚，颜色苍白/灰白，质地疏松，筛网样）
	3. 血管周围淋巴细胞浸润（血管套 perivascular cuffing）
	4. 胶质细胞增生：小胶质细胞结节
###### （2）病理临床联系
1. 嗜睡、昏迷：神经元广泛受累。 
2. 吞咽困难、面瘫、眼球运动障碍：颅神经核受损。 
3. 头痛、呕吐、严重时脑疝：脑血管扩张充血、脑水肿→颅内压升高。 
4. 脑膜刺激征、脑脊液中淋巴细胞增多：脑膜的炎症 反应
#### 6. 血吸虫病
###### （1）概述及基本病理变化
| 虫体引起病变           | 虫卵引起病变          |
| ---------------- | --------------- |
| 尾蚴：**尾蚴性皮炎**     | 急性虫卵结节          |
| 童虫：**血管炎、血管周围炎** | 慢性虫卵结节/假结核结节    |
| 成虫：**静脉内膜/周围炎**  | 循环免疫及免疫复合物引起的损害 |
- **虫卵引起的病变是血吸虫病的最主要病变**。 
- 血吸虫寿命：20～30年，每对成虫产卵量:150～3000个/每日 
- **50％虫卵沉积在结肠壁**，不足10％在小肠壁，23％随血流进入肝脏，17％虫 卵自粪便排出体外。 
- 肠壁的初产卵约经7～10天发育，经单细胞期、细胞分裂期与胚胎发育期，最后形成**内含毛蚴的成熟虫卵。** 
- 未成熟虫卵引起的病变轻微，含毛蚴的成熟虫卵往往引起严重的病变。
###### （2）肝、肠的病理变化及后果
1. **结肠**: 盲肠到直肠的全部结肠，**乙状结肠最为显著**。
	1. 急性期：多发性糜烂、溃疡（大小不一，边缘不规则的浅表溃疡）虫卵排入肠道 
		- 镜检：急性虫卵结节 
		- 临床：腹痛、腹泻和脓血便 
		- **血吸虫痢疾**（Schistosomal dysentery） 
		- 虫卵阳性检出率高
	2. 慢性期：
		- 病变新旧交替，大小不一**息肉**，大小深浅不一溃疡，肠壁增厚，肠腔狭窄、 肠梗阻。 
		- 镜检：粘膜下层，固有膜 
		- 急/慢性虫卵结节，部分钙化 晚期粪便虫卵检出率低
2. **肝脏**：干线型肝硬化，**血吸虫性肝硬变/干线型/管道型肝硬变**
### （十三）艾滋病、性传播疾病
#### 1. 艾滋病
- 致病机制：主要侵犯CD4+免疫细胞，形成前病毒并在细胞内复制
	- 对单核-巨噬细胞的损伤
		- HIV感染早期以嗜巨噬细胞性病毒占优势
		- 单核-巨噬细胞表面CD4和趋化因子受体CCR5分子分别是HIV的主要受体和辅助受体
		- 与CD4+T细胞不同，单核-巨噬细胞能抵抗HIV的溶细胞作用
	- 对CD4+T细胞的损伤
		- HIV亲嗜性逐渐由嗜巨噬细胞病毒转变为嗜T细胞病毒为主；CD4+T细胞是HIV感染的主要靶细胞
		- AIDS病人后期的免疫缺陷主要表现为CD4+T细胞数量减少及功能下降
