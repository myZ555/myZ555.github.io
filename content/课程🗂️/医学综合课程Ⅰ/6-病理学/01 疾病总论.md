### （一）细胞、组织的适应、损伤和修复
#### 1. 适应性改变
###### （1）萎缩的概念、类型及病理变化
- 概念：发育正常的实质细胞、组织、器官的体积变小
- 类型：
	- 生理性
	- 病理性
		- 营养不良性
		- 缺血性（脑血供不足--脑萎缩）
		- 失用性（骨折后）
		- 去神经性（麻风病）
		- 压迫性（肾盂积水--肾实质萎缩）
		- 内分泌性
- 病理变化：
	- 肉眼观NE：体积减小、重量减轻、色泽变深、质地较韧、边缘皱缩。【总体积不一定减小，如肾压迫性萎缩、老年乳腺，但实质部分一定减小。】脏器表面血管可有弯曲
	- 光镜LM：实质细胞数量、体积减小，细胞器退化，代谢降低，蛋白质合成减少。[^5]脂褐素堆积，细胞间质增多。
###### （2）肥大、增生和化生的概念、类型及病理变化
- 肥大：细胞体积增大及其引起的组织、器官体积增大
	- 生理性：【代偿】肌肉肥大、【内分泌】妊娠时子宫肥大
	- 病理性：【代偿】高血压时心脏肥大、一侧肾切除后对侧肾肥大；【内分泌】甲亢
- 增生：器官或组织的实质数目增多
	- 生理性：【代偿】损伤/部分切除后组织再生；【内分泌】青春期女性乳腺上皮、妊娠子宫平滑肌
	- 病理性：多为内分泌性
- 化生：一种分化成熟的细胞为另一种分化成熟的细胞所替代的过程。只出现在具有增生能力的组织细胞中
	- 上皮：鳞状上皮化生（磷化）；腺上皮化生
	- 间叶：成纤维细胞--成骨细胞/成软骨细胞
#### 2. 损伤
###### （1）可逆性损伤的类型、概念及病理变化
1. **细胞水肿/水变性**
	- 概念：由于细胞容积和胞质离子浓度调节机制功能下降引起细胞内水钠潴留【心肝肾常见】线粒体损伤，ATP↓，钠-钾泵↓，胞内水钠↑
	- 病理变化：
		- NE：浑浊肿胀，体积增大，边缘圆钝，包膜紧张，切面外翻，色泽苍白而无光泽。
		- LM：气球样变；细胞肿大，内含红染细颗粒物，细胞质高度疏松呈空泡状，核肿胀，质膜出现囊泡，微绒毛消失。胞质内可见细小空泡。
		- EM：线粒体、内质网肿胀（形成红染细颗粒物）细胞器肿胀
2. **脂肪变**
	- 概念：甘油三酯蓄积于非脂肪细胞的细胞质中；多发生于肝细胞（肥胖/中毒/淤血）、[^6][^7]心肌细胞（酒精中毒/缺氧）
	- 病理变化：
		- NE：体积增大，淡黄色，边缘圆钝，切面油腻
		- LM：HE染色：因被脂溶剂溶解而残留境界清楚的脂肪空泡。苏丹Ⅲ染色阳性（冰冻切片）
		- EM：脂质小体（有膜包绕）
3. **玻璃样变性**
	- 概念：细胞内或细胞间质中出现半透明的*蛋白质蓄积*
	- 病理变化：
		- 细小动脉硬化：血浆蛋白渗出、沉积，损伤血管内膜。肾、脑、脾多发
		- 结缔组织：胶原蛋白沉积；如：陈旧瘢痕中的胶原性纤维组织、肾小球纤维化、动脉粥样硬化的纤维性斑块
		- 细胞内：肾病综合征，Russel小体（浆细胞内），Mallory小体（肝细胞内）
		- HE染色红染、均匀、半透明
4. **淀粉样变性** ^fbee07
	- 概念：细胞间质内淀粉样蛋白粘多糖复合物蓄积，具有淀粉染色特征
	- 病理变化：多突起的星芒状纤维细胞，散在蓝灰色黏液基质中
5. **黏液样变性**
	- 概念：胞外粘多糖及蛋白质异常蓄积
	- 病理变化：疏松间质内有多突起的星芒状纤维细胞散在灰蓝色黏液基质中
6. **病理性色素沉着**
	- 碳：外源；肺
	- 含铁血黄素：巨噬细胞吞噬降解RBC中的血红蛋白产生的铁蛋白
	- 脂褐素：衰老心肌细胞/肝细胞内未被消化的细胞器膜碎片残体；常伴萎缩出现
	- 黑色素：慢性炎症、色素痣、黑色素瘤、基底细胞癌
7. **病理性钙化**
	- 概念：细胞内/外异常的钙盐沉积
	- 分类：
		- 营养不良性：继发于局部组织损伤/坏死/异物；碱性磷酸酶ALP↑；结核/血栓/瘢痕/瓣膜病变
		- 转移性钙化：全身性钙磷代谢障碍
	- 病理变化：
		- LM：蓝色颗粒状-片状钙盐【苏木素染色钙化物为蓝色】
		- NE：细小颗粒或团块，触摸有沙砾感或硬石感
###### （2）不可逆性损伤——细胞死亡的类型、结局、概念及病理变化
1. **细胞凋亡**
2. **细胞坏死**
	1. *凝固性坏死*
		- 坏死区呈灰黄、干燥、实质状态，坏死和正常组织之间界限明显【充血-出血带】
		- 细胞细微结构消失，但组织轮廓仍保留【胞质均一红染】
		- 蛋白质变性作用强，溶酶体水解作用弱。
		- 可发生于除脑以外的所有组织；多发生于实质器官（心、肝、肾、脾等）的缺血性坏死
		- 由于缺血缺氧、毒素、腐蚀剂引起
	2. *液化性坏死*
		-  最常见于化脓性炎 & 脑软化
		- 细胞坏死后发生溶解、液化，组织轮廓消失。
		- 坏死组织中可凝固的蛋白质少，坏死细胞、中性粒细胞大量释放水解酶，组织富含水分、磷脂。
		- 含脂质多（如脑）或含蛋白酶多（如胰腺）的组织。
		- 脓肿亦属于液化性坏死。
		- 胰腺、感染、水肿组织多见。
	3. *干酪性坏死*
		- 特殊类型的凝固性坏死。 无定形、粗糙的颗粒状嗜酸性细胞碎片。【均匀、红染、颗粒状、无定形结构】
		- NE：坏死区黄色，质松软细腻，似干酪。
		- LM：无结构、颗粒状、红染物。原有组织结构被完全破坏。
		- 结核病病灶，含脂质较多。
	4. *脂肪坏死*
		- 特殊的液化性坏死。脂肪坏死会释放游离脂肪酸，与钙离子形成灰白色“钙皂”。
		- 急性胰腺炎时，细胞释放胰酶分解脂肪酸（酶解性）；
		- 乳房创伤时脂肪细胞破裂（创伤性）
	5. *坏疽*(局部组织大块坏死并继发腐败菌感染)
		1. 干性坏疽：坏死区干燥皱缩，黑色，与正常组织界限清晰，腐败程度较轻；多发生于动脉阻塞但静脉回流通畅的四肢末端；多为凝固性坏死
		2. 湿性坏疽：水分较多，腐败菌易于繁殖，局部肿胀呈蓝绿色，与周围组织界限不清；多发生于与外界相通的内脏、动脉阻塞且静脉回流不畅的肢体
		3. 气性坏疽：组织肿胀，产生大量气体，有捻发感/握雪感；深度开放性创伤合并气荚膜杆菌感染，湿性坏疽
		![[坏疽总结表.png]]
	6. *纤维素样坏死*
		- 病变部位形成细丝状、颗粒状、条块状的无结构物质，与纤维素染色性质相似。 
		- 病变局部结构消失，形成边界不清的小条或小块状染色深红的有折光性的无结构物质
		- 常见于超敏反应性疾病。
		- 必须镜下观察。
		- 结缔组织、小血管壁：变态反应性疾病→胶原纤维崩解、免疫复合物沉积、血浆蛋白渗出。
#### 3. 修复
###### （1）再生的概念
###### （2）各种细胞的再生潜能及不同组织的再生过程
- **细胞类型**
	1. 不稳定细胞
	2. 稳定细胞
	3. 永久性细胞
	4. *干细胞*
		1. 胚胎干细胞 ESC
		2. 成体干细胞 ASC
- **不同组织**
	1. 上皮组织再生（被覆上皮、腺上皮）
	2. 显微组织再生
	3. 血管再生【生芽 budding】基底膜分解--内皮细胞再生、移动--形成管腔
	4. 神经组织再生（神经元不能再生；由胶质细胞取代--胶质瘢痕；外周神经可再生）
	5. 软骨和骨组织再生【起始于软骨膜增生；软骨母细胞--软骨基质--软骨细胞】
	6. 肌肉组织再生【横纹肌-较重瘢痕修复；平滑肌-分裂再生；心肌-一般为瘢痕】
###### （3）肉芽组织及瘢痕组织的形态与作用
- **肉芽组织**： 由①新生毛细血管、②增生的成纤维细胞构成，并伴有③炎症细胞浸润。
	- NE：鲜红，颗粒状，柔软湿润
	- LM：有许多新生的薄壁毛细血管；另可见大量炎症细胞及多量纤维母细胞、胶原纤维相对较少；部分白细胞漏出至血管外形成漏出性出血，被纤维细胞吞噬
	- 功能：抗感染保护创面；填补创口及其他组织缺损；机化或包裹坏死、血栓及异物
- **瘢痕组织**：肉芽组织经改建成熟形成的纤维结缔组织
	- NE：呈收缩状态，颜色苍白，半透明，缺乏弹性
	- LM：由大量平行或交错分布的胶原纤维束组成，纤维细胞较少
	- 结局： 胶原纤维可在胶原酶的作用下分解吸收，使瘢痕缩小软化
	- ①填补创口或缺损，保持组织器官完整性；②较强抗拉力。若无，则为疝
	- 收缩引起梗阻/瘢痕性黏连/过度增生、肥大性瘢痕
###### （4）皮肤创伤愈合和骨折愈合
- **创伤愈合：**
	- 一期愈合：组织缺损少，整齐，无感染；瘢痕小、修复快
	- 二期愈合：缺损大，不整齐，有感染；瘢痕大，修复慢
	- 清创缝合：二期愈合→一期愈合
- **骨折愈合：**
1. 血肿形成：骨组织和骨髓都有丰富的血管，大量出血
2. 纤维性骨痂形成：暂时性骨痂；梭形肿胀；血肿由肉芽组织取代而机化，纤维化形成纤维性骨痂，后进一步分化形成透明软骨
3. 骨性骨痂形成：编织骨；纤维性骨痂分化出成骨细胞
4. 骨痂改建：编织骨-板层骨【若骨折固定不好，只能形成纤维性骨痂】
### （二）局部血液循环障碍
[[局部血液循环障碍]]
#### 1. 充血和淤血
###### （1）充血的概念和类型
- 概念：器官、组织动脉血管扩张，血容量增加
- 类型：
	- 生理性：进食后、运动后
	- 炎症性：致炎因子刺激（组胺）
	- 减压后：长期受压解除后细动脉反射性扩张
###### （2）淤血的概念、原因、病理变化和对机体的影响
- 概念：静脉回流受阻、血液淤积于毛细血管、小静脉内所致
- 病因：
	1. 静脉受压（如肝硬化所致门脉高压）
	2. 静脉腔阻塞（如静脉内瘤栓阻塞）
	3. 心力衰竭（左心衰-肺淤血、右心衰-体循环淤血）
- 病理变化和对机体的影响
	- *淤血性水肿（多浆膜腔积液）、淤血性出血、淤血性硬化*
	1. ==肺淤血==（左心衰竭）
		1. 劳力性呼吸困难（首发）
		2. 肺泡壁内毛细血管、小静脉高度扩张淤血、肺间质水肿
		3. 肺泡腔内有少量红细胞、[^1]心衰细胞
		4. [^2]肺褐色硬化
	2. ==肝淤血==（右心衰竭/肝后段下腔静脉阻塞）
		1. 体循环淤血（眼睑水肿、双下肢水肿等）、腹胀
		2. 大体：肝脏体积、重量增加，质地变硬，肝包膜紧张，[^3]槟榔肝
		3. 镜下：
			1. 肝小叶外围肝细胞脂肪变
			2. 肝小叶中央区出血坏死
			3. 淤血性肝硬化
#### 2. 血栓形成
###### （1）血栓和血栓形成的概念
- 血栓：所形成的固体质块即为血栓（thrombus）
- 血栓形成：指在活体的心脏、血管内血液成分形成固体质块的过程。
###### （2）血栓形成的条件
1. 心血管内皮损伤
2. 血液状态紊乱（缓慢/涡流）
3. 血液高凝现象（遗传性/获得性）
###### （3）血栓的类型和形态

| 血流状态 | 较快                              | 较慢                                              | 缓慢        | 不明       |

###### （4）血栓的结局
1. 溶解吸收
2. 机化、再通
3. 钙化
4. 脱落形成栓子
###### （5）血栓对机体的影响
1. 阻塞血管（后果取决于有无充分侧支循环）
2. *栓塞*
3. 心瓣膜变形（狭窄、关闭不全）
4. 出血（DIC发生时形成大量微血栓，消耗大量凝血因子、血小板，造成血液低凝而出血）
#### 3. 栓塞
###### （1）栓塞及栓子的概念
- 栓塞：血管内出现的异常物质随血液至远处阻塞血管的现象
###### （2）栓子的运行途径
1. 顺行性栓塞
	1. 左心、体循环动脉
	2. 右心、体循环静脉
	3. 肠系膜静脉血栓
	4. 交叉性栓塞
2. 逆行性栓塞：
	- 罕见。如下腔静脉内栓子，在剧烈咳嗽、胸腹腔内压力骤增时可逆行血流方向运行栓塞下腔静脉所属分支
###### （3）栓塞的类型
1. **血栓栓塞 thromboembolism**
	1. *肺动脉栓塞*：栓子90%来自下肢深静脉
		1. 栓子巨大：栓塞肺动脉主干、大分支或骑跨于肺动脉分叉处，呼吸循环衰竭而猝死
		2. 栓子较少、数量多：少数小分支栓塞，如果栓塞前已有左心衰/肺淤血，则无法克服肺静脉阻力，造成局部肺组织缺血而发生出血性梗死
		3. 微栓子：可通过肺进入左心、体循环导致体循环动脉栓塞
	2. *体循环动脉栓塞*：栓子多源于感染性心内膜炎时左心瓣膜上的赘生物，以及二尖瓣狭窄的左心房血栓和心肌梗死时合并的附壁血栓
		4. 梗死（肾脑脾）
		5. 一过性缺血（肝脏有双重供血一般无梗死）
2. 气体栓塞
	1. 多量空气迅速进入血循环或溶解于血液内的气体迅速游离形成气泡阻塞血管
	2. 空气栓塞、减压病
3. 脂肪栓塞
4. 其他栓塞
###### （4）栓塞对机体的影响
#### 4. 梗死
###### （1）概念
- 由血管阻塞所引起的局部组织、器官的缺血性坏死
###### （2）梗死形成的原因和条件
- 原因：血栓形成（最常见）、动脉栓塞、动脉痉挛、血管受压闭塞
- 条件：器官组织供血血管类型、血流阻断速度、组织对缺氧的耐受性、血液含氧量
###### （3）梗死的类型和病理变化

|     | 贫血性梗死 anemic      | 出血性梗死 hemorrhagic      |
| --- | ----------------- | ---------------------- |
| 血管  | 动脉阻塞              | 静脉淤血基础上                |
| 器官  | 心肾脾等组织致密和侧支循环少的器官 | 肺肠双重血供或血管吻合丰富和组织疏松的器官  |
| 大体  | 灰白色--白色梗死         | 明显出血，红色--红色梗死          |
| 备注  | 脑梗死（大脑中动脉、前动脉供血区） | 肠梗死可发生于突发肠系膜动脉主干栓塞     |
|     |                   | 脑血栓栓塞和梗死后血液再灌注可发生出血性梗死 |

### （三）炎症
[[炎症]]
#### 1. 概述
###### （1）概念
- 概念：炎症是在**有血管分布的结缔组织**中发生的，以**体液和白细胞在血管外组织聚集为特征**的复杂反应。临床上表现为**红、肿、热、痛**等功能障碍。
###### （2）原因
- 生物因素：细菌、病毒、寄生虫
- 非生物因素：
	- 物理因素：高温、低温、机械损伤
	- 化学因素：外源性（强酸、强碱、氧化剂）、内源性（代谢产物）
- 变态反应（过敏、组织坏死）
###### （3）炎症的基本病理变化
1. **变质alteration**：炎症局部组织发生的变性和坏死，包括水样变性、脂肪变性、凝固性坏死、液化坏死、固缩坏死等。严重程度由致炎因子和机体的反应状态共同决定
2. **渗出exudation**：局部组织血管内的液体和细胞成分，通过血管壁进入间质、体腔、黏膜的过程
	1. 渗出是炎症最具特征性的变化
	2. 炎症时所渗出的液体和细胞成分总称为渗出液![[渗出液与漏出液对比.png]]
		|炎性水肿|渗出液集聚在组织间隙内|
		|炎性浆膜腔积液|渗出液集聚在浆膜腔内|
3. **增生proliferation**：包括实质细胞和间质细胞的增生，与生长因子的作用密切相关，能够限制炎症扩散和修复创伤
	1. 增生特点：多种细胞成分（多克隆性）、炎症刺激依赖性
	2. 意义：
		1. 积极-限制炎症扩散和修复损伤组织
		2. 不利-过度增生影响组织器官功能
###### （4）炎症的局部表现和全身反应
1. 局部表现
	|红：炎症初期动脉性充血，局部氧合血红蛋白增多
	|肿：渗出物；组织细胞增生
	|热：动脉充血、代谢增强
	|痛：局部炎症病灶内炎症介质的刺激、渗出物压迫
	|功能障碍：实质细胞变性坏死、渗出物压迫阻塞、局部疼痛
2. 全身反应
	1. **发热**
		- 病原微生物及其产物可作为发热激活物，作用于产生和释放内生致热源EP细胞，产生EP，作用于体温中枢
		- 增强防御功能/多系统功能紊乱
	2. **末梢血白细胞计数增高**
		- 白细胞计数可高达15k-20k/mm3
		- 核左移：幼稚中性粒细胞所占比例增加
			|细菌感染|中性粒细胞|
			|病毒感染|淋巴细胞|
			|寄生虫感染|嗜酸性粒细胞|
	3. **单核吞噬细胞系统增生**
	4. **实质器官病变**
			    
###### （5）炎症的分类和结局
|           |                   |                       |
| --------- | ----------------- | --------------------- |
| 变质性炎症     | 变质为主，多见于急性炎症      |                       |
| 渗出性炎症     | 浆液性炎症 serous      | 主要为白蛋白。水泡、体腔积液        |
|           | 纤维素性炎症 fibrinous  | 纤维蛋白渗出为主，机化形成瘢痕。绒毛心   |
|           | 化脓性炎症 suppurative | 中性粒细胞渗出为主。脓细胞、坏死细胞等   |
| 增生性炎症（慢性） | 非特异性增生性炎症         |                       |
|           | 肉芽肿性炎症            | 特异性，肉芽肿形成。上皮样细胞核多核巨细胞 |
#### 2. 急性炎症
###### （1）急性炎症过程中的血管反应
1. **血流动力学改变**
	1. 细动脉短暂收缩
	2. 血管扩张和血流加速（炎症介质作用于血管平滑肌）
	3. 血流速度减慢（通透性升高导致液体流失，最后扩张的小血管内挤满红细胞-血流停滞-有利于白细胞黏附于血管内皮并渗出到血管外）
2. **血管通透性增加**
	4. 内皮细胞收缩（导致内皮间缝隙增大；组胺、缓激肽、白细胞三烯）
	5. 内皮细胞穿胞作用增强（穿胞：富含蛋白质的液体通过穿胞通道穿越内皮细胞的现象）
	6. 血管内皮损伤（致炎因子直接损伤、白细胞黏附损伤、白细胞释放活性氧和水解酶间接损伤）
	7. 新生毛细血管的高通透性（内皮细胞连接不健全、较多活性介质的受体）
###### （2）急性炎症过程中的白细胞反应
1. **白细胞渗出**
	1. 边集--滚动--黏附在内皮细胞（P、E选择素、ICAM-1、LCAM-1）
	2. 白细胞游出血管：阿米巴运动方式由内皮细胞缝隙中逸出
	3. 通过趋化作用聚集到炎症病灶（白细胞表面特异性受体+趋化因子）
2. **白细胞激活**
	1. 吞噬作用：炎症灶内白细胞吞噬病原微生物、异物和组织碎片的过程。
		- 类上皮细胞（结核杆菌）、朗格汉斯巨细胞、异物巨细胞、泡沫巨细胞
		1. 识别和黏附（调理作用、调理素）
		2. 吞入
		3. 杀伤和降解
	2. 免疫作用
		![[抗原进入机体-免疫激活.png|300]]
3. **白细胞介导的组织损伤作用**
	1. 理化性致炎因子直接损伤组织
	2. 白细胞释放的酶损伤组织
	3. 病原微生物及其代谢产物
	4. 免疫反应
4. **白细胞功能缺陷**
	1. 黏附缺陷（LAD-1，LAD-2）
	2. 吞噬溶酶体形成障碍
	3. 杀菌活性障碍
	4. 骨髓白细胞生成障碍
	5. 白细胞激活障碍
###### （3）炎症介质的概念和主要作用
- 概念：对炎症发生发展起重要介导作用的化学性物质
- 主要作用：通过与靶细胞表面特异性受体结合发挥效应；可作用于靶细胞进一步引起细胞产生刺激炎症介质

###### （4）急性炎症的类型和病理变化
1. ==**浆液性炎**==
	- 黏液的浆液性炎：浆液性卡他炎
	- 浆液的浆液性炎：可引起体腔积液；浆液性渗出物在表皮内/下可形成水疱
	- 渗出物以*血浆成分*为主，主要为白蛋白
2. ==**纤维蛋白性炎**==
	- 以*纤维蛋白原*渗出为主，继而在病灶中形成纤维蛋白为主要特征
	- 纤维蛋白呈相互交织的红染物质，混有中性粒细胞/坏死碎片
	- 黏膜的纤维性炎：假膜性炎【假膜=渗出的纤维蛋白+坏死组织+炎症细胞】；细菌性痢疾；咽部白喉（固膜）；气管白喉（浮膜）
	- 浆膜的纤维性炎：胸膜、心包膜【[^4]绒毛心】
	- 肺：大叶性肺炎
3. ==**化脓性炎==**
	- 以*中性粒细胞*渗出为主，并由组织坏死和脓液形成为特征
	1. 表面化脓、积脓（黏膜/浆膜）黏膜化脓性炎：脓性卡他炎（中性粒向黏膜表面渗出）
	2. 蜂窝织炎（疏松结缔组织）弥漫性；LCT内大量中性粒弥漫性浸润，组织高度水肿，与周围组织界限不清但组织坏死不明显--预后好
	3. 脓肿（局限性化脓性炎）；组织溶解坏死形成充满脓液的腔（囊腔有壁、界限不清）；发生于皮下、内脏（肺脑肝）；中性粒浸润--崩解形成脓细胞--释放蛋白水解酶--组织液化性坏死--脓腔
4. ==**出血性炎==**
- 渗出物中含大量*红细胞*
	- 急性出血热
	- 钩端螺旋体病
	- 鼠疫
![[急性炎症的转归.png]]
#### 3. 慢性炎症
###### （1）一般慢性炎症的病理变化和特点
- 浸润：慢性炎症细胞（巨噬细胞、淋巴细胞、浆细胞、肥大细胞）
- 纤维结缔组织、血管等间质的增生及实质细胞增生
	- 炎性息肉polyp：局部黏膜上皮、腺体、肉芽组织增生
	- 炎性假瘤psedotumor：肺部/眼眶；炎细胞+增生结缔组织+实质细胞
- 伴有组织破坏（瘢痕形成）
###### （2）慢性肉芽肿性炎的概念、病因和病变特点
- 概念：巨噬细胞及其衍生细胞在局部增生形成的境界清楚的结节状病灶
- 病因：① 刺激因子长期存在；②刺激因子引起细胞免疫反应
2. 感染性肉芽肿：结核分枝杆菌--结核肉芽肿；中央干酪样坏死；朗格汉斯巨细胞
3. 异物肉芽肿：异物巨细胞
4. 原因不明肉芽肿：结节病肉芽肿
