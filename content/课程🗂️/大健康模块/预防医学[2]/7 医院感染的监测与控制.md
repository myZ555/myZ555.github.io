# 一、医院感染概论
#### 定义：
**住院患者在医院内获得的感染**
- 包括在医院期间发生的感染，在医院内获得、出院后发生的感染，医院工作人员在医院内获得的感染
- 不包括入院前已开始或已处于潜伏期的感染
#### 分类：
1. 按病原体的来源：
	1. 内源性：患者皮肤粘膜；胃肠道、口咽、泌尿生殖器；空腔脏器的内源性菌丛
	2. 外源性：医疗环境、诊疗用品；除患者以外的个体
2. 按病原体的种类
	- 细菌感染-主要-90%
	- 病毒感染-呼吸道传播的流感病毒、新冠病毒
	- 真菌感染-发病率上升趋势；最常见白色念珠菌
3. 按感染部位：
	1. 呼吸系统
	2. 腹部和消化系统
	3. 手术部位
	4. 泌尿系统
	5. 中枢神经系统
	6. 其他部位（心血管系统、血液系统、皮肤和软组织、生殖道、口腔等）
4. 按感染途径：
	1. 空气传播：肺结核、麻疹、水痘、流行性出血热
	2. 飞沫传播：百日咳、白喉、流行性感冒、病毒性腮腺炎
	3. 接触传播：肠道传染病、经血传播、多重耐药菌感染、皮肤感染患者
	4. 医源性传播、
	5. 其他：血液体液、生物媒介
#### 诊断标准：
##### 1. 临床诊断
1. 症状和体征
2. 辅助检查
##### 2. 病原学诊断
1. **微生物培养-金标准**
2. 其他：分子检测、抗原检测

| 属于医院感染                       | 不属于医院感染                            |
| ---------------------------- | ---------------------------------- |
| 有感染症状、无病原菌检出                 | 只有细菌定植而无炎症表现                       |
| 本次感染直接与上次住院有关                | 新生儿经胎盘获得（生后48h发病）的感染，如单纯疱疹、弓形体病、水痘 |
| 在原有感染基础上出现其他部位感染             | 患者原有的慢性感染在院内急性发作                   |
| 由于诊疗措施激活的潜在性感染（放化疗、免疫抑制剂、创伤） | 感染性疾病本身并发症                         |
| 医务人员在医院工作期间获得的感染             | 邻近部位感染的自然扩散                        |

#### 流行病学特征
- 传染源：患者、陪护人员、医院工作人员、医疗设备等被污染后
- 传播途径：接触传播、飞沫传播、空气传播、生物媒介传播
- 易感者：住院/门诊病人，探访者，医院工作人员等
#### 医防融合
1. 以个体健康为中心，同时提供医疗服务与公共卫生服务
2. 强化预防为主，通过服务关口前移，降低发病率
3. 医疗服务与公共卫生服务有效协同，根据个体具体健康情况与疾病所处阶段，有针对性提供医疗服务和公共卫生服务
# 二、医院感染监测
#### 定义
指长期、系统、连续地收集、分析医院感染在一定人群中的发生、分布及其影响因素，对监测资料定期进行整理分析，并将监测结果报送和反馈给有关部门和科室，为医院感染的预防、控制和管理提供科学依据
**目的和意义**：
1. 了解医院感染的发病情况和特点
2. 明确医院感染的危险因素和高危人群
3. 制定科学的医院感染预防和控制策略
4. 评价医疗机构的医疗质量
5. 提高患者的满意度和信赖度
#### 监测方法
##### 全院综合性监测
连续不断地对所有临床科室的全部住院患者和医院工作人员进行医院感染及其有关风险因素的监测
##### 目标性监测
针对高风险人群、高发感染部位、高感染风险部门等开展的医院感染及其风险因素监测
**ICU感染**：患者住进ICU时，该感染不存在也不处于潜伏期；患者 转出ICU到其他病房后，48h内发生的感染仍属ICU感染

**细菌耐药性监测**
**临床抗菌药物使用监测**
