**疾病的分布**是疾病在不同人群、地区和时间发生、发展和存在状态的群体现象和规律。通过回答“**谁得病 (Who)**、**哪里得病 (Where)**、**何时得病 (When)**”，即中国传统理念中的“天时、地利、人和”，来进行客观测量和比较，这是对疾病群体特征和水平认识的起点。

--------------------------------------------------------------------------------

## 一、 疾病流行强度

流行强度用于描述疾病在特定时间、特定地区的发生频率，主要分为四种形态：

1. **散发 (Sporadic)**：
    - 指某病发病呈历年一般水平，各病例在发病时间和地点上**无明显联系**。
    - 多用于描述范围较大的地区（区、县以上）。
    - 常见情况：常年流行且病后有持久免疫力的疾病（如麻疹）、隐性感染为主的疾病（如乙脑）、传播机制难实现的疾病（如斑疹伤寒）、潜伏期长的疾病（如麻风）。
2. **暴发 (Outbreak)**：
    - 指在**局部地区或集体单位**（如学校、托幼机构），**短时间内**某病发病人数突然大量增多。
    - 多数患者出现在该病的最长潜伏期内，通常有相同的传染源或传播途径。
3. **流行 (Epidemic)**：
    - 指某地区某病发病**显著超过历年发病水平**。
    - 注：存在“隐性流行”现象（隐性感染占绝大多数，实际感染率极高），如流行性乙型脑炎和脊髓灰质炎。
4. **大流行 (Pandemic)**：
    - 发病显著超过历年水平，且**蔓延迅速，涉及地域广**，短时间内跨越省界、国界甚至洲界（如新冠病毒感染、流感、霍乱等）。

--------------------------------------------------------------------------------

## 二、 疾病的分布特征（三间分布）

疾病的分布主要体现在时间、地区和人群三个维度，简称**“三间分布”**。

#### （一） 时间分布 (When - 天时)

观察疾病频率随时间推移的变化规律，有助于提供病因线索或评价防制效果。

1. **短期波动**：持续几天、几周或几个月的发病猛增。与暴发的区别在于，短期波动常用于**较大数量的人群**（如1952年伦敦大雾事件、食物污染引起的甲肝暴发）。
2. **季节性**：疾病在一定季节内呈现频率增高。
    - _严格季节性_：如虫媒传染病乙脑。
    - _季节性升高_：一年四季均可发病但特定季节升高，如呼吸道传染病或冠心病急性发作。
    - _原因_：病原体生长繁殖受气候影响、媒介昆虫消长、人类生活方式及暴露机会改变等。
3. **周期性**：每隔若干年出现一个流行高峰。
    - _原因_：取决于人口密度、易感者积累速度和病原体变异速度；病后形成稳固免疫的疾病易出现此特征。
4. **长期趋势 (长期变动)**：几年或几十年内疾病发病率/死亡率的持续上升或下降。
    - _原因_：病因改变、病原体变异、医疗水平提高或登记报告制度完善等。
5. **时间分布分析方法**：
    - **Joinpoint回归模型**：计算年百分比变化 (APC) 和平均年百分比变化率 (AAPC)，判断上升或下降趋势。
    - **自回归移动平均模型 (ARIMA)**：常用的时间序列预测方法。

#### （二） 地区分布 (Where - 地利)

差异的根本原因是不同地域的自然环境和社会环境（致病因素和保护因素）存在差异。流行病学史上第一张分析疾病特征的专题地图是1854年John Snow绘制的伦敦霍乱死亡分布图（证实水井污染）。

1. **国家间及国家内不同地区分布**：由于经济、文化、自然环境等综合因素，疾病（如结直肠癌、高血压）呈现显著的地域差异。
2. **城乡分布**：与生活条件、卫生状况、人口密度、交通及工业水平有关。
3. **地方聚集性与地方性**：
    - **地方聚集性**：某地发病率显著高于周边，提示存在特定病因（如我国西北部的黑热病）。
    - **地方性疾病 (Endemic disease)**：疾病经常存在于某地区，不需外地输入。判断依据包括：当地各民族均高发；外地相似人群低发；**外地人迁入后经过一段时间发病率与当地一致，迁出后发病率下降或自愈**。分为统计地方性、自然地方性和自然疫源性。

#### （三） 人群分布 (Who - 人和)

人群可按固有特征（年龄、性别）、生活方式、社会网络等进行分类。

1. **年龄 (★最重要因素)**：
    - 年龄与疾病关系最密切，慢性病随年龄增长而增多，急性传染病随年龄增长而减少。
    - **横断面分析 vs. 出生队列分析 (★核心重点)**：
        - **横断面分析**：分析同一时期各年龄组的发病率。缺点是**不能区分年龄效应和致病因子长期暴露的累积效应**（例如横断面看肺癌死亡率在70岁后下降，但这可能是假象）。
        - **出生队列分析**：将同一年代出生的人划为一组进行长期随访。它能**正确分辨年龄、时间、暴露经历对疾病频率的真实效应**，阐明致病因子与年龄的真实关系（如肺癌死亡率实际上随年龄增长而不断升高）。
2. **性别**：与遗传、内分泌、生理解剖、职业暴露机会及生活习惯嗜好有关。
3. **职业**：不同职业群体暴露于不同物理、化学、生物及精神紧张因素（如石棉工人易患间皮瘤）。
4. **其他人群特征**：民族/种族、婚姻与家庭、行为生活方式（吸烟、酗酒）、文化教育程度、流动人口及宗教信仰等均深刻影响疾病分布。

#### （四） 综合分布与时空分析

实际应用中需要将人群、地区、时间综合起来描述。

- **时空聚集性分析方法**：如SaTScan时空扫描、热点分析、时空泊松回归、深度学习及传染病动力学模型(SIR)等。
- **★ 移民流行病学 (Migration Epidemiology)**：
    - 核心目的：通过比较移民、移居地当地人、原居地当地人的发病率，**探讨疾病发生是受遗传因素还是环境因素主导**。
    - 逻辑判定：
        - 若 **移民发病率** ≈ **移居地当地居民** (且 = 原居住地)，说明**环境因素**是主因。
        - 若 **移民发病率** ≈ **原居住地居民** (且 = 移居地当地居民)，说明**遗传因素**是主因。

## 三、 实例解析

资料提供了一个经典实例：**SARS暴发后的第一个十年(2004-2013)中国法定传染病的流行特征与演变**。该研究完美体现了如何应用“三间分布”来分析公共卫生数据：

1. **总体情况**：10年间平均年发病率417.98/10万（手足口病、乙肝、结核病居前三）；病死比2.41/1000（狂犬病最高）。
2. **时间分布 (When)**：45种传染病中有35种具明显的**季节性**。运用**Joinpoint回归**发现，总发病率总体呈上升趋势(APC 5.9%)，但在2009年政策干预后，第二个五年的上升速度大幅放缓(APC降至2.3%，趋于平稳)。部分疾病如包虫病、丙肝快速增长，而鼠疫、霍乱显著下降。
3. **人群分布 (Who)**：男性发病率和病死比整体高于女性；10岁以下儿童发病率最高，但80岁以上老年人病死比最高（体现了**年龄与性别**特征）。
4. **地区分布 (Where)**：发病率和病死比存在显著的**地域差异**（北京、新疆、浙江发病率高；云南、广西病死比高），这提示了边远地区与高度城市化地区面临不同的公共卫生挑战。
