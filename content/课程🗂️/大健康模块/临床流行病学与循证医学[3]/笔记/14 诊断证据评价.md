## 一、 诊断试验的核心基础

- 诊断试验的目的是把患某病的病人从可疑患者及无病者中区分出来（确定或排除疾病）。
- 其基本程序为：确立金标准 ➔ 选择研究对象 ➔ 样本大小 ➔ 同步盲法评价 ➔ 诊断价值评价。

#### 1. 核心评价指标矩阵

一切评价都基于诊断试验与“金标准”对比得出的四格表（真阳性a、假阳性b、假阴性c、真阴性d）：

- **灵敏度 $(Se) = \frac{a}{(a+c)}$：** 真患者中，查出阳性的比例。漏诊率 = 1 - Se。
- **特异度 $(Sp) = \frac{d}{(b+d)}$：** 无病者中，查出阴性的比例。误诊率 = 1 - Sp。
- **阳性预测值 $(+PV) = \frac{a}{(a+b)}$：** 查出阳性的人中，真正有病的比例。
- **阴性预测值 $(-PV) = \frac{d}{(c+d)}$：** 查出阴性的人中，真正没病的比例。
- **$正确指数 (Youden Index) = Se + Sp - 1$：** 综合反映真实性。

#### 2. 诊断截断值 (Cut-off value) 的临床选择策略


- **严重疾病且早诊早治效果好：** 应稍**降低标准以提高灵敏度**，宁可错杀（假阳性）也不放过（减少漏诊）。
- **疗效不佳、治疗昂贵或假阳性带来巨大伤害：** 应**提高标准以提高特异度**，减少误诊。
- **漏诊与误诊重要性相近：** 定在 **Youden指数最大处**（最理想截断值）。

## 二、 诊断试验证据的严格评价（三大维度）

#### 维度 1：真实性评价 (Validity)

| 证据分级        | 研究设计类型                             |
| :---------- | :--------------------------------- |
| **1级 (最高)** | 采用相同金标准及盲法的横断面研究（诊断性队列研究）的**系统评价** |
| **2级**      | 采用相同金标准及盲法的**单个**横断面研究             |
| **3级**      | 非连续纳入受试者的研究，或金标准不一致的研究             |
| **4级**      | 病例对照研究，或金标准较差、非独立金标准               |
| **5级 (最低)** | 基于机制的推理                            |

- **真实性评价核心问题：**
    1. 是否用**盲法**将诊断试验与金标准做过**独立的、同步的**对比？
    2. 金标准是否合理？研究对象是否具有代表性（不能仅用完全健康人做对照）？
- **高级真实性指标：**
    - **诊断比值比 (DOR)：** $DOR=\frac{(真阳性×真阴性)}{(假阴性×假阳性)}=\frac{+LR}{−LR}$。它稳定性高，能更客观评价临床价值（等于1时无价值）。
    - **证实偏倚 (Verification Bias) 的纠正：** 如果阳性患者全部做了金标准，而阴性患者只有部分抽查了金标准，会**夸大灵敏度**。需通过系数 F (阳性做金标准的比例) 和 G (阴性做金标准的比例) 进行公式纠正：$纠正灵敏度=\frac{(a/F)}{(a/F+c/G)}$。

#### 维度 2：可靠性评价 (Reliability)

评价试验多次测量或多人测量结果是否一致。

- **符合率** = $\frac{(a+d)}{n}$。
- **Kappa值：** $Kappa=\frac{(p0​−pc​)}{(1−pc​)}$。常用于评估判断的一致性强度。

#### 维度 3：适用性与临床意义 (Applicability) ★

评价该检查能否用于自己的患者，以及检查结果能否改变临床决策。

1. 经典的“验前 ➔ 验后”概率推算流程（考试与实战重点）
	- **步骤一：** 估计**验前概率 (Pre-test probability)**，即受试者所在人群的**患病率**。
	- **步骤二：** 将验前概率转化为**验前比 (Pre-test odds)** = 验前概率/(1−验前概率)。
	- **步骤三：** 寻找该诊断试验的**似然比 (LR)**。
	    - _阳性似然比 (+LR)_ = $\frac{Se}{(1−Sp)}$。
	    - _阴性似然比 (-LR)_ = $\frac{(1−Se)}{Sp}$。
	- **步骤四：** 计算**验后比 (Post-test odds)** = 验前比×似然比(LR)。
	- **步骤五：** 将验后比转化为**验后概率 (Post-test probability)** = 验后比/(1+验后比)。_（这就是检查结果出来后，患者真正得病的概率）_
2. 核心经验法则 (Rules of Thumb)
	- **SnNout (高灵敏度排除法则)：** 当一个诊断试验具有极高灵敏度 (Sensitivity 近100%) 时，如果结果为**阴性 (Negative)**，则可以极大概率**排除 (Out)** 该疾病。
	- **SpPin (高特异度确诊法则)：** 当一个诊断试验具有极高特异度 (Specificity 近100%) 时，如果结果为**阳性 (Positive)**，则可以极大概率**确诊 (In)** 该疾病。

$表 2：似然比 (LR) 对验后概率的临床影响判别表$

| 似然比 (LR) 数值 | 对验后概率的影响程度 | 临床意义 | 
| :--- | :--- | :--- | 
| **> 10** 或 **< 0.1** | **决定性变化** | 具有极高的确诊或排除价值 | 
| **5 ~ 10** 或 **0.1 ~ 0.2** | 中等程度变化 | 有中等价值 | 
| **2 ~ 5** 或 **0.2 ~ 0.5** | 较小程度变化 | 价值有限 | 
| **1 左右** | 基本无变化 | **无诊断价值** |

## 三、 测试-治疗阈值模型

拿到验后概率后，如何决定下一步处理？

- **诊断/测试阈值 (Testing threshold)：** 如果验后概率低于此数值，直接**排除诊断（不采取措施）**。
- **治疗阈值 (Treat threshold)：** 如果验后概率高于此数值，直接**肯定诊断并给予治疗**。
- **处于两者之间：** 如果概率落在这两个阈值之间，说明现有检查不足以确诊或排除，必须进行**联合试验（进一步检查）**。
    - _联合试验计算：_ 总验后比 = 验前比×LR1​×LR2​…。

### 案例：儿童1型糖尿病 (T1DM) 的 IAA 筛查

- **PICO：** 有家族史儿童(P)，微量平板RIA法检测IAA(I)，对比WHO标准(C)，评价真实性与可靠性(O)。
- **文献评价结论：** 该试验与金标准做了盲法比较；灵敏度50%，特异度97%；阳性似然比(+LR)高达16.67，阴性似然比(-LR)为0.52。
- **临床应用计算：**
    - 假设患儿验前概率 (家族史估算) 为 5%。验前比 = 5%/95%=1/19。
    - 若检查结果为**阳性**：验后比 = $\frac{1}{19}×16.67=0.877$。验后概率 = $0.877/1.877≈46.72$%（患病可能性大幅飙升，跨过治疗/干预阈值）。
    - 若检查结果为**阴性**：验后比 = $\frac{1}{19}×0.52=0.0274$。验后概率 = $0.0274/1.0274≈2.67$%（患病可能性降低，暂不需特殊干预）。
