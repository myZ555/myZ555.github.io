# 骨关节炎OA
>- 骨关节炎（osteoarthritis，OA）是一种以<u>关节软骨退行性变</u>和<u>继发性骨质增生</u>为特征的慢性关节疾病。
>- 疾病累及关节软骨和关节内其他组织，包括软骨下骨、滑膜、关节囊等。
>- 好发于负重较大的膝关节、髋关节等部位。
>- 该病亦称为骨关节病或骨性关节炎。
##### 【病因】
- 不明；
- 机械性因素和生物性因素共同作用所致
- 年龄是明确高危因素
- 女性发病率高于男性
##### 【病理】
最主要的病理变化包括关节软骨变性、磨损、缺失，软骨下骨硬化或囊性变，关节边缘骨赘形成，关节滑膜增生，关节囊和周围韧带增厚挛缩，关节周围肌肉萎缩，最终关节面完全破坏，出现关节畸形

| 部位        | 病理                                                                                                 |
| --------- | -------------------------------------------------------------------------------------------------- |
| ==关节软骨==      | 早期关节软骨变为<u>淡黄色，失去光泽</u>，继而局部发生*软化*，失去弹性。<br>关节负重活动时软骨磨损，表面变粗糙，并发生碎裂、剥脱，形成*关节内游离体*，导致软骨下骨质外露        |
| ==软骨下骨==      | 软骨磨损最严重的<u>中央部位</u>软骨下<u>骨密度增加，骨小梁增粗</u>，形成“*象牙质改变*”。<br><u>外周部位</u>承受应力较小，软骨下骨骨质吸收，出现*囊性变*，囊壁可有硬化 |
| ==骨赘形成==      | 在软骨边缘或肌腱附着处，因为血管增生，软骨细胞代谢活跃，通过<u>软骨内成骨</u>而出现<u>骨质增生</u>。                                          |
| ==滑膜==        | 疾病早期的病理改变为*增殖型滑膜炎*，表现为<u>滑膜充血、水肿，滑液分泌增多</u>。<br>后期为*纤维型滑膜炎*，表现为滑液减少，增殖的滑膜被纤维组织条索物替代                |
| ==关节囊、韧带与肌肉== | 关节囊和韧带发生<u>纤维变性、增厚</u>，限制关节活动。<br>关节周围肌肉因疼痛产生*保护性痉挛*，进一步限制关节活动，后期出现*肌肉萎缩*。                         |
##### 【临床表现】
1. **关节疼痛与压痛**：初期为轻度或中度间歇性隐痛，休息时缓解，活动后加重，<u>疼痛常与天气变化有关</u>。后期可出现持续性疼痛或夜间痛。关节局部有压痛，在伴有关节肿胀时尤为明显。
2. **关节僵硬**：在早晨起床时关节僵硬，称之为晨僵，活动后可缓解。关节僵硬在<u>气压降低或空气湿度增加</u>时更明显，持续时间一般较短，一般不超过 30 分钟。 
3. **活动障碍**：关节疼痛、肌肉萎缩、软组织挛缩及关节内游离体可引起行走时打软腿或关节交锁、关节不能完全伸直和屈曲。 
4. **关节畸形**：<u>手部</u>骨关节炎关节肿大变形明显，可出现 *Heberden 结节*[^1]和 *Bouchard 结节*[^2]。膝关节可因关节积液和骨赘形成造成关节肿大，甚至出现内翻或外翻畸形。 
5. **骨擦音（感）**：由于关节软骨破坏、关节面不平，关节活动时可出现骨擦音（感），多见于膝关节。
##### 【辅助检查】
1. 实验室检查：滑膜炎症期可出现CRP轻度升高和ESR轻度加快
2. 影像学检查：<u>X线检查</u>最常用，表现为<u>非对称性关节间隙变窄、软骨下骨硬化和囊性变，关节边缘骨赘形成，部分关节内可见游离体，严重者出现关节畸形</u>如膝内翻
##### 【鉴别诊断】
1. *类风湿关节炎*[^3]：中青年女性；对称性多关节受累、常见双手小关节；ESR加快和CRP升高，<u>类风湿因子RF阳性</u>
2. *痛风性关节炎*[^4]：男性；非对称性急性发作的关节红肿、疼痛、皮温升高；第1跖趾关节、踝关节和膝关节；高嘌呤饮食；血尿酸升高
3. *强直性脊柱炎*[^5]：青年男性；双侧骶髂关节及下腰部->胸段和颈段；ESR升高、<u>HLA-B27阳性</u>；X线骶髂关节炎表现、脊柱<u>竹节样</u>改变
4. *色素沉着绒毛结节性滑膜炎*：20-30岁；膝关节、单发；关节疼痛肿胀，可扪及质韧的增生滑膜组织；MRI可见<u>增厚的滑膜和增厚的绒毛样组织</u>；关节穿刺可见<u>深色或咖啡色血性液体</u>
##### 【治疗】
1. 基础治疗（健康宣教、物理治疗、行动支持、支具辅助）
2. 药物治疗
	1) 外用镇痛药：NSAIDs
	2) 全身镇痛药：NSAIDs
	3) 关节腔注射药物：<u>透明质酸钠</u>（润滑关节、保护软骨、缓解疼痛）；<u>糖皮质激素</u>（快速缓解疼痛但加剧关节软骨退变）
3. 手术治疗：消除疼痛+矫正畸形+改善功能
	1) 关节镜清理术
	2) 关节力线矫正术
	3) 人工关节置换术
# 强直性脊柱炎AS
>- 强直性脊柱炎（ankylosing spondylitis，AS）是一种慢性炎症性疾病
>- 病变常从骶髂关节开始逐渐*向上蔓延*至脊柱，导致脊柱发生纤维性或骨性强直，并可累及下肢关节，最终发生畸形。
##### 【病理】
- 基本病理改变为**原发性、慢性、血管翳破坏性炎症**，**韧带骨化**是继发的修复过程。
- 病变一般自*骶髂关节*开始，缓慢沿着脊柱向上延伸，累及椎间小关节的滑膜和关节囊以及脊柱周围的软组织，晚期可使整个脊柱周围的软组织钙化、骨化，导致*严重的驼背畸形*。
- 病变也可同时向下蔓延，累及双侧髋关节，少数也可累及膝关节和踝关节。
##### 【临床表现】
- 本病好发于<u> 16～30 岁的青壮年</u>，<u>男性</u>占 90%，有明显的家族遗传史
- 早期主要表现为*下腰痛、骶髂部不适或疼痛、 发僵*。晨起或久坐起立时腰部僵硬明显，但<u>活动后减轻</u>。
- *髋关节*受累时表现为臀部和腹股沟酸痛不适，疼痛可向下肢放射。
- 症状在静止、休息时加重，活动后缓解。
- 半数病人以下肢大关节如髋、膝、踝 关节炎症为首发症状。
- 晚期脊柱严重僵硬可<u>致躯干和髋关节屈曲</u>， 最终发生*驼背畸形，胸椎明显后凸（大于 90°），不能平视，视野仅限于足下*。
- 由于颈椎和腰椎旋转活动受限，侧视时必须转动全身。
- 髋关节受累时行走呈*摇摆步态*。
- 也有病人病变始自颈椎，逐 渐向下波及胸椎和腰椎，称Bechterew 病，容易累及神经根而发生<u>上肢瘫痪和呼吸困难</u>，预后较差。
##### 【辅助检查】
1. 实验室检查：RF阴性；免疫球蛋白可轻度升高；HLA-B27阳性
2. 影像学检查[^6]：
	1) X线检查：早期表现为<u>骶髂关节骨质疏松</u>，关节边缘呈*虫蛀状改变*，间隙不规则增宽，软骨下骨硬化。随后关节面渐趋模糊，间隙逐渐变窄，直至*双侧骶髂关节完全融合*。椎间小关节出现类似改变，周围韧带发生骨化，形成广泛的“<u>骨桥</u>”，称为脊柱“*竹节样改变*“。病变晚期累及髋关节和膝关节时可见*关节间隙狭窄*，最终发生*骨性强直*
	2) CT 较 X 线可更清晰地显示骶髂关节结构性改变。
	3) MRI 可显示骶髂关节急性炎症性改变及结构改变，能更早地发现 AS 病人的骶髂关节病变。
##### 【诊断】
1. 腰背痛至少持续 3 个月，活动后疼痛可缓解。
2. 受累脊柱节段各个方向的活动均受限。
3. 胸廓扩展范围小于同年龄、同性别人群正常值。
4. 双侧骶髂关节炎 Ⅱ～Ⅳ级，或单侧骶髂关节炎Ⅲ～Ⅳ级。
5. 如具备第4条和1～3条中的任何 1 条，可诊断为强直性脊柱炎。
##### 【鉴别诊断】
1. *类风湿性关节炎*[^3]（女；AS均有骶髂关节受累、RA则很少；AS为<u>全脊柱自下而上受累</u>，RA只累及颈椎；AS外周关节以下肢大关节受累、而RA有手部小关节；RF阴性/阳性；HLA-B27阴性/阳性）
2. *髂骨致密性骨炎*：多见于青年女性、慢性腰骶部疼痛和晨僵
##### 【治疗】
目的：解除疼痛+防止畸形+改善功能
1. 非药物治疗
2. 药物治疗（*NSAIDs是一线药物*）
3. 手术治疗
# 类风湿性关节炎RA
[^3]
##### 【病理】
- 基本病理变化：*关节滑膜的慢性炎症*
- 早期滑膜充血、水肿，单核细胞、淋巴细胞浸润，*滑膜边缘部分增生形成肉芽组织血管翳*，并逐渐覆盖于关节软骨表面。
- 后期关节表面肉芽组织逐渐纤维化，形成*纤维性关节僵硬*，进一步发展为*骨性强直*。
- 关节周围的肌腱、腱鞘也有类似的肉芽组织 侵入，继而发生挛缩，进一步影响关节功能。
##### 【临床表现】
1. **关节肿痛**：受累关节表现为<u>持续性肿胀、疼痛和压痛</u>。绝大多数病人发病初期表现为*关节肿胀*。肿胀是由关节腔内<u>渗出液增多</u>及关节周围<u>软组织炎症</u>所致，表现为*关节均匀性肿大*，手指近侧指间关节的$梭形肿胀$是类风湿病人的典型表现之一。关节疼痛的程度通常与其肿胀的程度相关联，关节肿胀愈明显，疼痛愈重
2. **晨僵**：病变关节在夜间静止不动后，晨起时出现较长时间的僵硬与活动受限，<u>多持续超过 1 小时</u>。病情严重时全身关节均可出现僵硬感。 起床后经活动或温暖后晨僵症状可减轻或消失。晨僵常伴有*指（趾）发冷和麻木感*。
3. **关节活动受限与畸形**：病变持续发展，关节活动受限逐渐加重。晚期关节出现不同程度畸形，最常见的手部畸 形有*掌指关节尺偏畸形*和*手指鹅颈畸形*，后者表现为<u>掌指关节屈曲、近侧指间关节过伸、远侧指间关节屈曲</u>。 其他关节畸形包括膝关节内、外翻畸形和腕、肘关节强直等
##### 【辅助检查】
1. 实验室检查：RF阳性；ESR增快、CRP升高、血清 IgG、IgA、IgM 增高。<u>关节液混浊，黏稠度低，黏蛋白凝固力差，糖含量降低，细菌培养阴性</u>
2. 影像学检查：X线检查早期表现为关节周围软组织肿胀，关节间隙增宽，关节周围骨质疏松。随病变发展，关节周围骨质疏松加重，关节面边缘模糊不清，关节间隙逐渐变窄。晚期关节间隙消失，最终出现骨性强直
##### 【诊断】
1. 晨起关节僵硬至少 1 小时（≥6 周）
2. 3 个或 3 个以上关节肿胀（≥6 周）
3. 腕关节、 掌指关节或近侧指间关节肿胀（≥6 周）
4. 对称性关节肿胀（≥6 周）
5. 皮下类风湿结节
6. 手、腕关 节 X 线片有明确的骨质疏松或骨侵蚀
7. 类风湿因子阳性（滴度＞1∶32）
- 符合 4 条或 4 条以上标准即可诊断本病
##### 【鉴别诊断】
1. *风湿性关节炎*：<u>溶血性链球菌感染</u>引起的全身变态反应性疾病，常有咽峡炎、丹毒等感染病史。起病较急，多见于青少年。可侵犯心脏，引起<u>风湿性心脏病</u>[^7]，并有发热、皮下结节和皮疹等表 现。该病有两个特点：*一是关节红、肿、热、痛明显*，不能活动，发病部位常常是膝、髋、踝等下肢大关 节，其次是肩、肘、腕关节，手、足小关节少见。*二是多关节游走性疼痛*，但疼痛持续时间不长，几天可 消退。实验室检查示血沉增快，<u>抗链球菌溶血素 O 滴度升高，类风湿因子阴性。</u>治愈后很少复发，关节不遗留畸形。
2. *强直性脊柱炎*[^5]：脊柱；青年男性；骶髂关节；HLA-B27
3. *痛风性关节炎*[^4]：不再赘述
4. *骨关节炎*
##### 【治疗】
1. 非药物治疗
2. 药物治疗
	1. 尽早使用DMARDs，首选*甲氨蝶呤*
	2. 中/高疾病活动度，联合小剂量糖皮质激素
3. 手术治疗

[^1]: **赫伯登结节**（Heberden's nodes）是手指远端指间关节（最靠近指尖的关节）出现的骨性增生或硬结，是**骨关节炎**的常见体征。多见于中老年女性，主要由关节软骨磨损、退行性变及遗传因素引起。![[Heberden结节.png]]

[^2]: **布夏氏结节**（Bouchard's nodes）是近端指骨关节（手指或脚趾中间的关节）上坚硬的骨头增生或胶状囊肿。常见于手部患有骨关节炎的患者，是由关节软骨钙化增生的骨刺形成。在较少情况中，类风湿性关节炎患者会有这样的结节，但成因不同，它是因为抗体沉积在关节滑膜上引起发炎后的肿胀。
