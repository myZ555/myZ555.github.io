# 2. 炎症性肠病 IBD（克隆恩病和溃疡性结肠炎）
- 有许多共同特征、来源不明的慢性炎性疾病
- 在遗传易感的个体中，针对正常肠道菌群的一种不正常的局部免疫反应，可能针对某些自身抗原
- CD：克罗恩病；UC：溃疡性结肠炎![400](../Attachments/Pasted%20image%2020250604133923.png)
	- CD：主要是回肠末端和结肠起始段；整个消化道；最重是回盲部
	- UC：累及整个结肠，最明显在直肠和乙状结肠
- 终身疾病-为什么治不好?
	- 发病机制不清
	- 绝大多数药物只有部分病人有反应，且只是缓解不能根除
![Pasted image 20250604141115](../Attachments/Pasted%20image%2020250604141115.png)
### 克罗恩病 CD
- 一种**全身性炎症性疾病**，主要累及胃肠道
- 影响胃肠道的**任何部分**，从食道到肛门，最常累及**回肠**
- ==非干酪样肉芽肿性炎症==，50%
- ##### 流行病学：
	- 美国、中欧>亚非
	- 累及任何年龄，女>男
	- 新兴工业化国家发病率增加最快

#### 临床表现：
- 多变、不可预测
- 主要表现：
	- 腹泻、腹绞痛、发热
	- 黑便：结肠50%
- 无症状间隔
- 治疗后可能稳定，20%患者诊断后病情持续活跃
#### 结局：
- 瘘管（特点；UC不会发生；继发细菌感染--腹腔脓肿）

- 癌症（较低；UC较高）

### 溃疡性结肠炎 UC
- 系统性疾病
	- ==溃疡性炎症==
	- ==非肉芽肿性疾病==
- 局限于**结肠**，局限于黏膜和黏膜下层，很少纤维化
- 从结肠远端开始**连续性分布**，无跳跃病变，**癌发生风险高**
- #### 流行病学
	- 好发于任何年龄
	- 女=男
	- 家族关联性（≠遗传性；尚不清楚）

# 4. 肠伤寒 Typhoid fever
- 伤寒是由**伤寒沙门氏菌**引起的一种**急性传染病**，以**全身单核巨噬细胞系统**(mononuclear phagocyte system，MPS) （肠和肠系膜淋巴 结、肝脏、脾脏和骨髓等）网状内皮中**单核-巨噬细胞增生**为特征， 具有相应的临床表现。
### 病因和发病机制
- 病原体：**伤寒杆菌只感染人类**。 ==内毒素==(致病性强)，没有外毒素。
- 抗原：
	- 菌体：O抗原
	- 鞭毛抗原：H抗原--血清凝集
	- 表面抗原：Vi抗原
- 肥达反应检测
- 传染源：患者及健康携带者
- 感染途径：**粪口传播**
	- 由于有症状的伤寒病例或有症状带菌者的粪便污染了食物和水
	- 苍蝇是重要载体
- #### 机制：
	1. 被摄入的杆菌侵入**小肠黏膜**，在10天的潜伏期内被**巨噬细胞吸收**并运输到区域淋巴结 并在肠淋巴组织中繁殖。
	2. 在潜伏期结束时，杆菌进入血液（在第一周，内毒素血症和菌血症阶段），导致发烧、头痛 和肌肉疼痛。在这一阶段，许多组织可能受到感染，95%的病例**血培养呈阳性。**
	3. 伤寒沙门氏菌通过胆道排泄进入肠腔。该菌再次感染小肠淋巴组织，引起急性炎症、坏死和 溃疡（该菌直接侵入，杆菌释放内毒素，并伴有迟发性超敏反应）。
		1. 这个阶段的粪便和尿液培养呈阳性
		2. 约60%的患者血培养仍呈阳性
		3. 肥达反应阳性
	4. 愈合通常在第三周结束时开始，在无并发症的病例中，在第五周完成。
### 病理变化
- **急性增生性炎症**（Acute hyperplastic inflammation，**单核巨噬细胞增生**）
- 增生性巨噬细胞吞食细菌、红细胞和核碎片，往往在淋巴组织内形成小结节，这 种结节即为**伤寒肉芽肿**（typhoid granuloma），这种巨噬细胞称为**伤寒细胞** （typhoid cells）。

### 其他器官变化
伤寒与多种广泛分布的病变有关，这些病变可能在疾病过程中或之后出现：它们是由**内毒素血症和菌血症**引起的。
1. ==胆囊==：伤寒杆菌易在胆囊内大量繁殖，病愈后仍可由*胆汁长期排菌*，成为带菌者。
2. ==心肌==：中毒性心肌炎：心肌纤维浊肿，间质充血、水肿、单核细胞浸润。
3. ==横纹肌==：主要是腹直肌和大腿内收肌常发生**凝固性坏死**（亦称蜡样变性/坏死） 。
### 并发症
常发生于坏死期、溃疡期
1. 严重的**出血** （ hemorrhage ） 可能由肠道溃疡引起。广泛坏 死引起的更严重的并发症是小肠穿孔，通常伴有**致命的腹膜炎**（peritonitis）。
2. 喉炎、支气管炎和**支气管肺炎** （ bronchopneumonia ） 的 发展可能是由于其他细菌的入侵。

# 5. 细菌性痢疾 Bacillary dysentery
- 一种由**志贺氏菌**引起的**急性假膜性（pseudomembranous）炎症**， 是一种常见的**肠道传染病**（渗出性纤维素性炎）
### 病因和发病机制
- 病原体-志贺氏菌属，革兰氏阴性杆菌
- 四种：宋内、福氏、鲍氏、志贺菌
- 传染源：患者、健康带菌者
- 感染途径：非典型粪口传播。通过被志贺氏菌污染的食物和水传播
- **蝇类**是重要的传播媒介
- 发病环节：
	1. 微生物直接侵入肠黏膜
	2. 在固有层繁殖并释放毒素
	3. 毒血症
	4. 产生急性炎症和组织损伤
### 病理及临床表现
- 部位：**乙状结肠**(sigmoid colon)，**直肠**(rectum）是最常见的；在严重的情况下，所有的结肠和回肠末端都可能受到影响
#### 急性细菌性痢疾
- 最初的**急性卡他性炎症**（acute catarrh inflammation）
	- 产生急性炎症，伴有 弥漫性充血、水肿、点状出血、炎症细胞浸润（中性粒细胞）、粘液分泌过多和粘膜 形成浅表糜烂——
	- **急性假膜性炎症**（acute pseudomembranous inflammation） （纤维蛋白、粘液和炎症碎片构成的假膜）![500](../Attachments/Pasted%20image%2020250604145421.png)
	- 假膜本质上是**纤维化脓性渗出物**（fibrinsuppurative exudate） 假膜先在患病结肠粘膜表面斑片状分布，然后弥漫性覆盖粘膜，形成脏的膜状物质![500](../Attachments/Pasted%20image%2020250604145507.png)
	- 镜下，可见假膜由纤维蛋白、大量中性粒细胞混合红细胞、细胞和细菌组成(粘液 脓血便)。假膜脱落时，出现不规则浅表溃疡（**地图样溃疡**）
	>还有许多其他病原体可形成**假膜性肠炎**；滥用抗生素、形成耐药菌（e.g.,艰难梭菌）
		-肠道菌群失调的大部分炎症是假膜性炎


- ==临床表现：==
	- 毒血症—发热，外周血白细胞增多
	- 炎症— 痉挛(abdominal cramping)，刺激性黏液腹泻，粘液脓血便 (blood and pus in the stool) **里急后重**
	- 并发症：出血 穿孔 (不常见). 脱水 电解质紊乱（严重时）
	- 结局: 治愈（大多数情况下），在某些情况下，延长并演变成慢性疾病
#### 慢性细菌性痢疾
- 病程已超过**2个月**
- 这类慢性病例的特点是**反复损伤和修复，溃疡和愈合**，并可导致新老病变共存， 慢性溃疡形成，**息肉样粘膜不规则伴纤维瘢痕形成，随后肠狭窄**（stenosis）
- 临床表现： 腹痛，慢性腹泻，肠梗阻
#### 中毒型细菌性痢疾
- **儿童常见**
- 病原体**毒力较低**，疾病的发作是突然的
- 小肠病变表现为**卡他性肠炎或滤泡性肠炎**（catarrh enteritis or follicular enteritis）
- 有严重的**全身中毒症状（高热惊厥）**；发病数小时后，迅速出现中毒性休克、昏迷或脑水肿和呼吸衰竭。注意并发症
### 结局
- 大部分病例在急性期治愈，部分慢性病例可反复发作或成为慢性带菌者；
- 少数患者可出现肠出血、肠穿孔、肠狭窄及支气管炎等并发症；
- 中毒型菌痢的病死率较高。

- 6种血吸虫：**日本血吸虫**、曼氏、埃及、间插、湄公、马来血吸虫![Pasted image 20250604150802](../Attachments/Pasted%20image%2020250604150802.png)
### 基本病变

| 虫体引起病变           | 虫卵引起病变          |
| ---------------- | --------------- |
| 尾蚴：**尾蚴性皮炎**     | 急性虫卵结节          |

- **虫卵引起的病变是血吸虫病的最主要病变**。 
- 血吸虫寿命：20～30年，每对成虫产卵量:150～3000个/每日 
- **50％虫卵沉积在结肠壁**，不足10％在小肠壁，23％随血流进入肝脏，17％虫 卵自粪便排出体外。 
- 肠壁的初产卵约经7～10天发育，经单细胞期、细胞分裂期与胚胎发育期，最后形成**内含毛蚴的成熟虫卵。** 
- 未成熟虫卵引起的病变轻微，含毛蚴的成熟虫卵往往引起严重的病变。

- 代谢产物、分泌物、排泄物、分解产物等
- 循环抗原：肠相关抗原，膜相关抗原 可溶性虫卵抗原（SEA）
- 免疫复合物：肾损伤，血管关节损伤
### 主要器官病变及其后果
1. **结肠**: 盲肠到直肠的全部结肠，**乙状结肠最为显著**。
	1. 急性期：多发性糜烂、溃疡（大小不一，边缘不规则的浅表溃疡）虫卵排入肠道 
		- 镜检：急性虫卵结节 
		- 临床：腹痛、腹泻和脓血便 
		- **血吸虫痢疾**（Schistosomal dysentery） 
		- 虫卵阳性检出率高
	2. 慢性期：
		- 病变新旧交替，大小不一**息肉**，大小深浅不一溃疡，肠壁增厚，肠腔狭窄、 肠梗阻。 
		- 镜检：粘膜下层，固有膜 
		- 急/慢性虫卵结节，部分钙化 晚期粪便虫卵检出率低
2. **肝脏**：干线型肝硬化，**血吸虫性肝硬变/干线型/管道型肝硬变**

# 7. 肠阿米巴病
![Pasted image 20250604153224](../Attachments/Pasted%20image%2020250604153224.png)
包囊──→小滋养体──→包囊（强抵抗力）
（小肠下端及回盲部缺氧及碱性环境,肠腔型）
		↑     ↓（侵入肠粘膜）
		 大滋养体（组织型）
- 由**溶组织内阿米巴原虫**感染引起的
- 主要累及**结肠**，故又名阿米巴痢疾（amoebic dysentery）。还可以侵袭肝、肺、脑、 泌尿生殖系统及皮肤等，最常在**肝脏**引起**继发性阿米巴脓肿**(Amoebic abscess)
### 病理变化
- 最常见发生部位：**盲肠和升结肠** 其次：直肠、乙状结肠 
- 基本病变：**大滋养体的接触溶解作用，组织变质性炎症** 
#### 急性期病理变化
