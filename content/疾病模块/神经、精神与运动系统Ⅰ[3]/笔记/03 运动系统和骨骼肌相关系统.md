# 一、骨学总论

- 骨：以骨组织为主体构成的器官，在结缔组织或软骨基础上经骨化形成

| 骨的形状 | 分布     | 形态        | 功能   |
| ---- | ------ | --------- | ---- |
| 长骨   | 四肢     | 长管状，一体两端  | 运动功能 |
| 短骨   | 腕骨、跗骨  |           | 支持功能 |
| 扁骨   | 颅盖骨、肋骨 | 板状，保护内部脏器 | 保护功能 |
| 不规则骨 | 上颌骨、蝶骨 | 含气骨       |      |
| 籽骨   |        |           |      |
##### 骨的构造

| 骨质  | 骨密质 | 表面，致密坚硬，多层紧密骨板                                 |
| --- | --- | ---------------------------------------------- |
|     | 骨松质 | 内部，骨小梁交织形成，海绵状                                 |
| 骨膜  |     | DCT，被覆于除关节面以外的骨质表面；  <br>富含血管、神经，通过滋养孔分布于骨质和骨髓 |
| 骨髓  | 红骨髓 | 造血功能                                           |
|     | 黄骨髓 | 5岁后长骨骨髓内红骨髓被脂肪组织所代替                            |

##### 骨的化学成分和物理特征

| 有机质       | 无机质      |
| --------- | -------- |
| 骨胶原、粘多糖蛋白 | 无机盐（磷酸钙） |
| 韧性、弹性     | 硬度、脆性    |

##### 骨的表面标志
1. 骨面突起（肌腱/韧带牵拉所致）
2. 骨面凹陷（与邻位结构相接触或肌肉附着的影响而形成；血管或神经压迫后形成的沟）
3. 骨的腔洞（容纳某些结构或空气，结构穿行形成）
4. 骨端的标志（关节面）

# 二、关节学

##### 1. 骨连结的分类
1. 直接连结：间隙小或无，微动或不动
2. 间接连结：间隙大，活动性大（关节）
##### 2. 直接连结的分类和特征
- **纤维连结**
	- 韧带--致密结缔组织
	- 缝--少量结缔组织纤维
- **软骨连结**
	- 透明软骨连结
	- 纤维软骨连结
- **骨性连结**：由软骨结合经骨化演变而成，完全不能活动
##### 3. 关节的基本结构和辅助结构
- 基本构造：关节面、关节囊、关节腔
- 辅助结构：韧带、关节盘、关节唇、滑膜襞、滑膜囊
##### 4. 关节的类型和运动方式
1. 屈/伸
2. 内收/外展
3. 旋内/旋外

# 三、肌学
##### 1. 概述
1. 不随意肌：平滑肌、心肌；随意肌-骨骼肌
2. 骨骼肌是运动系统的组成部分
	- 横纹肌
	- 绝大部分附着于骨，仅部分皮肌不附于骨，名骨骼肌
	- 骨骼肌主要分布于躯干四肢，共600多块
	- 肌占体重40%，四肢肌占肌总重80%
	- 骨骼肌的作用：运动的动力、需要与骨和关节协同运动、需要形成分群互相协同运动
##### 2. 骨骼肌的构造和形态
1. 肌的构造
	- 肌腹 venter：肌的主体；色红，柔软，有收缩能力
	- 肌腱 tendon：色白，平行的胶原纤维束，*无收缩能力*
2. 肌的形态：长肌、短肌、扁肌、轮匝肌

##### 5. 肌的辅助装置
1. 筋膜 fascia
	1. 浅筋膜 ： 全身皮下组织，LCT；内含浅动静脉、浅淋巴结/管、皮神经；胸部乳腺 
	2. 深筋膜 ： 固有筋膜，DCT；包裹肌肉、血管神经束。肌肉分层时的*肌间隔* 
2. 滑液囊
3. 腱鞘
	- 长肌腱表面、伸向深部与骨膜连接，形成筒状纤维鞘
    - 外层：纤维层，由深筋膜形成的骨性纤维性管道
    - 内层：滑膜层，内层包在肌腱表面
    - *腱系膜*：腱纽（血管神经出入处）
	    - 临床意义：
		    1. 腱鞘炎--疼痛、运动受限
		    2. 腱鞘肿胀--压迫键钮血管--肌腱坏死
