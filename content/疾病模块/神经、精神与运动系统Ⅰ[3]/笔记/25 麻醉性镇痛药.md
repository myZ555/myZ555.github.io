# Opioid Analgesics 阿片类
- 阿片类镇痛易产生药物依赖性（dependence）或成瘾性，易导致药物滥用 （drug abuse）及戒断症状（withdrawal syndrome），故又将此类药物归为 **麻醉性镇痛药或成瘾性镇痛药**（addictive analgesics）。 绝大多数被归入管制药品之列

##### 阿片类受体
- 阿片类药物作用在受体上有高效性和选择性，立体结构特异性（左旋有效）
- 脑内阿片受体主要存在于下丘脑、中脑导水管周围灰质、蓝斑核和脊髓背角区

| Subtype  | Functions                                 |
| :------: | ----------------------------------------- |
|  $\mu$   | 介导吗啡镇痛效应的主要受体，也介导镇静、呼吸抑制、缩瞳、欣快及依赖性<br>等效应 |
| $\delta$ | 脊髓镇痛和镇静作用                                 |
| $\kappa$ | 镇痛效应不明显，但能引起抗焦虑和抗抑郁作用                     |

###### Morphine 吗啡
- 阿片受体激动药；主要作用于μ、κ受体；脂溶性低
- 经肾脏排泄，可经乳腺排泄，也可通过胎盘屏障
- 吗啡-6-葡萄糖醛酸活性强于吗啡
1. **药理学作用**
	1. 作用机制机制：作用于中枢阿片受体
		- 镇痛作用：Pain inhibitory system（同时通过直接源自脊髓背角的痛觉上行传入通路和激活源自中脑的痛觉下行控制环路来实现的。  直接抑制脊髓背角痛觉传导神经元，在脊髓水平发挥有力的镇痛作用。）
	2. 中枢神经系统：
		1. 镇痛：急性、慢性疼痛；对持续性钝痛>间断性锐痛， 脊髓胶质区、丘脑内侧、脑室及导水管周围灰质
		2. 欣快 ：满足感和飘然欲仙（边缘系统、蓝斑核，中脑腹侧背盖区）
		3. 镇静：消除疼痛引起的焦虑、进展、恐惧等情绪反应
		4. 抑制呼吸: 降低脑干呼吸中枢对 CO 2的敏感性，呼吸频 率减慢为主。（对延髓心血管运动中枢无抑制作用）
		5. 镇咳：抑制延髓咳嗽中枢（延髓孤束核阿片受体）
		6. 缩瞳：兴奋支配瞳孔的副交感神经，针尖样瞳孔。
		7. 其他
			1. 作用于下丘脑体温调节中枢，调节体温
			2. 兴奋延髓催吐化学感受区，引起恶心和呕吐
			3. 抑制下丘脑释放促性腺激素释放激素和促肾上腺皮质激 素释放激素
	3. 平滑肌：
		1. 胃肠道平滑肌：便秘
			1. 减慢胃蠕动，提高胃窦部和十二指肠上不的张力，可致胃 排空延迟、食物反流、减少药物吸收；
			2. 提高小肠及大肠平滑肌张力，减弱推进型如东，抑制消化 腺的分泌
			3. 提高回盲瓣和肛门括约肌张力，抑制中枢，降低便意
		2. 胆道平滑肌：收缩Oddi’s 括约肌
		3. 呼吸道平滑肌：大剂量可以引起支气管收缩
		4. 泌尿系统平滑肌：提高尿道外括约肌张力和膀胱容积， 引起尿潴留
		5. 子宫平滑肌：降低肌张力，抑制收缩，延长产程
	4. 心血管系统：
		1. Postural hypotension （直立性低血压）： 促进组胺释放，扩张血管，降低外周阻力
		2. 增加颅内压： 呼吸抑制 → CO2 增加 → 脑血管扩张 → 脑 血流增加
	5. 抑制免疫系统
		- 抑制自然杀伤细胞的细胞毒作用：抑制淋巴细胞 增殖，减少细胞因子的分泌
		- e.g.抑制HIV诱导的免疫细胞
	6. 影响内分泌
		- 中枢作用：抑制下丘脑释放促性腺激素释放激素 和促肾上腺皮质激素释放激素
2. **临床应用**
	1. 镇痛（Analgesia）：主要用于剧烈疼痛及疾病晚期的疼痛
	2. 心源性哮喘：急性左心衰竭突发急性肺水肿所致的呼吸困难
	3. 腹泻：急慢性消耗性腹泻
3. **不良反应**
	1. 急性和慢性不良反应
	2. 耐受性（tolerance）：阿片受体下调、Gi偶联减弱， 孤啡肽生成拮抗阿片类药物的作用
	3. 依赖性：生理依赖性和精神依赖性
		- 成瘾性，addiction
		- 戒断症状，withdrawal syndromes or abstinence syndromes
	4. 急性中毒：三联征：昏迷，深度呼吸抑制，针尖样瞳孔；伴低血压、体温降低、皮肤湿冷、尿潴留
	5. 痛觉过敏（hyperalgesia）
		- 增加疼痛感，导致痛觉过敏状态
		- 阿片受体适应性变化、NMDA受体的激活、胶质细胞 活化、下行易化系统增强。
		- 防吗啡治疗慢性疼痛的临床应用
	6. 禁忌症：
		1. 产妇和哺乳期妇女
		2. 支气管哮喘、肺心病患者
		3. 颅内压增高患者
		4. 肝功能严重减退患者
		5. 新生儿和婴儿

###### Codeine 可待因（甲基吗啡）
- 镇痛作用为吗啡的1/12～1/10，镇咳作用为吗啡的1/4，抑制呼吸作用也较轻，无明显的镇静作用
- 用于轻、中度疼痛和剧烈干咳
- 作用比吗啡弱，生物利用度增 加
- 无明显便秘、尿潴留、直立性 低血压等副作用
- 欣快症及成瘾性也低于吗啡

# Synthetic Analgesics
###### Pethidine(哌替啶，杜冷丁)
1. **药理学作用**
	1. Activates μ receptor；
	2. 镇静、呼吸抑制、致欣快和扩血管作用与吗啡相当；镇痛作用弱于吗啡
	3. 由于持续时间短（2-4小时），提高平滑肌和括约肌张力 引起的便秘和尿储留作用较弱；
	4. 可致支气管平滑肌收缩，但无明显中枢性镇咳作用
	5. 有轻微的子宫兴奋作用，但对妊娠末期子宫收缩无影响， 不对抗缩宫素的作用，故不延缓产程。
2. **临床应用**
	1. 镇痛、心源性休克
	2. 麻醉前用药或者人工冬眠（杜冷丁+氯丙嗪+异丙嗪）;
	3. 由于新生儿呼吸抑制的作用较为敏感，产妇临产前2-4小 时不宜使用。
3. **不良反应**：同吗啡
4. **药物相互作用**
	1. +MAOI，谵妄、高热、多汗，惊厥、严重的呼吸抑制、昏迷、死亡；
	2. +氯丙嗪、异丙嗪、TCA，呼吸抑制；
	3. 可增强双香豆素等的凝血作用；
###### Methadone （美沙酮）
- Activates μ receptor，左旋美沙酮起效;长效μ受体激动药
- 口服吸收好
- 镇痛作用与吗啡相当，持续时间长，不良反应较吗啡弱;
- 阿片类药物成瘾的**脱毒治疗**：口服美沙酮后再注射吗啡 不能引起原有的欣快感，亦不出现戒断症状，因而使吗 啡等的成瘾性减弱！
###### Fentanyl （芬太尼）
- μ受体激动药，镇痛效价强度为吗啡的100倍
- 短效镇痛药，静脉注射持续10min；肌肉注射维持1-2小时；透皮贴可 是血药浓度持续72小时。经肝脏CYP450酶系统代谢。
- 用于麻醉辅助用药、静脉复合麻醉，与氟哌利多合用 （1:50）用于神经安定镇痛术。
- 不良反应有眩晕、恶心、呕吐及胆道括约肌痉挛。静脉 注射过快可致呼吸抑制。反复用药产生依赖性。
- 透皮贴--经皮肤给药；芬太尼贴剂透皮给药具有血药浓度稳定、镇痛作用 持久（可达 72 小时）、胃肠道 反应轻的特点
==Sufentanil （舒芬太尼）==：镇痛作用是吗啡的1000倍
==Alfentanil （阿芬太尼）==：镇痛作用是吗啡的吗啡的40～50 倍；镇痛作用起效快（1分钟左右），作用时间短，称为超短效镇 痛药。血浆蛋白结合率高，肝脏代谢，肾脏排出。
==Remifentanil （瑞芬太尼）==：新型芬太尼衍生物，μ 受体激动药，镇痛作用为吗啡的 100～200 倍。注射后1分钟起效，持续5-10分钟，被体内的酯酶快速水解

###### 二氢埃托啡
镇痛作用是吗啡的6000-10000倍，依赖性强
###### 曲马多
- 弱 μ 受体激动药，能抑制NA和5-HT再摄取
- 镇痛作用与喷他佐辛相当，镇咳作用为可待因的1/2，呼吸 抑制作用弱，对胃肠道无影响，也无明显的心血管作用。
- 活性代谢产物的作用强与曲马多。
###### 布桂嗪（强痛定）
- 有较弱的μ阿片受体激动作，也可以影响DA、5-HT和NA等 神经递质发挥作用。
- 镇痛效价强度为吗啡的1/3，呼吸和胃肠道作用较轻。
- 临床多用于偏头痛、三叉神经痛、炎症性及外伤性疼痛、 关节痛、痛经及晚期癌症疼痛

# 阿片受体部分激动药
- 阿片受体混合型激动 - 拮抗药（mixed agonists-antagonists）；本类药物以镇痛作用为主，呼吸抑制作用较弱，成瘾性较小 ，有拟精神失常等副作用。
###### Pentazocine （喷他佐辛, 镇痛新）
- κ receptor agonist / μ receptor partial agonist;
- 成瘾性小
- 镇痛作用是吗啡的1/3，呼吸抑制作用为1/2，较为安全。
- 大剂量可产生烦躁不安、梦魇、幻觉等精神症状。也可增 加血中儿茶酚胺浓度，加快心率和升高血压。
###### 布托啡诺
- κ receptor agonist / μ receptor partial agonist;
- 镇痛作用是吗啡的3.5-7倍，呼吸抑制的作用不随剂量增加 而加重。
- 用于轻中度疼痛。
- 常见有镇静、乏力、出汗，个别出现嗜睡、头痛、眩晕、 飘浮感、精神错乱等。
- 可增加外周血管阻力和肺血管阻力
###### 丁丙诺啡
- 激动μ受体，对κ 受体有拮抗作用;
- 镇痛作用是吗啡的25倍，呼吸抑制的作用轻。
- 成瘾性比吗啡弱。
###### 纳布啡
- 对μ受体的拮抗作用强，对κ 的激动作用弱;
- 镇痛作用弱于吗啡，呼吸抑制的作用轻。
- 依赖性小，戒断症状轻。

# 其他中枢镇痛药
###### Rotundine （l-tetrahydropalmatine, 罗通定）
- 有镇静、安定、镇痛和中枢性肌肉松弛作用
- 镇痛作用较哌替啶弱，但较解热镇痛抗炎药作用强， 无明显的成瘾性
- 镇痛作用与阻断脑内DA受体，增加脑啡肽原和内啡肽 原的mRNA表达
###### Nefopam （奈福泮）
镇痛作用机制包括抑制5 - 羟色胺、去甲肾上腺素和多巴 胺等胺类递质再摄取，阻滞钠通道，导致与痛觉过敏有 关的突触后膜谷氨酸受体，如NMDA 受体激活减少。
###### Ziconotide （齐考诺肽）
- 是N 型钙通道拮抗剂ω - 芋螺毒素（ω-conotoxin） MVIIA 的合成类似物，能可逆性阻滞N 型钙通道，从 而抑制神经递质的释放，产生镇痛作用。
- 难透血脑屏障，只能用于鞘内注射。
