### 分级组织和平行联系

- 脊椎动物的运动控制是分级进行的，较低级的中枢控制较“自动”的运动，而较高级的中枢控制有目的的运动；同时，高级中枢可控制较低级的中枢。
- 运动系统由3个水平的神经结构组成：**脊髓、脑干的下行系统和大脑皮层的运动区

### 运动池内的运动单元按从小到大的顺序招募
- ![[Pasted image 20250312143544.png]]
 - motor unit 运动单位：一个运动神经元及其支配的所有肌纤维
 - motor pool 运动池：支配同一肌肉的运动神经元在空间上聚集在一起形成
 - 运动单位大小（即一个运动神经元所支配的肌纤维数量）在同一运动神经元池内的神经元之间存在很大差异。
 - **总而言之，一个运动神经元池内的运动单位遵循大小原则：运动单位较小的神经元会先于运动单位较大的神经元放电**
 - 
### 运动神经元接受多样而复杂的输入
- 伸肌-屈肌：拮抗肌，它们执行相反的动作，并且常常相继放电。

## 中枢模式发生器 central pattern generator

#### 中枢模式发生器协调运动时肌肉的节律性收缩
- 不同的腿部肌肉按照特定顺序被激活
- 当感觉反馈完好无损时，或者通过背根切断术去除感觉反馈后，四块肌肉呈现出相似的运动模式
- 这些实验催生了==中枢模式发生器 CPG==这个概念，它指的是：**中枢神经系统中的一个回路，该回路能在无需感觉反馈的情况下产生有节奏的输出信号，以协调不同肌肉的舒缩**
	- CPGs的存在并不意味着感觉反馈不重要；相反，感觉反馈会对其进行调节，甚至可以改变CPG的节律
	- CPG的概念已经不再局限于对运动的控制；研究认为，CPG还控制呼吸、吞咽以及其他许多节律性活动
#### 脊髓使用多个中枢模式发生器来控制运动
- *V0脊髓中间神经元的消融--> 破坏左右交替*![[Pasted image 20250312144454.png|400]]
#### 脑干包含特定的运动控制核
- 对脑干中一个名为中脑运动区（MLR）的区域进行电刺激，能促使中脑猫开始产生运动行为
- 应用：可以通过*rabies*病毒识别脑干运动神经元
- **兴奋性三叉运动神经运动核MdV**：支配前肢而非后肢运动的神经元![[Pasted image 20250312144850.png]]
- 使兴奋性三叉神经运动核MdV神经元沉默会扰乱熟练的运动任务

# 小脑
[[Neurobiology_神生_zsy_net.pdf#page=27&selection=29,0,31,8|Neurobiology_神生_zsy_net, 页面 27]]
## 一、小脑外形
| 小脑脚     | 纤维性质      | 联系部位                          |
| ------- | --------- | ----------------------------- |
| 上脚（结合臂） | 传入+**传出** | 脊髓小脑前束/**脑桥、中脑、间脑和端脑**        |
| 中脚（桥臂）  | 传入        | 大脑皮质、脑桥基底部（桥核）                |
| 下脚（绳状体） | **传入**+传出 | **脊髓、前庭核、下橄榄核、网状结构**/前庭核、网状结构 |
 - 小脑内部结构：皮质、髓质、小脑核
 - **小脑皮质**是传入纤维到达的主要部位
## 二、小脑皮质
1. 细胞
	- 神经元数量多（数十亿）
	- 细胞构筑均等一致
2. 分层![[Pasted image 20250312145830.png|700]]
## 三、纤维联系
 1. 小脑蚓部+绒球小结叶——近端躯干肌、前庭和眼的调控
2. 小脑半球中间部——远端四肢肌的调控
3. 小脑半球外侧部——四肢运动程序的规划

##  四、小脑功能

1. 维持身体平衡
2. 调节肌张力
	1. 肌张力高-僵硬（强直）
	2. 步态失常（共济失调步态-蹒跚步态）
3. 协调随意运动（辩距不良、快速轮替运动障碍）
4. 参与*运动性*学习记忆（体育运动）
5. 参与*感觉性*学习记忆
	- **感觉-运动学习反应**：小脑在协调肌肉运动方面起着核心作用，运动 员肢体协调的流畅性是由于他们习惯于自下而上 处理信息达成的。因此过多的自上而下的大脑处 理会扰乱认知流程，很可能导致一个过度思考的 运动员因 **"分析性瘫痪**"而无法正常发挥。
6. 参与摄食调控

## 三、基底核的环路
### 运动调节环路

1. 直接通路——**易化运动**
		*大脑皮质-纹状体-内侧苍白球-丘脑-大脑皮层*
2. 间接通路——**抑制运动**
		*大脑皮质-纹状体-外侧苍白球-丘脑底核-内侧苍白球-丘脑-大脑皮质*
3. 黑质-纹状体通路——**易化运动**
		黑质-*DA*-纹状体


- 功能：
	- 运动调节
	- 内脏神经功能
	- 感觉功能
	- 心理行为
## 四、基底核与临床

| **通路**                               | **作用**                   | **主要环路**                                                                                                                                                                                                           | **神经递质调控**                                                      |
| ------------------------------------ | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------- |
| **直接通路**                             | **促进运动** (Go Pathway)    | 1. **大脑皮层**（兴奋性，Glutamate） → **纹状体（尾状核+壳核）**（抑制，GABA）  <br>2. **内侧苍白球（GPi）& 网状部黑质（SNr）**（抑制，GABA）    <br>3. **丘脑（VL/VA核）**（兴奋性，Glutamate） → **大脑皮层运动区**（促进运动）                                                      | - **多巴胺（D1受体）**（来自黑质致密部，兴奋纹状体，增强运动）                             |
| **间接通路**                             | **抑制运动** (No-Go Pathway) | 1. **大脑皮层**（兴奋性，Glutamate） → **纹状体**（抑制，GABA）  <br>2. **外侧苍白球（GPe）**（抑制，GABA）  <br>3. **丘脑底核（STN）**（兴奋性，Glutamate） → **内侧苍白球（GPi）& 网状部黑质（SNr）**（抑制，GABA）  <br>4. **丘脑（VL/VA核）**（兴奋性，Glutamate） → **大脑皮层运动区**（抑制运动） | - **多巴胺（D2受体）**（来自黑质致密部，抑制纹状体，减少运动抑制，间接促进运动）                    |
| **黑质-纹状体通路** (Nigrostriatal Pathway) | 调节直接和间接通路的平衡，最终**促进运动**  | **黑质致密部（SNc） → 纹状体（释放多巴胺）**                                                                                                                                                                                        | - **D1受体（兴奋直接通路）** → 促进运动  <br>- **D2受体（抑制间接通路）** → 解除运动抑制，促进运动 |

- **直接通路 → 促进运动**
- **间接通路 → 抑制运动**
- **黑质-纹状体通路** 通过释放**多巴胺**调节两者的平衡，最终**促进运动**
    - **帕金森病**时，黑质-纹状体通路受损，多巴胺减少，导致运动减少
    - **亨廷顿病**时，间接通路受损，运动抑制减弱，导致不自主运动
