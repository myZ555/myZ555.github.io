# （一）NMJ 概念和结构
- 运动神经纤维末梢和它接触的骨骼肌细胞膜所构成。特化的突触
- 组成部分：
	- 接头前膜：嵌入细胞膜凹陷中的膜
	- 接头后膜：终板膜；对应的肌膜
	- 接头间隙：充满细胞外液

# （二）接头处兴奋传递的过程和基本特性
![[神经肌接头处兴奋传递的过程.png]]
- 去极化本身而非钠钾离子跨膜移动引起神经递质释放；接头前末梢去极化的增强（程度加大、持续时间延长、发生频率增高）均可引起Ach释放
- 去极化引起Ach释放是钙离子内流导致的
	- 钙离子是实现兴奋-分泌耦联的重要因子（双重作用：增强接头前膜去极化+触发递质释放）
	- 不同程度去极化可以激活不同程度的内向钙离子电流，引起不同程度释放（非线性）
	- 钙离子通道在接头前末梢分布最密集-囊泡集中和神经递质释放部位-活性区
	- 【钙离子回收途径：1. 细胞膜上钙泵；2. Na-Ca交换体；3. 内质网膜上钙泵】
###### 终板电位EPP
- 无“全或无”现象（MEPP的总和）
- 无电压门控钠通道，不产生AP，但可刺激周围肌膜电压门控钠通道产生AP
- 总和现象
- 与Ach释放量正相关
###### 微终板电位 MEPP
- 静息状态下肌细胞的终板膜区域仍可记录到一种自动散发的、波幅很小而固定、但特性类似于EPP的去极化电位
##### 量子式释放 quantal release
- EPP的幅度总是MEPP幅度的整数倍
- 神经末梢存在大量的囊泡和囊泡释放递质时特有的形态结构（Ω样结构）
- 量子 就是存在于接头前膜内、包含一定数目递质分子的突触囊泡

# （三）N型胆碱能受体介导NMJ快速突触传递
- nAChR只有阳离子选择性，无阴离子选择性
- 对Cl-没有通透性，对Na+、K+通透性最高，1 : 1.1
- 神经-肌接头处兴奋传递的特点：神经元1次兴奋—大量递质释放—及时被消除
	- 单向性传递（前-后）
	- 1:1 传递（1次神经AP-1次肌细胞AP）
	- 时间延搁 （化学传递）
	- 本质是电-化学-电的过程 （神经末梢AP—Ach+受体—EPP—肌膜AP）
	- 易受各种内外环境因素影响

| 影响神经肌接头兴奋传递的因素 |                                       |
| -------------- | ------------------------------------- |
| 阻断ACh受体        | 箭毒和α-银环蛇毒，肌松剂                         |
| 接头前膜ACh释放↓     | 肉毒杆菌中毒                                |
| 自身免疫性疾病        | 重症肌无力（破坏ACh受体）  <br>肌无力综合征（破坏神经末梢钙通道） |
| 抑制胆碱酯酶活性       | 有机磷农药                                 |
