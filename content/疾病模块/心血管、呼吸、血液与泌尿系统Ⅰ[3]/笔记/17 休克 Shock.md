**3. 正常的心泵功能**

###### 1. 微循环缺血期
![Pasted image 20250404103926](Pasted%20image%2020250404103926.png)
1. **变化特点**：少灌少流，灌少于流，组织呈缺血缺氧状态
2. **变化机制**：
	1. 交感神经兴奋
	2. 其他缩血管体液因子释放
		- 血管紧张素II（循环）：缩血管效应10倍于去甲肾上腺素
		- 血管升压素（PVN分泌入血）：血容量↓、疼痛，缩内脏小血管
		- 内皮素（内皮细胞产生）：强烈缩小、微血管
		- 血栓素A2（血小板、内皮细胞胞膜磷脂分解代谢）：强烈缩小、微血管，最强致聚剂
		- 白三烯（胞膜磷脂分解代谢）：缩腹腔内脏小血管
3. **代偿意义**：
	1. 有助于动脉血压维持
		- *回心血量增加*：
			- “自身输血”（肌性小/微静脉、肝、脾血管收缩） “第一道防线”
			- “自身输液”（Cap前括约肌收缩 > 微静脉收缩，Cap流体静压↓）“第二道防线”，多达1.5 L/d
		- 心排血量增加：心率↑、心肌收缩力↑
		- 外周阻力增高：小动脉痉挛收缩，有助升压
	2. 有助于心脑血管供应
		- 皮肤、骨骼肌、内脏血管： α受体为主 → 收缩
		- 冠状动脉：β受体为主 → 舒张
		- 脑动脉：血压 > 60 mmHg时可自身调节血流量
4. **主要临床表现**

###### 2. 微循环淤血期
![Pasted image 20250404105254](Pasted%20image%2020250404105254.png)
*休克原始病因未及时消除，有效循环血量未恢复，缺血缺氧持续存在*
1. **变化特点**：灌而少流，灌大于流，组织呈淤血性缺氧状态
2. **变化机制**：
	1. 微血管扩张
		1. *酸中毒*使血管平滑肌对儿茶酚胺的反应性降低（交感-肾上腺髓质系统 *持续兴奋* ）
		2. *扩血管物质生成增多*：ATP代谢产物腺苷堆积，肥大细胞释放组胺，BK↑，细胞释放K+ 肠源性内毒素或细菌入血 → iNOS↑ → NO、TNF-α↑
	2. 血液淤滞
		1. 白细胞活化，表达CAM，黏附于微静脉（**滚、集、黏**，见IRI部分）
		2. 血液**浓缩、泥化（sludge）**：组胺、BK、降钙素基因相关肽→毛细血管通透性↑，血 浆外渗，血黏度↑
3. **失代偿及恶性循环的产生**
	1. ***回心血量**急剧减少*：小、微动脉扩张，Cap网大量开放，静脉回流受阻
	2. *自身输液停止，有效**循环血量**减少*：Cap前阻力 < Cap后阻力 ，Cap流体静压↑， 血浆外渗、血液浓缩，加重微循环淤滞，形成恶性循环
	3. *心、脑血液**灌流量**减少*：动脉血压进行性↓，若 < 50 mmHg则自身调节作用丧失
4. **主要临床表现**

###### 3. 微循环衰竭期
*休克进入失代偿期后，治疗不及时或不当，进入衰竭/DIC/难治期*
![Pasted image 20250404110420](Pasted%20image%2020250404110420.png)
1. **微循环变换特点**：不灌不流，微血栓形成，无复流现象
2. **微循环变化机制**：
	1. 微血管麻痹性扩张（失去对神经体液调节的响应）
	2. DIC形成
3. **严重后果**：全身器官的持续低灌流，濒临死亡
4. **主要临床表现**：
	1. 循环衰竭：进行性顽固性低血压；心音低弱，脉搏细弱而频速；中心静脉压↓； 浅表静脉塌陷（找不到、输不进）
	2. 并发DIC：DIC时凝血物质消耗、纤溶系统激活等，引起出现出血、贫血、皮下 瘀斑等典型临床表现
	3. 重要器官功能障碍（MODS）→ 多器官衰竭 → 死亡

## （二）细胞分子机制

1. **细胞损伤**
	1. 细胞膜的变化：最早发生损伤的**部位**，膜蛋白功能障碍、膜通透性↑、细胞水肿
	2. 线粒体的变化：最早发生损伤的**细胞器**，钙盐沉着，ATP生成↓
	3. 溶酶体的变化：组织蛋白酶、胶原酶、弹性蛋白酶等释放，蛋白水解，细胞自溶；进入循环则增加血管通透性，促进炎症介质释放，雪上加霜
	4. 细胞死亡：凋亡、坏死
2. **炎症细胞活化及炎症介质表达增多**
	- 炎症细胞活化、炎性介质释放，引起全身炎症反应综合症 （systemic inflammatory response syndrome, SIRS）

# 3 机体代谢与功能变化

一、**物质代谢紊乱**
1. 脓毒性自身==分解代谢==：脓毒休克时氧耗↓，糖酵解↑，糖原、脂肪和蛋白分解代谢↑，合成代谢↓
2. 休克病因引发==应激==：一过性高血糖和糖尿，脂肪和蛋白分解代谢↑，尿氮增多、负氮平衡
3. ==氧债==（oxygen debt = 需求氧耗量 - 实测氧耗量）增大，原因：组织供应、利用氧↓，能量生成↓

二、**电解质与酸碱平衡紊乱**
1. ==代谢性酸中毒==：乳酸↑，肝代谢乳酸↓，肾排泄乳酸↓ → 高乳酸血症（心肌、血管）
2. ==呼吸性碱中毒==：早期诊断指标之一，休克病因导致过度通气（PaCO2↓）。后期因通换 气障碍，可出现呼吸性酸中毒，导致混合性酸碱失衡
3. ==高钾血症==：Na+-K+-ATP酶/钠泵障碍、H+-K+交换↑，影响心肌兴奋、自律、传导和收缩性

三、**器官功能障碍**：多器官功能障碍综合症（multiple organ dysfunction syndrome, **MODS**）
# 4 四种常见休克的特点

### （一）脓毒性休克 septic shock

- **脓毒症（sepsis）**：感染引起的SIRS，为宿主的抗感染反应失调，占MODS的70%
	- 脓毒症休克：脓毒症的特殊亚型，指伴有严重的循环、细胞功能代谢异常的脓毒症，表现为在充分液体复苏的情况下仍需要缩血管药物才能维持平均动脉压在65 mmHg以上，其血清 乳酸高于2 mmol/L，临床最常见（62%），死亡率高（60%）
		- *内毒素性休克（endotoxic shock）*：最常见的脓毒性休克，G-菌严重感染，释放内毒素（即LPS）
- 脓毒症休克**发生机制**：SIRS→血容量减少（cap通透性↑）、血管扩张、心泵功能受损

|          | 高动力型休克 hyperdynamic shock，先                                                           | 低动力型休克 hypodynamic shock，后                                                       |
| -------- | ------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| 定义       | 病原体或其毒素侵入机体引起**高代谢和高动力循环状态**，出现发热、心排血量增加、 外周阻力降低、脉压增大等，又称为**高排低阻型休克或暖休克（warm shock）** | 心排血量减少、外周阻力增高、脉压显著缩小等，又称**低排高阻型休克或称冷休克 （cold shock）**，表现为皮肤苍白、四肢湿冷、尿量减少、血压下降、酸中毒 |
| 机制       | β受体效应（心肌收缩力↑；动-静脉短路开放→回心血量↑→心排出量↑）<br>外周阻力下降（NO、PGE2、PGI2、BK、TNF-α、IL-1等扩血管物质增多）      | 毒素、酸中毒抑制心泵功能，交感-肾上腺髓质系统兴奋释放缩血管物质                                                 |
| 弊端       | 动-静脉短路开放，真毛细血管物质交换少，组织缺血缺氧                                                            |                                                                                  |
| **心输出量** | 高                                                                                     | 低                                                                                |
| **外周阻力** | 低                                                                                     | 高                                                                                |

### （二）心源性休克 cardiogenic shock

- 因心泵功能障碍，心排血量迅速↓，血压在休克早期即下降，死亡率80%
1. **低排高阻型**：多见；表现为外周阻力增高，与血压下降、减压反射抑制→交感-肾 上腺髓质系统兴奋、外周小动脉收缩
2. **低排低阻型**：少见；表现为外周阻力降低，心室充盈异常↑→牵张感受器兴奋→交 感-肾上腺髓质系统抑制。亦见于其他类型休克的晚期阶段，为休克失代偿表现。

### （三）过敏性休克 anaphylactic shock

- 过敏原首次暴露 → IgE持久吸附于肥大细胞、嗜碱性粒细胞、血小板，致敏；
- 过敏原再次暴露 → 过敏原-IgE复合物引起脱颗粒反应，释放组胺、激肽、补体等活性物质
- 过敏性休克又称变应性休克，属**I型变态反应即速发型超敏反应**，常伴有 皮肤（荨麻疹）、呼吸道和消化道的过敏症状，发病急骤
- **机制**
	1. 活性物质扩血管，血管床容量增大，血容量相对减少
	2. cap通透性增高，血浆外渗，血容量绝对减少
### （四）神经源性休克 neurogenic shock

- 根本机制：**失去神经支配**，血管张力丧失
- 常见情况：脑损伤、脑缺血、*深度麻醉，脊髓高位麻醉或脊髓损伤*，交感神经传出通路被阻断
- 过程与后果：小血管张力丧失、麻痹性扩张，大量血液淤滞于血管床，回心血量减少，血压下降

### 典型：失血性休克 hemorrhagic shock

- 失血是否引起休克，取决于失血的量和速度
- 15-20分钟内失血少于总血量的10-15%，机体可经代偿维持血压；15分钟内快速、大量失血超过总血量的20%（~1000 ml），超出机体代偿 能力，引起失血性休克；失血量超过总血量的45-50%，会很快导致死亡
- 分期明显、临床症状典型，易并发急性肾衰（休克肾）和肠源性内毒素血症

# **小结** 

1. 各种病因通过血容量减少、血管床容量增加或心泵功能障碍三个环节导致休克发生 

2. 有效循环血量急剧减少是各型休克共同的发病学基础 

3. 休克可分为微循环缺血期、微循环淤血期和微循环衰竭期。在微循环缺血期，交感神经兴奋和缩血管物质释放导致微血管强烈收缩，同时促使回心血量增加和血流发生重 新分布。在微循环淤血期，由于局部扩血管物质产生增多和白细胞黏附于微静脉而导 致微循环血流淤滞。微循环衰竭期表现为微血管麻痹性扩张、DIC形成或多器官功能障碍 

4. 休克病因也可直接作用于组织细胞，导致细胞内信号转导通路活化与炎症因子产生， 微血管内皮细胞活化及组织细胞损伤
