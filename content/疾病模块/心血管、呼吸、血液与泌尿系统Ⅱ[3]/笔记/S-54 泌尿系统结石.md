> [!NOTE] 学习要点
> 1. 流行病学与病因：代谢异常、局部病因、药物因素
> 2. 防治总则：大量饮水、饮食调节、药物预防
> 3. 肾结石：肾绞痛、血尿、影像学检查、ESWL/PCNL
> 4. 输尿管结石：输尿管绞痛、放射痛、URL治疗
> 5. 膀胱结石：排尿中断、膀胱镜取石
> 6. 尿道结石：排尿困难、尿潴留、取石方法
# 概述
- 泌尿系统结石/尿石症，*最常见的泌尿外科疾病之一*
- $上尿路结石$：肾结石、输尿管结石；男女比例相近
- $下尿路结石$：膀胱结石、尿道结石；男性明显多于女性
##### 【病因】
1. **代谢异常**
	1. 形成尿结石的物质排出增加（草酸、钙、尿酸、胱氨酸）
	2. 尿PH改变（碱性尿-磷酸镁铵及磷酸盐沉淀；酸性尿-尿酸和胱氨酸结晶）
	3. 尿中抑制晶体形成和聚集的物质减少（枸橼酸、焦磷酸盐、酸性粘多糖、镁等）
	4. 尿量减少（盐类和有机物质的浓度增高）
2. **局部病因**（尿路梗阻、感染、尿路存在异物）
3. **药物相关因素**
	1. 尿液的浓度高而溶解度低（氨苯蝶啶、磺胺类等）
	2. 诱发结石形成的药物（乙酰唑胺、VitD、VitC、皮质激素）
##### 【成分及特性】
常见度：草酸钙、磷酸盐、尿酸盐、碳酸盐、胱氨酸结石

| 成分       | 特征                                                      |
| -------- | ------------------------------------------------------- |
| 草酸钙      | 质硬，不易碎，粗糙，不规则， 呈桑葚样，棕褐色，尿路平片易显影                         |
| 磷酸钙、磷酸镁铵 | 与尿路感染 和梗阻有关，易碎，表面粗糙，不规则，常呈鹿角形，灰白色、黄色或棕色， 尿路平片可见*分层现象*   |
| 尿酸       | 与尿酸代谢异常有关，其质硬，光滑， 多呈颗粒状，黄色或红棕色，*纯尿酸结石在尿路平片上不显影*，属于阴性 结石 |
| 胱氨酸      | 罕见的家族性遗传性疾病所致，质坚，光滑，呈蜡样，淡黄至黄棕色，尿路平片上*亦不显影*              |
##### 【病理生理】
- $泌尿系统$结石在*肾和膀胱内*形成，绝大多数输尿管结石和尿道结石是结石排出过程中停留在该处所致。
	- 输尿管有**三个生理狭窄**：
		1. 肾盂输尿管连接处
		2. 输尿管跨过髂血管处
		3. 输尿管膀胱壁段
	- 以输尿管下 1/3 处最多见
- $肾结石$常先发生在*肾盏*，增大后向*肾盂*延伸。
	- 结石使<u>肾盏颈部梗阻</u>，会引起肾盏积液或积脓，进一步导致肾实质萎缩、瘢痕形成，甚至发展为肾周围感染。
	- 肾盏结石进入肾盂或输尿管后，结石可自然排出，或停留在尿路的任何部位。
	- 一旦结石堵塞肾盂输尿管连接处或输尿管，可引起急性完全性尿路梗阻或慢性不完全性尿路梗阻。
	- 前者在及时解除梗阻后，不影响肾功能；后者常导致渐进性肾积水，使肾实质受损、肾功能不全。
	- 结石在肾盏内慢慢长大，充满肾盂及部分 或全部肾盏，形成鹿角形结石。
	- 结石可合并感染，亦可无任何症状，少数继发恶性变

# 上尿路结石
##### 【临床表现】
主要是==疼痛和血尿==
1. **疼痛**
	- $肾结石$可引起*肾区疼痛伴肋脊角叩击痛*。肾盂内大结石及肾盏结石可无明显临床症状，偶有活动后上腹或腰部钝痛。
	- $输尿管结石$可引起*肾绞痛或输尿管绞痛*，典型的表现为疼痛剧烈难忍，阵发性发作，位于腰部或上腹部，并沿输尿管行径*放射至同侧腹股沟*，还可放射到同侧睾丸或阴唇。
		- 结石处于输尿管膀胱壁段，可伴有膀胱刺激症状及尿道和阴茎头部放射痛。
		- 肾绞痛常见于结石活动并引起输尿管梗阻的情况
2. **血尿**
	- 通常为镜下血尿，少数病人可见肉眼血尿。
	- *活动后出现镜下血尿*是部分上尿路结石病人的唯一临床表现。
	- 血尿的多少与结石对尿路黏膜损伤程度有关。
	- 如果结石引起尿路<u>完全性梗阻或固定不动</u>（如肾盏小结石），则可能没有血尿。
3. **恶心、呕吐**
	- 常与肾绞痛伴发。
	- 输尿管结石引起尿路梗阻时，使输尿管管腔内压力增高，<u>管壁局部扩张、痉挛和缺血</u>。
	- 由于*输尿管与肠有共同的神经支配*而导致恶心、呕吐。
4. **膀胱刺激症状**（伴感染或输尿管*壁段结石*时）
##### 【并发症及其表现】
- 畏寒、发热、寒战等全身症状：并发急性肾盂肾炎或肾积脓
- 肾积水：严重时可在上腹部扪及增大的肾
- 无尿->尿毒症：双侧上尿路结石引起双侧尿路完全性梗阻或孤立肾上尿路完全性梗阻
- *小儿上尿路结石以尿路感染为最重要的表现*
##### 【诊断】
1. 病史和体检
2. 实验室检查
	1. 血液分析
	2. 尿液分析
	3. 结石成分分析（物理分析法-*红外光谱法*）
3. 影像学检查
	1. 超声【首选】
	2. X线检查
	3. 磁共振尿路成像MRU（结石梗阻后肾输尿管积水的情况）
	4. 放射性核素肾显像
	5. 内镜检查
##### 【治疗】
1. 病因治疗
2. 药物治疗
	- 结石直径≤6cm、表面光滑、结石以下尿路无梗阻-$药物排石治疗$
	- 纯尿酸结石/胱氨酸结石-$药物溶石治疗$
		- 尿酸结石：枸橼酸氢钾钠、碳酸氢钠*碱化尿液*，口服[^1]别嘌醇及饮食调节等方法治疗
		- 胱氨酸结石：碱化尿液、摄入大量液体
	- α- 巯丙酰甘氨酸（α-MPG）和乙酰半胱氨酸有溶石作用
	- 卡托普利有预防胱氨酸结石形成的作用
	- 感染性结石需控制感染，*口服氯化铵酸化尿液*
	- 应用脲酶抑制剂，有控制结石长大作用
	- 限制食物中磷酸的摄入，应用氢氧化铝凝胶限制肠道对磷酸的吸收，有预防作用
	- **肾绞痛**：解痉镇痛；NSAIDs、阿片类；M型胆碱受体拮抗剂、CCB、黄体酮-解痉
3. $体外冲击波碎石ESWL$
	- 通过 X 线平片或超声对结石进行定位，利用冲击波聚焦后作用于结石，使结石裂解，直至粉碎成细砂，随尿液排出体外
	- *适应证*：直径≤2cm的肾结石几输尿管上段结石
	- *并发症*
		- 一过性肉眼血尿
		- 碎石排出过程中-可出现肾绞痛
		- 碎石过多积聚于输尿管内-$石街$-腰痛不适、有时可继发感染
4. $经皮肾镜取石术PCNL$
	- 在超声或 X 线定位下，经腰背部细针穿刺直达肾盏或肾盂，扩张并建立皮肤至肾内的通道，在肾镜下取石或碎石
	- 适用于大部分需手术干预的肾结石
	- 凝血功能障碍、过于肥胖导致穿刺针不能到达肾，或脊柱畸 形者不宜采用此法
	- *术中和术后出血*是 PCNL 最常见及最危险的并发症
5. $输尿管镜碎石取石术URL$
	- 经尿道置入输尿管镜，在膀胱内找到输尿管口，在安全导丝引导下进入输尿管，利用能量源将结石击碎并用工具将结石取出
	- 适用于中、下段输尿管结石，ESWL 失败的输尿管上段结石，X 线阴性的输尿管结石，停留时间长的嵌顿性结 石，亦用于 ESWL 治疗所致的“石街“
	- *输尿管撕脱或断裂*是术中严重并发症
	- *感染性休克*也是手术严重并发症
	- $输尿管软镜$主要用于肾结石（直径＜2cm）的治疗
6. $腹腔镜输尿管切开取石LUL$
	- 直径≥2cm的输尿管结石
	- 经腹腔、经腹膜后两种入路
7. 开放手术治疗（不再用）
##### 【预防】
1. 大量饮水（增加尿量、稀释尿中形成结石物质的浓度、减少晶体沉积）
2. 调节饮食
3. 特殊性预防
	1. 草酸盐结石：口服维生素B6以减少草酸盐排出
	2. 尿酸结石：口服别嘌醇和碳酸氢钠抑制结石形成
# 下尿路结石
>- 原发性膀胱结石多发于男孩
>- 尿道结石见于男性，绝大多数来自肾和膀胱
>- 多数尿道结石位于*前尿道*
##### 【临床表现】
- $膀胱结石$：*排尿突然中断*，疼痛放射至远端尿道及阴茎头部，伴排尿困难和膀胱刺激症状。
- $尿道结石$：排尿困难，*点滴状排尿，伴尿痛*，重者可发生急性尿潴留及会阴部剧痛。
- 除典型症状外，下尿路结石常伴发*血尿和感染*。
- 憩室内结石可仅表现为尿路感染
##### 【诊断】
- 典型症状+影像学检查诊断
- 前尿道结石可沿尿道扪及；后尿道结石经直肠指诊可触及
【辅助检查】超声检查；X线平片；平扫CT；膀胱尿道镜检查
##### 【治疗】
1. 经尿道膀胱镜取石或碎石（直径≤2cm）
2. $耻骨上膀胱切开取石术$（不再首选）

> 2. ==不显影的结石==：尿酸结石、胱氨酸结石
> 3. 最常见的结石类型：==草酸钙类==
> 4. 诊断尿路结石首选的X线检查：==腹部平片+静脉尿路造影==
> 5. 感染性结石的性质是：磷酸盐结石（细菌产脲酶->分解尿素->尿液强碱性->析出磷酸盐结石）
> 	- 草酸盐结石：内源性草酸合成增加等->高草酸血症
> 6. 静息状态下无明显临床症状的结石：==肾盂和肾盏结石==

> 8. 药物排石：直径≤0.6cm
> 9. ==0.6cm≤直径≤2cm==的结石，行体外冲击波碎石（ESWL）；==＞2cm==的结石，行经皮肾镜碎石取石术（PCNL）。
> 10. ==输尿管狭窄==不适用于体外碎石/输尿管镜碎石
> 11. ==肾结石==不是前列腺增生的并发症
> 12. 膀胱结石的典型表现是==排尿突然中断==
