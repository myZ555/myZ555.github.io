> - 主要临床表现为==出血==和==尿外渗==
> 	- 出血->血肿、血尿甚至休克
> 	- 尿外渗->继发感染；严重时导致脓毒症、周围脓肿、尿瘘或尿道狭窄
# 肾损伤
##### 【病因】
- 开放性损伤/穿透性损伤：锐器；有创口与外界相通
- 闭合性损伤/钝性损伤：多见；直接暴力或间接暴力、无创口
- 医源性损伤
- ==“自发性”破裂==：常合并肾积水、肾肿瘤等病理基础。 
##### 【病理】
[^1]

| 分级  | 类型   | 描述                                        |
| --- | ---- | ----------------------------------------- |
| I   | 挫伤   | 镜下/肉眼血尿，泌尿系统检查正常                          |
|     | 血肿   | 包膜下血肿，无实质损伤                               |
| II  | 血肿   | 局限于腹膜后肾区的肾周血肿                             |
|     | 裂伤   | 肾实质裂伤深度≤1cm，无尿外渗                          |
| III | 裂伤   | 肾实质裂伤深度≥1cm，无集合系统破裂或尿外渗<br>（如双侧肾损伤、应为IV级） |
| IV  | 裂伤   | 肾实质裂伤贯穿皮质、髓质和集合系统                         |
|     | 血管损伤 | 肾动脉或静脉主要分支损伤合并出血                          |
| V   | 裂伤   | 完全破裂的肾                                    |
|     | 血管损伤 | 肾门血管破裂、离断伴肾无血供                            |

|        |                                                                                                                                             |
| ------ | ------------------------------------------------------------------------------------------------------------------------------------------- |
| 肾挫伤    | 损伤仅局限于部分肾实质，形成肾瘀斑和(或)包膜下血肿，==肾包膜及肾盏肾盂黏膜完整==。损伤涉及肾集合系统可有少量血尿                                                                                 |
| 肾部分裂伤  | 肾近包膜部位裂伤==伴有肾包膜破裂==，可致肾周血肿。 <br>若肾近集合系统部位裂伤==伴有肾盏肾盂黏膜破裂==，则可有明显血尿。 <br>裂伤：<u>肾实质裂伤、深度小于1.0cm</u>，无尿外渗<u>或肾实质裂伤、深度 大于1.0cm ，无集合系统破裂或尿外渗</u>。 |
| 肾全层裂伤  | 肾实质深度裂伤，==外及肾包膜，内达肾盏肾盂黏膜==，此时常引起广泛 的肾周血肿、血尿和尿外渗。<br><u>肾实质裂伤深度往往超过1.0cm。 </u>                                                               |
| 肾蒂血管损伤 | 肾蒂或肾段血管的部分或全部撕裂，肾动脉内膜断裂及血栓形成，可 引起大出血、休克                                                                                                     |

【晚期病理改变】
- 持久的尿外渗可导致*尿囊肿*的形成
- 血肿、尿外渗引起*组织纤维化*，压迫肾盂输尿管交界处可导致*肾积水*
- 开放性肾损伤偶可发生*动静脉瘘*或*假性肾动脉瘤*
- 部分肾实质缺血或肾蒂周围纤维化压迫肾动脉，可引起*肾性高血压*
##### 【临床表现】
1. 休克（严重肾裂伤/肾蒂血管破裂/合并其他脏器损伤；失血->休克）
2. 血尿
	1) 肾挫伤：镜下血尿、轻度肉眼血尿
	2) 肾近集合系统部位裂伤伴有肾盏或肾盂黏膜破裂：明显血尿
	3) 肾全层裂伤：大量全程肉眼血尿
	- *血尿程度与肾损伤的严重程度有时并不一致*
3. 疼痛
	- 肾包膜下血肿、肾周围软组织损伤、出血或尿外渗：*病侧腰腹部疼痛*
	- 血液、尿液进入腹腔或合并腹内脏器损伤：*全腹疼痛和腹膜刺激症状*
	- 血块通过输尿管：*肾绞痛*
4. 腰腹部肿块（明显触痛和肌肉强直）
5. 发热（血肿吸收->发热；肾损伤->肾周血肿、尿外渗->继发感染/肾周脓肿/化脓性腹膜炎）
##### 【诊断】
1. 病史与查体（任何腹部、背部、下胸部创伤或受对冲力创伤的病人）
2. 实验室检查（血红细胞和血细胞比容持续下降提示有活动性出血）
3. 影像学检查[^2]
	1) 超声：肾损伤的部位和程度等；注意肾蒂血管情况
	2) CT：平扫及增强可显示肾实质裂伤程度、尿外渗和血肿范围【==首选==】
		- *对比剂增强CT* 是肾损伤中泌尿生殖系统成像的最佳方法
		- *CT尿路成像 CTU* 可评价肾损伤的范围和程度
		- *CT血管成像 CTA* 可显示肾动脉和肾实质损伤的情况
##### 【治疗】
==非手术：血流动力学稳定且创伤分期良好的I～III级肾损伤的标准治疗==
==IV、V级损伤：常需要手术探查==

1. 急诊处理
2. 保守治疗（绝对卧床休息2-4周；病情稳定、血尿消失后才允许离床活动）
3. 手术治疗
	- *绝对适应证*：①血流动力学不稳定，如发生休克；②扩大 / 搏动性肾血肿（通常提示肾动脉裂伤）； ③疑似肾血管蒂撕脱；④肾盂输尿管连接部破裂。
	- *相对适应证*：①尿外渗伴有明显的肾实质血供阻断；②肾损伤合并结肠 / 胰腺损伤；③动脉血栓形 成；④实质损伤引起的尿外渗
	- **手术方法**：经腹或经腰部切口施行手术
		- 怀疑腹腔脏器损伤时，<u>先探查并处理腹腔其他损伤脏器</u>， 再切开后腹膜，*显露并阻断肾蒂血管*，而后切开肾周筋膜和脂肪囊[^3]，探查伤侧肾，快速清除血肿，依具体情况选择作肾修补、肾部分切除或肾切除。
		- 只有在*严重肾全层裂伤或肾蒂血管损伤*，无法修复而危及生命时，才可考虑施行病侧肾切除。
		- 肾动脉造影和选择性血管栓塞是肾损伤中越来越常用的诊疗方式
		- 创伤性假性动脉瘤和动静脉瘘可通过血管造影及栓塞进行治疗
4. 并发症处理
	1. *腹膜后尿囊肿或肾周脓肿*需穿刺引流或切开引流
	2. *输尿管狭窄、肾积水*需施行成形术或肾切除术
	3. *恶性高血压*可作血管狭窄处扩张或肾切除术
	4. *持久性血尿且较严重者*可施行选择性肾动脉分支栓塞术
# 输尿管损伤
> - 输尿管位于<u>腹膜后间隙</u>，周围组织对其有良好的保护
> - 外界暴力所致的输尿管损伤少见，*多为医源性损伤*
> - 易被忽视漏诊、出现症状时发现，常延误治疗
##### 【病因】
1. 医源性损伤
	1. 输尿管腔内器械操作：经膀胱镜逆行输尿管插管等
	2. 输尿管腔外手术操作：常在盆腔、腹膜后的开放及腹腔镜手术
2. 外源性损伤（罕见）
3. 放射性损伤（放疗后）
##### 【病理】
- 输尿管被切断或管壁裂伤后可出现*腹膜后尿外渗或腹膜炎*；感染后有脓毒症的危险。
- 输尿管被结扎可致该侧*肾积水*，甚至肾萎缩。
- 输尿管被钳夹、外膜广泛剥离或被缝在阴道残端时，损伤部位输尿管可发生*缺血性坏死*，一般在1-2周内形成*尿外渗或尿瘘*，伴输尿管狭窄者可致*病侧肾积水*
##### 【临床表现】
1. 血尿（常见于输尿管黏膜损伤；==程度并不与输尿管损伤程度一致==）
2. 尿外渗（输尿管损伤处渗入腹膜后间隙甚至腹腔；引起腰痛、腹痛、腹胀、局部肿胀、肿块及触痛、感染）
3. 尿瘘
4. 梗阻症状（肾盂压力增高->病侧腰痛、腰肌紧张、肾区叩痛及发热；孤立肾/双侧输尿管的完全性梗阻->无尿）
##### 【诊断】
注意检查输尿管行径、手术野有无渗尿，输尿管有无损伤
1. *静脉注射靛胭脂检查*：如有裂口则可见蓝色尿液从损伤处流出
2. *静脉尿路造影*：可显示输尿管损伤处的尿外渗、尿漏或梗阻
3. *逆行肾盂造影*
4. 超声……
##### 【鉴别诊断】
1. $输尿管阴道瘘、膀胱阴道瘘$（经导尿管注亚甲蓝溶液至膀胱。膀胱-阴道内有蓝色液体流出；输尿管-流出液仍为澄清）
2. 结扎双侧输尿管引起无尿与$急性肾小管坏死$鉴别，需要作膀胱镜检 查及双侧输尿管插管,以明确有无梗阻存在
##### 【治疗】
1. 早期治疗：尽早处理；不能耐受手术->病侧肾穿刺造瘘以保护肾功能；尿外渗彻底引流
	1) 输尿管逆行插管所致的黏膜损伤：常不作特殊处理，或留置支架管1-2周。 
	2) 输尿管钳夹伤或轻度裂伤：宜从输尿管切口置入双J形输尿管支架引流管， *留置2周*后拔除。 
	3) 输尿管被误扎：术中发现误扎，应立即松解，如该处缺血坏死，则需切除缺 血段，作端端吻合，并留置双J形输尿管支架引流管*3～4周*。 
	4) 输尿管断离、部分缺损：若输尿管断离部位较高，两断端对合后无张力者可 施行端端吻合术。下1/3段损伤，部分缺损宜作输尿管膀胱吻合术或膀胱壁瓣 输尿管下段成形术。若输尿管缺损过多，可选做舌黏膜修补术、阑尾 / 回肠 代输尿管术
2. 晚期并发症治疗
	1) 输尿管狭窄：可试行输尿管插管、扩张或留置双J形输尿管支架引流管，或 进行输尿管周围粘连松解术或狭窄段切除端端吻合术。 
	2) 尿瘘：输尿管皮肤瘘或输尿管阴道瘘多应进行输尿管修复，或与膀胱吻合。 
	3) 输尿管完全梗阻：可先行肾造瘘术，3个月后再行输尿管修复。 
	4) 肾功能重度损害或丧失：ECT明确患肾功能重度损坏，如果对侧肾功能正常， 可行患肾切除术（谨慎！）
# 膀胱损伤
> - ==第二常见==的泌尿系统损伤
> - 膀胱空虚时位于骨盆深处，除贯通伤或==骨盆骨折==外，一般不易损伤
> - *膀胱充盈时其壁紧张而薄，易受损伤*
##### 【病因】
1. 开放性损伤（枪弹或锐器贯通所致；合并脏器损伤；形成瘘管）
2. 闭合性损伤（膀胱充盈时……）
3. 医源性损伤
##### 【病理】
1. 挫伤（仅伤及膀胱黏膜或浅肌层；膀胱壁未穿破；*无尿外渗、可发生血尿*）
2. 膀胱破裂
	1) 腹膜外型：单纯膀胱壁破裂、腹膜完整；*尿液极易外渗入膀胱周围组织及耻骨后间隙，沿骨盆筋膜到盆底，或沿输尿管周围疏松组织蔓延到肾区*。大多由膀胱前壁破裂引起，常伴骨盆骨折
	2) 腹膜内型：膀胱壁破裂伴有<u>腹膜破裂</u>，裂口与腹腔相通，*尿液流入腹腔，可引起腹膜炎*。多见于膀胱后壁和顶部损伤
##### 【临床表现】
1. ==血尿和排尿困难==（膀胱壁轻度挫伤->少量终末血尿；损伤较重->严重全程血尿）
2. 腹痛
	- 腹膜外破裂->尿外渗及血肿可引起下腹痛、压痛及肌紧张；直肠指诊-可触及直肠前壁饱满并有触痛
	- 腹膜内破裂->尿液流入腹腔常引起急性腹膜炎症状
3. 尿瘘（开放性-体表伤口漏尿；肛门/阴道漏尿；闭合性-尿瘘）
4. 局部症状（体表皮肤肿胀、血肿和瘀斑）
##### 【诊断】
1. 病史和查体
	1) 腹痛、血尿及排尿困难，耻骨上区压痛，直肠指诊触及直肠前壁有饱满感->腹膜外膀胱破裂
	2) 全腹剧痛，肌紧张，有压痛及反跳痛，并有移动性浊音->腹膜内膀胱破裂
2. $导尿试验$
	- <u>如引流出 300ml 以上的清亮尿液</u>，基本可排除膀胱破裂；
	- 如无尿液导出或仅导出少量血尿，则膀胱破裂的可能性大
		- 此时可经导尿管向膀胱内注入<u>灭菌生理盐水 200～300ml</u>，片刻后再引出：
		- 液体外漏时引出量会减少，腹腔液体回流时引出量会增多
		- *若液体出入量差异大，提示膀胱破裂*
3. $膀胱造影$
	- 自导尿管向膀胱内注入 *15% 泛影葡胺 300ml*，摄前后位片，抽出对比剂后再摄片
	- 如膀胱破裂，可发现对比剂漏至膀胱外，排液后的图像更能显示遗留于膀胱外的对比剂。
	- 腹膜内膀胱破裂时，则显示对比剂衬托的肠袢。
4. 逆行尿道造影
##### 【治疗】
1. 紧急处理：抗休克治疗、尽早合理使用抗生素预防感染
2. 保守治疗
	- 大多数*腹膜外膀胱损伤*可通过*持续引流*的非手术方式进行治疗
	- 从尿道插入导尿管持续引流尿液10 天左右+抗生素预防感染，多可自愈
3. 手术治疗
	- 腹膜外：下腹正中切口、腹膜外露并切开膀胱，清除外渗尿液，修补裂口
	- 腹膜内：行剖腹探查、其他脏器是否损伤
	- *膀胱修补术后应留置 Foley 导尿管或耻骨上膀胱造瘘，持续引流尿液 2 周*
4. 并发症（主要与感染有关；盆腔血肿应尽量避免切开；出血难以控制可行选择性盆腔血管栓塞术）
# 尿道损伤
> - 泌尿系统最常见的损伤
> - 多见于男性
> - 男性尿道以*尿生殖膈*为界，前尿道包括阴茎部和球部、后尿道包括膜部和前列腺部
> - ==球部和膜部的损伤最为多见==
> - 前尿道：骑跨伤、球部损伤
> - 后尿道：骨盆骨折、膜部损伤

> [!note] 会阴、阴茎的筋膜
> - 会阴：
> 	- 浅筋膜：浅层为Scarpa筋膜，包括前腹壁浅筋膜及阴茎浅筋膜 （Colles筋膜）；深层与前腹壁筋膜相延续。 
> 	- 深筋膜：浅层为尿生殖膈下筋膜，深层为尿生殖膈上筋膜
> - 阴茎：
> 	- 浅层：Colles筋膜
> 	- 深层：Buck筋膜
### 前尿道损伤（阴茎部、球部）
- 前尿道损伤多发生于==球部==；==**会阴部骑跨伤==时**，将尿道挤向耻骨联合下方、引起尿道球部损伤。
##### 【病理】
- ==挫伤==：仅有局部水肿和出血、愈合后一般不发生尿道狭窄
- ==裂伤==：尚有部分尿道壁完整，愈合后常有瘢痕性尿道狭窄
- ==断裂==：伤处完全离断、断端退缩分离

球部裂伤/断裂：血液及尿液渗入==会阴浅筋膜包绕==的会阴浅袋、可向上扩展至腹壁
- 会阴浅筋膜远侧附着于腹股沟部，近侧与腹壁浅筋膜深层相连，后方附着于尿生殖膈，<u>尿液不会外渗到两侧股部</u>
阴茎部损伤：若筋膜完整，外渗局限于筋膜内，表现为阴茎肿胀；若筋膜破裂，则与尿道球部损伤相同

##### 【临床表现】
1. *尿道出血*（==鲜血自尿道外口滴溢==，前尿道损伤最常见症状）
2. 疼痛（局部疼痛压痛，常见==排尿痛，向阴茎头及会阴部放射==）
3. 局部血肿（骑跨伤->会阴部、阴囊处肿胀、瘀斑及蝶形血肿）
4. 排尿困难
5. 尿外渗（感染后可形成尿道周围脓肿、脓肿破溃可形成尿瘘）
##### 【诊断】
1. 病史和查体
2. $诊断性导尿$：了解尿道完整性和连续性
3. $逆行尿道造影$：显示尿道损伤部位及程度
	- 对比剂外溢：部分裂伤
	- 对比剂未进入后尿道而大量外溢：前尿道有严重裂伤或断裂
##### 【治疗】
1. 紧急处理
	- 尿道*球部海绵体*严重出血可致休克、立即压迫会阴部止血+抗休克治疗
	- 尿潴留插入导尿管失败：行耻骨上膀胱造瘘术引流尿液
2. 尿道挫伤（尿道连续性存在：不需特殊治疗；留置导尿管1周）
3. 尿道裂伤（导尿顺利：留置引流2周；插入失败：立即行经会阴尿道修补术）
4. 尿道断裂（球部远端和阴茎部完全断裂：及时会阴切口探查+尿道端端吻合术）
5. 并发症处理
	1. $尿外渗$：尽早在外渗部位多处切开皮肤、深达浅筋膜以下，置多孔引流管引流。同时作*耻骨上膀胱造瘘*，3个月后修补尿道
	2. $尿道狭窄$：常见晚期并发症
	3. $尿瘘$：前尿道狭窄所致尿瘘多见于会阴部/阴囊部；解除狭窄+切除/清理瘘管
### 后尿道损伤（膜部、前列腺部）
- 膜部尿道穿过尿生殖膈，*骨盆骨折时*，附着于耻骨下支的尿生殖膈突然移位产生剪切暴力，使膜部尿道撕裂，甚至在前列腺尖处撕断。
- 耻骨前列腺韧带撕裂致前列腺向上后方移位。
- 骨折及盆腔血管丛损伤可引起大出血，在前列腺和膀胱周围形成大的血肿。
- 后尿道断裂后，尿液沿前列腺尖处可外渗到**耻骨后间隙**和**膀胱周围**
##### 【临床表现】
1. 休克（骨盆骨折后所致后尿道损伤通常严重、合并大出血）
2. 疼痛（下腹痛、局部肌紧张、伴压痛）
3. 排尿困难（尿道连续性中断/血块堵塞->==排尿困难和尿潴留==）
4. ==尿道出血==（尿道外口无或仅有少量血液）
5. 尿外渗及血肿（后尿道损伤时尿外渗一般进入耻骨后间隙和膀胱周围，但是当尿生殖膈撕裂时，会阴、阴囊部会出现血肿及尿外渗）
##### 【诊断】
1. 病史和查体（骨盆挤压伤出现尿潴留；直肠指诊）*直肠指检可触及直肠前方有柔软的血 肿并有压痛，前列腺尖端可浮动*
2. X线平片检查/造影（骨盆前后位片--骨盆骨折；逆行尿道造影联合经耻骨上膀胱造瘘管的膀胱造影）
##### 【治疗】
1. 紧急处理（平卧；抗休克治疗）
2. 早期处理
	1. 导尿（损伤较重者不宜导尿、以免加重局部损伤及血肿感染）
	2. $膀胱造瘘$
		- 尿潴留者可行局麻下耻骨上高位膀胱穿刺造瘘
		- 经膀胱尿道造影明确尿道无狭窄及尿外渗后才可拔出造瘘管
	3. $尿道会师复位术$（早期恢复尿道连续性、避免尿道断端远离形成瘢痕假道）[^4]
3. 并发症处理
	- *常并发尿道狭窄*
	- 为预防狭窄，去除导尿管后*每周进行1次尿道扩张、持续1个月后仍需定期行尿道扩张术*

| ------ | ---------- | -------------- |

# 阴茎损伤
##### 【临床表现】
1. 阴茎挫伤（肿胀、皮下出血与血肿）
2. 阴茎折断（勃起时受直接暴力造成*白膜及海绵体破裂*，伴有响声、剧痛、勃起消退、血肿、皮肤青紫）
3. 阴茎绞窄
4. 阴茎脱位（从耻骨韧带撕脱而脱位于会阴部或股内侧）
5. 尿道损伤
##### 【诊断】
1. 病史和查体
	- 通常在阴茎勃起状态下，病人听见*清脆声响*，随之伴有疼痛，其后出现阴茎勃起障碍、肿胀、变色及阴茎偏位，可形成典型“**茄子畸形**”。
	- 若海绵体白膜破裂，血肿可延伸到阴囊、会阴和耻骨上区域。
	- 由于血肿影响，瘀斑性阴茎常移位到白膜撕裂对面，可触及白膜中的“骨折”线。
2. 超声检查（首选）
3. 磁共振成像
4. 阴茎海绵体造影
##### 【治疗】
1. 手术处理
	- 疑似阴茎折断应及时探查和修复，常选择*腹侧垂直切开*阴茎，充分显露折断位置，小的横行切口可用于清除局部血肿。
	- 当折断部位不明时，可作*远端包皮环行切口*，可以充分显露三个 阴茎区室。
2. 并发症处理
# 睾丸损伤
> 主要为钝性损伤
##### 【临床表现】
1. 创伤后睾丸剧痛，疼痛可放射至下腹、腰部或上腹部，阴囊皮肤有肿胀或血肿
2. 恶心、呕吐甚至疼痛性休克
3. 睾丸肿大或界限难以触清
4. 睾丸*扭转*时可触及精索增粗
5. 睾丸脱位者，自诉伤后原有睾丸的阴囊空虚、而在其他部位扪及类似睾丸的肿块
##### 【诊断】
1. 病史和查体
	- 主诉多为阴囊剧痛和恶心
	- 阴囊肿胀、瘀斑、血肿的程度与睾丸损伤的严重程度无相关性
2. 超声检查（名曲睾丸完整性和血运情况）【==睾丸实质组织回声不均及白膜不完整可提示睾丸破裂==】
3. MRI检查
##### 【治疗】
==清除失去活力的睾丸组织、闭合睾丸包膜（白膜）和修复阴囊==
1. 保守治疗
2. 手术处理
	- 目的：挽救睾丸、预防感染、控制出血及缩短恢复期
	- ==横向切开阴囊==
