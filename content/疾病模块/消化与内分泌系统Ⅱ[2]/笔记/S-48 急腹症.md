##### 【病因】
1. 空腔脏器病变
	1. 穿孔
	2. 梗阻
	3. 炎症
2. 实质性脏器病变
	1. 破裂出血
	2. 炎症
3. 血管病变
	1. 腹主动脉夹层
	2. 腹部脏器血管栓塞/血栓形成
##### 【临床诊断和分析】
###### 病史
1. 现病史
	1. 腹痛
		- 内脏痛/躯体痛/牵涉痛
			- 转移性腹痛-急性阑尾炎[^1]
			- 右肩/肩胛下角放射痛-胆绞痛、急性胆囊炎[^2]
			- 同侧下腹/腹股沟放射痛-肾或输尿管上段结石所致肾绞痛
			- 会阴部放射痛-输尿管下段结石
		- 空腔脏器穿孔起病急、感染性疾病起病缓慢
		- 性质
			【持续性钝痛/隐痛】炎症、出血
			【阵发性绞痛】空腔脏器梗阻初期
			【持续性疼痛阵发性加剧】炎症与梗阻并存或空腔脏器绞窄时
		- 程度
			- 炎症初期-不剧烈、隐痛、定位不确切
			- 空腔脏器穿孔-剧烈、腹膜刺激征强烈
			- 内脏破裂出血程度轻于消化道穿孔
	2. 消化道症状
		1) 厌食
		2) 恶心、呕吐
			- 反射性呕吐发生较早、可在腹痛发作后随即发生
			- 引起腹痛病变位置越高、发生呕吐早且频繁
			【呕吐宿食不含胆汁】幽门梗阻
			【含胆汁】梗阻位于十二指肠乳头开口以下
			【咖啡色】提示消化道出血
			【粪水】低位完全性肠梗阻
		3) 排便
			【排便排气停止】机械性肠梗阻
			【腹泻】胃肠炎；盆腔脓肿、阑尾炎
			【血便】肠绞窄坏死、肠套叠、急性出血坏死性肠炎
	3. 其他伴随症状
		- 急性梗阻型化脓性胆管炎：寒战、高热、黄疸[^3]
		- 泌尿系疾病：尿频、尿急、尿痛、血尿、排尿困难
		- 外科急腹症-腹痛->发热；若先发热-内科
2. 月经史（卵巢滤泡/黄体破裂多发生在2次月经之间）
3. 既往史（消化性溃疡/胆囊结石/腹部手术者）
###### 体格检查
1. 全身情况和体位
	【腹腔出血】面色苍白、贫血貌
	【腹膜炎】面容痛苦、体位屈曲、不敢伸展
	【脱水】眼眶凹陷、皮肤皱缩、弹性下降
	【胆道梗阻】巩膜皮肤黄染
2. 腹部检查
	1) $视$：肠梗阻-膨隆；PU穿孔-[^4]凹陷舟状腹；肠扭转-局部隆起伴肠型；腹股沟区肿块突出-[^5]嵌顿疝
	2) $触$
		- 典型腹膜炎体征：压痛、肌紧张、反跳痛
		- 压痛最明显部位通常就是病变部位
		- 肌紧张-反映腹腔炎症的程度
			- 高度-*板状腹*-胃十二指肠溃疡穿孔
	3) $叩$
		- 鼓音：气体、肠袢
		- 移动性浊音：腹水、腹腔积血
		- 消化道穿孔-<u>肝浊音界消失</u>
	4) $听$
		- 机械性肠梗阻初起时肠鸣音增加，音质高亢，常伴有气过水声
		- 伴有麻痹性肠梗阻、急性腹膜炎、低血钾时肠鸣音常减弱或消失
3. 直肠指诊
###### 辅助检查
1. 实验室检查
	- 尿液白细胞计数升高-泌尿系感染
	- 尿红细胞计数升高-泌尿系结石
	- 尿胆红素阳性-黄疸为梗阻性
	- 血、尿或腹腔穿刺液淀粉酶明显升高-胰腺炎诊断
	- 人绒毛膜促性腺激素（HCG）测定-异位妊娠
	- *降钙素原PCT检测*-评价急腹症和外科严重感染性疾病进程及预后
2. 影像学检查
	1) 超声：实质性脏器破裂、肿块以及结石
	2) X线平片/透视：立位平片-肠道液气平面和肠袢分布；卧位片-肠腔扩张程度
	3) CT、MRI
	4) 选择性动脉造影（腹部动脉栓塞性疾病）
3. 十二指肠镜检查（急性胆管炎）
4. 诊断性腹腔穿刺（左/右髂前上棘与脐连线中外1/3处）
5. 腹腔镜检查
##### 【常见急腹症的诊断与鉴别诊断】

| 急腹症         | 诊断/鉴别诊断                                                            |
| ----------- | ------------------------------------------------------------------ |
| 胃十二指肠溃疡急性穿孔 | ==突发剧烈腹痛、“板状腹”和膈下游离气体==<br>既往病史、突发上腹部*刀割样*疼痛、明显腹膜刺激征、肝浊音界消失        |
| 急性胆囊炎       | 油腻食物、右肩胛下角放射<br>右上腹压痛、反跳痛、肌紧张<br>肝区叩痛阳性、Murphy征阳性                  |
| 急性胆管炎       | 上腹痛伴寒战、高热、黄疸<br>细菌/毒素易突破肝窦入血-脓毒血症和感染性休克                            |
| 急性胰腺炎       | 左上腹、剧烈、持续性<br>血尿淀粉酶升高-定性诊断                                         |
| 急性阑尾炎       | ==转移性右下腹痛和右下腹固定压痛==                                                |
| 急性小肠梗阻      | ==腹痛、腹胀、肛门停止排气排便、呕吐==<br>高位-呕吐为主；低位-腹胀明显<br>初期：蠕动活跃、肠鸣音增强、气过水声<br> |
| 腹部闭合伤       | 实质-破裂出血<br>空腔-腹膜炎表现                                                |
| 急性盆腔炎       | 年轻人；下腹部疼痛、压痛、反跳痛<br>涂片镜检WBC内有革兰阴性双球菌                               |
| 卵巢肿瘤蒂扭转     | 卵巢囊肿病史、疼痛突然发作                                                      |
| 异位妊娠        | 输卵管妊娠破裂最常见<br>腹腔或后穹窿穿刺抽得不凝血确诊                                      |
##### 【急腹症处理原则】
1. 尽快明确诊断，针对病因采取相应措施。
2. 诊断尚未明确时，禁用吗啡等强效镇痛药，以免掩盖病情变化，延误诊断。 
3. 需进行手术探查或治疗者，必须根据病情进行相应的术前准备。 
4. 诊断虽不能明确，但有下列情况者原则上需行急诊手术探查[^6] 
5. 手术原则是救命放在首位，其次是根治疾病。手术方案力求简单有效，条件许可时，尽量一次性手术解决问题；病情危重者，宜遵循损伤控制原则，先解决危及生命的主要问题，待病人全身情况平 稳后再行根治性手术。急诊腹腔镜手术现已广泛应用于急腹症外科，具有创伤小、恢复快等优点，对 部分疑难复杂病人的诊疗尤有助益。
