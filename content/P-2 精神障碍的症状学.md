# 常见精神症状和体征
## 感知觉障碍
### 感觉障碍
$感觉（sensation）$：是大脑对客观刺激作用于感觉器官所产生的对事物个别属性的反映

| 类型        | 表现                                                          | 疾病                             |
| --------- | ----------------------------------------------------------- | ------------------------------ |
| ==感觉减退==  | 对刺激的感受性降低，感觉阈值增高，表现为对外界强烈的刺激产生轻微的感觉体验或完全不能感知                | 神经系统疾病<br>抑郁障碍、木僵状态、意识障碍和分离性障碍 |
| ==感觉过敏==  | 对刺激的感受性增高，感觉阈值降低，表现为对外界一般强度的刺激产生强烈的感觉体验                     | 神经系统疾病<br>分离性障碍、躯体不适障碍         |
| ==内感性不适== | 又称体感异常，是躯体内部产生的不舒适和难以忍受的异样感觉，如咽喉部堵塞感、胃肠扭转感、腹部气流上涌感等，可继发疑病观念 | 躯体不适障碍、精神分裂症和抑郁障碍              |

### 知觉障碍
- $知觉（perception）$：是在感觉基础上，大脑对事物的各种不同属性进行整合，并结合以往经验，形成的整体印象。
- $错觉（illusion）$：是对客观事物歪曲的知觉。病理性错觉常在意识障碍或其他精神障碍时出现，不容易被纠正，多表现为错视和错听，并常带有恐怖色彩。多见于谵妄状态、精神分裂症
- $幻觉（hallucination）$：是没有现实刺激作用于感觉器官时出现的知觉体验，是一种虚幻的知觉。

| 涉及器官     | 表现                                                                                        |
| -------- | ----------------------------------------------------------------------------------------- |
| ==幻听==   | 一种虚幻的听觉，即患者听到了并不存在的声音；<br>最常见的幻觉<br>评论性幻听、议论性幻听和命令性幻听是精神分裂症的典型症状                          |
| ==幻视==   | 患者看到了并不存在的事物<br>意识清晰状态下出现的幻视多见于精神分裂<br>意识障碍时的幻视多见于谵妄状态，内容常常形象、生动、鲜明，且多具有恐怖性质              |
| ==幻味==   | 患者尝到食物或水中并不存在的某种特殊的怪味道<br>常与被害妄想同时存在；多见于精神分裂症                                             |
| ==幻嗅==   | 患者闻到环境中并不存在的某种难闻的气味<br>幻嗅和幻味往往同时出现，并经常与被害妄想结合在一起，多见于精神分裂症<br>单一出现的幻嗅，多见于颞叶癫痫或颞叶器质性损害      |
| ==幻触==   | 在没有任何刺激时，患者感到皮肤上有某种异常的感觉                                                                  |
| ==本体幻觉== | 内脏幻觉：是患者身体内部某一部位或某一脏器虚幻的知觉体验；常伴疑病妄想出现、多见于精神分裂症和抑郁障碍<br>运动幻觉：指患者处于静止状态时自觉身体某个部位在动；多见于精神分裂症 |

| 真性幻觉                                         | 假性幻觉                                                       |
| -------------------------------------------- | ---------------------------------------------------------- |
| 是来自于外部客观空间，通过感觉器官而获得的幻觉。                     | 是存在于自己的主观空间内，不通过感觉器官而获得的幻觉。                                |
| 其特点为幻觉内容就像感知外界真实事物一样生动形象，故患者常常述说是亲耳听到或亲眼看到的。 | 其特点为幻觉内容往往比较模糊、不清晰和不完整，故患者常常描述为没有通过耳朵或眼睛，大脑内就隐约出现了某种声音或影像。 |
| 患者对幻觉内容深信不疑，并可做出相应的情感与行为反应                   | 虽然此类幻觉与一般知觉不同，但患者往往仍然比较肯定地相信幻觉内容。                          |

| 产生条件      | 表现                                                                        |
| --------- | ------------------------------------------------------------------------- |
| ==功能性幻觉== | 一种伴随现实刺激而出现的幻觉，即当某种感觉器官处于功能活动状态时出现涉及该器官的幻觉，正常知觉与幻觉并存。<br>常见功能性幻听、多见于精神分裂症 |
| ==反射性幻觉== | 一种伴随现实刺激而出现的幻觉，但涉及两个不同的感觉器官，即当某一感官处于功能活动状态时，出现涉及另一感官的幻觉。<br>多见于精神分裂症      |
| ==入睡前幻觉== | 出现在入睡前的幻觉，多为幻视。表现为患者闭上眼睛就能看见许多幻觉形象，如各种动物、风景或人体的某部分等，与睡梦时的体验相近             |
| ==心因性幻觉== | 强烈心理因素影响下出现的幻觉<br>多见于应激相关障碍、分离性障碍等                                        |


| 感知综合障碍       | 表现                                        |
| ------------ | ----------------------------------------- |
| ==视物变形症==    | 视物显大/显小症；多见于癫痫                            |
| ==自身感知综合障碍== | 自己身体的某一部分在大小形状等方面发生变化<br>可见于精神分裂症、癫痫      |
| ==时间感知综合障碍== | 对时间的快慢出现不正确的感知体验<br>可见于抑郁障碍、躁狂发作、精神分裂症等   |
| ==空间感知综合障碍== | 对周围事物的距离、空间位置等感知错误                        |
| ==现实解体==     | 患者感到周围事物和环境变得不真实、犹如隔着窗纱<br>可见于抑郁障碍、精神分裂症等 |
## 思维障碍
$思维$ 是人脑对客观事物间接概括的反映，它可以揭露事物内在的、本质的特征，是人类认识活动的最高形式。<u>正常特征：目的性、连贯性、逻辑性、实践性
</u>
### 思维形式障碍
- 形式障碍：思维过程的联想及逻辑障碍
- $思维联想障碍$：思维的速度、数量、表现形式或思维内容连接形式的异常
- $思维逻辑障碍$：逻辑推理过程的异常

| 类型      | 表现                                                                                         |
| ------- | ------------------------------------------------------------------------------------------ |
| 思维奔逸    | 思维联想速度加快、数量增多和转换加速<br>表现为特别健谈、主题极易改变<br>多见于躁狂发作                                            |
| 思维迟缓    | 思维联想速度减慢、数量减少和转换困难<br>表现为语量少、语速慢、语音低和反应迟缓<br>多见于抑郁障碍                                       |
| 思维贫乏    | 指联想概念与词汇贫乏<br>表现为寡言少语，谈话时言语内容空洞单调或词穷句短，回答问题简单<br>多见于精神分裂症、痴呆及智力发育障碍                        |
| 思维散漫    | 思维的连贯性障碍，即联想概念之间缺乏必要的联系<br>表现为在交谈时联想松弛、内容散漫、缺乏主题，话题转换缺乏必要的联系<br>多见于精神分裂症及智力发育障碍            |
| 思维破裂    | 思维的连贯性障碍，即联想概念之间缺乏必要的联系<br>表现为患者的言语或书写内容有结构完整的句子，但各句含意不想管、变成了语句堆积                          |
| 语词杂拌    | 思维的连贯性障碍，即联想概念之间缺乏必要的联系<br>表现为患者言语支离破碎，句子结构不完整，为一些不相干字词的堆积<br>多见于精神分裂症                     |
| 思维不连贯   | 在意识障碍背景下出现的言语支离破碎和杂乱无章状态<br>多见于谵妄状态                                                        |
| 思维中断    | 思维联想过程突然发生中断<br>表现为患者在无意识障碍、无外界干扰时，言语突然停顿，片刻后又重新开始，但所谈主题已经转换<br>多见于精神分裂症                   |
| 思维被夺    | 思维联想障碍；表现为患者感到自己的思想被某种外力突然抽走<br>多见于精神分裂症                                                   |
| 思维插入    | 思维联想障碍；表现为患者感到有某种不属于自己的思想被强行塞入自己脑中<br>多见于精神分裂症                                             |
| 强制性思维   | 思维联想的自主性障碍<br>表现为患者感到脑内涌现大量无现实意义、不属于自己的联想；突然出现、突然消失、内容多变；患者认为思维活动已经完全不受自己的意愿支配<br>多见于精神分裂症 |
| 病理性赘述   | 思维联想活动迂回曲折、联想枝节过多<br>表现为不必要的过分详尽的描述、言语啰嗦，但最终能够回答出有关问题<br>见于癫痫、痴呆等                          |
| 思维化声    | 同时包含思维障碍和感知觉障碍的症状<br>感到自己的思想在脑子里变成了言语声、自己和他人均能听到<br>多见于精神分裂症                               |
| 语词新作    | 概念的融合、浓缩和无关概念的拼凑<br>自创奇特文字、符号、图形或语言并赋予特殊的意义，他人无法理解<br>多见于精神分裂症                             |
| 象征性思维   | 概念转换障碍；患者以无关的具体概念代替某一抽象概念<br>多见于精神分裂症                                                      |
| 逻辑倒错性思维 | 以推理缺乏逻辑性为特点，表现为患者推理过程缺乏前提依据或因果倒置<br>多见于精神分裂症                                               |
| 强迫性思维   | 1)反复出现某些想法; 2)强迫怀疑; 3)强迫回忆; 4)强迫性对立思维; 5)强迫性穷思竭虑<br>多见于强迫症、也可见于精神分裂症                       |
### 思维内容障碍
- $妄想（delusion）$：在病态推理和判断基础上形成的一种病理性的歪曲的信念
	1) 妄想内容通常与事实不符、缺乏客观现实基础
	2) 妄想内容通常涉及患者本人
	3) 妄想内容常具有个体独特性
	4) 妄想内容与患者的文化背景和经历有关

| 原发性妄想                   | 继发性妄想                        |
| ----------------------- | ---------------------------- |
| 没有发生基础的妄想<br>精神分裂症的典型症状 | 发生在其他病理心理基础上的妄想<br>可见于多种精神障碍 |

| 系统性妄想                                  | 非系统性妄想                               |
| -------------------------------------- | ------------------------------------ |
| 内容前后相互联系、结构严密的妄想<br>与现实具有一定联系或围绕某一核心思想 | 片段、零散、内容不固定、结构不严密的妄想<br>明显脱离现实、缺乏逻辑性 |


| 妄想的主要内容    | 表现                                                    |
| ---------- | ----------------------------------------------------- |
| ==关系妄想==   | 认为周围环境中所发生的与自己无关的事情均与自己有关<br>多见于精神分裂症和其他妄想性障碍         |
| ==被害妄想==   | 坚信自己被某些人或某组织进行陷害<br>主要见于精神分裂症和其他妄想性障碍                 |
| ==夸大妄想==   | 认为自己拥有非凡的才能、智慧、财富、权力、地位等<br>可见于躁狂发作、精神分裂症和痴呆等         |
| ==罪恶妄想==   | 又称自罪妄想。坚信自己犯了严重的错误<br>多见于抑郁障碍、可也见于精神分裂症               |
| ==疑病妄想==   | 坚信患了某种严重的躯体疾病或不治之症<br>多见于抑郁障碍、精神分裂症及躯体不适障碍等           |
| ==虚无妄想==   | 又称否定妄想。认为客观存在的事物已不复存在、一切都虚假<br>多见于抑郁障碍、精神分裂症等         |
| ==钟情妄想==   | 坚信自己被某异性或许多异性钟情<br>多见于精神分裂症                           |
| ==嫉妒妄想==   | 坚信自己的配偶对自己不忠诚、另有外遇<br>多见于精神分裂症、痴呆等                    |
| ==非血统妄想==  | 坚信自己不是父母亲生的<br>多见于精神分裂症                               |
| ==物理影响妄想== | 又称被控制感<br>感到自己的思想、情感和意志行为受到某种外界力量的控制而身不由己<br>多见于精神分裂症 |
| ==内心被揭露感== | 又称被洞悉感<br>多见于精神分裂症                                    |

### 超价观念
$超价观念$：超价观念是一种具有强烈情感色彩的错误观念，其发生一般均有一定事实依据，不十分荒谬离奇，也没有明显的逻辑推理错误。此种观念片面而偏激，可明显地影响患者的行为及其他心理活动，多见于人格障碍
## 注意障碍
- $注意（attention）$：是指个体精神活动集中指向一定对象的心理过程
	- 主动注意
	- 被动注意
- 正常人注意的特征：集中性、稳定性、转移性

| 分类   | 表现                                            |
| ---- | --------------------------------------------- |
| ==注意增强== | 为主动注意的兴奋性增高<br>多见于精神分裂症、躯体不适障碍等               |
| ==注意减退== | 为主动及被动注意的兴奋性减弱和注意稳定性降低<br>多见于抑郁障碍、精神分裂症等      |
| ==注意涣散== | 为被动注意兴奋性增强和注意稳定性降低<br>多见于注意缺陷多动障碍、焦虑障碍、精神分裂症等 |
| ==注意狭窄== | 为注意广度和范围的显著缩小<br>多见于意识障碍、智能障碍等                |
| ==注意转移== | 为注意转换性增强和稳定性降低<br>多见于躁狂发作等                    |

## 记忆障碍
- $记忆（memory）$：是既往事物经验在大脑中的重现。记忆是在感知觉和思维基础上建立起来的精神活动，包括识记、保持及再认和回忆三个基本过程。
	1. 识记
	2. 保持
	3. 再认和回忆

| 记忆障碍分类   | 表现                                                                              |
| -------- | ------------------------------------------------------------------------------- |
| ==记忆增强== | 病理性的记忆力增强<br>多见于躁狂发作和精神分裂症等                                                     |
| ==记忆减退== | 记忆各个基本过程功能的普遍减退<br>多见于痴呆、也可见于正常老年人                                              |
| ==遗忘==   | 暂时性/永久性遗忘<br>部分性/完全性遗忘                                                          |
| ==虚构==   | 指在遗忘的基础上，患者以想象的、未曾亲身经历的事件来填补记忆的缺损<br>多见于各种原因引起的痴呆及酒精所致精神病性障碍                    |
| ==错构==   | 指在遗忘的基础上，患者对过去所经历过的事件，在发生的地点、情节，特别是在时间上出现错误的回忆，并坚信不疑<br>多见于各种原因引起的痴呆及酒精所致精神病性障碍 |

| 遗忘分类      | 表现                                                                   |
| --------- | -------------------------------------------------------------------- |
| ==顺行性遗忘== | 指对紧接着疾病以后一段时间内发生的经历不能回忆<br>如脑挫伤患者不能回忆受伤后一段时间内所发生的事                   |
| ==逆行性遗忘== | 指对疾病发生之前一段时间内的经历不能回忆<br>多见于脑外伤、脑卒中发作、急性意识障碍等                         |
| ==界限性遗忘== | 指对某一特定时间段的经历不能回忆，遗忘的发生通常与该时间段内的不愉快事件有关，又称心因性遗忘<br>多见于分离性障碍，又称为分离性遗忘症 |
| ==进行性遗忘== | 指随着疾病的发展，遗忘逐渐加重<br>主要见于痴呆                                            |

## 智能障碍
$智能（intelligence）$：是人们获得和运用知识解决实际问题的能力，包括在经验中学习或理解的能力、获得和保持知识的能力、迅速而又成功地对新情境做出反应的能力、运用推理有效地解决问题的能力等。它涉及感知、记忆、注意和思维等一系列认知过程。

| 智力发育障碍                                            | 痴呆                                |
| ------------------------------------------------- | --------------------------------- |
| 是指先天或发育成熟以前（18 岁以前），由于各种原因影响智能发育所造成的智能低下和社会适应困难状态 | 指智力发育成熟以后，由于各种原因损害原有智能所造成的智能低下状态。 |

| 痴呆分类      | 表现                                                                                   |
| --------- | ------------------------------------------------------------------------------------ |
| ==全面性痴呆== | 表现为大脑弥散性损害，智能活动的各个方面均受累及，从而影响患者全部的精神活动<br>常出现人格改变、定向力障碍及自知力缺乏<br>多见于阿尔茨海默病和梅毒所致痴呆等   |
| ==部分性痴呆== | 大脑的病变只侵犯脑的局部，患者可只产生记忆力减退、理解力削弱或分析综合困难等，但其人格仍保持良好，定向力完整，有一定的自知力<br>可见于血管性痴呆和脑外伤后痴呆的早期 |
| ==假性痴呆==  | 在强烈的精神创伤后，部分患者可产生一种类似痴呆的表现，而大脑组织结构无任何器质性损害<br>可见于分离性障碍及应激障碍等                         |
|           | ==刚塞Ganser综合征==：又称为心因性假性痴呆，表现为对简单问题给予近似而错误的回答，往往给人以故意或开玩笑的感觉；但对某些复杂问题，患者却往往能正确应付     |
|           | ==童样痴呆==：成人患者的言行类似儿童                                                                 |
## 定向力障碍
- $定向力（orientation）$：指一个人对时间、地点、人物以及自身状态的认识能力
	- $对周围环境的定向力$：时间定向、地点定向、人物定向
	- $自我定向力$：对自己姓名、性别、年龄及职业等状况的认知
- 定向力障碍多见于意识障碍时，它是意识障碍的一个重要标志。但有定向力障碍者并不一定存在意识障碍
## 情感障碍
- $情感（affection）$：主要是指与人的社会性需要相联系的体验，具有稳定性、持久性，不一定有明显的外部表现，如爱与恨等
- $情绪（emotion）$：主要是指与人的自然性需要相联系的体验，具有情景性、暂时性和明显的外部表现，如喜与怒等
- $心境（mood）$：是指一种较微弱而持续的情绪状态，是一段时间内精神活动的基本背景

| 情感障碍            | 表现                                                               |
| --------------- | ---------------------------------------------------------------- |
| ==情感高涨elation== | 正性情感活动的明显增强<br>多见于躁狂发作                                           |
| ==欣快euphoria==  | 在智能障碍基础上出现的与周围环境不协调的愉快体验<br>往往会给人以呆傻、愚蠢的感觉；多见于痴呆                 |
| ==情感低落==        | 负性情感活动的明显增强<br>多见于抑郁障碍                                           |
| ==情感淡漠==        | 对外界刺激缺乏相应的情感反应，缺乏内心体验<br>多见于晚期精神分裂症                              |
| ==焦虑==          | 在缺乏相应客观刺激的情况下出现的内心不安状态<br>多见于焦虑障碍                                |
| ==恐惧==          | 面临某种事物或处境时出现的紧张不安反应<br>多见于恐惧障碍                                   |
| ==易激惹==         | 情感活动的激惹性增高<br>多见于人格障碍、躁狂发作等                                      |
| ==情绪不稳==        | 情感活动的稳定性障碍，表现为患者的情感反应极易发生变化，从一个极端波动至另一个极端<br>多见于脑器质性损害所致的精神障碍    |
| ==情感倒错==        | 情感表现与其内心体验或处境明显不相协调，甚至截然相反<br>多见于精神分裂症                           |
| ==情感矛盾==        | 在同一时间对同一人或事物产生两种截然不同的情感反应，但患者并不感到这两种情感的矛盾和对立，没有痛苦和不安<br>多见于精神分裂症 |
## 意志障碍
- $意志（volition）$：是人自觉地确定目标，并根据目标调节和支配自身的行动，克服困难，实现预定目标的心理过程。一般把意志品质归纳为<u>自觉性、果断性、自制性和坚持性</u>四个方面

| 意志障碍     | 表现                                                             |
| -------- | -------------------------------------------------------------- |
| ==意志增强== | 指意志活动增多。表现为在病态情感或妄想的支配下，患者持续地坚持某些行为，具有极大的顽固性<br>多见于精神分裂症、躁狂发作等 |
| ==意志减退== | 指意志活动的减少。表现为动机不足，缺乏积极主动性及进取心<br>多见于抑郁障碍和精神分裂症                  |
| ==意志缺失== | 指意志活动缺乏。表现为对任何活动都缺乏动机、要求，生活处于被动状态<br>多见于精神分裂症、智力发育障碍和痴呆        |
| ==矛盾意向== | 表现为对同一事物，同时出现两种完全相反的意向，但患者并不感到这两种意向的矛盾和对立<br>多见于精神分裂症          |
## 动作行为障碍
- $动作（action）$：是指简单的随意和不随意运动，如挥手、点头等
- $行为（behavior）$：是一系列动作的有机组合，是为达到一定目的而进行的复杂的随意运动

| 动作行为障碍          | 表现                                                                     |
| --------------- | ---------------------------------------------------------------------- |
| *精神运动性兴奋*       |                                                                        |
| ==协调性精神运动性兴奋==  | 精神运动性兴奋；患者的整个精神活动比较协调，行为具有目的，可以被周围人理解<br>多见于躁狂发作                       |
| ==不协调性精神运动性兴奋== | 患者增多的动作行为及言语与思维、情感、意志等精神活动不相协调，脱离周围现实环境<br>多见于精神分裂症、谵妄状态               |
| *精神运动性抑制*       |                                                                        |
| ==木僵==          | 动作行为和言语活动完全抑制。不语、不动、不饮、不食，肌张力增高，面部表情固定，保持固定姿势<br>可见于精神分裂症、严重抑郁障碍、应激障碍等 |
| ==蜡样屈曲==        | 在木僵的基础上，患者肢体可任人摆布；“空气枕头”<br>可见于精神分裂症                                   |
| ==缄默症==         | 缄默不语、不回答任何问题；通过书写交流<br>多见于分离性障碍及精神分裂症                                  |
| ==违拗症==         | 患者对他人的要求加以抗拒<br>主动违拗：不但拒绝执行且做出与要求截然相反的行为<br>被动违拗：对他人的各种要求一概拒绝执行        |
| ==模仿动作==        | 患者无目的地模仿别人的动作                                                          |
| ==刻板动作==        | 机械刻板地重复某一单调的动作                                                         |
| ==装相、作态==       | 患者用一种不常用的表情、姿势或动作来表达某一有目的的行为                                           |
| ==强迫动作==        | 患者明知没有必要却难以克制地去重复做某种动作行为<br>常见于强迫症                                     |
## 意识障碍
- $意识（consciousness）$：个体对周围环境、自身状态感知的清晰程度及认识反应能力
	- *大脑皮质及脑干的网状上行激活系统的兴奋性* 对维持意识起重要作用
- 意识障碍：意识清晰度的降低、意识范围的缩小、意识内容的变化

| 分类                  | 表现                                                                                   |
| ------------------- | ------------------------------------------------------------------------------------ |
| *以意识清晰度降低为主*        |                                                                                      |
| ==嗜睡==              | 意识清晰度降低较轻微；给予刺激后可立即转醒、停止刺激后又进入睡眠状态                                                   |
| ==混浊==              | 意识清晰度轻度受损<br>可出现强握反射、吸吮反射等原始反射                                                       |
| ==昏睡==              | 意识清晰度较混浊更低；患者的周围环境定向力和自我定向力均丧失、无言语功能<br>只有强刺激才引起防御性反应<br>深反射亢进、病理反射阳性<br>可出现不自主运动及震颤 |
| ==昏迷==              | 意识完全丧失，以痛觉反应和随意运动消失为特征<br>吞咽反射、防御反射甚至对光反射均消失，并可出现病理反射                                |
| *意识清晰度降低伴范围缩小或内容变化* |                                                                                      |
| ==朦胧状态==            | 在意识清晰度降低的同时伴有意识范围缩小<br>患者在狭窄的意识范围内，可有相对正常的感知觉，以及协调连贯的复杂行为，但对除此范围以外的事物却不能进行正确感知。      |
| ==谵妄==              | 患者在意识清晰度降低的同时出现大量的幻觉、错觉，这些幻觉和错觉以形象鲜明的恐怖性幻视和错视为主<br>谵妄状态往往夜间加重，具有昼轻夜重的规律              |
| ==梦样状态==            | 指在意识清晰程度降低的同时出现梦样的体验                                                                 |
# 常见精神疾病综合征
1. ==幻觉妄想综合征 （hallucinatory-paranoid syndrome）== 是一组以幻觉为主，并在幻觉的基础上产生相应妄想的综合征。幻觉和妄想联系紧密，且相互影响。如一患者耳边出现同学议论的声音（幻听），在此之后逐渐怀疑同学对其进行跟踪迫害（妄想）。多见于精神分裂症，也可见于脑器质性损害和精神活性物质所致精神障碍等。
2. ==躁狂综合征 （manic syndrome）== 以情感高涨、思维奔逸和活动增多为特征。主要见于躁狂发作，也可见于脑器质性损害所致精神障碍。另外，某些药物（如糖皮质激素、抗抑郁药物等）也可引起类似发作。
3. ==抑郁综合征 （depressive syndrome）== 以情感低落、思维迟缓和活动减少为特征。主要见于抑郁障碍，也可见于脑器质性损害所致精神障碍。另外，某些药物如利血平等也可引起类似发作。
4. ==紧张综合征 （catatonic syndrome）== 最突出的症状是患者全身肌张力增高，包括紧张性木僵和紧张性兴奋两种状态。前者常有违拗症、刻板言语及刻板动作、模仿言语及模仿动作、蜡样屈曲等症状，后者表现为突然爆发的兴奋激动和暴烈行为。紧张性木僵状态可持续数日或数年，可无任何原因地转入兴奋状态。而兴奋状态持续较短暂，发作后往往再次进入木僵状态或缓解。紧张综合征可见于精神分裂症、抑郁障碍、急性应激障碍、脑器质性损害所致精神障碍、药物中毒等。
5. ==遗忘综合征 （amnestic syndrome）== 又称为科尔萨科夫综合征（Korsakoff syndrome），患者通常无意识障碍，智能相对完好，主要表现为近事记忆障碍、定向力障碍和虚构。多见于慢性酒精中毒所致精神障碍、颅脑损伤所致精神障碍、脑肿瘤及其他脑器质性精神障碍。
